# <a name="visual-studio-documentation-for-microsoft-c"></a>Microsoft C++용 Visual Studio 설명서

환영합니다. 이 리포지토리에는 개발 중인 Microsoft C++(MSVC 또는 Visual C++) 기술 설명서의 원본 파일이 포함됩니다. 이 문서는 [Visual Studio 설명서 사이트의 C++](https://docs.microsoft.com/cpp)에 게시됩니다.

Visual Basic 및 Visual C#에 대한 설명서는 [https://github.com/dotnet/core-docs](https://github.com/dotnet/core-docs)의 별도 리포지토리에 위치하고, Visual Studio 설명서는 [https://github.com/Microsoft/visualstudio-docs](https://github.com/Microsoft/visualstudio-docs)의 리포지토리에 위치합니다.

## <a name="contributing-to-the-documentation"></a>설명서에 기여

MSVC 문서 개선에 도움이 되는 모든 기여를 환영합니다. docs.microsoft.com에 참여하는 포괄적인 가이드는 [Microsoft Docs 기여자 가이드 개요](https://docs.microsoft.com/contribute)를 참조하세요. MSVC 설명서에 기여하는 방법에 대한 자세한 내용은 [기여 가이드](CONTRIBUTING.md)를 참조하세요.

MSVC의 여러 가지 기능 영역은 이 리포지토리에서 고유한 폴더를 사용합니다. 예를 들어 C++ 표준 라이브러리에 대한 문서의 경우 `standard-library`, Visual Studio IDE(대화형 개발 환경)에 대한 C++ 특정 문서의 경우 `ide`를 사용합니다. 각 폴더의 `/media` 하위 폴더에는 해당 문서에 대한 아트 파일이 포함되어 있습니다. 자세한 내용은 [기여 가이드](CONTRIBUTING.md)를 참조하세요.
