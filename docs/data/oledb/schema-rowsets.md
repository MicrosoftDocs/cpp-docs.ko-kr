# <a name="schema-rowsets"></a>스키마 행 집합

스키마 (또는 메타 데이터)는 데이터베이스의 구조 또는 조직에 대 한 정보 (예: 공급자, 행 집합, 테이블, 열 또는 실제 데이터베이스 내용 외의 기타 정보)입니다. OLE DB를 사용 하면 스키마 행 집합을 사용 하 여 행 집합을 열지 않고도 이러한 정보를 얻을 수 있습니다.

스키마 행 집합을 사용 하는 방법에 대 한 자세한 내용은 [스키마 행 집합으로 메타 데이터 가져오기](../../data/oledb/obtaining-metadata-with-schema-rowsets.md)를 참조 하세요.
