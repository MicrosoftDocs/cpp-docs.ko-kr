# <a name="cbulkrowset-class"></a>CBulkRowset 클래스
인출 하 고 한 번의 호출으로 여러 행 핸들을 검색 하 여 대량으로 데이터에서 작동 하도록 행을 조작 합니다.  
  
## <a name="syntax"></a>구문

```cpp
template <class TAccessor>  
class CBulkRowset : public CRowset<TAccessor>  
```  
  
#### <a name="parameters"></a>매개 변수  
 `TAccessor`  
 접근자 클래스입니다.
