# <a name="cmultipleresults-class"></a>CMultipleResults 클래스
여러 결과 집합을 처리 하는 명령을 사용 `CMultipleResults` 에 대 한는 `CCommand` 템플릿 인수 *TMultiple*합니다.  
  
## <a name="syntax"></a>구문

```cpp
class CMultipleResults  
```  
  
## <a name="remarks"></a>설명  
 여러 결과 처리 하도록 설정한 경우 [CCommand](../../data/oledb/ccommand-class.md) 이 클래스에서 상속 해야 합니다.

 **헤더:** atldbcli
