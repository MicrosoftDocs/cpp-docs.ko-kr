# <a name="debugging-your-provider"></a>공급자 디버깅

공급자는 다음과 같은 두 가지 방법으로 디버그할 수 있습니다.

- 공급자는 프로세스에서 만들어지기 때문에 OLE DB 소비자 템플릿을 사용 하 여 일부 소비자 코드를 만들고 공급자를 정상적으로 한 단계씩 실행 할 수 있습니다.

- Visual C++와 함께 제공 되는 다양 한 유틸리티를 사용할 수 있습니다.

## <a name="to-use-debugging"></a>디버깅을 사용 하려면

1. 공급자 프로젝트를 엽니다.

1. **프로젝트** 메뉴에서 **속성** 을 클릭 합니다.

1. **속성 페이지** 대화 상자에서 **디버깅** 탭을 클릭 합니다.

1. 필요에 따라 옵션을 선택 하 고 **확인** 을 클릭 합니다.

1. 중단점을 설정 하 고 일반적인 방식으로 디버깅 합니다.
