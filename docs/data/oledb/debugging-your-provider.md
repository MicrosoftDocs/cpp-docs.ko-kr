# <a name="debugging-your-provider"></a>공급자 디버깅

공급자 디버깅 하는 방법은 두 가지:

- 공급자, 프로세스에서 생성 되므로 일반적으로 OLE DB 소비자 템플릿 및 단계 공급자를 사용 하 여 일부 소비자 코드를 만들 수 있습니다.

- Visual c + +를 사용 하 여 제공 되는 다양 한 유틸리티를 사용할 수 있습니다.

## <a name="to-use-debugging"></a>디버깅을 사용 하려면

1. 프로젝트를 엽니다.

1. 에 **프로젝트** 메뉴에서 클릭 **속성**합니다.

1. 에 **속성 페이지** 대화 상자에서 클릭 합니다 **디버깅** 탭 합니다.

1. 필요한 경우 클릭 옵션을 선택 **확인**합니다.

1. 중단점을 설정 하 고 디버깅 합니다.
