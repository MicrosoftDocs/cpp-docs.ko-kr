# <a name="debugging-your-provider"></a>공급자 디버깅

다음 두 가지 방법으로 공급자를 디버깅할 수 있습니다.

- 공급자는 단계별로 만들어지므로 OLE DB 소비자 템플릿을 사용하여 일부 소비자 코드를 만든 다음 공급자 단계를 정상적으로 진행할 수 있습니다.

- Visual C++에 제공되는 ITEST 유틸리티를 사용할 수 있습니다.

## <a name="to-use-debugging"></a>디버깅을 사용하려면

1. 공급자 프로젝트를 엽니다.

1. **프로젝트** 메뉴에서 **속성**을 클릭 합니다.

1. **속성 페이지** 대화 상자에서 **디버깅** 탭을 클릭 합니다.

1. 필요에 따라 옵션을 선택 하 고 **확인**을 클릭 합니다.

1. 중단점을 설정하고 디버깅합니다.
