## <a name="syntax"></a>구문  
  
```cpp
      STDMETHOD(AddRefAccessor)(HACCESSOR hAccessor,  
   DBREFCOUNT* pcRefCount);  
```

 참조 [IAccessor::AddRefAccessor](https://msdn.microsoft.com/en-us/library/ms714978.aspx) 에 *OLE DB Programmer's Reference*합니다.
