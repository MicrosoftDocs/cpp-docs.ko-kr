## <a name="syntax"></a>구문  
  
```cpp
      STDMETHOD(GetRecordCount )(ULONG *pcRecords);  
```

 참조 [IErrorRecords::GetRecordCount](https://msdn.microsoft.com/en-us/library/ms722724.aspx) 에 *OLE DB Programmer's Reference*합니다.
