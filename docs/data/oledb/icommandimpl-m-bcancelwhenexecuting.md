# <a name="icommandimplmbcancelwhenexecuting"></a>ICommandImpl::m_bCancelWhenExecuting
명령을 실행할 때 취소할 수 있는지 여부를 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```cpp
unsigned m_bCancelWhenExecuting:1;  
  
```  
  
## <a name="remarks"></a>설명  
 기본적으로 **true** (취소할 수 있습니다).  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** atldb.h
