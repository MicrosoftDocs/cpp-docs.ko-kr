# <a name="creating-a-project-for-the-provider"></a>공급자용 프로젝트 만들기
#### <a name="to-create-a-project-in-which-the-ole-db-provider-will-reside"></a>OLE DB 공급자 상주할 프로젝트를 만들려면  
  
1.  **파일** 메뉴를 클릭 하 여 `New`, 클릭 하 고 **프로젝트**합니다.  
  
     **새 프로젝트** 대화 상자가 나타납니다.  
  
2.  프로젝트 형식 창에서 Visual c + + 프로젝트 폴더를 클릭 합니다. 템플릿 창에서 클릭 **ATL 프로젝트**합니다.  
  
3.  에 **이름** 상자, 프로젝트에 대 한 이름을 입력 한 다음 클릭 **확인**합니다.  
  
     ATL 프로젝트 마법사가 나타납니다.  
  
4.  ATL 프로젝트 마법사에서 선택 **동적 연결 라이브러리 (DLL)** 에 대 한 **서버 유형**합니다.  
  
5.  **마침**을 클릭합니다.
