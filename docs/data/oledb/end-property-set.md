# <a name="endpropertyset"></a>END_PROPERTY_SET

## <a name="syntax"></a>구문  
  
```cpp
END_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>매개 변수  
 `guid`  
 [in] GUID 속성입니다.
