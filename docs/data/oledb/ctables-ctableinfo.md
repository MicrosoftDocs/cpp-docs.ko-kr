# <a name="ctables-ctableinfo"></a>CTables, CTableInfo
Typedef 클래스 호출 **CTables** 해당 매개 변수 클래스를 구현 하려면 **CTableInfo**합니다.  
  
## <a name="remarks"></a>설명  
 참조 [스키마 행 집합 클래스 및 Typedef 클래스](../../data/oledb/schema-rowset-classes-and-typedef-classes.md) typedef 클래스 사용에 대 한 자세한 내용은 합니다.  
  
 이 클래스는 카탈로그에 정의 된를 사용할 수 있거나 지정된 된 사용자가 부여 하는 테이블에 대 한 권한을 식별 합니다.  
  
 다음 표에는 클래스 데이터 멤버 및 해당 OLE DB 열이 나열됩니다. 참조 [TABLES 행 집합](https://msdn.microsoft.com/en-us/library/ms716980.aspx) 에 *OLE DB Programmer's Reference* 스키마 및 열에 대 한 자세한 내용은 합니다.  
  
|데이터 멤버|OLE DB 열|  
|------------------|--------------------|  
|m_szCatalog|TABLE_CATALOG|  
|m_szSchema|TABLE_SCHEMA|  
|m_szName|TABLE_NAME|  
|m_szType|TABLE_TYPE|  
|m_guidTable|TABLE_GUID|  
|m_szDescription|설명|
