# <a name="endcolumnmap"></a>END_COLUMN_MAP
열 맵 항목의 끝을 표시 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>설명  
 행 집합에 단일 접근자 사용 됩니다. `BEGIN_COLUMN_MAP` 매크로는 `END_COLUMN_MAP` 매크로로 완료됩니다.  
  
## <a name="example"></a>예제  
 참조 [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md)합니다.
