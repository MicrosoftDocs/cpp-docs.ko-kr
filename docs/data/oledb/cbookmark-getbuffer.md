# <a name="cbookmarkgetbuffer"></a>CBookmark::GetBuffer
책갈피 버퍼에 대 한 포인터를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp
virtual BYTE* GetBuffer() const throw();  
  
```  
  
## <a name="return-value"></a>반환 값  
 책갈피 버퍼에 대 한 포인터입니다.
