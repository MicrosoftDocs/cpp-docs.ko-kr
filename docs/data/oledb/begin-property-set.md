# <a name="beginpropertyset"></a>BEGIN_PROPERTY_SET
속성에는 속성의 시작 부분을 설정 하는 부호 맵을 설정 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp
BEGIN_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>매개 변수  
 `guid`  
 [in] GUID 속성입니다.
