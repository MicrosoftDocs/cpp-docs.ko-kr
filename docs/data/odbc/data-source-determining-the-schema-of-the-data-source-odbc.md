# <a name="data-source-determining-the-schema-of-the-data-source-odbc"></a>데이터 소스: 데이터 소스의 스키마 확인(ODBC)

이 토픽은 MFC ODBC 클래스에 적용됩니다.

개체에서 데이터 멤버를 설정 하려면 `CRecordset` 연결 하는 데이터 원본의 스키마를 알고 있어야 합니다. 데이터 원본의 스키마를 확인 하려면 데이터 원본의 테이블 목록, 각 테이블의 열 목록, 각 열의 데이터 형식 및 인덱스 유무를 확인 해야 합니다.
