 MFC 없이 ODBC API를 사용 하는 방법에 대 한 자세한 내용은 참조는 [ODBC Open Database Connectivity ()](https://msdn.microsoft.com/en-us/library/ms710252.aspx) SDK MSDN에 있습니다.
