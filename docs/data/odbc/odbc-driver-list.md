# <a name="odbc-driver-list"></a>ODBC 드라이버 목록

Visual c + +는 다음 데이터베이스에 대 한 ODBC 드라이버를 제공합니다.  
  
- SQL Server  
  
- Microsoft Access  
  
- Microsoft Excel  
  
- dBASE  
  
- Paradox  
  
- Microsoft Oracle ODBC  
  
- 텍스트 파일  
  
Microsoft ODBC Driver 팩을 비롯 한 다른 회사에서 사용할 수 있는 ODBC 드라이버에 대 한 내용은 Microsoft 기술 지원 서비스에 문의 합니다.
