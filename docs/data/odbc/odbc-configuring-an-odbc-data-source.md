# <a name="odbc-configuring-an-odbc-data-source"></a>ODBC: ODBC 데이터 소스 구성

개발한 응용 프로그램과 함께 [데이터 원본을](../../data/odbc/data-source-odbc.md) 사용하려면 ODBC 관리자를 사용하여 데이터 원본을 구성해야 합니다. ODBC 관리자는 Windows 레지스트리에서 사용 가능한 데이터 원본과 연결 정보를 추적합니다. ODBC 관리자를 사용하여 **데이터 원본** 대화 상자에서 데이터 원본을 추가, 수정 및 삭제하고 ODBC 드라이버를 추가 및 삭제합니다.

> [!NOTE]
> 이 정보는 ODBC 액세스를 위해 MFC 데이터 액세스 개체(DAO) 클래스를 사용하고 MFC ODBC 클래스를 사용할 때 적용됩니다.

ODBC 관리자는 MFC(Microsoft Foundation 클래스) 라이브러리 데이터베이스 지원과 함께 자동으로 설치됩니다. ODBC 관리자 프로그램에 대한 자세한 내용은 [ODBC 관리자](../../data/odbc/odbc-administrator.md) 및 온라인 ODBC API 참조 도움말 시스템을 참조하십시오.

MFC 데이터베이스 응용 프로그램에 대한 ODBC 설치 및 관리 프로그램을 작성하는 방법에 대한 자세한 내용은[기술 참고 48](../../mfc/tn048-writing-odbc-setup-and-administration-programs.md).
