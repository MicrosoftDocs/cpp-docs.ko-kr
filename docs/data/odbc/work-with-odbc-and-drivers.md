# <a name="work-with-odbc-and-drivers"></a>ODBC 및 드라이버 작업

ODBC를 이용하여 특정 데이터베이스 관리 시스템(DBMS)에 종속되지 않는 응용 프로그램을 작성할 수 있습니다. 사용자에게 특정 DBMS에 적합한 ODBC 드라이버가 있는 경우 다중 DBMS에서 동작하는 응용 프로그램을 작성할 수 있습니다.
