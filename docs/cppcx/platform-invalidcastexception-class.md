# <a name="platforminvalidcastexception-class"></a>Platform::InvalidCastException 클래스

캐스트 또는 명시적 변환이 잘못된 경우 throw됩니다.

## <a name="syntax"></a>구문

```cpp
public ref class InvalidCastException : COMException,    IException,    IPrintable,    IEquatable
```

### <a name="remarks"></a>설명

자세한 내용은 [COMException](../cppcx/platform-comexception-class.md) 클래스를 참조하세요.

**지원 되는 최소 서버:** Windows Server 2012

**네임스페이스:** Platform
