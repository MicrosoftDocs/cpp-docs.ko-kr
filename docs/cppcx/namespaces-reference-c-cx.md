# <a name="namespaces-reference-ccx"></a>네임스페이스 참조(C++/CX)
설명서의이 섹션의 문서 설명 네임 스페이스 컴파일러를 지 원하는 C + + /cli CX 합니다.  
  
## <a name="compiler-supplied-namespaces"></a>컴파일러가 제공하는 네임스페이스  
 Windows 런타임은 C + 대상으로 하는 프로그램의 코딩을 간소화 하기 위해 + CX 컴파일러 및 지원 헤더 파일에서는 광범위 한 형식 정의 하는 네임 스페이스를 제공 합니다. 네임 스페이스는 기본 제공 숫자 형식, 정의 문자열, 배열 및 컬렉션 Windows 런타임 오류를 나타내는 visual c + + 예외 표준 Windows 런타임 형식에 대 한 언어별 향상 하 고 있습니다.
