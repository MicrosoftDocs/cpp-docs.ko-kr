# <a name="namespaces-reference-ccx"></a>네임스페이스 참조(C++/CX)

설명서의이 섹션에 있는 문서에서는/Cx에 대 한 C++컴파일러를 지 원하는 네임 스페이스에 대해 설명 합니다.

## <a name="compiler-supplied-namespaces"></a>컴파일러가 제공하는 네임스페이스

Windows 런타임를 대상으로 하는 프로그램의 코딩을 간소화 하기 C++위해/cx 컴파일러와 지원 헤더 파일은 다양 한 형식의 형식을 정의 하는 네임 스페이스를 제공 합니다. 네임 스페이스는 기본 제공 숫자 형식을 정의 합니다. 문자열, 배열 및 컬렉션 C++ Windows 런타임 오류를 나타내는 예외 표준 Windows 런타임 형식에 대 한 언어별 향상 된 기능
