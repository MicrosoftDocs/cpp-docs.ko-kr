# <a name="platformruntimeinteropservices-namespace"></a>Platform::Runtime::InteropServices 네임스페이스

이 네임스페이스는 내부 전용이며 개발용으로 사용되지 않아야 합니다.

## <a name="syntax"></a>구문

```cpp

   namespace InteropServices {
}}
```
