# <a name="platformnullreferenceexception-class"></a>Platform::NullReferenceException 클래스

null 개체 참조를 역참조하려고 할 때 throw됩니다.

## <a name="syntax"></a>구문

```cpp
public ref class NullReferenceException : COMException,    IException,    IPrintable,    IEquatable
```

### <a name="remarks"></a>설명

자세한 내용은 [COMException](../cppcx/platform-comexception-class.md) 클래스를 참조하세요.
