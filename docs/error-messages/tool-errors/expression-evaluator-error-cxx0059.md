# <a name="expression-evaluator-error-cxx0059"></a>식 계산기 오류 CXX0059

왼쪽 피연산자는 함수 이름이 아닌 클래스입니다.

함수 호출의 왼쪽 피연산자가 클래스 이름이 며 함수 호출로 확인 될 수 없습니다. 식에서 멤버 함수의 이름을 생략 하면이 오류가 발생할 수 있습니다.

이 오류는 CAN0059와 동일 합니다.
