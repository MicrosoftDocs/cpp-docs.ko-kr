# <a name="command-line-error-d8045"></a>명령줄 오류 D8045
C 파일 'file' /clr 옵션으로 컴파일할 수 없습니다.  
  
 C + + 소스 코드 파일에만 사용 하는 컴파일에 전달할 수 **/clr**합니다.  사용 하 여 **/TP** .c 파일을.cpp 파일로 컴파일하는 데 참조 [/Tc, /Tp, /TC, /TP (소스 파일 형식 지정)](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) 자세한 정보에 대 한 합니다.  
  
 자세한 내용은 [/clr(공용 언어 런타임 컴파일)](../../build/reference/clr-common-language-runtime-compilation.md)을 참조하세요.  
  
 D8045 Visual c + +를 사용 하 여 ATL 응용 프로그램을 컴파일하는 경우에 발생할 수 있습니다. 참조 [하는 방법: /clr으로 마이그레이션](../../dotnet/how-to-migrate-to-clr.md) 자세한 정보에 대 한 합니다.
