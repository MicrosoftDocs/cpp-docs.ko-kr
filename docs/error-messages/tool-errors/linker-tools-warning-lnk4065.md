# <a name="linker-tools-warning-lnk4065"></a>링커 도구 경고 LNK4065

패키지 함수로 지정된 된 함수를 컴파일되지 않았습니다. 사용 하 여 다시 컴파일할 [/Gy](../../build/reference/gy-enable-function-level-linking.md)합니다.
