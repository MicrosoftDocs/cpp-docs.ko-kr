# <a name="project-build-error-prj0004"></a>프로젝트 빌드 오류 PRJ0004

> '*도구*' 도구에 대 한 명령줄을 생성할 수 없습니다.

*도구* 에 대 한 호출의 구문이 잘못 되도록 하는 방법으로 하나 이상의 속성이 지정 되었습니다.

- 잘못 된 형식이 나 알 수 없는 매크로를 지정 했을 수 있습니다.

- 컴퓨터의 사용 가능한 디스크 공간이 부족할 수 있습니다.

[명령줄 속성 페이지](../../build/reference/command-line-property-pages.md)에서 도구에 대 한 설정을 검토 하는 것이 좋습니다.
