# <a name="project-build-error-prj0004"></a>프로젝트 빌드 오류 PRJ0004
'도구' 도구에 대 한 명령줄을 생성 하지 못했습니다.  
  
 하나 이상의 속성에 대 한 호출 구문을 확인 하는 방식에 지정 된 ***도구*** 잘못 되었습니다.  
  
-   잘못 구성 되었거나 알 수 없는 매크로 지정 했습니다.  
  
-   컴퓨터에서 사용 가능한 디스크 공간이 부족할 수 있습니다.  
  
 이 도구에 대 한 설정을 확인 하 여 검토 하려는 경우는 [명령줄 속성 페이지](../../ide/command-line-property-pages.md)합니다.
