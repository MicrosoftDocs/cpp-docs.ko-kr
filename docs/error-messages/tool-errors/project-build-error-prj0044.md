# <a name="project-build-error-prj0044"></a>프로젝트 빌드 오류 PRJ0044
'Rule' 'file' 파일에 할당 된 사용자 지정 빌드 규칙에 대 한 ' 추가 종속성 ' 속성이 잘못 되었습니다. 이 속성에 'string'이 포함되어 있으며 이것은 'value'로 계산됩니다.  
  
 **추가 종속성** 속성이 빈 문자열 이거나 잘못 된 문자 (파일 또는 디렉터리 이름에 사용할 수 없는 문자)를 포함 하는 문자열입니다. 사용자 지정 빌드 규칙이 빌드 작업의 출력을 필요로 합니다.  
  
 자세한 내용은 참조 [사용자 지정 빌드 도구 지정](../../ide/specifying-custom-build-tools.md)합니다.
