# <a name="project-build-error-prj0044"></a>프로젝트 빌드 오류 PRJ0044

' File ' 파일에 할당 된 ' rule ' 사용자 지정 빌드 규칙의 ' 추가 종속성 ' 속성이 잘못 되었습니다. 이 속성에 'string'이 포함되어 있으며 이것은 'value'로 계산됩니다.

**추가 종속성** 속성은 빈 문자열이 나 잘못 된 문자가 포함 된 문자열 (파일 또는 디렉터리 이름에 포함할 수 없는 문자)로 계산 됩니다. 사용자 지정 빌드 규칙이 빌드 작업의 출력을 필요로 합니다.

자세한 내용은 [사용자 지정 빌드 도구 지정](../../build/specifying-custom-build-tools.md)을 참조 하세요.
