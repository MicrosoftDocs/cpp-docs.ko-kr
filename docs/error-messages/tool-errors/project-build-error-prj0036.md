# <a name="project-build-error-prj0036"></a>프로젝트 빌드 오류 PRJ0036

웹 배포 도구의 ' 추가 파일 ' 속성에 잘못 된 항목이 포함 되어 있습니다.

웹 배포 속성 페이지의 추가 파일 속성에 오류가 포함 되어 있습니다 (매크로 평가 문제 때문일 수 있음). 이 오류는 경로 형식이 잘못 된 것을 의미할 수도 있습니다 .이 경우 파일 경로에 잘못 된 문자 또는 문자 조합이 포함 됩니다.

이 오류를 해결 하려면 매크로를 수정 하거나 경로 지정을 수정 합니다. 계산 된 경로는 프로젝트 디렉터리의 절대 경로입니다.

이 오류는 참조 된 파일 중 하나가 존재 하지 않음을 의미할 수도 있습니다.
