# <a name="linker-tools-error-lnk1158"></a>링커 도구 오류 LNK1158

' filename '을 실행할 수 없습니다.

[Link](../../build/reference/linking.md) 에 의해 호출 된 지정 된 실행 파일은 PATH 환경 변수에 지정 된 디렉터리 또는 링크를 포함 하는 디렉터리에 없습니다.

예를 들어 32 비트 운영 체제가 설치 된 컴퓨터에서 [/ltcg](../../build/reference/ltcg-link-time-code-generation.md) 링커 옵션에 PGOPTIMIZE 매개 변수를 사용 하려고 하면이 오류가 발생 합니다.
