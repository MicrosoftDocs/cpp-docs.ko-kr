# <a name="nmake-fatal-error-u1077"></a>NMAKE 심각한 오류 U1077

' program ': ' value ' 반환 코드

NMAKE에 의해 호출 된 지정 된 명령 또는 프로그램이 실패 하 여 지정 된 종료 코드를 반환 했습니다.

이 오류를 억제 하 고 NMAKE 세션을 계속 하려면/I 옵션인를 사용 합니다 **. IGNORE** dot 지시문 또는 대시 ( **-** ) 명령 한정자입니다. 종속성 트리의 관련이 없는 부분에 대해 NMAKE 세션을 계속 하려면/K 옵션을 사용 합니다.
