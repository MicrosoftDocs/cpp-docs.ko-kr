 콜론 (**:**)에 유추 규칙을 따라야 합니다 다음 중 하나가:  
  
-   줄 바꿈 문자  
  
-   세미콜론 (**;**), 뒤에 명령이 오는  
  
-   숫자 기호 (**#**), 뒤에 주석이 오는
