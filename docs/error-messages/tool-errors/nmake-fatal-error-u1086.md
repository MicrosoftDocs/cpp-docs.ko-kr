# <a name="nmake-fatal-error-u1086"></a>NMAKE 심각한 오류 U1086

**유추 규칙은 종속 항목을 가질 수 없습니다.**

유추 규칙의 콜론 (**:**) 다음에 다음 중 하나가와 야 합니다.

- 줄 바꿈 문자

- 세미콜론 (**;**), 명령 뒤에 올 수 있습니다.

- 숫자 기호 ( **#** ), 주석이 뒤에 올 수 있습니다.
