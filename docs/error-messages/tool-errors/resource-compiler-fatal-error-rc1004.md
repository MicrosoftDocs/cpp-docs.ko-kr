# <a name="resource-compiler-fatal-error-rc1004"></a>리소스 컴파일러 심각한 오류 RC1004
예기치 않은 파일의 끝  
  
 텍스트 파일의 마지막 줄에서 캐리지 리턴 문자가 및이 오류는 줄 바꿈 없는 경우에 발생할 수 있습니다.
