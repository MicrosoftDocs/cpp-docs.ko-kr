# <a name="resource-compiler-warning-rc4002"></a>리소스 컴파일러 오류 RC4002
'identifier' 매크로의 실제 매개 변수가 너무 많습니다.  
  
 이 식별자로 지정 된 실제 매개 변수 수가 식별자의 매크로 정의에 지정 된 형식 매개 변수 개수 보다 큽니다.  
  
 실제 매개 변수 추가 된 수집 되지만 매크로 확장 하는 동안 무시 됩니다.
