# <a name="resource-compiler-warning-rc4002"></a>리소스 컴파일러 오류 RC4002

'identifier' 매크로의 실제 매개 변수가 너무 많습니다.

주어진된 식별자를 사용 하 여 지정 된 실제 매개 변수 수가 식별자의 매크로 정의에 지정 된 정식 매개 변수 수보다 큽니다.

추가 실제 매개 변수 수집 있었지만 매크로 확장 하는 동안 무시 됩니다.
