# <a name="resource-compiler-fatal-error-rc1017"></a>리소스 컴파일러 심각한 오류 RC1017

정수 계열 상수 식이 잘못되었습니다.

지시문의 식이 `#if` 없거나 상수로 계산 되지 않았습니다.
