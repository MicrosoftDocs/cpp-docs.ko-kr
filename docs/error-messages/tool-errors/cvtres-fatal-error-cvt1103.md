# <a name="cvtres-fatal-error-cvt1103"></a>CVTRES 심각한 오류 CVT1103

파일을 읽을 수 없습니다.

지정 된 파일을 읽는 데 cvtres를 복구할 수 없는 오류가 발생 했습니다.

이 오류는 파일 손상으로 발생할 수 있습니다.
