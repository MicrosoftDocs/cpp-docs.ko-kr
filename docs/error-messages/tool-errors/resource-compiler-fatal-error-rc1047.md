# <a name="resource-compiler-fatal-error-rc1047"></a>리소스 컴파일러 심각한 오류 RC1047

"너무 많은 옵션 옵션, ' 문자열 '"

지정 된 옵션이 너무 여러 번 지정 되었습니다. 지정 된 문자열은 오류를 발생 시킨 옵션에 대 한 인수입니다.
