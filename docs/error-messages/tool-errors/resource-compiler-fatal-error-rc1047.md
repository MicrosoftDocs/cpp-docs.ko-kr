# <a name="resource-compiler-fatal-error-rc1047"></a>리소스 컴파일러 심각한 오류 RC1047

너무 많은 옵션 "옵션", 'string'

지정 된 옵션을 너무 여러 번 지정 되었습니다. 지정된 된 문자열에는 오류를 발생 시키는 옵션 인수입니다.
