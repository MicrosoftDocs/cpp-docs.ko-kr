# <a name="nmake-fatal-error-u1036"></a>NMAKE 심각한 오류 U1036

구문 오류: ' = ' 왼쪽에 이름이 너무 많습니다.

매크로 정의의 왼쪽에는 하나의 문자열만 사용할 수 있습니다.
