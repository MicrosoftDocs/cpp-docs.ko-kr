# <a name="math-error-m6205"></a>수학 오류 M6205

'function': _TLOSS 오류

중요 (전체 자릿수)는 전체 손실이 발생 했습니다.

이 오류는 cos, sin의 피연산자로 매우 큰 숫자를 지정 하 여 것일 수 있습니다 또는 0과 2 사이의 숫자를 줄여야 하는 피연산자 tan * pi입니다.
