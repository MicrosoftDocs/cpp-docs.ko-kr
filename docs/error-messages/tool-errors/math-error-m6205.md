# <a name="math-error-m6205"></a>수학 오류 M6205

' function ': _TLOSS 오류

중요 (전체 자릿수)의 전체 손실이 발생 했습니다.

이 오류는 0에서 2 * pi 사이의 숫자로 피연산자를 줄여야 하므로 sin, cos 또는 tan의 피연산자로 매우 큰 숫자를 제공 하는 경우에 발생할 수 있습니다.
