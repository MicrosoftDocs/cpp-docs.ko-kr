# <a name="resource-compiler-fatal-error-rc1021"></a>리소스 컴파일러 심각한 오류 RC1021

'string' 전처리기 명령이 잘못되었습니다.

숫자 기호 () 뒤의 문자가 **#** 유효한 전처리기 지시문을 형성 하지 않았습니다.

숫자 기호는 식별자의 첫 문자로 사용할 수 없습니다.
