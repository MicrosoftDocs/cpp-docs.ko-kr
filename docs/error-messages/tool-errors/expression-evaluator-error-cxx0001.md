# <a name="expression-evaluator-error-cxx0001"></a>식 계산기 오류 CXX0001

사용자 함수를 실행 하는 동안 예외 발생

실행 되는 코드에서 일반 보호 오류가 발생 했습니다. 이 오류는 CAN0001와 동일 합니다.
