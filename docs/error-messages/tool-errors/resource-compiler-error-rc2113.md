# <a name="resource-compiler-error-rc2113"></a>리소스 컴파일러 오류 RC2113

대화 상자에 끝이 필요 합니다.

**End** 키워드는 **DIALOG** 문의 끝에 있어야 합니다. 앞의 문에서 왼쪽에 열려 있는 따옴표가 없는지 확인 합니다.
