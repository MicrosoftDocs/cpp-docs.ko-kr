# <a name="resource-compiler-error-rc2113"></a>리소스 컴파일러 오류 RC2113

종료 대화 상자에서 필요 합니다.

**끝** 키워드의 끝에 있어야 합니다.는 **대화 상자** 문입니다. 위의 문에서 왼쪽 열린 따옴표 없음 없는지 확인 합니다.
