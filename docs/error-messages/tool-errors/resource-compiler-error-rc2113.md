# <a name="resource-compiler-error-rc2113"></a>리소스 컴파일러 오류 RC2113
종료 대화 상자에서 필요 합니다.  
  
 **끝** 키워드의 끝에서 수행 되어야 합니다는 **대화** 문. 왼쪽 위의 문에서 열린 따옴표 없는 없는지 확인 합니다.
