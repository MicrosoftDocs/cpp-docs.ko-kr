# <a name="nmake-fatal-error-u1007"></a>NMAKE 심각한 오류 U1007

큰따옴표를 표시 이름에 사용할 수 없습니다

지정 된 대상 이름 또는 파일 이름이 포함 된 큰따옴표 (**"**).

큰따옴표 파일 이름을 둘러쌀 수 있지만 그 안에 포함 될 수 없습니다.
