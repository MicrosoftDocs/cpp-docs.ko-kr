# <a name="nmake-fatal-error-u1007"></a>NMAKE 심각한 오류 U1007

이름에 큰따옴표를 사용할 수 없습니다.

지정 된 대상 이름 또는 파일 이름에 큰따옴표 (**"**)가 포함 되어 있습니다.

큰따옴표는 파일 이름을 포함할 수 있지만 그 안에는 포함할 수 없습니다.
