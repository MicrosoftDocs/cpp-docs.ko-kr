# <a name="expression-evaluator-error-cxx0046"></a>식 계산기 오류 CXX0046

멤버 함수에 필요한 인수 목록

식을 멤버 함수를 호출 하지만 실제 매개 변수를 지정 하지 않았습니다.

이 오류는 can0046과 동일 합니다.
