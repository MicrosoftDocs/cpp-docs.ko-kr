# <a name="expression-evaluator-error-cxx0061"></a>식 계산기 오류 CXX0061
함수 중첩 한도 초과  
  
 식을 중첩 한도 보다 큰 함수를 포함 합니다. 중첩 깊이 줄이기 위해 식을 수정 합니다.  
  
 이 오류는 can0061과 동일 합니다.
