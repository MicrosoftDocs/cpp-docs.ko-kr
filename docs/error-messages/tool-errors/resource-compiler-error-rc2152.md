# <a name="resource-compiler-error-rc2152"></a>리소스 컴파일러 오류 RC2152
잘못 된 제어 문자  
  
 **ACCELERATORS** 문의 제어 문자가 잘못되었습니다. 올바른 제어 문자 (전용)에 캐럿을 다음 자 씩 이루어져 (**^**).
