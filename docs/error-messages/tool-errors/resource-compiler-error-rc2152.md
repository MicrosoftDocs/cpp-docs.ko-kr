# <a name="resource-compiler-error-rc2152"></a>리소스 컴파일러 오류 RC2152

잘못 된 제어 문자

**ACCELERATORS** 문의 제어 문자가 잘못되었습니다. 유효한 제어 문자는 캐럿 ( **^** ) 뒤에 한 문자 (만)로 구성 됩니다.
