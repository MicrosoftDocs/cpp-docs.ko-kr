# <a name="nmake-fatal-error-u1087"></a>NMAKE 심각한 오류 U1087

동일한 대상에 대해: 및:: 종속 파일을 사용할 수 없습니다.

단일 콜론 ( **:** )과 더블 콜론 (`::`) 종속성 모두에 대상을 지정할 수 없습니다.

여러 설명 블록의 대상을 지정 하려면 각 종속성 줄에 `::`를 사용 합니다.
