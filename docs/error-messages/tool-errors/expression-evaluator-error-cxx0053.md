# <a name="expression-evaluator-error-cxx0053"></a>식 계산기 오류 CXX0053

중단점을 바인딩하는 동안 바인딩하 기호 일치

중단점으로 사용 되는 기호가 함수가 아닙니다. 데이터 멤버를 중단점으로 지정 하면이 오류가 발생할 수 있습니다.

이 오류는 CAN0053와 동일 합니다.
