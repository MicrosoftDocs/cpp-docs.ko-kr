# <a name="expression-evaluator-error-cxx0053"></a>식 계산기 오류 CXX0053
중단점을 바인딩하는 동안 비 함수 기호 일치  
  
 중단점으로 사용 되는 기호는 함수 없습니다. 중단점으로 데이터 멤버를 지정 하면이 오류가 발생할 수 있습니다.  
  
 이 오류는 can0053과 동일 합니다.
