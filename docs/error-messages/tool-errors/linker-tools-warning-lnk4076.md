# <a name="linker-tools-warning-lnk4076"></a>링커 도구 경고 LNK4076

잘못 된 증분 상태 파일 'filename'; 비증분 링크

링크 (.ilk) 증분 상태 파일을 쓸 수 없습니다. 두 `filename` 손상 되었거나는 증분 연결 데이터베이스가 아닙니다. 파일 및 다시 연결을 제거 합니다.
