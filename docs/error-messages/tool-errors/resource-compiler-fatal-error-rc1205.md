잘못 된 코드 페이지

코드 페이지 지정 (/c) 옵션 뒤에 잘못 된 코드 페이지입니다. 참조 [코드 페이지](../../c-runtime-library/code-pages.md) 에 *런타임 라이브러리 참조* 자세한 내용은 합니다.
