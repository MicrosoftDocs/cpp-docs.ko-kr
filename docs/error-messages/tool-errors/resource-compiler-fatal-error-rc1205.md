# <a name="resource-compiler-fatal-error-rc1205"></a>리소스 컴파일러 심각한 오류 RC1205
잘못 된 코드 페이지  
  
 코드 페이지 지정 (/c) 옵션 뒤 잘못 된 코드 페이지입니다. 참조 [코드 페이지](../../c-runtime-library/code-pages.md) 에 *런타임 라이브러리 참조* 자세한 정보에 대 한 합니다.
