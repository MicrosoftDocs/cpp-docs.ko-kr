# <a name="resource-compiler-fatal-error-rc1205"></a>리소스 컴파일러 심각한 오류 RC1205

잘못 된 코드 페이지

코드 페이지 지정 (/c) 옵션 뒤에 잘못 된 코드 페이지가 있습니다. 자세한 내용은 [런타임 라이브러리 참조](../../c-runtime-library/code-pages.md) 에서 *코드 페이지* 를 참조하세요.
