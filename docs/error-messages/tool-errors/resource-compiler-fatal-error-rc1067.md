# <a name="resource-compiler-fatal-error-rc1067"></a>리소스 컴파일러 심각한 오류 RC1067

컴파일러 한계: 식별자에는 내부 버퍼 오버플로가 발생 했습니다.

내부 컴파일러 한계를 초과 했습니다.
