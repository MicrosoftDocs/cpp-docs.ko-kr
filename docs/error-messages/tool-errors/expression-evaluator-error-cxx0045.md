# <a name="expression-evaluator-error-cxx0045"></a>식 계산기 오류 CXX0045

not 함수

함수 이름이 아닌 프로그램의 기호에 대 한 인수 목록이 제공 되었습니다.

## <a name="example"></a>예제

```
queue( alpha, beta )
```

때 `queue` 함수가 아닙니다.

이 오류는 can0045와 동일 합니다.
