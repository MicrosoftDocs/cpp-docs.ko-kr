# <a name="expression-evaluator-error-cxx0045"></a>식 계산기 오류 CXX0045

함수가 아닙니다.

프로그램에서 함수 이름이 아닌 기호에 대 한 인수 목록을 제공 했습니다.

## <a name="example"></a>예제

```
queue( alpha, beta )
```

`queue`이 함수가 아닌 경우

이 오류는 CAN0045와 동일 합니다.
