# <a name="linker-tools-error-lnk1240"></a>링커 도구 오류 LNK1240

IDL 내용을 컴파일하지 못했습니다.

MIDL 컴파일 포함된 IDL을 생성 하는 링커 했지만 문제가 발생 했습니다. MIDL에 의해 지정 된 오류를 확인 합니다.
