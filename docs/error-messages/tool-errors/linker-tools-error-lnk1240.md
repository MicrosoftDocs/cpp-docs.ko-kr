# <a name="linker-tools-error-lnk1240"></a>링커 도구 오류 LNK1240

IDL 콘텐츠를 컴파일하지 못했습니다.

링커가 MIDL을 생성 하 여 포함 된 IDL을 컴파일 했지만 문제가 발생 했습니다. MIDL에 지정 된 오류를 확인 합니다.
