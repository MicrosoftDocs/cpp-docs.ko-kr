# <a name="resource-compiler-error-rc2148"></a>리소스 컴파일러 오류 RC2148

하위 언어 ID 너무 큼

하위 언어 ID 값이 범위를 벗어났습니다.

**LANGUAGE** 문에서는 다음 구문을 사용해야 합니다.

**언어** *primary_language_ID*,*secondary_language_ID*

유효한 하위 언어 Id는 WINNT.EXE 파일에서 **SUBLANG_** 상수로 정의 됩니다.
