# <a name="project-build-error-prj0046"></a>프로젝트 빌드 오류 PRJ0046

지정 된 항목이 비어 있으므로 명령줄을 생성할 수 없습니다.

빈 명령줄 메이크파일 구성;에 지정 된 명령줄이 필요 합니다.
