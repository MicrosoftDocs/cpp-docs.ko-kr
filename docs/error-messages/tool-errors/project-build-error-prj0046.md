# <a name="project-build-error-prj0046"></a>프로젝트 빌드 오류 PRJ0046

지정 된 항목은 비어 있으므로 명령줄을 생성할 수 없습니다.

메이크파일 구성에 대해 빈 명령줄이 지정 되었습니다. 명령줄이 필요 합니다.
