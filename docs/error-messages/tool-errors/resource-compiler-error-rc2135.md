# <a name="resource-compiler-error-rc2135"></a>리소스 컴파일러 오류 RC2135

파일을 찾을 수 없습니다. filename

리소스 컴파일러 명령줄에 지정된 파일을 찾을 수 없습니다. 파일이 다른 디렉터리로 이동 되었는지와 파일 이름과 경로가 올바르게 입력 되었는지 확인 하십시오.

**INCLUDE** 환경 변수 또는 Visual C++ **include** 설정을 사용 하 여 파일을 검색 합니다.
