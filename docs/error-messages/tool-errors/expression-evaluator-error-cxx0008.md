# <a name="expression-evaluator-error-cxx0008"></a>식 계산기 오류 CXX0008
**문자열의 끝에 "누락 된**  
  
 리터럴 문자열의 끝에 있어야 할 큰따옴표가 없습니다. 문자열 리터럴을 큰따옴표로 묶어 식을 다시 입력 합니다.  
  
 이 오류는 can0008과 동일 합니다.
