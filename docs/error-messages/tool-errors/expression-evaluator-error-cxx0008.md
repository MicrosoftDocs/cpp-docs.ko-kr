# <a name="expression-evaluator-error-cxx0008"></a>식 계산기 오류 CXX0008

**문자열 끝에 "가 없습니다.**

문자열 리터럴의 끝에 필요한 큰따옴표가 누락 되었습니다. 문자열 리터럴을 큰따옴표로 묶어 식을 다시 입력 합니다.

이 오류는 CAN0008와 동일 합니다.
