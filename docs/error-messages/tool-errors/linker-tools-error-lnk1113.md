# <a name="linker-tools-error-lnk1113"></a>링커 도구 오류 LNK1113

잘못 된 컴퓨터 유형 유형

개체 헤더에 지정 된 컴퓨터 유형이 잘못 되었습니다.

파일이 손상 된 경우이 오류가 발생할 수 있습니다. 개체를 다시 빌드합니다.
