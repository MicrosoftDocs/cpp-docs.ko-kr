# <a name="linker-tools-error-lnk1113"></a>링커 도구 오류 LNK1113
잘못 된 컴퓨터 형식 유형  
  
 개체 헤더에 지정 된 시스템 유형이 올바르지 않습니다.  
  
 이 오류는 파일이 손상 된 경우 발생할 수 있습니다. 개체를 다시 작성 합니다.
