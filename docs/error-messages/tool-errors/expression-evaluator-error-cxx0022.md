# <a name="expression-evaluator-error-cxx0022"></a>식 계산기 오류 CXX0022

_main 전의 함수 호출

디버거가 **_main**함수를 입력 하기 전에는 C 식 계산기가 함수를 계산할 수 없습니다. **_Main** 가 호출 될 때까지 프로그램이 올바르게 초기화 되지 않았습니다.

이 오류는 CAN0022와 동일 합니다.
