# <a name="expression-evaluator-error-cxx0056"></a>식 계산기 오류 CXX0056

줄 번호에 오류가 있습니다.

잘못 된 줄 번호가 지정 되었습니다.

이 오류는 CAN0056와 동일 합니다.
