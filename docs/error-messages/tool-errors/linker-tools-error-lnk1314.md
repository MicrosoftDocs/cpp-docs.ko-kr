# <a name="linker-tools-error-lnk1314"></a>링커 도구 오류 LNK1314

손상 되었거나 잘못 된 COFF 기호 테이블 (정의 되지 않은 정적 또는 레이블 기호)입니다.

지정 된 개체의 섹션에 대해 컴파일러에서 생성 된 콘텐츠가 손상 된 것 같습니다.

이 오류는 컴파일러 오류를 나타낼 수 있습니다.
