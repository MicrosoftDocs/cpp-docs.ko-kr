# <a name="expression-evaluator-error-cxx0016"></a>식 계산기 오류 CXX0016

상수가 너무 큽니다.

C 식 계산기는 4294967295 (0FFFFFFFF 16 진수) 보다 큰 부호 없는 정수 상수 또는 크기가 약 1.8 E + 308 보다 큰 부동 소수점 상수를 받아들일 수 없습니다.

이 오류는 CAN0016와 동일 합니다.
