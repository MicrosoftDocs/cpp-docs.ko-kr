# <a name="resource-compiler-fatal-error-rc1018"></a>리소스 컴파일러 심각한 오류 RC1018

예기치 않은 ' #elif '

`#elif`지시문 `#if` 이, **#ifdef** 또는 **#ifndef** 구문에 표시 되지 않았습니다.

`#if`이 문 앞에, **#ifdef** 또는 **#ifndef** 문이 적용 되는지 확인 합니다.
