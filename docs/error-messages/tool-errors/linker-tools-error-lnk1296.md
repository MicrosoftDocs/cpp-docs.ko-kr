# <a name="linker-tools-error-lnk1296"></a>링커 도구 오류 LNK1296

파일 이름을 로드할 수 없습니다.

지정 된 DLL을 사용할 수 없습니다.

DLL의 가능한 위치는 현재 디렉터리, 시스템 디렉터리, Windows 디렉터리 및 PATH 환경 변수에 지정 된 디렉터리입니다.
