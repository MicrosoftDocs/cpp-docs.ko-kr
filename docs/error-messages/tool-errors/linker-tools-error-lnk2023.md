# <a name="linker-tools-error-lnk2023"></a>링커 도구 오류 LNK2023

dll 또는 진입점 \<dll 또는 진입점이 잘못 되었습니다 >

링커가 잘못 된 버전의 msobj90를 로드 하 고 있습니다. 경로의 링크 .exe 및 msobj90가 동일한 버전 인지 확인 합니다.

Msobj90의 종속성이 없을 수 있습니다. Msobj90에 대 한 종속성 목록은 다음과 같습니다.

- Msvcr90.dll

- Kernel32.dll

컴퓨터에 오래 된 msobj90의 다른 복사본이 있는지 확인 합니다.
