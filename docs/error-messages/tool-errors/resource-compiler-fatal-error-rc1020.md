# <a name="resource-compiler-fatal-error-rc1020"></a>리소스 컴파일러 심각한 오류 RC1020
예기치 않은 ' #endif'  
  
 `#endif` 지시문 짝이 되는 없이 나타난 `#if`, **#ifdef**, 또는 **#ifndef** 지시문입니다.  
  
 짝이 되는 없는지 확인 `#endif` 각 `#if`, **#ifdef**, 및 **#ifndef** 문.
