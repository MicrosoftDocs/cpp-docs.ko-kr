# <a name="resource-compiler-fatal-error-rc1020"></a>리소스 컴파일러 심각한 오류 RC1020

예기치 않은 ' #endif '

`#endif`짝 `#if` 이 되는, **#ifdef** 또는 **#ifndef** 지시문 없이 지시어가 나타났습니다.

`#endif`각 `#if` , **#ifdef** 및 **#ifndef** 문에 대해 일치 하는 항목을 확인 해야 합니다.
