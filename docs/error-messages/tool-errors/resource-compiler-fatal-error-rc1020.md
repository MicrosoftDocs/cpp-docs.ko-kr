# <a name="resource-compiler-fatal-error-rc1020"></a>리소스 컴파일러 심각한 오류 RC1020

예기치 않은 ' #endif '

짝이 되는 `#if`, **#ifdef**또는 **#ifndef** 지시문이 없으면 `#endif` 지시문이 표시 됩니다.

각 `#if`, **#ifdef**및 **#ifndef** 문에 대해 일치 하는 `#endif` 있는지 확인 합니다.
