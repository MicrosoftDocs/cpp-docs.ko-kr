# <a name="resource-compiler-fatal-error-rc1020"></a>리소스 컴파일러 심각한 오류 RC1020

예기치 않은 ' #endif'

`#endif` 일치 하는 없이 지시문 나타난 `#if`를 **#ifdef**, 또는 **#ifndef** 지시문입니다.

있는지는 일치 하는 `#endif` 마다 `#if`를 **#ifdef**, 및 **#ifndef** 문입니다.
