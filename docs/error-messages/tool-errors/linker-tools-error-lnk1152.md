# <a name="linker-tools-error-lnk1152"></a>링커 도구 오류 LNK1152

하나 이상의 데코레이팅되지 않은 기호를 확인할 수 없습니다.

이 오류 [앞에는](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) 확인할 수 없는 각 데코레이팅되지 않은 기호와 데코레이팅되지 않은 기호에 대해 발견 된 중복 기호에 대 한 두 개 이상의 경고 [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) 이 있습니다.
