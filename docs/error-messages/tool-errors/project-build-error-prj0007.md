# <a name="project-build-error-prj0007"></a>프로젝트 빌드 오류 PRJ0007
'Directory' 출력 디렉터리를 만들 수 없습니다.  
  
 Visual c + +를 출력 디렉터리를 만들지 못했습니다. 가능한 원인은 다음과 같습니다.  
  
-   만들 출력 디렉터리 되었습니다 디렉터리는 읽기 전용입니다.  
  
-   컴퓨터는 디스크 공간이 부족 합니다.  
  
-   잘못 된 디렉터리 경로 지정 했습니다.  
  
-   사용자 권한을 디렉터리를 만들 충분 하지 않습니다.
