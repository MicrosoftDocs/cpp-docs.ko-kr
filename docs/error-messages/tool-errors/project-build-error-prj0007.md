# <a name="project-build-error-prj0007"></a>프로젝트 빌드 오류 PRJ0007

'directory' 출력 디렉터리를 만들 수 없습니다. 

Visual C++ 출력 디렉터리를 만들지 못했습니다. 가능한 이유:

- 출력 디렉터리를 만들 디렉터리가 읽기 전용입니다.

- 컴퓨터의 디스크 공간이 부족 합니다.

- 잘못 된 디렉터리 경로를 지정 했습니다.

- 사용자 권한으로는 디렉터리를 만들 수 없습니다.
