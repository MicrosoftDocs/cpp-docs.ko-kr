# <a name="expression-evaluator-error-cxx0067"></a>식 계산기 오류 CXX0067

함수를 실행할 수 없습니다

식에는 함수 호출을 포함 합니다. 일부 식 계산기는 함수 호출을 지원 하지 않습니다.

이 오류는 can0067과 동일 합니다.
