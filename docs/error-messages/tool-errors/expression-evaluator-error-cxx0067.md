# <a name="expression-evaluator-error-cxx0067"></a>식 계산기 오류 CXX0067

함수 실행이 지원 되지 않습니다.

식에 함수 호출이 포함 되어 있습니다. 일부 식 계산기는 함수 호출을 지원 하지 않습니다.

이 오류는 CAN0067와 동일 합니다.
