# <a name="linker-tools-error-lnk2008"></a>링커 도구 오류 LNK2008

픽스업 대상이 ' symbol_name '에 맞춰지지 않았습니다.

링크가 제대로 정렬 되지 않은 개체 파일의 픽스업 대상을 찾았습니다.

이 오류는 사용자 지정 secton 맞춤 (예: #pragma [팩](../../preprocessor/pack.md)), [align](../../cpp/align-cpp.md) 한정자 또는 secton alignment를 수정 하는 어셈블리 언어 코드를 사용 하 여 발생할 수 있습니다.

코드에서 위의 코드를 사용 하지 않는 경우이는 컴파일러에 의해 발생할 수 있습니다.
