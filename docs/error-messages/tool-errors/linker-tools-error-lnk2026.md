# <a name="linker-tools-error-lnk2026"></a>링커 도구 오류 LNK2026

모듈이 SAFESEH 이미지에 대해 안전 하지 않습니다.

[/Safeseh](../../build/reference/safeseh-image-has-safe-exception-handlers.md) 가 지정 되었지만 모듈이 안전한 예외 처리 기능과 호환 되지 않습니다. 이 모듈을 **/safeseh** 와 함께 사용 하려는 경우에는 모듈을 다시 컴파일해야 합니다.
