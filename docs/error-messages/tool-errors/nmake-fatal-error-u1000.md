> 구문 오류: 매크로 호출에 ') '가 없습니다.

매크로 호출에서 왼쪽 괄호 **(** 가 일치 하는 오른쪽 괄호 없이 나타났습니다. **)** 입니다. 올바른 형식은 **$ (** <em>name</em> **)** ;입니다. **$** <em>n</em> 은 한 문자 이름에 사용할 수 있습니다.
