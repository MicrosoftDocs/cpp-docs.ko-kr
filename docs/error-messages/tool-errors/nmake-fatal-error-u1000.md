 왼쪽된 괄호 **(**일치 하는 오른쪽 괄호 없이 나타난, **)**, 매크로 호출 합니다. 올바른 형식은 **$(***이름***)**; `$`  *n*  한 문자로 된 이름을 사용할 수 있습니다.
