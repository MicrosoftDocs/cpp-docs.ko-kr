# <a name="resource-compiler-error-rc2111"></a>리소스 컴파일러 오류 RC2111

잘못 된 컨트롤 형식

각 CONTROL 문의 **대화 상자** 문을 다음 중 하나 여야 합니다. 3STATE AUTO3, 여, AUTORADIO, 편집, 확인란, 콤보 상자, 컨트롤, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, 아이콘, IEDIT, LISTBOX, LTEXT, PUSHBOX, 누름 단추, 라디오 단추, RTEXT, 스크롤 막대, USERBUTTON 합니다.

이러한 제어 문을 제대로 입력 했는지 확인 합니다.
