# <a name="resource-compiler-error-rc2111"></a>리소스 컴파일러 오류 RC2111

컨트롤 형식이 잘못 되었습니다.

**DIALOG** 문의 각 제어 문은 다음 중 하나 여야 합니다. 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BEDIT, CHECKBOX, COMBOBOX, CONTROL, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, \DIT, ICON, IEDIT, LISTBOX, LTEXT, PUSHBOX, 누름, RADIOBUTTON, RTEXT, SCROLLBAR, userbutton.

이러한 제어 문의 철자가 올바른지 확인 합니다.
