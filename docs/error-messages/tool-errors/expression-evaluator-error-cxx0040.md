# <a name="expression-evaluator-error-cxx0040"></a>식 계산기 오류 CXX0040

함수에 암시적 변환이 필요 합니다.

C 식 계산기는 생성자 호출과 관련 된 암시적 변환을 지원 하지 않습니다.

이 오류는 CAN0040와 동일 합니다.
