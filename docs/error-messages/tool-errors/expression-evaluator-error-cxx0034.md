# <a name="expression-evaluator-error-cxx0034"></a>식 계산기 오류 CXX0034

**연산자를 사용 하 여 호환 되지 않는 형식**

지정 된 피연산자 형식과 작업에 대해 올바르지 않습니다.

예를 들어, 임의의 값으로 포인터를 곱할 수 없습니다.

입력 해야의 피연산자는 연산자와 호환 되는 형식으로 캐스팅 합니다.
