# <a name="linker-tools-error-lnk1189"></a>링커 도구 오류 LNK1189

LIBTOOMANYMEMBERS:: 라이브러리의 한계 개체 수를 초과했습니다.

라이브러리의 개체 또는 멤버 수 한계인 65535개를 초과했습니다.

이 문제를 해결하려면 라이브러리를 더 적은 수의 개체나 멤버를 포함하는 둘 이상의 라이브러리로 분할합니다.
