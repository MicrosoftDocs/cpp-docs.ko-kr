# <a name="expression-evaluator-error-cxx0021"></a>식 계산기 오류 CXX0021

구조체 또는 공용 스칼라로 사용

식에서 구조체 또는 공용 구조체를 사용 했지만 요소가 지정 되었습니다.

구조체 또는 공용 구조체 변수를 조작 하는 경우 변수 이름을 필드 한정자 없이 단독으로 나타날 수 있습니다. 구조체 또는 공용 구조체에 식을 사용 하면 원하는 특정 요소를 사용 하 여 한정 되어야 합니다.

식에 사용할 값을 갖는 요소를 지정 합니다.

이 오류는 can0021과 동일 합니다.
