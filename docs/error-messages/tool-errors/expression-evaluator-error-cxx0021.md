# <a name="expression-evaluator-error-cxx0021"></a>식 계산기 오류 CXX0021

스칼라로 사용 되는 구조체 또는 공용 구조체

구조체 또는 공용 구조체가 식에 사용 되었지만 요소가 지정 되지 않았습니다.

구조체 또는 공용 구조체 변수를 조작 하는 경우에는 필드 한정자 없이 변수의 이름이 자체로 표시 될 수 있습니다. 식에서 구조체 또는 공용 구조체를 사용 하는 경우 원하는 특정 요소로 정규화 되어야 합니다.

식에 사용할 값이 포함 된 요소를 지정 합니다.

이 오류는 CAN0021와 동일 합니다.
