# <a name="cvtres-fatal-error-cvt1105"></a>CVTRES 심각한 오류 CVT1105

파일에서 검색할 수 없습니다.

CVTRES에서 파일의 위치로 이동할 수 없습니다.

이 오류는 파일이 손상 된 경우에 발생할 수 있습니다.
