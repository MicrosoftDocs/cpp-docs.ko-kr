# <a name="resource-compiler-fatal-error-rc1105"></a>리소스 컴파일러 심각한 오류 RC1105
잘못 된 스위치, 옵션: 너무 많은 /d 스위치  
  
 너무 많은 기호가 기호 정의 사용 하 여 정의 된 (/d) 옵션입니다. 원본 및 재컴파일에 일부 기호를 정의 합니다.
