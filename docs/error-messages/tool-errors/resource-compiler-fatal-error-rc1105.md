# <a name="resource-compiler-fatal-error-rc1105"></a>리소스 컴파일러 심각한 오류 RC1105

잘못 된 스위치, 옵션:/d 스위치가 너무 많습니다.

기호 정의 (/d) 옵션을 사용 하 여 너무 많은 기호가 정의 되었습니다. 원본에서 일부 기호를 정의 하 고 다시 컴파일하십시오.
