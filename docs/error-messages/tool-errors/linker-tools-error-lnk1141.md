# <a name="linker-tools-error-lnk1141"></a>링커 도구 오류 LNK1141

내보내기 파일을 빌드하는 동안 오류가 발생 했습니다.

[LINK](../../build/reference/linking.md) 에서 내보내기 파일 (.exp)을 빌드할 수 없습니다.

이 오류의 원인으로는 옵션 또는 모듈 정의 문에서 메모리 부족 및 구문 오류와 같은 시스템 문제가 있습니다.

이 오류 앞에는 추가 정보를 제공 하는 다른 오류가 발생 합니다.
