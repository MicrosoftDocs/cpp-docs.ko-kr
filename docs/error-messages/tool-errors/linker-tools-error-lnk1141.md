# <a name="linker-tools-error-lnk1141"></a>링커 도구 오류 LNK1141

내보내기 파일 빌드 동안 오류가 발생 했습니다

[링크](../../build/reference/linker-command-line-syntax.md) 내보내기 (.exp) 파일을 만들지 못했습니다.

이 오류의 원인은 옵션 또는 모듈 정의 문의 부족 한 메모리 및 구문 오류와 같은 시스템 문제를 포함 합니다.

이 오류는 추가 정보를 제공 하는 다른 오류 뒤에 옵니다.
