# <a name="resource-compiler-error-rc2169"></a>리소스 컴파일러 오류 RC2169

리소스 파일 파일 이름이 2.03 형식이 아닙니다.

지정 된 리소스에서 버전 2.03 이전의 형식을 사용 했습니다. 버전 3.00 이상에 대 한 형식을 사용 하 여 리소스 파일을 변환 하거나 다시 만들어야 합니다.
