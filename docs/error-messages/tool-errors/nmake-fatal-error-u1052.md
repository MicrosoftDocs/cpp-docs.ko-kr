# <a name="nmake-fatal-error-u1052"></a>NMAKE 심각한 오류 U1052

> '*filename*' 파일을 찾을 수 없습니다.

NMAKE에서 다음 중 하나를 사용 하 여 지정 된 파일을 찾을 수 없습니다.

- **/F** 옵션

- **!** 전처리 지시문 포함

- **\@** 지시 파일에 대 한 At 기호 () 지정자

파일이 있고 파일 이름의 철자가 올바른지 확인 합니다.
