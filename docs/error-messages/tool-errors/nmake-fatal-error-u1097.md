# <a name="nmake-fatal-error-u1097"></a>NMAKE 심각한 오류 U1097

파일 이름 부분 구문에는 종속이 필요 합니다.

현재 종속성에 명시적 종속 또는 암시적 종속 관계가 없습니다. Percent () 지정자를 사용 하는 파일 이름 부분 구문은 `%` 현재 대상의 첫 번째 종속 파일의 구성 요소를 나타냅니다.
