# <a name="nmake-fatal-error-u1097"></a>NMAKE 심각한 오류 U1097
파일 이름 부분 구문에는 종속 파일이 있어야합니다.  
  
 현재 종속성 중 하나는 명시적 또는 암시적 종속이 없습니다. 백분율을 사용 하 여 파일 이름 부분 구문 (`%`) 지정자의 현재 대상의 첫 번째 종속 구성 요소를 나타냅니다.
