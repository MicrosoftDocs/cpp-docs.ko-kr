# <a name="linker-tools-error-lnk1287"></a>링커 도구 오류 LNK1287

관리 되는 진입점 함수가 잘못 되었습니다.

진입점이 관리 되는 이미지에 적합 하지 않습니다. 관리 되는 진입점 함수의 반환 형식은 `void` 또는 `int`만 가능 합니다. 관리 되는 진입점 함수의 매개 변수 형식은 `void` 또는 `String []`만 가능 합니다.
