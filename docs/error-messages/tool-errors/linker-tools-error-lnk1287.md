# <a name="linker-tools-error-lnk1287"></a>링커 도구 오류 LNK1287
잘못 된 관리 되는 진입점 함수  
  
 진입점 관리 되는 이미지에 대해 올바르지 않습니다. 관리 되는 진입점 함수의 반환 형식은 수 `void` 또는 `int`합니다. 관리 되는 진입점 함수의 매개 변수 형식은 수 `void` 또는 `String []`합니다.
