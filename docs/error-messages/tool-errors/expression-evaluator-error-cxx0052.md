# <a name="expression-evaluator-error-cxx0052"></a>식 계산기 오류 CXX0052

멤버 함수가 없습니다.

멤버 함수가 중단점으로 지정 되었지만 찾을 수 없습니다. 인라인 된 함수에 중단점을 설정 하면이 오류가 발생할 수 있습니다.

이 함수에서 중단점을 설정 하려면 인라인 강제 해제 (/Sv0)를 사용 하 여 파일을 다시 컴파일하십시오.

식이 정의 되지 않은 함수를 호출 했습니다.

이 오류는 CAN0052와 동일 합니다.
