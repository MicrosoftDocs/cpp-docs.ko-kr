# <a name="expression-evaluator-error-cxx0052"></a>식 계산기 오류 CXX0052

멤버 함수가 없습니다.

멤버 함수 중단점으로 지정 되었지만 찾을 수 없습니다. 인라인 된 함수에 중단점을 설정 하면이 오류가 발생할 수 있습니다.

사용 하 여 파일을 다시 컴파일하고 인라인 해제 강제 (/ Ob0)이이 함수에 중단점을 설정 하려면.

정의 되지 않은 함수를 호출 하는 식입니다.

이 오류는 can0052와 동일 합니다.
