# <a name="command-line-warning-d9028"></a>명령줄 경고 D9028

> 최소 다시 빌드 실패, 일반 빌드로 되돌리기

프로젝트 .idb 파일이 손상 되었습니다. 파일을 삭제 하 고 다시 빌드합니다.

최소 다시 빌드 및 .idb 파일에 대 한 자세한 내용은 사용 되지 않는 [/gm 컴파일러 옵션](../../build/reference/gm-enable-minimal-rebuild.md)을 참조 하세요.
