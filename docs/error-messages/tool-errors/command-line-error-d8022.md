# <a name="command-line-error-d8022"></a>명령줄 오류 D8022
'messagefile'를 열 수 없습니다.  
  
 지정된 된 파일에는 PATH 환경 변수에 지정 된 디렉터리 또는 현재 디렉터리에 있지 했습니다. 메시지 파일의 한 간단한 요약 컴파일러 명령줄 구문 및 옵션을 포함합니다.  
  
 현재 디렉터리 또는 현재 경로에 디렉터리에이 파일을 이동 합니다. 이 파일을 찾을 수 없는 경우 설치 디스크에서 복사 하기 위한 설치 프로그램을 실행 합니다.
