# <a name="command-line-error-d8022"></a>명령줄 오류 D8022

' messagefile '를 열 수 없습니다.

지정 된 파일이 현재 디렉터리 또는 PATH 환경 변수에 지정 된 디렉터리에 없습니다. 메시지 파일에는 컴파일러 명령줄 구문 및 옵션에 대 한 간략 한 요약이 포함 되어 있습니다.

이 파일을 현재 디렉터리 또는 현재 경로에 있는 디렉터리로 이동 합니다. 이 파일을 찾을 수 없는 경우 설치 프로그램을 실행 하 여 배포 디스크에서 복사 합니다.
