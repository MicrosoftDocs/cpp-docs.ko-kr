# <a name="linker-tools-error-lnk1000"></a>링커 도구 오류 LNK1000

> 알 수 없는 오류입니다. 기술 지원 옵션에 대 한 설명서를 참조 하세요.

오류가 발생 한 상황을 확인 하 고 문제를 격리 하 여 재현 가능한 테스트 사례를 만듭니다. 이러한 오류를 조사 및 보고 하는 방법에 대 한 자세한 내용은 [시각적 C++ 도구 집합 또는 설명서를 사용 하 여 문제를 보고 하는 방법](../../overview/how-to-report-a-problem-with-the-visual-cpp-toolset.md)을 참조 하세요.

표준 헤더 파일 (예: Windows)과 고유한 파일을 함께 사용할 경우이 오류가 발생할 수 있습니다. 미리 컴파일된 헤더 (있는 경우)를 포함 한 다음 표준 헤더와 사용자 고유의 헤더 파일을 포함 합니다.
