# <a name="resource-compiler-fatal-error-rw1016"></a>리소스 컴파일러 심각한 오류 RW1016

 리소스 컴파일러는 다른 오류로 인해 중단 되었습니다. 다른 오류를 해결 하 고 다시 컴파일하십시오.
