# <a name="project-build-error-prj0005"></a>프로젝트 빌드 오류 PRJ0005
'Directory' 디렉터리에 임시 파일을 만들 수 없습니다.  
  
 임시 파일을 만들 호출이 실패 했습니다. 실패 한 이유는 다음과 같습니다.  
  
-   임시 파일 이름이 부족합니다.  
  
-   임시 디렉터리는 읽기 전용입니다.  
  
-   임시 디렉터리 또는 TMP 환경 변수에 없을 합니다.  
  
-   이 컴퓨터에서는 사용 가능한 디스크 공간이 부족 합니다.
