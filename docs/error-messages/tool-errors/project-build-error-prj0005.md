# <a name="project-build-error-prj0005"></a>프로젝트 빌드 오류 PRJ0005

'Directory' 디렉터리에 임시 파일을 만들 수 없습니다.

임시 파일을 만들를 호출 하지 못했습니다. 오류의 원인은 다음과 같습니다.

- 임시 파일 이름이 부족합니다.

- 임시 디렉터리에는 읽기 전용입니다.

- 임시 디렉터리 또는 TMP 환경 변수가 필요 하지 않습니다.

- 컴퓨터는 사용 가능한 디스크 공간이 부족합니다.
