# <a name="bscmake-error-bk1517"></a>BSCMAKE 오류 BK1517

/Yc 및 /Yu를 사용 하 여 컴파일된 sbrfile에 대 한 소스 파일

.Sbr 파일 자체를 가리킵니다. /Yc로 컴파일한 후 /Yu를 사용 하 여 다시 컴파일 했습니다. 소스 파일에 대 한 컴파일러 옵션을 /Yc로 다시 설정한 다음 선택 **Rebuild** 새.sbr 파일을 생성 합니다. /Yu를 사용 하 여 원본 파일을 컴파일하지 않습니다.
