# <a name="bscmake-error-bk1517"></a>BSCMAKE 오류 BK1517

/Yc와/Yu를 모두 사용 하 여 컴파일된 sbrfile의 원본 파일

.Sbr 파일은 자신을 참조 합니다. /Yc.를 사용 하 여 컴파일한 후/Yu를 사용 하 여 다시 컴파일할 수 있습니다. 원본 파일에 대 한 컴파일러 옵션을/Yc로 다시 설정 하 고 **다시 빌드** 를 선택 하 여 새 .sbr 파일을 생성 합니다. /Yu.를 사용 하 여 소스 파일을 다시 컴파일하지 마세요.
