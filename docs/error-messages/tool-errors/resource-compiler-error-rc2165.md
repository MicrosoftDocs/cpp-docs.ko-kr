# <a name="resource-compiler-error-rc2165"></a>리소스 컴파일러 오류 RC2165

DLGINCLUDE 문에서 문자열을 찾을 수 없습니다.

문에서 올바른 포함 파일을 지정 하지 않았습니다.

**DLGINCLUDE** 문은 다음 구문을 사용 해야 합니다.

**DLGINCLUDE "**<em>filename</em>**.h"**
