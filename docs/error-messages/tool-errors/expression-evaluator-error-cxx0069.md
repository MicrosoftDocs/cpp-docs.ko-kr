# <a name="expression-evaluator-error-cxx0069"></a>식 계산기 오류 CXX0069

변수 스택 프레임이 필요합니다.

식 계산기 스택 프레임에 없는 변수를 계산할 수 없습니다. 이 인라인 함수의 일부로 선언 된 변수에서 발생할 수 있습니다.
