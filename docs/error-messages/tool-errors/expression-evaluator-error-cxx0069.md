# <a name="expression-evaluator-error-cxx0069"></a>식 계산기 오류 CXX0069

변수에 스택 프레임이 필요 합니다.

식 계산기는 스택 프레임에서 발생 하지 않으므로 변수를 계산할 수 없습니다. 이는 인라인 함수의 일부로 선언 된 변수에 의해 발생할 수 있습니다.
