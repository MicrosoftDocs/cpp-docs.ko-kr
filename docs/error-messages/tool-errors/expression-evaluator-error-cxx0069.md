# <a name="expression-evaluator-error-cxx0069"></a>식 계산기 오류 CXX0069
변수에 스택 프레임이 필요합니다.  
  
 스택 프레임에 발생 하지 않으므로 식 계산기에서 변수를 계산할 수 없습니다. 인라인 함수의 일환으로 선언 된 변수 때문일 수 있습니다.
