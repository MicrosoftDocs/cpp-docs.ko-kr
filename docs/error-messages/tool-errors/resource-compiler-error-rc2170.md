# <a name="resource-compiler-error-rc2170"></a>리소스 컴파일러 오류 RC2170

비트맵 파일의 파일 이름이 3.00 형식이 아닙니다.

Windows 버전 2.x 형식을 사용하는 비트맵은 버전 3.x 리소스 파일에서 사용할 수 없습니다.

이 문제를 해결하려면 비트맵을 3.x 형식으로 다시 그리거나 변환합니다.
