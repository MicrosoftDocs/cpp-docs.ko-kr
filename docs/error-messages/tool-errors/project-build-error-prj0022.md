# <a name="project-build-error-prj0022"></a>프로젝트 빌드 오류 PRJ0022

> 알 수 없는 도구, '*property*' 속성에 잘못 된 파일 이름 '*file*'이 있습니다.

속성 *속성* 에 지정 된 파일 이름 *파일이* 잘못 되었습니다.

알 수 없거나 잘못 된 매크로를 사용 했을 수 있습니다.

프로젝트의 속성에 액세스 하는 방법에 대 한 자세한 내용은 [Visual C++ 프로젝트 속성 설정](../../build/working-with-project-properties.md) 을 참조 하세요.
