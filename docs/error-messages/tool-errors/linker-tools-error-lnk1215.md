# <a name="linker-tools-error-lnk1215"></a>링커 도구 오류 LNK1215

메타 데이터 작업이 실패 했습니다. (HRESULT): 오류

.NET 런타임을 통해 메타 데이터 업데이트를 시도 하는 동안 링커가 .NET 런타임에서 오류를 수신 했습니다.

`HRESULT`은 .NET 런타임 메서드의 HRESULT입니다. `error`입니다. NET에서 제공 하는 텍스트입니다.

일치 하지 않는 링커 및 .NET 런타임이 있을 수 있습니다. 시각적 개체 C++를 다시 설치 합니다.
