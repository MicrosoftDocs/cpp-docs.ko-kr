# <a name="project-build-error-prj0047"></a>프로젝트 빌드 오류 PRJ0047

일시 중단 된 프로세스를 계속할 수 없습니다.  빌드가 실패 했습니다.

개발 환경에서 오류가 발생 했습니다.

이 오류를 해결 하려면 닫고 프로젝트를 다시 로드 합니다. 필요한 경우에 종료 하 고 Visual Studio를 다시 시작 합니다.
