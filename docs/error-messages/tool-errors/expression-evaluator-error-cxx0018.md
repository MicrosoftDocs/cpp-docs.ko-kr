# <a name="expression-evaluator-error-cxx0018"></a>식 계산기 오류 CXX0018
잘못 된 레지스터 이름  
  
 지정 된 레지스터 존재 하지 않거나를 표시할 수 없습니다.  
  
 조사식 창에는 다음에 레지스터 표시 수 있습니다.  
  
 AX SP DS IP  
  
 BX BP ES FL  
  
 CX SI SS GS  
  
 DX DI CS SS  
  
 EAX ESP DS EIP  
  
 EBX EBP ES EFL  
  
 ECX ESI FS  
  
 EDX EDI CS  
  
 이 오류는 can0018과 동일 합니다.
