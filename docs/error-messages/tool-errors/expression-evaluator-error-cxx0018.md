# <a name="expression-evaluator-error-cxx0018"></a>식 계산기 오류 CXX0018

레지스터 이름이 잘못 되었습니다.

지정 된 레지스터가 없거나 표시할 수 없습니다.

조사식 창는 다음 레지스터를 표시할 수 있습니다.

AX SP DS IP

BX BP ES FL

CX SI SS GS

DX DI CS SS

EAX ESP DS EIP

EBX EBP ES EFL

ECX ESI FS

EDX EDI CS

이 오류는 CAN0018와 동일 합니다.
