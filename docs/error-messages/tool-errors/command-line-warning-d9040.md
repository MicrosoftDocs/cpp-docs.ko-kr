# <a name="command-line-warning-d9040"></a>명령줄 경고 D9040

> '/analyze ' 옵션을 무시 합니다. 이 버전의 컴파일러에서는 코드 분석 경고를 사용할 수 없습니다.

**`/analyze`** 명령줄 옵션은 모든 버전의 Visual Studio에서 사용할 수 없습니다. 이 경고를 해결하려면, 지원되는 Visual Studio 버전으로 전환하거나 명령줄 옵션을 제거합니다.
