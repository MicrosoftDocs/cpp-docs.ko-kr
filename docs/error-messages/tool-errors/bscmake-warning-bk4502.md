# <a name="bscmake-warning-bk4502"></a>BSCMAKE 경고 BK4502

잘라낼. ' Filename ' .SBR 파일 이름에 없습니다.

업데이트 중 원래 .bsc 파일에 속하지 않은 길이가 0 인 .sbr 파일이 지정 되었습니다.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. 지정한 파일 이름이 잘못 되었습니다.

1. 파일이 삭제 되었습니다. (오류 [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) 결과)

1. 파일이 손상 되어 BSCMAKE에서 전체 빌드를 수행 해야 합니다.
