# <a name="bscmake-warning-bk4502"></a>BSCMAKE 경고 BK4502
잘립니다. 'Filename' 파일 이름에 없는 SBR 파일  
  
 길이가 0 인.sbr 파일.bsc 파일에 포함 된 원래을 업데이트 하는 동안 지정 되었습니다.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면  
  
1.  지정한 이름이 잘못 되었습니다.  
  
2.  파일을 삭제할 수 있습니다. (오류 [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) 결과입니다.)  
  
3.  파일 손상 전체 빌드를 수행 하려면.
