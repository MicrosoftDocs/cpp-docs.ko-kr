# <a name="bscmake-warning-bk4502"></a>BSCMAKE 경고 BK4502

잘립니다. 'Filename' 파일에 없는 SBR 파일

포함 되지 않은 원래.bsc 파일의 길이가 0 인.sbr 파일을 업데이트 하는 동안 지정 되었습니다.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. 지정한 이름이 잘못 되었습니다.

1. 파일을 삭제 합니다. (오류 [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) 결과입니다.)

1. 파일이 손상 BSCMAKE 전체 빌드를 수행 하도록 요구 합니다.
