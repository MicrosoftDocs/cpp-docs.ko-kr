# <a name="math-error-m6107"></a>수학 오류 M6107

에뮬레이션 되지 않음

잘못 되었거나 에뮬레이터에서 지원 하지 않는 보조 프로세서 명령을 실행 하려고 한 경우

프로그램이 종료 코드 135를 사용 하 여 종료 됩니다.
