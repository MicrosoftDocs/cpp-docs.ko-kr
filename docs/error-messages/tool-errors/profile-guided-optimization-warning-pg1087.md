# <a name="profile-guided-optimization-warning-pg1087"></a>프로필 기반 최적화 경고 PG1087

PGC 파일 ' file '이 잘못 되었습니다.

프로 파일링 세션이 비정상적으로 종료 되어 .pgc 파일이 손상 되었을 수 있습니다.  프로 파일링 시나리오를 다시 실행 하 고 정상적으로 종료 합니다.
