이 섹션은 빌드 도구에 의해 생성된 오류에 대한 참조입니다. 특정 오류 메시지에 도움말을 보려면 출력 창의 오류 번호를 마우스로 클릭 하 고 F1 키를 하거나 입력에 오류 번호는 **찾아보십시오** 인덱스 상자로 합니다.
