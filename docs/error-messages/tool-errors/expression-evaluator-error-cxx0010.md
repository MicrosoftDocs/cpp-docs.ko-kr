# <a name="expression-evaluator-error-cxx0010"></a>식 계산기 오류 CXX0010

왼쪽 대괄호가 없습니다.

식에 불균형 대괄호가 포함 되어 있습니다. 짝이 되는 대괄호를 사용 하 여 식을 다시 입력 합니다.

이 오류는 CAN0010와 동일 합니다.
