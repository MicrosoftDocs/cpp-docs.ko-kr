# <a name="expression-evaluator-error-cxx0010"></a>식 계산기 오류 CXX0010

왼쪽된 괄호 누락

식 불균형된 대괄호를 포함합니다. 균형 있는 대괄호를 사용 하 여 식을 다시 입력 하십시오.

이 오류는 can0010과 동일 합니다.
