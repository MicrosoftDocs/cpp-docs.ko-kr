# <a name="project-build-error-prj0013"></a>프로젝트 빌드 오류 PRJ0013

시스템 리소스가 심각하게 부족할 수 있습니다. 빌드를 시작하는 데 필요한 파이프를 만들 수 없습니다.

이 오류는 시스템 리소스가 부족함을 나타냅니다. 이 오류를 해결하려면 다른 프로세스/애플리케이션의 시스템 리소스 사용을 줄입니다.

이 오류는 보안 수준이 파이프를 만들 수 없는 경우에도 발생할 수 있습니다 ( [CreatePipe](/windows/win32/api/namedpipeapi/nf-namedpipeapi-createpipe)참조).
