# <a name="resource-compiler-error-rc2015"></a>리소스 컴파일러 오류 RC2015

상수에 문자가 너무 많습니다.

문자 상수에 문자가 두 개 이상 포함 되어 있습니다.

문자 상수는 한 문자 (표준 문자 상수) 또는 두 문자 (긴 문자 상수)로 제한 됩니다.

이스케이프 시퀀스 (예: \t for tab)는 단일 문자로 변환 됩니다.
