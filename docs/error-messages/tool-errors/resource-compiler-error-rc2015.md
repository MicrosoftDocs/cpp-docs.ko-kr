# <a name="resource-compiler-error-rc2015"></a>리소스 컴파일러 오류 RC2015

상수에 문자가 너무 많습니다.

두 개 이상의 문자를 포함 하는 문자 상수입니다.

문자 상수 (표준 문자 상수) 문자를 한 개 또는 두 개의 문자 (긴 문자 상수)으로 제한 됩니다.

참고는 이스케이프 시퀀스 (예: 탭 \t)는 단일 문자 변환 됩니다.
