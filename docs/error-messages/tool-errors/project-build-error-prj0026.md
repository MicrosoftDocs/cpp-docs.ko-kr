# <a name="project-build-error-prj0026"></a>프로젝트 빌드 오류 PRJ0026

> 응답 파일 '*파일*' 사용자의 ANSI 코드 페이지로 변환할 수 없는 유니코드 내용이 있습니다.
>
> *파일의 유니코드 콘텐츠*

프로젝트 시스템이 유니코드 내용이 제대로 사용자의 현재 ANSI 코드 페이지로 변환할 수 없는 응답 파일을 찾았습니다.

이 오류에 대 한 해상도를 ANSI를 사용 하거나 코드 페이지를 컴퓨터에 설치 하 고 시스템 기본값으로 설정 하는 응답 파일의 내용을 업데이트 하는 것입니다.
