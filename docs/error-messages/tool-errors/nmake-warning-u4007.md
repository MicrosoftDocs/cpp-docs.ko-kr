# <a name="nmake-warning-u4007"></a>NMAKE 경고 U4007

파일 이름이 너무 깁니다. ' filename' 8.3으로 잘립니다.

지정된 된 파일의 기본 이름이 8 자 또는 확장에는 세 개 이상의 문자입니다. Nmake는 8 자 자료에 3 자로 확장 이름.

긴 파일 이름 파일 시스템에서 지원 되는 경우 큰따옴표로 이름을 묶습니다 (**"**).
