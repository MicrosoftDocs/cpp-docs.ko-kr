# <a name="nmake-warning-u4007"></a>NMAKE 경고 U4007

' filename ' 파일 이름이 너무 깁니다. 8.3로 자르기

지정 된 파일의 기본 이름이 8 자를 초과 하거나 확장에 3 자 이상이 있습니다. NMAKE는 이름을 8 자 기준으로, 3 자 확장으로 자릅니다.

파일 시스템에서 긴 파일 이름을 지 원하는 경우 이름을 큰따옴표 (**"**)로 묶습니다.
