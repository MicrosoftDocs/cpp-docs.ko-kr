# <a name="resource-compiler-error-rc2124"></a>리소스 컴파일러 오류 RC2124
빈 메뉴를 사용할 수 없습니다  
  
 **끝** 키워드가 나타나는 메뉴 항목을 모두에 정의 되기 전에 **메뉴** 문. 리소스 컴파일러는 빈 메뉴를 허용 하지 않습니다. 내에서 여는 따옴표가 없는 있는지 확인은 **메뉴** 문.
