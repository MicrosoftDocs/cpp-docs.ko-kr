# <a name="command-line-warning-d9036"></a>명령줄 경고 D9036

> '*option \_ 1*' 대신 '*option \_ 2*'

## <a name="remarks"></a>설명

경고 D9036는 [명령줄 경고 D9035](../../error-messages/tool-errors/command-line-warning-d9035.md)의 연속입니다. 사용 되지 않는 옵션 대신 하나의 컴파일러 옵션을 사용 하는 것이 좋습니다 .이 옵션은 여기에 나열 되어 있습니다. 자세한 정보에 대 한 링크는 [사용 되지 않는 컴파일러 옵션](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options)을 참조 하세요.
