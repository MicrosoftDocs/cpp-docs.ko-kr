# <a name="command-line-warning-d9036"></a>명령줄 경고 D9036

> '*옵션\_2*' of '*옵션\_1*'

## <a name="remarks"></a>설명

경고 D9036 연속 된 것 [명령줄 경고 D9035](../../error-messages/tool-errors/command-line-warning-d9035.md)합니다. 이상의 컴파일러 옵션을 사용 되지 않는 옵션 대신 선호 하는 경우 여기에 나열 되어 있습니다. 자세한 정보에 대 한 링크를 참조 하세요 [제거 컴파일러 옵션과 사용 되지 않는](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options)합니다.
