# <a name="project-build-warning-prj0018"></a>프로젝트 빌드 경고 PRJ0018

다음 환경 변수를 찾을 수 없습니다.

환경 변수 정의 되지 않았습니다. 이 오류는 정의 되지 않은 환경 변수를 나열 합니다.

참조 [사용자 지정 빌드 명령에 대 한 매크로](../../ide/common-macros-for-build-commands-and-properties.md) 빌드 프로세스에서 매크로 대 한 정보에 대 한 합니다.
