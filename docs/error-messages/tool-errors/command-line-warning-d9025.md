   프로젝트의 속성 페이지에서 옵션을 설정 (솔루션 탐색기에서 선택한 프로젝트 노드)와 컴파일러의 전처리기 속성 페이지에서 확인 합니다.  여기서 설정, 솔루션 탐색기에서 각 소스 코드 파일에 대 한 전처리기 속성 페이지 설정 되었는지 확인 하는 옵션을 표시 되지 않으면 있습니다 추가 되지 않습니다.
