# <a name="resource-compiler-fatal-error-rc1109"></a>리소스 컴파일러 심각한 오류 RC1109

리소스 이름을 만드는 동안 오류가 발생 했습니다.

지정 된 .res 파일을 만들 수 없습니다. 읽기 전용 드라이브에서 생성 되 고 있지 않은지 확인 합니다. /V 옵션을 사용 하 여 파일을 만들고 있는지 여부를 확인 합니다.
