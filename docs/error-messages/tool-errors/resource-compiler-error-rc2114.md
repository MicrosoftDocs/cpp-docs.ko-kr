# <a name="resource-compiler-error-rc2114"></a>리소스 컴파일러 오류 RC2114

컨트롤 클래스 이름이 필요 합니다.

`class` **DIALOG** 문의 CONTROL 문 필드는 BUTTON, COMBOBOX, EDIT, LISTBOX, SCROLLBAR, STATIC 또는 사용자 정의 형식 중 하나 여야 합니다. 클래스의 철자가 정확한 지 확인 합니다.
