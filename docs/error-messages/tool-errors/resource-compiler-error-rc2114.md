# <a name="resource-compiler-error-rc2114"></a>리소스 컴파일러 오류 RC2114
예상된 컨트롤 클래스 이름  
  
 `class` 에서 제어 문의 필드는 **대화** 문은 다음 형식 중 하나 이어야 합니다: 단추, 콤보 상자, 편집, 목록 상자, 스크롤 막대, STATIC, 또는 사용자 정의 합니다. 클래스는 제대로 입력 했는지 확인 합니다.
