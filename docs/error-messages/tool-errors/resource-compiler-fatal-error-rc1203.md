# <a name="resource-compiler-fatal-error-rc1203"></a>리소스 컴파일러 심각한 오류 RC1203

잘못 된 16 진수 기본 언어 ID를 지정 했습니다.

기본 언어 지정 (/l) 옵션 뒤에 잘못 된 16 진수 언어 ID가 있습니다. 유효한 언어 Id 목록은 *런타임 라이브러리 참조* 에서 [언어 및 국가/지역 문자열](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) 을 참조 하세요.
