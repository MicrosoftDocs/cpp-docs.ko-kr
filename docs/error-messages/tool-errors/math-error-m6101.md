# <a name="math-error-m6101"></a>수학 오류 M6101

잘못됨

잘못된 작업입니다.

이 오류는 피연산자가 NaN (숫자가 아님) 이거나 무한대 일 때 발생할 수 있습니다.

프로그램이 종료 코드 129를 사용 하 여 종료 됩니다.
