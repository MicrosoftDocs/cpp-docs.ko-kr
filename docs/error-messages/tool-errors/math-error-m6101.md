# <a name="math-error-m6101"></a>수학 오류 M6101

잘못된

잘못 된 작업입니다.

이 오류는 무한대 또는 NaN (숫자가 아님)는 피연산자가 발생할 수 있습니다.

프로그램은 129 종료 코드로 종료 됩니다.
