# <a name="bscmake-error-bk1514"></a>BSCMAKE 오류 BK1514
모든 합니다. SBR 파일 잘림 파일 이름에 찾을 수 없습니다.  
  
 업데이트에 대 한 지정한.sbr 파일이 원래 찾아보기 정보 (.bsc) 파일의 부분 이었습니다. 이 오류를 일으킨.sbr 파일의 이름을 찾으려고 읽기는 [BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) 앞에 있는 경고 합니다.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면  
  
1.  지정 된.sbr 또는.bsc 파일 이름이 잘못 되었습니다.  
  
2.  .Bsc 파일이 손상 된 다시 BSCMAKE 필요 합니다.
