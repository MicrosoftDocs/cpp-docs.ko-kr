# <a name="bscmake-error-bk1514"></a>BSCMAKE 오류 BK1514

모든 합니다. SBR 파일 잘리는 경우 파일 이름에 찾을 수 없습니다.

업데이트에 대해 지정 된.sbr 파일이 원래 찾아보기 정보 (.bsc) 파일의 일부가 되었습니다. 이 오류를이 발생 시키는.sbr 파일 이름을 찾으려고 읽기를 [BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) 앞에 경고입니다.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. 지정 된.sbr 또는.bsc 파일 이름이 잘못 되었습니다.

1. .Bsc 파일이 손상 된 다시 BSCMAKE 필요 합니다.
