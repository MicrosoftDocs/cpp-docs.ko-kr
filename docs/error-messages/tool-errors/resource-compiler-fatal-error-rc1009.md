# <a name="resource-compiler-fatal-error-rc1009"></a>리소스 컴파일러 심각한 오류 RC1009

컴파일러 한계: 매크로가 너무 많이 중첩 되었습니다 '매크로'

파일 중첩 매크로 대 한 리소스 컴파일러 한계를 초과합니다. 중첩 깊이 해당 매크로를 줄이기 위해 지정한 원본 파일을 수정 합니다.
