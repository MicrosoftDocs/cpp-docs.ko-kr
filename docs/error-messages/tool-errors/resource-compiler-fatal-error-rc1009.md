# <a name="resource-compiler-fatal-error-rc1009"></a>리소스 컴파일러 심각한 오류 RC1009
컴파일러 한계: 매크로가 너무 많이 중첩 '매크로'  
  
 파일은 매크로 중첩에 대 한 리소스 컴파일러 한계를 초과합니다. 해당 매크로의 중첩 깊이 줄이려면 지정 된 소스 파일을 수정 합니다.
