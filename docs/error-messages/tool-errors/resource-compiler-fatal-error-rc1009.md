# <a name="resource-compiler-fatal-error-rc1009"></a>리소스 컴파일러 심각한 오류 RC1009

컴파일러 한계: 매크로가 너무 많이 중첩 되었습니다. ' macro '

파일이 매크로 중첩에 대 한 리소스 컴파일러 한계를 초과 합니다. 지정 된 소스 파일을 수정 하 여 해당 매크로의 중첩 수준을 줄입니다.
