# <a name="expression-evaluator-error-cxx0060"></a>식 계산기 오류 CXX0060

등록을 사용할 수 없습니다.

식에서 사용할 수 없는 레지스터를 지정 했습니다. 이 오류는를 실행 하는 컴퓨터에 존재 하지 않는 레지스터에 액세스 하려고 할 때 발생할 수 있습니다.

이 오류는 CAN0060와 동일 합니다.
