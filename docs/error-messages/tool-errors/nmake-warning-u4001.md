# <a name="nmake-warning-u4001"></a>NMAKE 경고 U4001

> 명령줄 에서만 명령 파일을 호출할 수 있습니다.

At 기호 ( **\@** ) 지정자에 의해 호출 되는 명령 파일은 다른 명령 파일에 대 한 사양을 포함할 수 없습니다. 이러한 중첩은 허용 되지 않습니다. 사양이 무시 되었습니다.
