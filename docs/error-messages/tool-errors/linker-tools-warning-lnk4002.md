# <a name="linker-tools-warning-lnk4002"></a>링커 도구 경고 LNK4002

개체에 정의 된 기호

데코레이팅된 형식으로 표시 된 기호가 `object`의 데코레이팅되지 않은 형식으로 지정 되었지만 데코레이팅된 기호와 고유한 일치 항목을 찾을 수 없습니다. 이 경고는 항상 경고 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) 로 시작 하 고 그 다음에 심각한 오류 [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)가 옵니다.
