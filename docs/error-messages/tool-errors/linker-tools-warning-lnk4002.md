# <a name="linker-tools-warning-lnk4002"></a>링커 도구 경고 LNK4002
개체에 정의 된 기호  
  
 데코레이팅된 폼에 표시 되는 기호, 데코레이팅되지에 지정 된 `object`, 하지만 데코레이팅된 기호에 일치 하는 고유한 항목을 찾을 수 없습니다. 이 경고는 경고 앞에 항상 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) 치명적인 오류인 [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)합니다.
