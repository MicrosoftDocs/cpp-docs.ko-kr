# <a name="linker-tools-warning-lnk4002"></a>링커 도구 경고 LNK4002

개체에 정의 된 기호

트 데코 레이 된 형태로 표시 되는 기호는 데코 레이트 되지 않은 형태로에 지정 된 `object`, 하지만 데코레이팅된 기호에만 일치를 찾을 수 없습니다. 이 경고는 경고 앞에 항상 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) 치명적인 오류인 [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)합니다.
