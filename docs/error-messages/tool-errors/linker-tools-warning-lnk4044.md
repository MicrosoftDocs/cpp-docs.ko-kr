# <a name="linker-tools-warning-lnk4044"></a>링커 도구 경고 LNK4044

인식할 수 없는 옵션 'option'; 무시

지정 된 옵션이이 도구에 대 한 올바른 옵션이 아닙니다. 도구 옵션을 무시 합니다.

유효한 옵션의 목록을 보려면, 인수 또는 옵션 없이 도구를 실행 합니다.
