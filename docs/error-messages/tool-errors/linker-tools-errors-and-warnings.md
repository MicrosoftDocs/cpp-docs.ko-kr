# <a name="linker-tools-errors-and-warnings-lnkxxxx"></a>링커 도구 오류 및 경고(LNKxxxx)

링커 도구 링크, LIB, DUMPBIN, 및 EDITBIN 이러한 오류 및 경고를 생성 하는 일반적인 실행 파일을 공유 합니다. LNK 폼의 경고를 생성 하는 도구*xxxx*, 여기서 *xxxx* 는 4 자리 숫자입니다.
