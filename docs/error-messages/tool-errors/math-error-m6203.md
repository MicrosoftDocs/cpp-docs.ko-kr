# <a name="math-error-m6203"></a>수학 오류 M6203
'function': _OVERFLOW 오류  
  
 지정 된 함수 결과를 표현할 너무 큽니다.  
  
 호출 하는이 오류는 `_matherr` 함수 함수 이름, 인수 및 오류 유형을 포함 합니다. 다시 작성할 수 있습니다는 `_matherr` 특정 런타임 부동 소수점 수학 오류 처리 사용자 지정 하는 함수입니다.
