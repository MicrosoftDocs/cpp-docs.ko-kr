# <a name="expression-evaluator-error-cxx0005"></a>식 계산기 오류 CXX0005
지원 되지 않는 연산자  
  
 지원 되지 않는 C 연산자가 식에 지정 되었습니다. 지원 되는 C 연산자를 사용 하 여 해당 하는 식을 작성 합니다.  
  
 이 오류는 can0005와 동일 합니다.
