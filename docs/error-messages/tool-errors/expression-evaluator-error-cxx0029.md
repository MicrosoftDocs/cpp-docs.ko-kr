# <a name="expression-evaluator-error-cxx0029"></a>식 계산기 오류 CXX0029

구조체 포인터가 아닙니다

멤버 선택 연산자 (**->**) 구조에 대 한 포인터가 아닌 식에 적용 되었습니다.

전체 식을 괄호로 묶인 올바르게 이거나 형식 캐스팅 적절 한 구조 포인터 형식으로 주소 식을 확인 합니다.

이 오류는 can0029와 동일 합니다.
