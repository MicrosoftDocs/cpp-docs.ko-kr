# <a name="linker-tools-error-lnk1218"></a>링커 도구 오류 LNK1218

경고를 오류로 처리 됩니다. 출력 파일이 생성 되지 않았습니다

연결할 때 **/WX**, 링커 경고를 오류로 간주 됩니다 및 출력 파일이 생성 됩니다.
