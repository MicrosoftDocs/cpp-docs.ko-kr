# <a name="linker-tools-error-lnk1218"></a>링커 도구 오류 LNK1218

경고가 오류로 처리 되었습니다. 출력 파일이 생성 되지 않았습니다.

**/Wx**와 연결 하면 모든 링커 경고가 오류로 처리 되 고 출력 파일이 생성 되지 않습니다.

자세한 내용은 [/wx (링커 경고를 오류로 처리)](../../build/reference/wx-treat-linker-warnings-as-errors.md) 를 참조 하세요.
