# <a name="resource-compiler-error-rc2109"></a>리소스 컴파일러 오류 RC2109

숫자 대화 상자 상수가 필요 합니다.

**DIALOG** 문에는 *x, y, width*및 *height* 필드의 정수 값이 필요 합니다. 이러한 값이 **대화 상자** 키워드 다음에 포함 되 고 음수가 아닌지 확인 합니다.
