하나 이상의 기호 정의가 여러 개 인해 빌드가 실패 했습니다. 이 오류는 오류를 뒤 [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)합니다.

합니다 [/force](../../build/reference/force-force-file-output.md) 하거나 **/FORCE:MULTIPLE** 옵션이이 오류를 무시 합니다.
