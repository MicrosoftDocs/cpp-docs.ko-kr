# <a name="linker-tools-error-lnk1169"></a>링커 도구 오류 LNK1169

하나 이상의 곱하기 정의 기호가 있습니다.

하나 이상의 기호에 대 한 여러 정의로 인해 빌드에 실패 했습니다. 이 오류는 오류 [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)앞에 나옵니다.

[/Force](../../build/reference/force-force-file-output.md) 또는 **/force: MULTIPLE** 옵션은이 오류를 무시 합니다.
