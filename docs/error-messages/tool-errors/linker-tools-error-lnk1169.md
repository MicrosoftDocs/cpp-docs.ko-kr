# <a name="linker-tools-error-lnk1169"></a>링커 도구 오류 LNK1169
여러 번 정의 된 기호가 있습니다.  
  
 하나 이상의 기호의 정의가 여러 개 있으므로 빌드하지가 못했습니다. 이 오류에 오류 옵니다 [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)합니다.  
  
 [/force](../../build/reference/force-force-file-output.md) 또는 **/FORCE:MULTIPLE** 옵션이이 오류를 무시 합니다.
