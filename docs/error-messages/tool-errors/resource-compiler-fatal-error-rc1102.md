# <a name="resource-compiler-fatal-error-rc1102"></a>리소스 컴파일러 심각한 오류 RC1102

내부 오류: RCPP에 대 한 인수가 너무 많습니다.

리소스 컴파일러 전처리기에 너무 많은 인수가 전달 되었습니다. 원본에서 정의 하 여 기호 정의 (/d) 옵션으로 정의 된 기호 수를 줄입니다. 이 오류는 포함 검색 경로 옵션 (/i)을 사용 하 여 너무 많은 포함 파일 검색 경로를 지정 하는 경우에도 발생할 수 있습니다.
