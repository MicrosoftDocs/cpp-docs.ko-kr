# <a name="resource-compiler-fatal-error-rc1102"></a>리소스 컴파일러 심각한 오류 RC1102

내부 오류: RCPP 인수가 너무 많습니다.

리소스 컴파일러 전처리기에 너무 많은 인수가 전달 되었습니다. 기호 정의 사용 하 여 정의 된 기호 수를 줄입니다 (/d) 원본에 정의한 옵션입니다. 이 오류를 지정 하 여 너무 많은 포함 파일 검색 경로 포함 검색 경로 옵션을 사용 하 여도 발생할 수 있습니다 (/ i).
