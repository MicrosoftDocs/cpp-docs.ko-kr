# <a name="expression-evaluator-error-cxx0044"></a>식 계산기 오류 CXX0044

_based (void) 포인터 사용에는 > 연산자가 필요 합니다.

`void` 기반 포인터는 직접 사용할 수 없습니다. **: >** 연산자를 사용 하 여 완전 한 포인터를 형성 해야 합니다.
