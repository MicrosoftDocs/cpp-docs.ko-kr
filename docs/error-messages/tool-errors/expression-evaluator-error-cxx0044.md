# <a name="expression-evaluator-error-cxx0044"></a>식 계산기 오류 CXX0044
_based(void) 포인터를 사용하려면 :> 연산자가 필요합니다.  
  
 기반 포인터 `void` 직접 사용할 수 없습니다. 사용 하 여 완전 한 포인터를 구성 해야는 **: >** 연산자입니다.  
  
 이 오류는 can0044와 동일 합니다.
