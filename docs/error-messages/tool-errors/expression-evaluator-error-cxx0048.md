# <a name="expression-evaluator-error-cxx0048"></a>식 계산기 오류 CXX0048

호출 시퀀스는 지원 되지 않습니다.

식에 지정 된 함수가 C 식 계산기에서 지원 하지 않는 호출 시퀀스를 사용 합니다. 조사식 창 식에서는이 함수를 호출할 수 없습니다.

이 오류는 CAN0048와 동일 합니다.
