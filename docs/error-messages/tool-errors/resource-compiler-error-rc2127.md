# <a name="resource-compiler-error-rc2127"></a>리소스 컴파일러 오류 RC2127

쉼표로 구분 된 버전 단어 필요

버전 리소스의 버전 번호는 **단어**형식 이어야 하며 쉼표로 구분 해야 합니다.
