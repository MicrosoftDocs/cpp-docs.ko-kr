# <a name="resource-compiler-error-rc2112"></a>리소스 컴파일러 오류 RC2112

대화 상자에 BEGIN이 필요 합니다.

**BEGIN** 키워드는 **DIALOG** 키워드 바로 뒤에와 야 합니다.
