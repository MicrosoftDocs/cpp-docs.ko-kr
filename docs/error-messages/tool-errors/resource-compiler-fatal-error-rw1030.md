# <a name="resource-compiler-fatal-error-rw1030"></a>리소스 컴파일러 심각한 오류 RW1030

출력 오류

리소스 컴파일러에서 대상 파일을 만들 수 없는 경우이 오류를 발생할 수 있습니다. 디스크 공간이 충분 인지 하 고 볼륨에 대 한 쓰기 권한이 있는지 확인 합니다.
