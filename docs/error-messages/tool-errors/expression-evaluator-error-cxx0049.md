# <a name="expression-evaluator-error-cxx0049"></a>식 계산기 오류 CXX0049

사용 되지 않는 OMF-프로그램을 다시 연결 하세요.

프로그램에서 이전 OMF (개체 모듈 형식)를 사용 했습니다.

현재 링커 버전을 사용 하 여 프로그램을 다시 연결 합니다.

이 오류는 CAN0049와 동일 합니다.
