# <a name="expression-evaluator-error-cxx0009"></a>식 계산기 오류 CXX0009

누락 된 ' 후 문자 상수

작은따옴표 문자 상수 끝에 누락 되었습니다. 문자 상수는 작은따옴표로 묶어 식을 입력 합니다.

이 오류는 can0009와 동일 합니다.
