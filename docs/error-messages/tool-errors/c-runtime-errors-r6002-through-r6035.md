# <a name="c-runtime-errors-rxxxx"></a>C 런타임 오류(Rxxxx)

C 런타임 라이브러리 (CRT) 앱이 로드 되거나 실행 중일 때 런타임 오류를 보고할 수 있습니다. 각 메시지가 Microsoft 시각적 개체를 참조 하는 경우에 C++ 런타임 라이브러리를 의미 하지는 않습니다 라이브러리에 버그가 있습니다. 이러한 오류는 앱의 코드에서 버그 또는 메모리와 같은 런타임 라이브러리를 처리할 수 없는 상태를 나타냅니다. 앱의 최종 사용자는에 쓰기를 방지 하기 위해 또는 오류를 캡처하고 친숙 한 오류를 표시 하도록 앱에 메시지를 사용자에 게 대신 하지 않는 한 이러한 오류가 표시 될 수 있습니다.
