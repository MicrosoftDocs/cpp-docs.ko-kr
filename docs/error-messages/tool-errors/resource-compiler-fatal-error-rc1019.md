# <a name="resource-compiler-fatal-error-rc1019"></a>리소스 컴파일러 심각한 오류 RC1019

예기치 않은 ' #else '

`#else` 지시문이 `#if`, **#ifdef**또는 **#ifndef** 구문에 표시 되지 않았습니다.

이 문 앞에 `#if`, **#ifdef**또는 **#ifndef** 문이 적용 되는지 확인 합니다.
