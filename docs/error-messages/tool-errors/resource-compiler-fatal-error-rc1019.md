# <a name="resource-compiler-fatal-error-rc1019"></a>리소스 컴파일러 심각한 오류 RC1019

예기치 않은 ' #else '

합니다 `#else` 지시문 내에서 표시 되지 않은 `#if`를 **#ifdef**, 또는 **#ifndef** 생성 합니다.

있는지 확인 하십시오는 `#if`, **#ifdef**, 또는 **#ifndef** 문을이 문 앞에 적용 합니다.
