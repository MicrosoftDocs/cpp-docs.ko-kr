# <a name="project-build-error-prj0002"></a>프로젝트 빌드 오류 PRJ0002

> '*명령줄*'에서 오류 결과가 반환 되었습니다.

**속성 페이지** 대화 상자의 사용자 입력에서 구성 된 명령줄 *(명령줄)* 이 오류 코드를 반환 했지만 **출력** 창에 정보가 표시 되지 않습니다.

이 오류에 대 한 해결 방법은 오류를 생성 한 도구에 따라 달라 집니다. MIDL의 경우/o (리디렉션 출력)가 정의 된 경우 무엇이 잘못 되었는지 파악 하 게 됩니다.

오류 조건에 대 한 정보를 제공 하지 않는 사용자 지정 빌드 단계 또는 빌드 이벤트와 같은 배치 파일은이 오류의 원인이 될 수도 있습니다.
