# <a name="expression-evaluator-error-cxx0057"></a>식 계산기 오류 CXX0057

줄 번호에 코드가 없습니다.

지정 된 줄 번호에 대 한 코드가 생성 되지 않았습니다. 중단점으로 사용할 수 없습니다.

이 오류는 CAN0057와 동일 합니다.
