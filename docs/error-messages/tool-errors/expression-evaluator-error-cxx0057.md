# <a name="expression-evaluator-error-cxx0057"></a>식 계산기 오류 CXX0057

줄 번호에 코드가 없습니다.

지정된 된 줄 번호에 대 한 생성 된 코드가 없습니다. 중단점으로 사용할 수 없습니다.

이 오류는 can0057과 동일 합니다.
