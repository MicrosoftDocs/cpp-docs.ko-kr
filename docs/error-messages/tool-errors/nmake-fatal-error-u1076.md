# <a name="nmake-fatal-error-u1076"></a>NMAKE 심각한 오류 U1076

이름이 너무 깁니다.

문자열이 다음 제한 중 하나를 초과 했습니다.

- 매크로 이름에는 1024 자입니다.

- 대상 경로 이름에는 256 자입니다.

- 명령의 경우 2048 자입니다.
