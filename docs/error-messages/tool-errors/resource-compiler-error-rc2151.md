# <a name="resource-compiler-error-rc2151"></a>리소스 컴파일러 오류 RC2151

문자열 상수를 다시 사용할 수 없습니다.

**Stringtable** 문에서 같은 값을 두 번 사용 합니다. 겹치는 소수 및 16 진수 값을 혼합 하지 않아야 합니다.

**Stringtable** 의 각 ID는 고유 해야 합니다. 최대 효율성을 위해 16의 배수로 시작 하는 연속 상수를 사용 합니다.
