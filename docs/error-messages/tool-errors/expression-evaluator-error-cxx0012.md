# <a name="expression-evaluator-error-cxx0012"></a>식 계산기 오류 CXX0012
왼쪽된 중괄호가 없습니다.  
  
 식은 불균형된 중괄호를 포함합니다. 한 식이 중괄호의 짝된을 다시 입력 합니다.  
  
 이 오류는 can0012와 동일 합니다.
