# <a name="expression-evaluator-error-cxx0072"></a>식 계산기 오류 CXX0072

오류: 형식 정보가 없거나 알 수 없습니다.

.Pch 파일이에 연결 되지 않았거나, 코드에/Ziiif로 컴파일되지 않은 모듈에 있는 형식에 대 한 참조가 있습니다.

이 오류는 CAN0067와 동일 합니다.
