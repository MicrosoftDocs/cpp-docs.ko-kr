# <a name="expression-evaluator-error-cxx0072"></a>식 계산기 오류 CXX0072
오류: 형식 정보가 없거나 알 수 없음  
  
 .Pch 파일을 링크 되지 않았거나 또는 코드에 /Zi로 컴파일되지 않은 모듈에 있는 형식에 대 한 참조입니다.  
  
 이 오류는 can0067과 동일 합니다.
