# <a name="resource-compiler-warning-rc4214"></a>리소스 컴파일러 오류 RC4214

코드 페이지가 잘못 됨: 무시 됨

.Rc 파일에 codepage 인수가 포함 되어 있고 지정 된 코드 페이지가 잘못 되었습니다. 자세한 내용은 [Is유효한 코드 페이지](/windows/win32/api/winnls/nf-winnls-isvalidcodepage) 를 참조 하세요.
