# <a name="resource-compiler-warning-rc4214"></a>리소스 컴파일러 오류 RC4214
유효 하지 않은 코드 페이지: 무시  
  
 .Rc 파일에 코드 페이지 인수가 포함 된 및 지정 된 코드 페이지가 올바르지 않습니다. 참조 [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674) 자세한 정보에 대 한 합니다.
