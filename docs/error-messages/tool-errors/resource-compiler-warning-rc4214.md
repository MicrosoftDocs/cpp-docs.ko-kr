코드 페이지 유효 하지 않습니다: 무시

.Rc 파일에 코드 페이지 인수를 포함 하 고 지정 된 코드 페이지 유효 하지 않은 합니다. 참조 [IsValidCodePage](/windows/desktop/api/winnls/nf-winnls-isvalidcodepage) 자세한 내용은 합니다.
