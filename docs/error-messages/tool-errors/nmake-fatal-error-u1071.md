# <a name="nmake-fatal-error-u1071"></a>NMAKE 심각한 오류 U1071

' targetname ' 대상에 대 한 종속성 트리에서 순환

지정 된 대상에 대 한 종속성 트리에 순환 종속성이 있습니다. 지정 된 대상이 지정 된 대상의 종속 항목 중 하나에 종속 되어 있습니다. 순환 종속성이 잘못 되었습니다.
