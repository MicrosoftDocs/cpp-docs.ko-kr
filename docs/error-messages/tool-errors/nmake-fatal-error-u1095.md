# <a name="nmake-fatal-error-u1095"></a>NMAKE 심각한 오류 U1095

확장 된 명령줄 ' commandline '이 너무 깁니다.

매크로 확장 후 지정 된 명령줄이 운영 체제에 대 한 명령줄 길이 제한을 초과 했습니다.

MS-DOS는 명령줄에서 최대 128 문자를 허용 합니다.

파일이 파일에서 명령줄 입력을 받을 수 있는 프로그램인 경우 명령을 변경 하 고 디스크의 파일 또는 인라인 파일의 입력을 제공 합니다. 예를 들어 LINK와 LIB는 지시 파일의 입력을 허용 합니다.
