# <a name="project-build-warning-prj0029"></a>프로젝트 빌드 경고 PRJ0029

프로젝트 수준의 사용자 지정 빌드 단계에 대 한 ' Outputs ' 속성이 설정 되지 않았습니다. 사용자 지정 빌드 단계를 건너뜁니다.

출력이 지정 되지 않았으므로 사용자 지정 빌드 단계가 실행 되지 않았습니다.

이 오류를 해결 하려면 다음 중 하나를 수행 합니다.

- 빌드에서 사용자 지정 빌드 단계를 제외 합니다.

- 출력을 추가 합니다.

- 사용자 지정 빌드 단계의 명령 콘텐츠를 삭제 합니다.
