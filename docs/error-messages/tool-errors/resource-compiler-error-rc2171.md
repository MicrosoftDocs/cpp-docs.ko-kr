# <a name="resource-compiler-error-rc2171"></a>리소스 컴파일러 오류 RC2171

알 수 없는 DIB 헤더 형식입니다.

비트맵 헤더는 **BITMAPCOREHEADER** 또는 **BITMAPINFOHEADER** 구조체가 아닙니다.
