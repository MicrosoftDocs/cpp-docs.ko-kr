# <a name="resource-compiler-error-rc2171"></a>리소스 컴파일러 오류 RC2171

알 수 없는 DIB 헤더 형식

비트맵 헤더가 없는 **BITMAPCOREHEADER** 하거나 **BITMAPINFOHEADER** 구조입니다.
