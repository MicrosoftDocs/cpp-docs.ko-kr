# <a name="resource-compiler-fatal-error-rw1023"></a>리소스 컴파일러 심각한 오류 RW1023

파일을 쓰는 동안 I/o 오류가 발생 했습니다. 드라이브 꽉 찼습니다.

사용 가능한 공간은 만들려는 실행 파일 크기의 두 배 이상 이어야 합니다.
