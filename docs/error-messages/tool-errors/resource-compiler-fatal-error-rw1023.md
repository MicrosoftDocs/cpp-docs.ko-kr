# <a name="resource-compiler-fatal-error-rw1023"></a>리소스 컴파일러 심각한 오류 RW1023

전체 드라이브에 파일을 작성 하는 I/O 오류

사용 가능한 공간을 만들려는 실행 파일의 크기 보다 두 배 이상의 같아야 합니다.
