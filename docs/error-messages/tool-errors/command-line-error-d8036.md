# <a name="command-line-error-d8036"></a>명령줄 오류 D8036

여러 소스 파일에 '/option '를 사용할 수 없습니다.

이러한 컴파일러 옵션은 여러 소스 파일과 함께 사용할 수 없습니다.

- 이름 어셈블리 파일 목록 (/Fa)

- 개체 파일 이름 바꾸기 (/Fo)

- 지역 변수 없이 소스 브라우저 정보 만들기 (/Fr)

- 지역 변수를 포함 하는 원본 브라우저 정보 만들기 (/FR)
