# <a name="expression-evaluator-error-cxx0062"></a>식 계산기 오류 CXX0062

생성자 호출은 지원 되지 않습니다.

식에서 생성자를 호출 했습니다. 식은 생성자를 명시적으로 호출 하거나 생성자를 호출 해야 하는 변환을 수행할 수 없습니다.

이 오류는 CAN0062와 동일 합니다.
