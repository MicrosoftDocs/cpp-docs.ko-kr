# <a name="linker-tools-warning-lnk4039"></a>링커 도구 경고 LNK4039

/SECTION 옵션으로 지정 된 ' name ' 섹션이 없습니다.

[DUMPBIN](../../build/reference/dumpbin-reference.md) 또는 [EDITBIN](../../build/reference/editbin-reference.md) 이 입력 파일에서 `name` 이라는 섹션을 찾을 수 없습니다.

파일에서 섹션을 보려면 [/sd](../../build/reference/headers.md) 옵션을 사용 하 여 DUMPBIN을 실행 합니다.
