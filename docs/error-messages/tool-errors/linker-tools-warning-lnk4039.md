# <a name="linker-tools-warning-lnk4039"></a>링커 도구 경고 LNK4039

 [DUMPBIN](../../build/reference/dumpbin-reference.md) 또는 [EDITBIN](../../build/reference/editbin-reference.md) 라는 섹션을 찾을 수 없습니다 `name` 입력된 파일에 있습니다.  
  
 파일의 섹션을 보려면와 DUMPBIN 실행 된 [/HEADERS](../../build/reference/headers.md) 옵션입니다.
