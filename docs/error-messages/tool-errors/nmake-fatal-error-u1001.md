# <a name="nmake-fatal-error-u1001"></a>NMAKE 심각한 오류 U1001
구문 오류: 잘못 된 문자 'character' 매크로  
  
 지정 된 문자는 매크로에 나타나지만 문자, 숫자 또는 밑줄 아닙니다.  
  
 이 오류는 매크로 확장에서 누락 된 콜론으로 발생할 수 있습니다.  
  
```  
syntax error : illegal character '=' in macro  
```
