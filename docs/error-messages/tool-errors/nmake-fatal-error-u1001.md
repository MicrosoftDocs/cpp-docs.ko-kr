# <a name="nmake-fatal-error-u1001"></a>NMAKE 심각한 오류 U1001

구문 오류: 매크로에 ' character ' 문자가 잘못 되었습니다.

지정 된 문자가 매크로에 나타나지만 문자, 숫자 또는 밑줄이 아닙니다.

이 오류는 매크로 확장에 콜론이 없는 경우 발생할 수 있습니다.

```
syntax error : illegal character '=' in macro
```
