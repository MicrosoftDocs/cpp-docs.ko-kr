# <a name="project-build-error-prj0050"></a>프로젝트 빌드 오류 PRJ0050
출력을 등록하지 못했습니다. 레지스트리를 수정 하려면 적절 한 권한이 있는지 확인 하십시오.  
  
 Visual c + + 빌드 시스템에서 빌드 (dll 또는.exe)의 출력을 등록할 수 없습니다. 레지스트리를 수정 하려면 관리자 권한으로 로그온 해야 합니다.  
  
 .Dll을 작성 하는 regsvr32.exe를 사용 하 여 수동으로.dll을 등록 하려고 해도,이 빌드 실패 원인에 대 한 정보를 표시 해야 합니다.  
  
 .Dll을 빌드하지 않는 경우 오류가 발생 하는 명령에 대 한 빌드 로그를 확인 합니다.
