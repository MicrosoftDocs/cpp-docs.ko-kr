# <a name="resource-compiler-fatal-error-rc1002"></a>리소스 컴파일러 심각한 오류 RC1002

힙 공간이 부족 합니다.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>아래의 해결 방법 따라 수정합니다.

1. Windows 스왑 파일 공간을 늘립니다. 스왑 파일 공간을 높이는 방법에 대 한 자세한 내용은 Windows 도움말의 가상 메모리를 참조 하십시오.

1. 현재 파일을 더 작은 파일로 분할 하 고 별도로 컴파일합니다.

1. 시스템에서 실행 중인 다른 프로그램 또는 드라이버를 제거 합니다.
