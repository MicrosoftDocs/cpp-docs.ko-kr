# <a name="resource-compiler-fatal-error-rc1002"></a>리소스 컴파일러 심각한 오류 RC1002

힙 공간 부족

### <a name="to-fix-by-using-the-following-possible-solutions"></a>아래의 해결 방법 따라 수정합니다.

1. Windows 스왑 파일 공간을 늘립니다. 파일 스왑 공간을 늘리는 방법에 대 한 자세한 내용은 Windows 도움말에서 가상 메모리를 참조 하세요.

1. 현재 파일을 더 작은 파일로 분할 하 고 개별적으로 컴파일할 수입니다.

1. 다른 프로그램 또는 시스템에서 실행 중인 드라이버를 제거 합니다.
