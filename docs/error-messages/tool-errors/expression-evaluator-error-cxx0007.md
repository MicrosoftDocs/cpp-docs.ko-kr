# <a name="expression-evaluator-error-cxx0007"></a>식 계산기 오류 CXX0007

오른쪽 괄호가 없습니다.

식에 불균형 괄호가 있습니다. 짝이 되는 괄호를 사용 하 여 식을 다시 입력 합니다.

이 오류는 CAN0007와 동일 합니다.
