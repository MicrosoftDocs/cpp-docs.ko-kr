# <a name="nmake-fatal-error-u1023"></a>NMAKE 심각한 오류 U1023
식에 구문 오류가 있습니다.  
  
 식의 형식이 올바르지 않습니다.  
  
 사용할 수 있는 연산자 및 연산자 우선 순위를 확인 합니다.
