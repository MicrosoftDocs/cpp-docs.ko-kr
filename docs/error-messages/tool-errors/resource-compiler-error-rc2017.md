# <a name="resource-compiler-error-rc2017"></a>리소스 컴파일러 오류 RC2017

이스케이프 시퀀스가 잘못 되었습니다.

이스케이프 시퀀스가 예상과 다른 것으로 나타났습니다.

백슬래시 ( **\\** ) 뒤에 숫자나 문자가 오는 이스케이프 시퀀스는 문자 또는 문자열 상수 에서만 발생할 수 있습니다.
