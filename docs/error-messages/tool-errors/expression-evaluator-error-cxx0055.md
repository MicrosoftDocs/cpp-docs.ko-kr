# <a name="expression-evaluator-error-cxx0055"></a>식 계산기 오류 CXX0055

컨텍스트 연산자에서 모호한 기호

컨텍스트 연산자에서 기호 (**{}**) 프로그램에 둘 이상의 기호를 가리킵니다.

범위 결정 연산자 (`::`) 모호성을 해결 하는 일을 할 수 있습니다.

이 오류는 can0055와 동일 합니다.
