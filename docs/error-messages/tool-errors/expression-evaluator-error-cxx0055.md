# <a name="expression-evaluator-error-cxx0055"></a>식 계산기 오류 CXX0055

컨텍스트 연산자에 모호한 기호가 있습니다.

컨텍스트 연산자 ()의 기호가 **{}** 프로그램에서 둘 이상의 기호를 참조 합니다.

범위 확인 연산자 ( `::` )는 모호성을 해결할 수 있습니다.

이 오류는 CAN0055와 동일 합니다.
