# <a name="expression-evaluator-error-cxx0043"></a>식 계산기 오류 CXX0043

이 포인터는 멤버 함수 외부에서 사용 되었습니다.

**`this`** 포인터는 비정적 멤버 함수에만 사용할 수 있습니다.

이 오류는 CAN0043와 동일 합니다.
