# <a name="expression-evaluator-error-cxx0028"></a>식 계산기 오류 CXX0028
구조체/공용 구조체 요소가 아닙니다  
  
 양식 'Struct.Member' 또는 'pstruct->member' 식을 지정 하지만 \<멤버 >는 구조체의 요소가 아닙니다.  
  
 식이 아니어야 괄호로 묶인 올바르게 합니다.  
  
 이 오류는 can0028과 동일 합니다.
