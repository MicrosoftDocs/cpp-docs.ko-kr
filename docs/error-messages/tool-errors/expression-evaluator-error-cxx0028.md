# <a name="expression-evaluator-error-cxx0028"></a>식 계산기 오류 CXX0028

구조체/공용 구조체 요소가 아닙니다.

' Struct. Member ' 또는 ' pStruct->Member ' 형식의 식이 지정 되었지만 \<member> 구조체의 요소가 아닙니다.

식의 괄호를 올바르게 지정할 수 없습니다.

이 오류는 CAN0028와 동일 합니다.
