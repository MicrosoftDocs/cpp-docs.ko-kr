# <a name="expression-evaluator-error-cxx0004"></a>식 계산기 오류 CXX0004

구문 오류

식의 구문을 잘못 되었습니다.

올바른 구문 사용 하 여 식을 입력 합니다.

이 오류는 can0004와 동일 합니다.
