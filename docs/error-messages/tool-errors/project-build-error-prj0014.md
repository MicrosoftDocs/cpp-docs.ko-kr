# <a name="project-build-error-prj0014"></a>프로젝트 빌드 오류 PRJ0014

생성 된 프로세스를 제어 하는 데 사용 된 작업 개체가 실패 했습니다.  빌드를 계속할 수 없습니다.

개발 환경에서 오류가 발생 했습니다.

이 오류를 해결 하려면 프로젝트를 닫았다가 다시 로드 합니다. 필요한 경우 Visual Studio를 종료 하 고 다시 시작 합니다.
