# <a name="linker-tools-error-lnk1282"></a>링커 도구 오류 LNK1282

/REBASE 파일 수 없습니다. 서명 되었습니다.

에 대 한 /REBASE 옵션을 사용 하 여 서명된 된 어셈블리의 기준 주소를 변경 하려고 [editbin](../../build/reference/editbin-reference.md)합니다. 이렇게 하려면 먼저 기본 주소를 변경 하 고 어셈블리에 서명 합니다.
