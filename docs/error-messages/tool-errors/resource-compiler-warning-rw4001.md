# <a name="resource-compiler-warning-rw4001"></a>리소스 컴파일러 오류 RW4001
. EXE 처리 옵션 (/L /M /P /T /K /E /31 또는/30)  
  
 EXE 처리 옵션을 지정 했지만 실행 파일이 처리할 수 없습니다. .Res 파일 함께 이러한 옵션을 사용 하지 마십시오.
