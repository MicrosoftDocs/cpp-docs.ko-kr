# <a name="project-build-error-prj0025"></a>프로젝트 빌드 오류 PRJ0025
배치 파일 'file' 사용자의 ANSI 코드 페이지로 변환할 수 없는 유니코드 내용이 있습니다.  
  
 ***파일의 내용을 유니코드***  
  
 프로젝트 시스템이 유니코드 내용이 사용자 지정에서 빌드 규칙 또는 빌드에 사용자의 현재 ANSI 코드 페이지로 제대로 변환 될 수 없는 이벤트를 찾았습니다.  
  
 이 오류에 대 한 해결책은 빌드를 ANSI 사용 하거나 코드 페이지를 컴퓨터에 설치 하 고 시스템 기본값으로 설정 하는 이벤트 나 빌드 규칙의 내용을 업데이트 하는 것입니다.  
  
 사용자 지정에 대 한 자세한 내용은 빌드 단계 및 빌드 이벤트, 참조 [이해 사용자 지정 빌드 단계 및 빌드 이벤트](../../ide/understanding-custom-build-steps-and-build-events.md)합니다.
