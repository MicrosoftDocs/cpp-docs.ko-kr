# <a name="resource-compiler-fatal-error-rc1101"></a>리소스 컴파일러 심각한 오류 RC1101

> 리소스 이진 파일 이름을 지정 하지 않았습니다.

출력 이름 바꾸기 (/fo) 옵션 뒤에 파일 이름이 없습니다.

/Fo 옵션에 대해 다음 구문을 사용 합니다.

> **Rc/fo** *outfilename* <em>infilename</em> **.rc**
