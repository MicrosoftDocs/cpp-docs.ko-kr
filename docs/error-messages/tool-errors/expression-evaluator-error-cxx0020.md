# <a name="expression-evaluator-error-cxx0020"></a>식 계산기 오류 CXX0020

이 작업에 대 한 피연산자 형식이 잘못 되었습니다.

연산자가 해당 연산자에 대해 잘못 된 형식의 식에 적용 되었습니다.

예를 들어 레지스터의 주소를 사용 하거나 부동 소수점 식이 있는 배열의 첨자를 사용 하는 것은 유효 하지 않습니다.

이 오류는 CAN0020와 동일 합니다.
