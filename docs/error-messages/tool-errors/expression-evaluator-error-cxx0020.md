# <a name="expression-evaluator-error-cxx0020"></a>식 계산기 오류 CXX0020
이 작업에 대 한 잘못 된 피연산자 형식  
  
 연산자는 해당 연산자가 잘못 된 유형의 식에 적용 되었습니다.  
  
 예를 들어, 레지스터 또는 부동 소수점 식 사용 하 여 배열 첨자의 주소를 가져올 올바르지 않습니다.  
  
 이 오류는 can0020과 동일 합니다.
