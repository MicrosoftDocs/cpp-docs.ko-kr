# <a name="expression-evaluator-error-cxx0023"></a>식 계산기 오류 CXX0023

기수가 잘못 되었습니다.

C 식 계산기는 지정 된 기 수를 인식 하지 못합니다. 10 진수, 16 진수 및 8 진수 기수만 유효 합니다.

이 오류는 can0023과 동일 합니다.
