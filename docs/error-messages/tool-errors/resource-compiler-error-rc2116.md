# <a name="resource-compiler-error-rc2116"></a>리소스 컴파일러 오류 RC2116

ID에 수가 있어야합니다.

에 대 한 숫자가 있어야 합니다.는 `id` 제어문에 필드를 **대화** 문입니다. 숫자로 했는지 확인 하거나 `#define` 문을 컨트롤 id입니다.
