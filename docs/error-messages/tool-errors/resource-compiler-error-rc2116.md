# <a name="resource-compiler-error-rc2116"></a>리소스 컴파일러 오류 RC2116

ID에 숫자가 필요 합니다.

**DIALOG** 문에서 control 문의 `id` 필드에는 숫자가 필요 합니다. 컨트롤 ID에 대해 숫자나 `#define` 문이 있는지 확인 합니다.
