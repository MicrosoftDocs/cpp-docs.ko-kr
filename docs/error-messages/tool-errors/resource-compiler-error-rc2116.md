# <a name="resource-compiler-error-rc2116"></a>리소스 컴파일러 오류 RC2116
ID에 대 한 숫자가 있어야합니다.  
  
 에 대 한 숫자가 있어야 합니다.는 `id` 에서 제어 문의 필드는 **대화** 문. 번호를 지정 했는지 확인 또는 `#define` 문을 컨트롤 id입니다.
