# <a name="resource-compiler-fatal-error-rc1022"></a>리소스 컴파일러 심각한 오류 RC1022

' #endif '가 필요 합니다.

`#if`, **#ifdef**또는 **#ifndef** 지시문이 `#endif` 지시문으로 종료 되지 않았습니다.

이 문 앞에 `#if`, **#ifdef**또는 **#ifndef** 문이 적용 되는지 확인 합니다.
