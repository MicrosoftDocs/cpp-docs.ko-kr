# <a name="resource-compiler-fatal-error-rc1022"></a>리소스 컴파일러 심각한 오류 RC1022

예상된 ' #endif'

`#if`, **#ifdef**, 또는 **#ifndef** 지시문으로 종료 되지 않았습니다.는 `#endif` 지시문입니다.

있는지 확인 하십시오는 `#if`, **#ifdef**, 또는 **#ifndef** 문을이 문 앞에 적용 합니다.
