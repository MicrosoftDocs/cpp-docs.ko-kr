# <a name="expression-evaluator-error-cxx0054"></a>식 계산기 오류 CXX0054

중단점에 등록 식이 잘못 되었습니다.

레지스터는 중단점 식에 사용할 수 없습니다.

이 오류는 CAN0054와 동일 합니다.
