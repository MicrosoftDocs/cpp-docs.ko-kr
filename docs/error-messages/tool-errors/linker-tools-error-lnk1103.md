# <a name="linker-tools-error-lnk1103"></a>링커 도구 오류 LNK1103

디버깅 정보가 손상 되었습니다. recompile 모듈

이 오류는 올바른 개체 파일을 만든 전에 컴파일이 종료 된 때문에 발생할 수 있습니다.

지정된 된 개체 파일을 다시 컴파일하십시오. 다시 컴파일 문제가 해결 되지 않으면, 오류 상황을 확인, 문제를 격리 하 여 재현 가능한 테스트 사례를 만듭니다. 다음 기술 지원 도움말 파일 또는 설명서의의 기술 지원 섹션을 참조 하세요.
