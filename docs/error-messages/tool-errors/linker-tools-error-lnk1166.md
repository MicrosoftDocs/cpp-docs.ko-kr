# <a name="linker-tools-error-lnk1166"></a>링커 도구 오류 LNK1166

오프셋에서 코드를 조정할 수 없는 오프셋, va = = value

링크 필요에 따라 코드를 채울 수 없습니다.

일부 프로세서 페이지 경계를 교차 하는 특정 지침 허용 되지 않습니다. 링크는이 문제를 해결 하는 패드를 추가 하려고 합니다. 이 경우에 링크 문제를 해결 하지 못했습니다.
