# <a name="bscmake-warning-bk4503"></a>BSCMAKE 경고 BK4503

.SBR 파일 이름의 사소한 오류가 무시되었습니다.

.sbr 파일에 포함된 오류로 인해 빌드가 중지되지는 않았지만 결과 .bsc 파일이 잘못될 수 있습니다. 이 문제를 해결하려면 다시 컴파일하여 새 .sbr 파일을 만듭니다.
