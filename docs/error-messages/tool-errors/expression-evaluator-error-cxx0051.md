# <a name="expression-evaluator-error-cxx0051"></a>식 계산기 오류 CXX0051

중단점에 지정 된 오버 로드 된 기호가 두 개 이상 있습니다.

식 계산기에서 중단점으로 사용할 오버 로드 된 둘 이상의 기호를 확인할 수 없습니다.

이 오류는 CAN0051와 동일 합니다.
