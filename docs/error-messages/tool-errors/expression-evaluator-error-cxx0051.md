# <a name="expression-evaluator-error-cxx0051"></a>식 계산기 오류 CXX0051
중단점에 지정 된 둘 이상의 오버 로드 된 기호  
  
 식 계산기는 중단점으로 사용할 둘 이상의 오버 로드 된 기호를 확인 하지 못했습니다.  
  
 이 오류는 can0051과 동일 합니다.
