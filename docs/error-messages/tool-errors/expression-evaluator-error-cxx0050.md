# <a name="expression-evaluator-error-cxx0050"></a>식 계산기 오류 CXX0050

::의 왼쪽은 클래스/구조체/공용 구조체 여야 합니다.

범위 결정 연산자 (`::`)의 왼쪽에 있는 기호가 클래스, 구조체 또는 공용 구조체가 아닙니다.

이 오류는 CAN0050와 동일 합니다.
