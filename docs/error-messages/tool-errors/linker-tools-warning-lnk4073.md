# <a name="linker-tools-warning-lnk4073"></a>링커 도구 경고 LNK4073

.ilk 파일에 대 한 맵을 만들 수 없습니다. 비증분 링크

증분 상태 (.ilk) 파일을 만들기 위한 링크에 대 한 공유 메모리에 충분 한 공간이 부족 합니다. 비증분 빌드를 수행 하는 링크입니다.
