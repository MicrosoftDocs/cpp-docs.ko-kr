# <a name="linker-tools-error-lnk1221"></a>링커 도구 오류 LNK1221

하위 시스템을 유추할 수 없으므로 정의 해야 합니다.

링커에 대상 하위 시스템을 유추 하기에 충분 한 정보가 없습니다.

이 오류를 해결 하려면 [/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)를 사용 합니다.
