# <a name="c-runtime-error-r6024"></a>C 런타임 오류 R6024

_onexit/atexit 테이블을 위한 공간이 부족 합니다.

> [!NOTE]
> 앱을 실행 하는 동안이 오류 메시지가 표시 되는 경우 내부 메모리 문제가 있기 때문에 앱이 종료 되었습니다. 일반적으로이 오류는 프로그램의 버그가 나 사용 하는 Visual C++ 라이브러리의 손상으로 인 한 메모리 부족 상태 이거나 거의 발생 하지 않습니다.
>
> 이 오류를 해결하려면 다음 단계를 시도할 수 있습니다.
>
> - 실행 중인 다른 응용 프로그램을 닫거나 컴퓨터를 다시 시작 하 여 메모리를 확보 하십시오.
> - **제어판** 의 **앱 및 기능** 또는 **프로그램 및 기능** 페이지를 사용 하 여 프로그램을 복구 하거나 다시 설치 합니다.
> - **제어판** 의 **앱 및 기능** 또는 **프로그램 및 기능** 페이지를 사용 하 여 Microsoft Visual C++ 재배포 가능 패키지의 모든 복사본을 복구 하거나 다시 설치 합니다.
> - **제어판** 의 소프트웨어 업데이트에 대 한 **Windows 업데이트** 를 확인 합니다.
> - 앱의 업데이트 된 버전을 확인 합니다. 문제가 지속 되 면 앱 공급 업체에 문의 하세요.

**프로그래머를 위한 정보**

이 오류는 또는 함수에 사용할 수 있는 메모리가 없기 때문에 발생 `_onexit` `atexit` 합니다. 이 오류는 메모리 부족 상태로 인해 발생 합니다. 앱 시작 시 버퍼를 미리 할당 하 여 사용자 데이터를 저장 하 고 메모리 부족 상태에서 정리 된 앱 종료를 수행할 수 있습니다.
