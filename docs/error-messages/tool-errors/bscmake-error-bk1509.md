# <a name="bscmake-error-bk1509"></a>BSCMAKE 오류 BK1509

힙 공간이 부족 합니다.

가상 메모리를 포함 하 여 BSCMAKE에 메모리가 부족 합니다.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>다음 해결 방법을 사용하여 수정하려면

1. 디스크 공간을 확보 합니다.

1. 스왑 파일의 크기를 늘립니다.

1. Windows 스왑 파일의 크기를 늘립니다.

1. /Ei 또는/Es를 사용 하 여 일부 입력 파일이 나/Em 매크로 본문을 제거 하는 데 필요한 메모리를 줄입니다.
