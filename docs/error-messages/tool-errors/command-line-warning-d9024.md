# <a name="command-line-warning-d9024"></a>명령줄 경고 D9024

인식할 수 없는 소스 파일 형식 ' filename ', 개체 파일을 가정 합니다.

지정 된 파일의 확장명을 인식할 수 없습니다. 파일은 개체 파일로 간주 되어 링커에 전달 된 것으로 간주 됩니다.

다음 확장이 인식 됩니다.

- .c (C 소스 파일)

- . .cxx (C++ 소스 파일)

- .cpp (C++ 소스 파일)

- .obj (개체 파일)

- .lib (라이브러리 파일)

- .def (모듈 정의 파일)

- .exp (링커 내보내기 파일, LINK/LIB로 만듦)
