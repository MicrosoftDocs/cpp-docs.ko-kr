# <a name="linker-tools-warning-lnk4199"></a>링커 도구 경고 LNK4199

/DELAYLOAD: dllname가 무시 되었습니다. dllname에서 가져오기를 찾을 수 없습니다.

링커에서는를 `dllname` 내보내는 함수가 필요 하지 않으므로를 무시 `dllname` 합니다.
