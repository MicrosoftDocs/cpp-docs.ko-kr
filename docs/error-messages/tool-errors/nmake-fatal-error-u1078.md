# <a name="nmake-fatal-error-u1078"></a>NMAKE 심각한 오류 U1078

' expression '의 상수가 오버플로 되었습니다.

지정 된 식에-2147483648 ~ 2147483647 범위를 초과한 상수가 포함 되어 있습니다. 상수는 다음 상황 중 하나에서 나타났습니다.

- 전처리 지시문을 사용 하 여 지정 된 식

- 대시 ( **-** ) 명령 한정자로 지정 된 오류 수준
