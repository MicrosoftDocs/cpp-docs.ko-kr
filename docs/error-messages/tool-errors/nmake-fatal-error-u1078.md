# <a name="nmake-fatal-error-u1078"></a>NMAKE 심각한 오류 U1078
'expression'에서 상수가 오버플로  
  
 지정 된 식은 범위-2147483648과 2147483647 사이 초과 하는 상수를 포함 합니다. 상수는 다음과 같은 상황 중 하나에 나타납니다.  
  
-   전처리 지시문으로 지정 된 식  
  
-   대시로 지정 된 오류 수준 (**-**) 명령 한정자
