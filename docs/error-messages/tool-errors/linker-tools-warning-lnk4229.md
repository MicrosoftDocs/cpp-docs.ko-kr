# <a name="linker-tools-warning-lnk4229"></a>링커 도구 경고 LNK4229

잘못 된 지시문/지시문이 있습니다. 무시

[주석](../../preprocessor/comment-c-cpp.md) pragma를 통해 전달 된 지시문이 잘못 되었습니다. 링커가를 무시 `/directive` 합니다.
