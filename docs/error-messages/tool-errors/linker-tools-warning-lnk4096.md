# <a name="linker-tools-warning-lnk4096"></a>링커 도구 경고 LNK4096

/BASE 값 "number"가 Windows 95 및 Windows 98에 적합 하지 않습니다. 이미지가 실행 되지 않을 수 있습니다.

지정한 기본 주소가 잘못 되었습니다. Windows 95 및 Windows 98 실행 파일의 기본 주소는 0x400000 보다 커야 합니다. 기본 주소에 대 한 자세한 내용은 [/base](../../build/reference/base-base-address.md) 링커 옵션을 참조 하세요.
