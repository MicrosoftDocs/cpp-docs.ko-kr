# <a name="bscmake-error-bk1512"></a>BSCMAKE 오류 BK1512
filename: 용량을 초과 했습니다  
  
 BSCMAKE 정의 참조, 모듈 또는 기타 정보 수가 제한을 초과 하기 때문에 찾아보기 정보 파일을 빌드할 수 없습니다.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>아래의 해결 방법 따라 수정합니다.  
  
1.  /Em, /Es 또는 /Ei를 사용 하 여 일부 정보를 제외 합니다.  
  
2.  /Iu 옵션을 생략 합니다.
