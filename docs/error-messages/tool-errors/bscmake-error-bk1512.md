# <a name="bscmake-error-bk1512"></a>BSCMAKE 오류 BK1512

파일 이름: 용량 초과

정의, 참조, 모듈 또는 기타 정보 수가 제한을 초과 하므로 BSCMAKE에서 찾아보기 정보 파일을 빌드할 수 없습니다.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>다음 해결 방법을 사용하여 수정하려면

1. /Em,/Es 또는/Ei.를 사용 하 여 일부 정보를 제외 합니다.

1. /Iu 옵션을 생략 합니다.
