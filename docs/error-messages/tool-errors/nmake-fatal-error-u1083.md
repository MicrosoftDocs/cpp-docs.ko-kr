# <a name="nmake-fatal-error-u1083"></a>NMAKE 심각한 오류 U1083

' target ' 대상 매크로가 확장 되지 않습니다.

지정 된 대상은 정의 되지 않았거나 null 값을 포함 하는 매크로의 호출입니다. NMAKE는 null 대상을 처리할 수 없습니다.
