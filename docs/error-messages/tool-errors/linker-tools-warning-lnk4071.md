# <a name="linker-tools-warning-lnk4071"></a>링커 도구 경고 LNK4071

다음 링크에 대해 증분 링크할 수 없습니다.

링크에서 하나 이상의 기호에 대 한 여러 정의를 찾았지만 [/force](../../build/reference/force-force-file-output.md) 또는 **/force: multiple** 은 오류와 관계 없이 출력 파일을 만드는 데 사용 되었습니다. 링크가 증분 상태 파일 (.ilk)을 삭제 했습니다.
