# <a name="resource-compiler-error-rc2164"></a>리소스 컴파일러 오류 RC2164

RCDATA에 예기치 않은 값이 있습니다.

*RCDATA* 문의 **raw-data** 값은 각각 쉼표로 구분된 정수 또는 문자열이어야 합니다. 쉼표를 빠뜨렸거나 문자열 주위의 따옴표를 빠뜨리지 않았는지 확인합니다.
