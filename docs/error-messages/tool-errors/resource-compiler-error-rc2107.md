# <a name="resource-compiler-error-rc2107"></a>리소스 컴파일러 오류 RC2107
**숫자 명령 값을 필요 합니다.**  
  
 리소스 컴파일러는 숫자가 필요한 데 *idvalue* 필드에 **액셀러레이터** 문. 사용 했는지 확인 한 `#define` 상수 값을 지정 하 고 상수를 올바르게 입력 합니다.
