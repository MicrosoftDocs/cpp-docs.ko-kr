# <a name="profile-guided-optimization-warning-pg1039"></a>프로필 기반 최적화 경고 PG1039

자동 시간 비우기를 기다리지 못했습니다.

응용 프로그램에서 [pgosweep](../../build/pgosweep.md) 를 실행할 수 없습니다.  가능 하면 응용 프로그램이 유휴 상태일 때 pgosweep를 실행 합니다.
