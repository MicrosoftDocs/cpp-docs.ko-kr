# <a name="nmake-fatal-error-u1088"></a>NMAKE 심각한 오류 U1088

유추 규칙에 대 한 ':: ' 구분 기호가 잘못 되었습니다.

유추 규칙 뒤에는 단일 콜론 ( **:** )이와 야 합니다.
