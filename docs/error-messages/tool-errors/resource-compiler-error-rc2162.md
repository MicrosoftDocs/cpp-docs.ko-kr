# <a name="resource-compiler-error-rc2162"></a>리소스 컴파일러 오류 RC2162

예상된 매크로 정식 매개 변수

문자열 화 연산자 뒤의 토큰이 (**#**) 정식 매개 변수 이름이 없습니다.
