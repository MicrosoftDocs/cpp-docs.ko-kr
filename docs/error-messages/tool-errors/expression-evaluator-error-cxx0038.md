# <a name="expression-evaluator-error-cxx0038"></a>식 계산기 오류 CXX0038

함수 인수 개수 및/또는 형식 불일치

지정 된 대로 함수 호출 함수에 대 한 프로토타입을 일치 하지 않습니다.

올바른 인수 개수를 사용 하 여 호출을 다시 입력 합니다. 형식 캐스트는 필요에 따라 프로토타입을 맞게 각 인수.

이 오류는 can0038과 동일 합니다.
