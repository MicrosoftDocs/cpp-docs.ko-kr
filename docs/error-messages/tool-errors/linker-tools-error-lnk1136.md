# <a name="linker-tools-error-lnk1136"></a>링커 도구 오류 LNK1136

파일이 잘못 되었거나 손상 되었습니다.

입력 파일의 헤더가 손상 되었거나 크기가 0 이거나 비정상적으로 작습니다.

파일이 추가 종속성으로 잘못 지정 된 경우에도 LNK1136 발생할 수 있습니다.
