# <a name="bscmake-error-bk1516"></a>BSCMAKE 오류 BK1516

bscfile 손상 되었습니다. 증분 업데이트할 수 없습니다.

가능한 경우.bsc 파일 빌드 중 시스템 오류로 인해 손상 되었습니다. .Bsc 파일을 삭제, 모든.sbr 파일이 다시 작성 한 다음.bsc 파일을 다시 빌드해야 합니다.
