# <a name="math-error-m6102"></a>수학 오류 M6102

denormal

작업에서 매우 작은 부동 소수점 숫자를 생성 했습니다 .이 숫자는 중요 하지 않은 경우 유효 하지 않습니다. Denormal 부동 소수점 예외는 일반적으로 마스크 되어 트랩 되 고 작동 합니다.

프로그램이 종료 코드 130를 사용 하 여 종료 됩니다.
