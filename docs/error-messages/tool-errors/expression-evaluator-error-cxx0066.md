# <a name="expression-evaluator-error-cxx0066"></a>식 계산기 오류 CXX0066

정적 멤버가 없습니다.

클래스의 정적 멤버를 찾을 수 없거나 정의 되지 않았습니다. 이 오류는 선언 되었지만 정의 되지 않은 정적 클래스 멤버가 발생 하거나 디버그 정보를 포함 하지 않는 모듈에만 정의 되 고 참조 되는 경우에 발생할 수 있습니다.

이 오류는 CAN0066와 동일 합니다.
