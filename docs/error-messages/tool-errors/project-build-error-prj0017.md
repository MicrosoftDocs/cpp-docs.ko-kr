# <a name="project-build-error-prj0017"></a>프로젝트 빌드 오류 PRJ0017

현재 작업 디렉터리가 잘못 되었습니다.

현재 작업 디렉터리의 경로를 _MAXPATH 보다 깁니다. 이 오류를 해결 하려면 이러한 중첩 수준에서 프로젝트를 만들 하지 않습니다.
