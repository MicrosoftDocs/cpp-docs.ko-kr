# <a name="linker-tools-error-lnk1224"></a>링커 도구 오류 LNK1224

잘못 된 이미지의 기준 주소

이미지에 대 한 잘못 된 기본 주소를 지정 했습니다. 기본 주소에는 정렬 하는 64 KB를 사용 해야 합니다. (마지막 네 개의 16 진수는 0 이어야 함) 및 기본 이미지는 32 비트 부호 있는 정수나 부호 없는 값 내에 담겨 야 합니다.
