# <a name="linker-tools-error-lnk1224"></a>링커 도구 오류 LNK1224

잘못 된 이미지 기준 주소

이미지에 대 한 잘못 된 기본 주소를 지정 했습니다. 기준 주소는 64KB로 정렬 되어야 합니다 (마지막 4 개의 16 진수는 0 이어야 함). 이미지 밑은 32 비트의 부호 있는 값 이나 부호 없는 값에 맞아야 합니다.
