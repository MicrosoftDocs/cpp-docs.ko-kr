# <a name="linker-tools-error-lnk1224"></a>링커 도구 오류 LNK1224
잘못 된 이미지의 기준 주소  
  
 이미지에 대 한 잘못 된 기본 주소를 지정 합니다. 기본 주소에 정렬 하는 64 KB 이어야 합니다 (마지막 4 자리 16 진수는 0 이어야 함) 및 기본 이미지는 32 비트 부호 있는 또는 부호 없는 값에 맞도록 해야 합니다.
