# <a name="linker-tools-error-lnk1248"></a>링커 도구 오류 LNK1248

> 이미지 크기 ('*size*')가 최대 허용 크기 (8000만)를 초과 합니다.

링커가 출력 파일의 크기가 32 비트 프로그램 이미지에 대해 가능한 최대 크기를 초과 하는 것으로 확인 했습니다. 프로그램을 여러 Dll로 만드는 것이 좋습니다.
