# <a name="linker-tools-error-lnk1248"></a>링커 도구 오류 LNK1248

> 이미지 크기 ('*크기*') 최대 허용 크기 (80000000)를 초과 합니다.

링커에서 확인 출력 파일의 크기는 32 비트 프로그램 이미지에 대 한 가능한 가장 큰 크기를 초과 했습니다. 프로그램을 여러 개의 Dll 해야 할 수 있습니다.
