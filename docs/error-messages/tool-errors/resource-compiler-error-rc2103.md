# <a name="resource-compiler-error-rc2103"></a>리소스 컴파일러 오류 RC2103

문자열 리터럴에 예기치 않은 파일의 끝이 있습니다.

문자열의 끝에 파일의 끝이 있습니다. 문자열에 닫는 큰따옴표 ( **"** )가 없을 수 있습니다.
