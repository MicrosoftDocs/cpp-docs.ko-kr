# <a name="resource-compiler-error-rc2103"></a>리소스 컴파일러 오류 RC2103

예기치 않은 파일의 끝에 문자열 리터럴

문자열의 종료 되기 전에 파일 끝 찾을 수 있습니다. 문자열이 닫는 큰따옴표가 없습니다 (**"**).
