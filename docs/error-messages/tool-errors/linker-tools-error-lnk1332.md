# <a name="linker-tools-error-lnk1332"></a>링커 도구 오류 LNK1332

\<count>한 모듈에서 가져와 다른 모듈에서 정의한 Windows 런타임 유형을 검색 했습니다.

현재 대상이 생성 될 때 링커는 `count`> Windows 런타임 형식 <를 검색 합니다. 각 형식은 한 모듈에서 가져오고 다른 모듈에도 정의 되어 있습니다.

### <a name="to-correct-this-error"></a>이 오류를 해결하려면

- 오류 메시지의 제안에 따라 빌드에서 각 LNK2039 오류를 해결합니다.
