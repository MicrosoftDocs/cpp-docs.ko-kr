# <a name="expression-evaluator-error-cxx0026"></a>식 계산기 오류 CXX0026

형식 문자열을 잘못 지정 했습니다. 식의 구문을 확인합니다.

이 오류는 CAN0026와 동일 합니다.

자세한 내용은 [c + +의 형식 지정자](/visualstudio/debugger/format-specifiers-in-cpp)를 참조 하세요.
