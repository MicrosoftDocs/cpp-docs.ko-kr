# <a name="expression-evaluator-error-cxx0058"></a>식 계산기 오류 CXX0058

오버 로드 된 연산자를 찾을 수 없음

클래스 형식 식에서 왼쪽 피연산자로 지정 되었지만 오버 로드 된 연산자는 클래스에 정의 되지 않았습니다.

이 오류는 can0058과 동일 합니다.
