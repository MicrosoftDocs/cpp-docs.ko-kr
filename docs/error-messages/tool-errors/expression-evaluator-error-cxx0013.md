# <a name="expression-evaluator-error-cxx0013"></a>식 계산기 오류 CXX0013

누락 연산자

식에 연산자가 필요 하지만 찾을 수 없습니다. 식의 구문을 확인합니다.

이 오류는 CAN0013와 동일 합니다.
