# <a name="linker-tools-warning-lnk4205"></a>링커 도구 경고 LNK4205

' filename '에 참조 모듈에 대 한 현재 디버깅 정보가 없습니다. 디버그 정보가 없는 것 처럼 개체를 링크 합니다.

.Pdb 파일에는 최신 정보가 없습니다. 링커는 디버그 정보 없이 개체를 계속 연결 합니다. [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md) 옵션을 사용 하 여 개체 파일을 다시 컴파일하는 것이 좋습니다.
