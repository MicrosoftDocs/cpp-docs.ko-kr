# <a name="bscmake-error-bk1510"></a>BSCMAKE 오류 BK1510

없거나. .SBR 파일 파일 이름

지정 된 .sbr 파일에 필요한 형식이 없습니다. 다시 컴파일하여 새 .sbr를 만듭니다.
