# <a name="linker-tools-warning-lnk4010"></a>링커 도구 경고 LNK4010
잘못 된 하위 시스템 버전 번호 번호입니다. 기본 하위 시스템 버전이 선택 됩니다.  
  
 이미지의 하위 시스템에 대 한 버전을 지정할 수 있습니다 ([/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)). 각 하위 시스템에는 최소 버전 요구 사항이 있습니다. 지정 된 버전이 최소값 보다 낮은 경우이 경고가 발생 하 고 링커 기본 하위 시스템을 사용 하면 됩니다.
