# <a name="nmake-fatal-error-u1055"></a>NMAKE 심각한 오류 U1055

환경 공간이 부족 합니다.

운영 체제에 환경 변수에 대 한 공간이 부족 합니다.

환경 공간을 늘리거나 더 작은 환경 변수를 설정 하십시오.
