# <a name="nmake-fatal-error-u1055"></a>NMAKE 심각한 오류 U1055
환경 공간이 부족 합니다.  
  
 운영 체제 환경 변수를 위한 공간이 부족합니다.  
  
 환경 공간이 증가 하거나 더 적은 수의 환경 변수를 설정 합니다.
