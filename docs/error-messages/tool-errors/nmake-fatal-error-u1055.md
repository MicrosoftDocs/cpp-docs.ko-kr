# <a name="nmake-fatal-error-u1055"></a>NMAKE 심각한 오류 U1055

환경 공간이 부족 합니다.

운영 체제 환경 변수에 대 한 공간이 부족합니다.

환경 공간 늘리기 또는 더 적은 수의 환경 변수를 설정 합니다.
