# <a name="compiler-warning-level-1-c4402"></a>컴파일러 경고(수준 1) C4402

PTR 연산자를 사용 해야 합니다.

형식 참조 또는 인라인 어셈블리 코드의 형식으로 캐스팅 하는 경우에 PTR 연산자 없이 피연산자에 사용 됩니다.

컴파일러에는 PTR 연산자 라고 가정 합니다.
