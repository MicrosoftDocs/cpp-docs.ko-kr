# <a name="compiler-warning-level-1-c4402"></a>컴파일러 경고(수준 1) C4402

PTR 연산자를 사용 해야 합니다.

인라인 어셈블리 코드에서 형식을 참조 하거나 형식을 캐스팅 하는 경우 PTR 연산자가 없는 피연산자에서 형식이 사용 됩니다.

컴파일러는 PTR 연산자를 가정 합니다.
