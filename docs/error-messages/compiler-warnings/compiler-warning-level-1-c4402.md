# <a name="compiler-warning-level-1-c4402"></a>컴파일러 경고(수준 1) C4402
PTR 연산자를 사용 해야 합니다.  
  
 으로 참조 하거나 인라인 어셈블리 코드의 형식으로 캐스팅할 때 PTR 연산자 없이 피연산자 형식이 사용 됩니다.  
  
 컴파일러는 PTR 연산자를 가정합니다.
