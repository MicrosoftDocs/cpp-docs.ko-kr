# <a name="compiler-warning-level-1-c4381"></a>컴파일러 경고(수준 1) C4381

'function1': 인터페이스 메서드 'function2' public이 아닌 메서드에서 구현 되지 것입니다

클래스는 인터페이스의 모든 함수를 구현 해야 합니다. 클래스는 기본 클래스 중 하나는 함수를 구현 하는 경우이 조건이 충족 수 있습니다. 그러나 함수는 공용 함수로 구현 되어야 합니다.
