# <a name="compiler-warning-level-1-c4168"></a>컴파일러 경고(수준 1) C4168

컴파일러 한계: 디버거 형식을 벗어났습니다. 'database' 프로그램 데이터베이스를 삭제하고 다시 빌드합니다.

프로그램의 모든 형식에 맞게 프로그램 데이터베이스 파일을 다시 작성해야 합니다.
