# <a name="compiler-warning-level-1-c4045"></a>컴파일러 경고(수준 1) C4045

'array': 배열 범위에 오버플로가 발생했습니다.

배열에 이니셜라이저가 너무 많습니다. 불필요한 이니셜라이저가 무시됩니다.

배열 요소와 이니셜라이저의 크기 및 수량이 일치하는지 확인합니다.
