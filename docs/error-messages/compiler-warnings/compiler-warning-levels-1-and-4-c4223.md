# <a name="compiler-warning-levels-1-and-4-c4223"></a>컴파일러 경고(수준 1 및 4) C4223

비표준 확장이 사용 됨: lvalue가 아닌 배열을 포인터로 변환 했습니다.

표준 C에서는 lvalue가 아닌 배열을 포인터로 변환할 수 없습니다. 기본 Microsoft 확장 ([/ze](../../build/reference/za-ze-disable-language-extensions.md))을 사용 하면 가능 합니다.
