# <a name="compiler-warning-level-1-c4410"></a>컴파일러 경고(수준 1) C4410

피연산자 크기가 잘못 되었습니다.

명령에서 피연산자 중 하나가 크기가 잘못 되었습니다. 피연산자에 대 한 유효한 최소 크기 사용 되었습니다.
