# <a name="compiler-warning-level-1-c4410"></a>컴파일러 경고(수준 1) C4410

피연산자의 크기가 잘못 되었습니다.

명령의 피연산자 중 하나에 잘못 된 크기가 있습니다. 피연산자에 대해 가장 작은 올바른 크기를 사용 했습니다.
