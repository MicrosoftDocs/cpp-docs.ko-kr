# <a name="compiler-warning-level-3-c4306"></a>컴파일러 경고(수준 3) C4306

> '*identifier*': 더 큰 크기의 '*type1*'에서 '*type2*' (으)로의 변환입니다.

식별자는 더 큰 포인터에 대 한 형식 캐스트입니다. 새 형식의 채워지지 않은 상위 비트는 0으로 채워집니다.

이 경고는 원치 않는 변환을 나타낼 수 있습니다. 결과 포인터는 유효 하지 않을 수 있습니다.
