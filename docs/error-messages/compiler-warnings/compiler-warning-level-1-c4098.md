# <a name="compiler-warning-level-1-c4098"></a>컴파일러 경고 (수준 1) C4098

' function ': void 함수가 값을 반환 합니다.

반환 형식 [void](../../cpp/void-cpp.md) 로 선언 된 함수에는 값을 반환 하는 `return` 문이 있습니다. 컴파일러는 함수가 `int`형식의 값을 반환 한다고 가정 합니다.
