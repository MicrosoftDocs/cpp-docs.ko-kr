# <a name="compiler-warning-level-1-c4098"></a>컴파일러 경고 (수준 1) C4098
'function': 값을 반환 하는 void 함수  
  
 함수 반환 형식으로 선언 [void](../../cpp/void-cpp.md) 에 `return` 값을 반환 하는 문입니다. 컴파일러는 함수 형식의 값을 반환 하는 것으로 가정 `int`합니다.
