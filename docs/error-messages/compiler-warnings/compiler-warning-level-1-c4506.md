# <a name="compiler-warning-level-1-c4506"></a>컴파일러 경고(수준 1) C4506

' function ' 인라인 함수에 대 한 정의가 없습니다.

지정 된 함수가 선언 되었으며 인라이닝 되도록 표시 되었지만 정의 되지 않았습니다.

컴파일러가 함수를 인라인 하지 않았습니다.

인라인 될 외부 함수가 키워드로 선언 되었는지 확인 **`extern`** 합니다.
