# <a name="compiler-warning-level-1-c4506"></a>컴파일러 경고(수준 1) C4506

'function' 인라인 함수에 대 한 정의가 없습니다

지정된 된 함수 선언 및 인라인 처리 하지만 정의 되지 않은 표시 되었습니다.

컴파일러는 함수를 인라인 하지 않았습니다.

사용 하 여 선언 된는 외부 함수 인라인 될 수 있는지 확인 합니다 `extern` 키워드입니다.
