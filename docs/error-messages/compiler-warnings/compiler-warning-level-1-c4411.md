# <a name="compiler-warning-level-1-c4411"></a>컴파일러 경고(수준 1) C4411

' identifier ': 기호가 치환 레지스터로 확인 됩니다.

이 식별자는 치환 레지스터를 확인 하는 로컬 기호 이므로 다른 기호와 함께 피연산자에서 사용할 수 있습니다.
