# <a name="compiler-warning-level-1-c4120"></a>컴파일러 경고(수준 1) C4120

based 또는 unbased가 일치하지 않습니다.

한 포인터는 based이고 다른 포인터는 based가 아니므로 컴파일러에서 두 포인터 간에 변환할 수 없습니다.
