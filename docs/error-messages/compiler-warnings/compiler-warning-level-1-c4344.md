# <a name="compiler-warning-level-1-c4344"></a>컴파일러 경고(수준 1) C4344

동작 변경: 명시적 템플릿 인수를 사용 하면 ' function ' 호출이 발생 합니다.

명시적 인수를 지정 하지 않은 경우와 같은 명시적 템플릿 인수를 사용 하 여 함수를 호출 하면 다른 함수가 호출 됩니다.
