# <a name="compiler-warning-level-1-c4375"></a>컴파일러 경고(수준 1) C4375
'method1' 2' public이 아닌 메서드를 재정의 하지 않습니다.  
  
 다른 형식을 구현 하는 형식을 재정의 메서드를 정의 하지만 재정의 공개 아닙니다. 따라서 메서드는 기본 형식 메서드를 재정의 하지 않습니다.
