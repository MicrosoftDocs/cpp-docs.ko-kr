# <a name="compiler-warning-level-1-c4024"></a>컴파일러 경고 (수준 1) C4024

'function': 형식 및 실제 매개 변수 'r'에 대 한 다양 한 종류

해당 형식 및 실제 매개 변수 형식이 다른 합니다. 컴파일러는 변경 하지 않고도 실제 매개 변수를 전달 합니다. 수신 함수 예상 되는 형식 매개 변수 형식을 변환 합니다.
