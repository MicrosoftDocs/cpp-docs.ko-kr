# <a name="compiler-warning-level-1-c4024"></a>컴파일러 경고 (수준 1) C4024

' function ': 형식 및 실제 매개 변수 ' number '의 형식이 서로 다릅니다.

해당 하는 형식 및 실제 매개 변수의 형식은 서로 다릅니다. 컴파일러가 실제 매개 변수를 변경 하지 않고 전달 합니다. 수신 함수는 매개 변수 형식을 예상한 형식으로 변환 합니다.
