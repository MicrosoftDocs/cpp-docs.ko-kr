# <a name="compiler-warning-level-1-c4711"></a>컴파일러 경고(수준 1) C4711

인라인 확장을 위해 'function' 함수가 선택되었습니다.

컴파일러가 인라인으로 표시 되지 않았지만 지정 된 함수에서 인라이닝을 수행 했습니다.

C4711은 [/Sv2](../../build/reference/ob-inline-function-expansion.md) 가 지정 된 경우에 사용할 수 있습니다.

인라인은 컴파일러의 판단에 의해 수행 됩니다. 이 경고는 정보 제공용입니다.

기본적으로 이 경고는 해제되어 있습니다. 경고를 사용 하도록 설정 하려면 [#pragma 경고](../../preprocessor/warning.md)를 사용 합니다. 자세한 내용은 [기본적으로 해제되어 있는 컴파일러 경고](../../preprocessor/compiler-warnings-that-are-off-by-default.md) 를 참조하세요.
