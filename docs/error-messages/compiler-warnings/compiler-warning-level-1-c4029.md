# <a name="compiler-warning-level-1-c4029"></a>컴파일러 경고 (수준 1) C4029

선언 된 정식 매개 변수 목록이 정의와 다릅니다.

형식 매개 변수 형식이 함수 선언에서 함수 정의의 설정과 일치 하지 않습니다. 컴파일러는 정의에서 매개 변수 목록을 사용 합니다.
