# <a name="compiler-warning-level-2-c4653"></a>컴파일러 경고(수준 2) C4653

'option' 미리 컴파일된 헤더와 일치 하지 않는 컴파일러 옵션 현재 명령줄 옵션이 무시 됩니다

미리 컴파일된 헤더 사용을 사용 하 여 지정 된 옵션 ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) 옵션이 미리 컴파일된 헤더가 만들어졌을 때 지정 된 옵션을 사용 하 여 일관 되지 않았습니다. 이 컴파일 미리 컴파일된 헤더를 만들 때 지정한 옵션을 사용 합니다.

팩 구조 옵션에 대해 다른 값이 경고가 발생할 수 있습니다 ([/Zp](../../build/reference/zp-struct-member-alignment.md)) 미리 컴파일된 헤더를 컴파일하는 동안 지정 되었습니다.
