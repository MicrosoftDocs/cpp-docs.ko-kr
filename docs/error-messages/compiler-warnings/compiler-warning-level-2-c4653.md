# <a name="compiler-warning-level-2-c4653"></a>컴파일러 경고(수준 2) C4653

' option ' 컴파일러 옵션이 미리 컴파일된 헤더와 일치 하지 않습니다. 현재 명령줄 옵션이 무시 됩니다.

미리 컴파일된 헤더 사용 ([/yu](../../build/reference/yu-use-precompiled-header-file.md)) 옵션으로 지정 된 옵션이 미리 컴파일된 헤더를 만들 때 지정한 옵션과 일치 하지 않습니다. 이 컴파일에서는 미리 컴파일된 헤더를 만들 때 지정 된 옵션을 사용 했습니다.

이 경고는 미리 컴파일된 헤더를 컴파일하는 동안 Pack 구조체 옵션 ([/zp](../../build/reference/zp-struct-member-alignment.md))의 다른 값이 지정 된 경우에 발생할 수 있습니다.
