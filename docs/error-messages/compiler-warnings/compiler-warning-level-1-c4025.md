# <a name="compiler-warning-level-1-c4025"></a>컴파일러 경고(수준 1) C4025
'number': 기반 포인터가 가변 인수를 갖는 함수로 전달되었습니다. 매개 변수 number  
  
 가변 인수를 갖는 함수에 기반 포인터를 전달하면 포인터가 정규화되어 예기치 않은 결과가 발생합니다. 가변 인수를 갖는 함수에 기반 포인터를 전달하지 마세요.
