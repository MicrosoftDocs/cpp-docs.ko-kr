# <a name="compiler-warning-level-1-c4652"></a>컴파일러 경고(수준 1) C4652
미리 컴파일된 헤더와 일치 하지 않는 ' option' 컴파일러 옵션 현재 명령줄 옵션이 미리 컴파일된 헤더에 정의 된 것을 재정의 합니다.  
  
 주어진된 명령줄 옵션이 미리 컴파일된 헤더 (.pch)를 만들 때 다릅니다. 현재 명령줄에 지정 된 옵션이 사용 되었습니다.  
  
 이 경고는 주어진된 명령줄 옵션이 미리 컴파일된 헤더를 다시 생성 하 여 방지할 수 있습니다.
