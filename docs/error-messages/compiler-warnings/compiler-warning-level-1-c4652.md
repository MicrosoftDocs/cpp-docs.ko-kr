'option' 미리 컴파일된 헤더와 일치 하지 않는 컴파일러 옵션 현재 명령줄 옵션이 미리 컴파일된 헤더에 정의 된 재정의

지정 된 명령줄 옵션이 미리 컴파일된 헤더 (.pch) 만들어질 때 지정 된 된 다릅니다. 현재 명령줄에 지정 된 옵션을 사용 했습니다.
