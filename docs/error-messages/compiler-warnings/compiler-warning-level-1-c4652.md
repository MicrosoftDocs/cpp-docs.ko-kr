# <a name="compiler-warning-level-1-c4652"></a>컴파일러 경고(수준 1) C4652

' option ' 컴파일러 옵션이 미리 컴파일된 헤더와 일치 하지 않습니다. 현재 명령줄 옵션이 미리 컴파일된 헤더에 정의 된 옵션을 재정의 합니다.

지정 된 명령줄 옵션이 미리 컴파일된 헤더 (.pch)를 만들었을 때 지정 된 옵션과 다릅니다. 현재 명령줄에 지정 된 옵션이 사용 되었습니다.

지정 된 명령줄 옵션을 사용 하 여 미리 컴파일된 헤더를 다시 생성 하면이 경고를 방지할 수 있습니다.
