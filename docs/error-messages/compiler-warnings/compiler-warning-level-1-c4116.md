# <a name="compiler-warning-level-1-c4116"></a>컴파일러 경고 (수준 1) C4116

괄호 안에 명명 되지 않은 형식 정의

이름이 없는 구조체, 공용 구조체 또는 열거 형식이 괄호 식에 정의 되어 있습니다. 형식 정의는 의미가 없습니다.

C 함수 호출에서는 정의가 전역 범위입니다. C++ 함수 호출에서 정의의 범위는 호출 되는 함수와 동일 합니다.
