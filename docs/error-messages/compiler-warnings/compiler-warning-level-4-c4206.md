# <a name="compiler-warning-level-4-c4206"></a>컴파일러 경고(수준 4) C4206

**비표준 확장이 사용 됨: 변환 단위가 비어 있습니다.**

전처리 후 파일이 비어 있습니다.

이 확장을 통해 코드를 다른 컴파일러로 이식할 수 없게 될 수 있습니다. ANSI 호환성 ([/za](../../build/reference/za-ze-disable-language-extensions.md))에서 오류를 생성 하 고 C 소스 코드에만 적용 됩니다.
