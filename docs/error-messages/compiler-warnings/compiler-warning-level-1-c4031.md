# <a name="compiler-warning-level-1-c4031"></a>컴파일러 경고 (수준 1) C4031

두 번째 정식 매개 변수 목록이 첫째 목록 보다 깁니다.

함수는 다른 형식 매개 변수를 사용 하 여 다시 선언 되었습니다. 컴파일러는 첫 번째 선언에 지정된 형식 매개 변수를 사용합니다.
