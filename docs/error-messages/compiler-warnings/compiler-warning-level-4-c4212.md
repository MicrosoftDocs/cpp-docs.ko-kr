# <a name="compiler-warning-level-4-c4212"></a>컴파일러 경고(수준 4) C4212
비표준 확장이 사용 됨: 줄임표를 사용 하는 함수 선언  
  
 함수 프로토타입에 인수의 변수 수 있습니다. 함수 정의 그렇지 않습니다.  
  
 다음 샘플에서는 C4212 오류가 생성 됩니다.  
  
```  
// C4212.c  
// compile with: /W4 /Ze /c  
void f(int , ...);  
void f(int i, int j) {}  
```
