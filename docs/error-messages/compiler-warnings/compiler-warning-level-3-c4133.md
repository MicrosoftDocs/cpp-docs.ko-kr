# <a name="compiler-warning-level-3-c4133"></a>컴파일러 경고 (수준 3) C4133

'type': 'type2' 'type1'에서 호환 되지 않는 형식

이 경고는 서로 다른 형식의 두 포인터를 뺄 시도 하 여 발생할 수 있습니다.

이 경고를 방지 하려면 적절 한 형식 캐스트를 제공 합니다.
