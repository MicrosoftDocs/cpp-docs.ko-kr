# <a name="compiler-warning-level-3-c4133"></a>컴파일러 경고 (수준 3) C4133
'type': 'type1' 'type2' (으)에서 호환 되지 않는 유형-  
  
 이 경고는 서로 다른 형식의 두 포인터 빼기 하려는 경우에 발생할 수 있습니다.  
  
 이 경고를 방지 하려면 적절 한 형식 변환을 제공 합니다.
