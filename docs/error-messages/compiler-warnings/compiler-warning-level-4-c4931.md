# <a name="compiler-warning-level-4-c4931"></a>컴파일러 경고(수준 4) C4931

> 형식 라이브러리가 *숫자* 비트 포인터에 대해 빌드된 것으로 가정 합니다.

[#Import](../../preprocessor/hash-import-directive-cpp.md) 지시어의 **ptrsize** 특성과 함께 명시적 정보가 제공 되지 않았습니다. 컴파일러는 형식 라이브러리의 포인터 크기가 *number* 임을 결론 합니다.

기본적으로 이 경고는 해제되어 있습니다. 자세한 내용은 [기본적으로 해제되어 있는 컴파일러 경고](../../preprocessor/compiler-warnings-that-are-off-by-default.md) 를 참조하세요.
