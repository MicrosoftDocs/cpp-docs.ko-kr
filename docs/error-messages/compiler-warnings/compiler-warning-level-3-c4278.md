# <a name="compiler-warning-level-3-c4278"></a>컴파일러 경고(수준 3) C4278

> '*identifier*': 형식 라이브러리 '*tlb*'의 식별자가 이미 매크로입니다. ' rename ' 한정자 사용

[#Import](../../preprocessor/hash-import-directive-cpp.md)를 사용 하는 경우 가져오는 typelib의 식별자가 식별자 *식별자* 를 선언 하려고 합니다. 그러나이는 이미 유효한 기호입니다.

`#import` **이름 바꾸기** 특성을 사용 하 여 형식 라이브러리에 있는 기호에 별칭을 할당 합니다.
