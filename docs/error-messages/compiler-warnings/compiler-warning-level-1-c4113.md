# <a name="compiler-warning-level-1-c4113"></a>컴파일러 경고 (수준 1) C4113

' identifier1 '는 ' identifier2 '의 매개 변수 목록과 다릅니다.

함수 포인터가 다른 함수 포인터에 할당 되었지만 함수의 정식 매개 변수 목록이 일치 하지 않습니다. 할당은 수정 하지 않고 컴파일됩니다.
