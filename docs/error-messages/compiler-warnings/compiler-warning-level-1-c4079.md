# <a name="compiler-warning-level-1-c4079"></a>컴파일러 경고 (수준 1) C4079
예기치 않은 'token' 토큰입니다.  
  
 예기치 않은 구분 토큰 pragma 인수 목록에서 발생합니다. Pragma의 나머지 부분에서는 무시 되었습니다.  
  
 다음 샘플에서는 C4079 오류가 생성 됩니다.  
  
```  
// C4079.cpp  
// compile with: /W1  
#pragma warning(disable : 4081)  
#pragma pack(c,16)   // C4079  
  
int main() {  
}  
```
