# <a name="compiler-warning-level-2-c4051"></a>컴파일러 경고(수준 2) C4051
형식을 변환하면서 데이터가 손실될 수 있습니다.  
  
 식에 기본 형식이 각기 다른 두 데이터 항목이 포함되어 있습니다. 한 형식으로 변환하면 데이터 항목이 잘립니다.  
  
 적절한 형식으로 데이터 항목을 캐스팅하여 이 경고를 해결할 수 있습니다.
