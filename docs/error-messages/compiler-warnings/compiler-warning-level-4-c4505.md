# <a name="compiler-warning-level-4-c4505"></a>컴파일러 경고(수준 4) C4505

' function ': 참조 되지 않은 지역 함수를 제거 했습니다.

지정 된 함수는 로컬 이며 모듈 본문에서 참조 되지 않습니다. 따라서 함수는 데드 코드입니다.

컴파일러가이 비활성 함수에 대 한 코드를 생성 하지 않았습니다.
