# <a name="compiler-warning-level-4-c4505"></a>컴파일러 경고(수준 4) C4505
'function': 참조 되지 않은 로컬 함수를 제거 했습니다  
  
 지정된 된 함수는 로컬 및 모듈의 본문에서 참조 하지 않았으므로 따라서 함수는 데드 코드가 있음.  
  
 컴파일러는이 불필요 한 함수에 대 한 코드를 생성 하지 않았습니다.
