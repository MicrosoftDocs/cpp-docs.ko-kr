# <a name="compiler-warning-level-4-c4672"></a>컴파일러 경고(수준 4) C4672

'identifier1': 모호합니다. 우선 'identifier2'로 표시됩니다.

블록에서 throw 되는 지정 된 개체가 **`try`** 모호 합니다. 발생할 경우 개체를 명확하게 지정할 수 없습니다.
