# <a name="compiler-warning-level-1-c4229"></a>컴파일러 경고(수준 1) C4229
사용 하는 오래 된 구문: 데이터의 한정자는 무시 됩니다.  
  
 와 같은 Microsoft 한정자를 사용 하 여 `__cdecl` 데이터 선언은 쿼리로 합니다.  
  
## <a name="example"></a>예  
  
```  
// C4229.cpp  
// compile with: /W1 /LD  
int __cdecl counter;   // C4229 cdecl ignored  
```
