# <a name="compiler-warning-level-1-c4541"></a>컴파일러 경고(수준 1) C4541

/GR-에서 다형 형식 ' type '에 ' identifier '를 사용 합니다. 예기치 않은 동작이 발생할 수 있습니다.

런타임 형식 정보를 사용 하지 않고 런타임 형식 정보를 요구 하는 기능을 사용 하려고 했습니다. [/Gr](../../build/reference/gr-enable-run-time-type-information.md)를 사용 하 여 다시 컴파일하십시오.
