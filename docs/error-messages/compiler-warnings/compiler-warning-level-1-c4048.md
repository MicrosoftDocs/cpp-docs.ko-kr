# <a name="compiler-warning-level-1-c4048"></a>컴파일러 경고 (수준 1) C4048
배열 첨자를 다른 선언: 'identifier1' 및 'identifier2'  
  
 식에는 크기가 다른 배열에 대 한 포인터가 포함 됩니다. 포인터 변환 없이 사용 됩니다.  
  
 배열 동일 하거나 동등한 형식으로 명시적으로 캐스팅 하는 경우이 경고를 해결할 수 있습니다.
