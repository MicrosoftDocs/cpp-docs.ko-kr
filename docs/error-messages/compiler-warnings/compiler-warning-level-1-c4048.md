# <a name="compiler-warning-level-1-c4048"></a>컴파일러 경고 (수준 1) C4048

다른 배열 첨자를 선언 합니다. 'identifier1' 및 'identifier2'

식을 다른 크기의 배열에 대 한 포인터를 포함 합니다. 포인터 변환 없이 사용 됩니다.

동일 하거나 동등한 형식 배열은 명시적으로 캐스팅 하는 경우이 경고를 해결할 수 있습니다.
