# <a name="compiler-warning-level-1-c4406"></a>컴파일러 경고(수준 1) C4406

지시문의 피연산자가 무시 되었습니다.

지시문은 피연산자를 사용 하지 않지만 피연산자가 지정 되었습니다.

컴파일러가 지정 된 피연산자 또는 피연산자를 무시 했습니다.
