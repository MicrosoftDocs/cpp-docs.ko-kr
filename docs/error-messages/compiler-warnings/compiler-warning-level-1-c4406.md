# <a name="compiler-warning-level-1-c4406"></a>컴파일러 경고(수준 1) C4406
무시 지시문의 피연산자  
  
 지시문에 피연산자 하지만 피연산자가 지정 되어 있습니다.  
  
 컴파일러는 지정 된 피연산자를 무시 합니다.
