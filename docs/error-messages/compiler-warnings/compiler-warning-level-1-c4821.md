# <a name="compiler-warning-level-1-c4821"></a>컴파일러 경고(수준 1) C4821

> 시그니처 (BOM)를 사용 하 여 파일을 저장 하세요 유니코드 인코딩 형식을 확인할 수 없습니다.

컴파일러는 파일의 인코딩 형식을 확인 하지 못했습니다. 이 경고를 해결 하는 바이트 순서 표식을 사용 하 여 파일을 저장 합니다. 참조 [인코딩 사용 하 여 파일 관리](/sql/ssms/solution/manage-files-with-encoding) 자세한 내용은 합니다.
