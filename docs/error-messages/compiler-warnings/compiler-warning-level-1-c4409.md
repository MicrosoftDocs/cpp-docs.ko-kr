# <a name="compiler-warning-level-1-c4409"></a>컴파일러 경고(수준 1) C4409
잘못 된 명령 크기  
  
 지정된 된 크기를 사용 하 여 폼 명령을 사용할 수 없습니다. 유효한 최소 크기 사용 되었습니다.
