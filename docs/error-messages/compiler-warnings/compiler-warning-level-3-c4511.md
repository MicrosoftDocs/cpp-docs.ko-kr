# <a name="compiler-warning-level-3-c4511"></a>컴파일러 경고(수준 3) C4511

'class': 복사 생성자를 생성할 수 없습니다

컴파일러는 클래스에 대 한 기본 복사 생성자를 생성할 수 없습니다. 기본 클래스에 전용 복사 생성자 있을 수 있습니다.
