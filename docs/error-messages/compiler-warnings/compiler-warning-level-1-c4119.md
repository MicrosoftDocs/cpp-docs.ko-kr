# <a name="compiler-warning-level-1-c4119"></a>컴파일러 경고(수준 1) C4119

서로 다른 기준 'base1'과 'base2'를 지정했습니다.

두 기준 포인터가 서로 다른 기준을 사용하므로 호환되지 않습니다. 컴파일러에서 서로 변환할 수 없습니다.
