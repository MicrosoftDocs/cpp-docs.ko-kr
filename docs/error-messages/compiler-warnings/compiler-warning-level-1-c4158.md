# <a name="compiler-warning-level-1-c4158"></a>컴파일러 경고(수준 1) C4158

> #pragma pointers_to_members(full_generality, inheritance)로 간주합니다.

함께 제공 된 **#pragma pointers_to_members (full_generality)** 를 사용 하지 않고 **#pragma pointers_to_members (** *단일* &#124; *다중* &#124; *가상* **)** 이 실행 되었습니다.
