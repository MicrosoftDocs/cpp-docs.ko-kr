# <a name="compiler-warning-level-4-c4513"></a>컴파일러 경고(수준 4) C4513

' class ': 소멸자를 생성할 수 없습니다.

컴파일러가 지정 된 클래스에 대 한 기본 소멸자를 생성할 수 없습니다. 소멸자를 만들지 않았습니다. 소멸자가 파생 클래스에서 액세스할 수 없는 기본 클래스에 있습니다. 기본 클래스에 전용 소멸자가 있는 경우 public 또는 protected로 설정 합니다.
