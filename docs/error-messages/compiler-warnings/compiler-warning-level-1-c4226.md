# <a name="compiler-warning-level-1-c4226"></a>컴파일러 경고(수준 1) C4226

> 비표준 확장이 사용 됨: '*키워드*'는 사용 되지 않는 키워드

시각적 개체의 현재 버전 C++ 이 키워드를 사용 하지 않습니다.

이 경고는 오류를 자동으로 승격 됩니다.
