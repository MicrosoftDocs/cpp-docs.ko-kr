# <a name="compiler-warning-level-1-c4226"></a>컴파일러 경고(수준 1) C4226

> 비표준 확장이 사용 됨: '*keyword*'는 사용 되지 않는 키워드입니다.

Visual C++의 현재 버전은이 키워드를 사용 하지 않습니다.

이 경고는 자동으로 오류로 승격 됩니다.
