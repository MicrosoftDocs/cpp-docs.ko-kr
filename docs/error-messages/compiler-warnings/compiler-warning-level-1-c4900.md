# <a name="compiler-warning-level-1-c4900"></a>컴파일러 경고(수준 1) C4900

'tool1' 버전 'version1'과 'tool2' 버전 'version2' 사이에 중간 언어가 일치하지 않습니다.

*tool1* 과 *tool2* 에서 사용되는 중간 언어가 일치하지 않습니다. 각 도구의 현재 버전이 설치되어 있는지 확인합니다.
