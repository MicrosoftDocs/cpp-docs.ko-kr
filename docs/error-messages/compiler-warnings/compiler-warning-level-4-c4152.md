# <a name="compiler-warning-level-4-c4152"></a>컴파일러 경고(수준 4) C4152

비표준 확장입니다. 식에서 함수/데이터 포인터 변환이 있습니다.

함수 포인터가 데이터 포인터로 변환되거나 그 반대로 변환됩니다. 이 변환은 Microsoft 확장(/Ze)에서는 허용되지만 ANSI C에서는 허용되지 않습니다.
