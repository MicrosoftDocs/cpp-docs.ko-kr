# <a name="compiler-warning-level-1-c4679"></a>컴파일러 경고(수준 1) C4679

> '*멤버*': 멤버를 가져올 수 없습니다

컴파일러가 메타 데이터에서 가져올 수 없는, 지원 되지 않는 구문을 발견 했습니다.

구조를 사용 하지 마십시오.
