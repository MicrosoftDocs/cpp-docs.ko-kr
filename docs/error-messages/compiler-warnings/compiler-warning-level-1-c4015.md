# <a name="compiler-warning-level-1-c4015"></a>컴파일러 경고 (수준 1) C4015

' identifier ': 비트 필드의 형식은 정수 계열 이어야 합니다.

비트 필드가 정수 형식으로 선언 되지 않았습니다. 컴파일러는 비트 필드의 기본 형식을 unsigned로 간주 합니다. 비트 필드는 부호 없는 정수 형식으로 선언 해야 합니다.
