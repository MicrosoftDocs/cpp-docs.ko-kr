# <a name="compiler-warning-level-1-c4179"></a>컴파일러 경고(수준 1) C4179

> ' / / *':으로 구문 분석 '/' 및 ' /\*': 혼동 표준 ' / /' 주석

__//\*__ 잘못 된 주석 구분 기호입니다. 사용 하 여 __//__ 하거나 __/ \*__ 대신 합니다.
