# <a name="compiler-warning-level-1-c4179"></a>컴파일러 경고(수준 1) C4179

> ' / / *':로 구문 분석할 '/' 및 ' /\*': 혼동 표준 ' / /' 주석  
  
__//\*__ 잘못 된 주석 구분 기호입니다. 사용 하 여 __//__ 또는 __/ \*__ 대신 합니다.
