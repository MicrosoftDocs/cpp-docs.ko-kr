# <a name="compiler-warning-level-1-c4179"></a>컴파일러 경고(수준 1) C4179

> '//* ': '/' 및 '/\*'로 구문 분석 됩니다. 표준 '//' 주석과 혼동 됩니다.

__//\*__ 는 잘못된 주석 구분 기호입니다. 대신 __//__ 또는 __/\*__ 를 사용하세요.
