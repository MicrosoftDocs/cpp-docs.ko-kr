# <a name="compiler-warning-level-3-c4543"></a>컴파일러 경고(수준 3) C4543

' No_injected_text ' 특성에 의해 삽입 된 텍스트가 표시 되지 않습니다.

[No_injected_text](../../windows/attributes/no-injected-text.md) 특성이 소스 코드에 표시 됩니다. 즉, 컴파일러가 특성을 삽입 하는 것을 방지 합니다.

이 경고는 특성에서 코드를 삽입할 수 없다는 알림을 표시 하기 위한 것입니다.
