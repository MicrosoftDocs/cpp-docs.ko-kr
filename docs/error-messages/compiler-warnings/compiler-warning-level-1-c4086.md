# <a name="compiler-warning-level-1-c4086"></a>컴파일러 경고(수준 1) C4086
pragma 매개 변수는 '1', '2', '4', '8' 또는 '16'이어야 합니다.  
  
 pragma 매개 변수에 필수 값(1, 2, 4, 8 또는 16)이 없습니다.  
  
## <a name="example"></a>예  
  
```  
// C4086.cpp  
// compile with: /W1 /LD  
#pragma pack( 3 ) // C4086  
```
