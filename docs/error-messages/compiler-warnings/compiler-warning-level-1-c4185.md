# <a name="compiler-warning-level-1-c4185"></a>컴파일러 경고(수준 1) C4185
알 수 없는 #import 특성 'attribute'를 무시합니다.  
  
 attribute는 `#import`의 유효한 특성이 아닙니다. 무시됩니다.  
  
## <a name="example"></a>예  
  
```  
// C4185.cpp  
// compile with: /W1 /c  
#import "stdole2.tlb" no_such_attribute   // C4185  
```
