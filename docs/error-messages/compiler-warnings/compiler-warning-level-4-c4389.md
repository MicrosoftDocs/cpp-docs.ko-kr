# <a name="compiler-warning-level-4-c4389"></a>컴파일러 경고(수준 4) C4389
'operator': signed 또는 unsigned 일치 하지 않습니다.  
  
 작업 서명 되 고 서명 되지 않은 변수를 포함 합니다. 이 인해 데이터가 손실 될 수 있습니다.  
  
 다음 샘플에서는 C4389 오류가 생성 됩니다.  
  
```  
// C4389.cpp  
// compile with: /W4  
#pragma warning(default: 4389)  
  
int main()  
{  
   int a = 9;  
   unsigned int b = 10;  
   if (a == b)   // C4389  
      return 0;  
   else  
      return 0;  
};  
```
