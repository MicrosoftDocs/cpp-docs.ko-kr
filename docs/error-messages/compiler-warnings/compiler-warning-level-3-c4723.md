# <a name="compiler-warning-level-3-c4723"></a>컴파일러 경고(수준 3) C4723

> `potential divide by 0`

분할 작업에서 두 번째 피연산자가 컴파일 시간에 0으로 계산 되어 정의 되지 않은 결과를 제공 합니다.

이 경고는 최적화를 사용 하도록 설정한 경우에만 실행 됩니다.

컴파일러가 0 피연산자를 생성 했을 수 있습니다.
