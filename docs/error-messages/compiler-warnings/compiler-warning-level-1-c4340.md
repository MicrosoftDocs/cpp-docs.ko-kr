# <a name="compiler-warning-level-1-c4340"></a>컴파일러 경고(수준 1) C4340
'value': 값이 음수 값으로 양의에서 래핑 되었습니다  
  
 `enum` 값이 가장 큰 값 보다 크면 `enum` 양수 값을 음수 값으로 연결 되었습니다.
