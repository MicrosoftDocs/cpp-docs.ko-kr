# <a name="compiler-warning-level-1-c4187"></a>컴파일러 경고(수준 1) C4187

\#import 특성 'attribute1' 및 'attribute2' 호환 되지 않습니다. 모두 무시

[#import](../../preprocessor/hash-import-directive-cpp.md) 문이 `no_implementation` 및 `implementation_only` 특성을 지정했습니다. 모두 무시됩니다.
