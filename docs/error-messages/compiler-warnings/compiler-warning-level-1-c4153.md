# <a name="compiler-warning-level-1-c4153"></a>컴파일러 경고(수준 1) C4153

식에서 함수/데이터 포인터 변환이 있습니다.

함수 포인터가 데이터 포인터로 변환되거나 그 반대로 변환됩니다. 이 변환은 Microsoft 확장(/Ze)에서는 허용되지만 ANSI C에서는 허용되지 않습니다.
