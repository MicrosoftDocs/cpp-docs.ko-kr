# <a name="compiler-warning-level-1-c4403"></a>컴파일러 경고(수준 1) C4403
PTR 연산자가 잘못 되었습니다  
  
 PTR 연산자는 인라인 어셈블러 코드에 부적절 하 게 사용 됩니다.  
  
 컴파일러에서 PTR 연산자를 무시 합니다.
