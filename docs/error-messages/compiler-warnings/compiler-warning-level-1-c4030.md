# <a name="compiler-warning-level-1-c4030"></a>컴파일러 경고(수준 1) C4030
**첫째 정식 매개 변수 목록이 둘째 목록보다 깁니다.**  
  
 다른 형식 매개 변수를 사용하여 함수가 다시 선언되었습니다. 컴파일러는 첫 번째 선언에 지정된 형식 매개 변수를 사용합니다.
