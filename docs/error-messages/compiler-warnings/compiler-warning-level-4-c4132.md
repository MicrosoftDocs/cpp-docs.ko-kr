# <a name="compiler-warning-level-4-c4132"></a>컴파일러 경고(수준 4) C4132

'object': const 개체를 초기화해야 합니다.

상수가 초기화되지 않았습니다. **const** 특성이 있는 기호 값은 정의된 후에 변경할 수 없으며 초기화 값을 지정해야 합니다.
