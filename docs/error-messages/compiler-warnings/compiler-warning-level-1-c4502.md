# <a name="compiler-warning-level-1-c4502"></a>컴파일러 경고(수준 1) C4502

' 링크 사양 '은 ' extern ' 키워드를 사용 해야 하며 다른 모든 지정자 앞에와 야 합니다.

`extern` 키워드 없이 링크를 지정 했습니다. 링크는 extern이 아닌 형식과 관련이 없습니다.

컴파일러는 `extern` 키워드를 가정 합니다.
