# <a name="compiler-warning-level-1-c4502"></a>컴파일러 경고(수준 1) C4502
'extern' 키워드를 사용 해야 하 고 다른 모든 지정자 앞에 야 하는 '링크 사양이'  
  
 링크 하지 않고 지정 된 된 `extern` 키워드입니다. 링크는 extern이 아닌 형식과 관련이 없습니다.  
  
 가정 컴파일러는 `extern` 키워드입니다.
