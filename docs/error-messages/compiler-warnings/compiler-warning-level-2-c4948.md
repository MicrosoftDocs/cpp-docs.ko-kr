# <a name="compiler-warning-level-2-c4948"></a>컴파일러 경고(수준 2) C4948

' accessor '의 반환 형식이 해당 setter의 마지막 매개 변수 형식과 일치 하지 않습니다.

컴파일러가 인덱싱된 속성에 대해 가져오고 설정 되는 데이터 형식 간의 불일치를 발견 했습니다.

C4948는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
