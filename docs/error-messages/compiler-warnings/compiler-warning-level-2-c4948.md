# <a name="compiler-warning-level-2-c4948"></a>컴파일러 경고(수준 2) C4948

'accessor'의 반환 형식이 해당 setter의 마지막 매개 변수 형식에 맞지 않습니다.

컴파일러가 어떤 데이터 형식이 가져오기 되 고 인덱싱된 속성에 대 한 설정 불일치를 발견 했습니다.

C4948 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
