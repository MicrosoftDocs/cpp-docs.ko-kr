# <a name="compiler-warning-level-1-c4230"></a>컴파일러 경고(수준 1) C4230
사용 하는 오래 된 구문: 오래; 한정자가 무시 되었습니다  
  
 와 같은 Microsoft 한정자 전에 한정자를 사용 하 여 `__cdecl` 쿼리로 됩니다.  
  
## <a name="example"></a>예  
  
```  
// C4230.cpp  
// compile with: /W1 /LD  
int __cdecl const function1();   // C4230 const ignored  
```
