# <a name="compiler-warning-level-1-c4164"></a>컴파일러 경고(수준 1) C4164

'identifier': 내장 함수를 선언하지 않았습니다.

지정된 내장 함수를 선언하지 않았습니다. 적절한 헤더 파일을 #include해야 할 수도 있습니다.
