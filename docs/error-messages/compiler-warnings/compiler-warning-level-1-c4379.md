# <a name="compiler-warning-level-1-c4379"></a>컴파일러 경고(수준 1) C4379
이 컴파일러 버전의 공용 언어 런타임 버전을 지원 되지 않습니다. 이 버전을 사용 하 여 예기치 않은 결과가 발생할 수 있습니다.  
  
 이전 버전의 공용 언어 런타임 컴퓨터에 있지만 현재 버전이 아닌 해야합니다. C4379를 해결 하려면 컴파일러와 함께 제공 된 공용 언어 런타임의 버전을 설치 합니다.
