# <a name="compiler-warning-level-1-c4034"></a>컴파일러 경고 (수준 1) C4034

sizeof가 0을 반환 합니다.

`sizeof` 연산자는 크기가 0 인 피연산자 (빈 구조체, 공용 구조체, 클래스 또는 열거형 형식 또는 `void`형식)에 적용 됩니다.
