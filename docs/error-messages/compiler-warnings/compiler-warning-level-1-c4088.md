# <a name="compiler-warning-level-1-c4088"></a>컴파일러 경고 (수준 1) C4088

' function ': 실제 매개 변수 ' number ', 정식 매개 변수 ' number '의 포인터가 일치 하지 않습니다.

해당 하는 형식 및 실제 매개 변수는 서로 다른 수준의 간접 참조를 가집니다. 실제 매개 변수는 변경 없이 전달 됩니다. 호출 된 함수는 해당 값을 포인터로 해석 합니다.
