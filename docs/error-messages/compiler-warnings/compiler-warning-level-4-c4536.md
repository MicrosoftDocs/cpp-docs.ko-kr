# <a name="compiler-warning-level-4-c4536"></a>컴파일러 경고(수준 4) C4536
'type name': 형식-이름이 메타 데이터 한계인 '한계' 자 초과 합니다  
  
 관리 되는 형식 인 경우 형식 이름이 메타 데이터에서 잘립니다. 참조 [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) 자세한 정보에 대 한 합니다.  
  
 기본적으로 이 경고는 해제되어 있습니다. 자세한 내용은 [기본적으로 해제되어 있는 컴파일러 경고](../../preprocessor/compiler-warnings-that-are-off-by-default.md) 를 참조하세요.
