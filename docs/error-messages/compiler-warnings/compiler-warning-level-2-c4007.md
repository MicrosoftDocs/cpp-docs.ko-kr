# <a name="compiler-warning-level-2-c4007"></a>컴파일러 경고 (수준 2) C4007

' function ': ' attribute ' 여야 합니다.

함수에 필요한 특성은 명시적으로 명시 되지 않습니다. 예를 들어 **main** 함수에는 특성이 있어야 합니다 **`__cdecl`** . 컴파일러는 특성을 강제로 적용 합니다.
