# <a name="compiler-warning-level-2-c4007"></a>컴파일러 경고 (수준 2) C4007
'function': '특성' 이어야 합니다  
  
 함수에 대 한 필수 특성을 명시적으로 지정 되지 않습니다. 예를 들어 함수 **주** 있어야는 `__cdecl` 특성입니다. 컴파일러 분해할 수 없습니다.
