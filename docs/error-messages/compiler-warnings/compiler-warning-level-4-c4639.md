# <a name="compiler-warning-level-4-c4639"></a>컴파일러 경고(수준 4) C4639

MSXML 오류, XML 문서 주석이 처리 되지 않습니다. reason

이 경고는 여러 가지 이유로 발생할 수 있습니다.

이 경고를 해결 하려면:

- 컴파일하고.

- 공용 언어 런타임을 다시 설치 하 여 MSXML을 다시 설치 합니다.

- 경고를 발생 시키고 다시 컴파일하는 문서 주석을 편집 하거나 제거 합니다.

C4639가 실행 되 면 모든 추가 XML 주석 처리가 사용 하지 않도록 설정 되 고 .xdc 파일이 생성 되지 않습니다.
