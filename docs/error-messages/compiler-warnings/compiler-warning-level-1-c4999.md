# <a name="compiler-warning-level-1-c4999"></a>컴파일러 경고(수준 1) C4999

알 수 없는 경고 도움말 메뉴에서 기술 지원 명령을 선택하거나 기술 지원 도움말 파일을 참조하세요.

오류 상황을 확인하고 문제를 파악하여 재현 가능한 테스트 사례를 만든 다음 [Microsoft 기술 지원 서비스](/visualstudio/ide/talk-to-us)에 문의합니다.
