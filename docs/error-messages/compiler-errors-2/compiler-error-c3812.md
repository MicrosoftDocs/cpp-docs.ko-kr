# <a name="compiler-error-c3812"></a>컴파일러 오류 C3812

' property '는 속성 선언에서 첫 번째 토큰 이어야 합니다.

속성을 선언할 때 키워드는 `__property` 줄의 첫 번째 토큰 이어야 합니다.

C3812는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
