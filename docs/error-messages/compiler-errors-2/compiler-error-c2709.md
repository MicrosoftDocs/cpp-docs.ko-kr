# <a name="compiler-error-c2709"></a>컴파일러 오류 C2709

'identifier': 정식 매개 변수 길이(바이트)가 이전 선언과 다릅니다.

지정된 함수에 대한 호출의 서명은 프로토타입과 다릅니다.
