# <a name="compiler-error-c3817"></a>컴파일러 오류 C3817

' 선언 ': 속성은 함수에만 적용할 수 있습니다.

**`property`** 키워드는 함수 정의에만 적용 될 수 있습니다.

C3817는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
