# <a name="compiler-error-c3817"></a>컴파일러 오류 C3817
'declaration': 속성 함수에만 적용 될 수 있습니다  
  
 `property` 키워드는 함수 정의에 적용 될 수 있습니다.  
  
 C3817은 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
