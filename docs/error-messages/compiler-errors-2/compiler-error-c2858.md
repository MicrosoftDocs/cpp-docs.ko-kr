# <a name="compiler-error-c2858"></a>컴파일러 오류 C2858
명령줄 옵션 ' /Yc (/ Fdfilename)'는 데 사용 하는 미리 컴파일된 헤더와 일치 하지 않는 ' / Fdfilename'  
  
 미리 컴파일된 헤더 사용 하 여 지정 된 프로그램 데이터베이스 ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) 옵션은 이전 만들 미리 컴파일된 헤더에 의해 지정 된 없습니다 ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) 옵션입니다.
