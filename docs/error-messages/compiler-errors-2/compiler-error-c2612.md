# <a name="compiler-error-c2612"></a>컴파일러 오류 C2612
'char' 기본/멤버 이니셜라이저 목록에 잘못 된 후행  
  
 마지막 기본 또는 멤버 이니셜라이저 목록에서 다음에 문자가 표시 합니다.  
  
 다음 샘플에서는 C2612 오류가 생성 됩니다.  
  
```  
// C2612.cpp  
class A {  
public:  
   int i;  
   A( int ia ) : i( ia ) + {};   // C2612  
};  
```
