# <a name="compiler-error-c3832"></a>컴파일러 오류 C3832

' type library ': 형식 라이브러리는 32 비트 포인터에 대해 빌드된 것 처럼 보입니다. ' ptrsize ' 한정자를 변경 하세요.

`ptrsize` [#Import](../../preprocessor/hash-import-directive-cpp.md) 지시문의 특성과 함께 제공 된 명시적 정보가 형식 라이브러리에 있는 컴파일러의 내용과 일치 하지 않습니다.
