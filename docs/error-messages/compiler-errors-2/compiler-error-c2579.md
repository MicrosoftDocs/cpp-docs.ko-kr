# <a name="compiler-error-c2579"></a>컴파일러 오류 C2579

' type ' 형식을 확인할 수 없습니다 (오프셋). 파일에 필요 합니다.

C2579는 항상 C4691 뒤에 나옵니다. 자세한 내용은 [컴파일러 경고 (수준 1) C4691](../../error-messages/compiler-warnings/compiler-warning-level-1-c4691.md)를 참조 하세요.
