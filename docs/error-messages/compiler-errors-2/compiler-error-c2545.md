# <a name="compiler-error-c2545"></a>컴파일러 오류 C2545

' operator ': 오버 로드 된 연산자를 찾을 수 없습니다.

연산자는 제공 된 피연산자와 함께 사용할 수 없습니다. 필요한 피연산자를 사용 하 여 오버 로드 된 연산자를 제공 해야 합니다.

피연산자의 형식이 잘못 된 경우이 오류가 발생할 수 있습니다.

단일 매개 변수를 사용 하는 변환 연산자나 생성자를 정의 하는 경우이 오류를 해결할 수 있습니다.
