# <a name="compiler-error-c3715"></a>컴파일러 오류 C3715

' pointer ': ' class '에 대 한 포인터 여야 합니다.

[`__hook`](../../cpp/hook.md)유효한 클래스를 가리키지 않는 또는에서 포인터를 지정 [`__unhook`](../../cpp/unhook.md) 했습니다. 이 오류를 해결 하려면 **`__hook`** 및 **`__unhook`** 호출에서 유효한 클래스에 대 한 포인터를 지정 해야 합니다.
