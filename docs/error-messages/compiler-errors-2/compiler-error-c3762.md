# <a name="compiler-error-c3762"></a>컴파일러 오류 C3762

' attribute ' 특성을 처리할 수 없습니다.

에서 상속 되는 사용자 정의 특성은 `System.Security.Permissions.SecurityAttribute` 보안 특성을 정의 하는 데 사용 됩니다. 이러한 특성은 정의 된 동일한 어셈블리에서 사용할 수 없습니다.
