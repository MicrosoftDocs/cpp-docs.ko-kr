# <a name="compiler-error-c3762"></a>컴파일러 오류 C3762
' attribute '특성을 처리할 수 없습니다.  
  
 상속 하는 사용자 정의 특성 `System.Security.Permissions.SecurityAttribute` 보안 특성을 정의 하는 데 사용 되 고 있습니다. 이러한 특성이 정의 되어 있는 동일한 어셈블리에 사용할 수 없습니다.
