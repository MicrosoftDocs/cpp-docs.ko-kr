# <a name="compiler-error-c2823"></a>컴파일러 오류 C2823  
  
> typedef 템플릿이 잘못 되었습니다.  
  
서식 파일에서 허용 되지 않는 `typedef` 정의 합니다.  
  
## <a name="example"></a>예  
  
다음 샘플에서는 C2823, 경고가 발생 하 고를 해결 하는 방법을 보여 줍니다.  
  
```cpp  
// C2823.cpp  
template<class T>  
typedef struct x {  
   T i;   // C2823 can't use T, specify data type and delete template  
   int i;   // OK  
} x1;  
```
