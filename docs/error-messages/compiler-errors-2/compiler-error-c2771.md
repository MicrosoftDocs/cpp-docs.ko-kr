# <a name="compiler-error-c2771"></a>컴파일러 오류 C2771

\#가져오기는 전역 또는 네임 스페이스 범위 에서만 사용할 수 있습니다.

`#import` 지시문은 함수나 구조체 등에 사용할 수 없습니다.
