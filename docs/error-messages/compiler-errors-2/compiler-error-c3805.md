# <a name="compiler-error-c3805"></a>컴파일러 오류 C3805

'token': 예기치 않은 토큰을 예상 하거나 '을 (를) ' 또는 식별자

속성을 정의 하는 경우에 잘못 된 토큰이 발견 되었습니다. 잘못 된 토큰을 제거 합니다.
