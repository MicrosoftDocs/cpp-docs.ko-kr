# <a name="compiler-error-c2526"></a>컴파일러 오류 C2526
'identifier1': C 링크 함수 c + + 클래스 'identifier2' 반환할 수 없습니다  
  
 C 링크가 있는 정의 된 함수는 사용자 정의 형식을 반환할 수 없습니다.
