# <a name="compiler-error-c2839"></a>컴파일러 오류 C2839
오버로드된 'operator ->'의 'type' 반환 형식이 잘못되었습니다.  
  
 `->` 연산자는 클래스, 구조체 또는 공용 구조체 또는 하나에 대 한 참조를 반환 해야 합니다.
