# <a name="compiler-error-c2654"></a>컴파일러 오류 C2654

' identifier ': 멤버 함수 외부에서 멤버에 액세스 하려고 했습니다.

선언에서 멤버에 액세스 합니다. 멤버 데이터는 멤버 함수 에서만 액세스할 수 있습니다.

선언에서 변수를 초기화 하려고 할 때이 오류가 발생할 수 있습니다. 이 목적을 위해 생성자를 사용 합니다.
