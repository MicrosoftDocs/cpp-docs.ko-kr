# <a name="compiler-error-c3173"></a>컴파일러 오류 C3173
idl 병합에서 버전이 일치 하지 않습니다.  
  
 이 오류는 개체 파일 포함된 idl 컴파일러의 이전 버전으로 생성 된 때 발생 합니다. 컴파일러가는.obj 파일에 포함 된 idl 내용을 생성 하는 데 동일한 컴파일러 컴파일러 포함된 idl 병합 하는 데 사용 하도록 버전 번호를 인코딩합니다.  
  
 발표 된 최신 버전에서 모든 도구는 Visual c + + 설치를 업데이트 합니다.
