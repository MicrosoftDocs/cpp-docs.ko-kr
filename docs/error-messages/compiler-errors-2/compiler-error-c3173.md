# <a name="compiler-error-c3173"></a>컴파일러 오류 C3173

idl 병합에서 버전이 일치 하지 않습니다.

이 오류는 이전 버전의 컴파일러를 사용 하 여 생성 된 포함 idl이 개체 파일에 포함 된 경우에 발생 합니다. 컴파일러는 버전 번호를 인코딩하여 .obj 파일에 포함 된 idl 콘텐츠를 생성 하는 데 사용 되는 것과 동일한 컴파일러가 포함 idl을 병합 하는 데 사용 되는 컴파일러도 되도록 합니다.

모든 도구가 최신 C++ 릴리스 버전에서 제공 되도록 시각적 개체 설치를 업데이트 합니다.
