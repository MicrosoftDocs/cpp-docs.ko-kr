# <a name="compiler-error-c2692"></a>컴파일러 오류 C2692
'function_name': C 컴파일러에 필요한 완전히 프로토타입화 된 함수가 ' / clr' 옵션  
  
 .NET에 대 한 컴파일을 관리 코드에 C 컴파일러에 ANSI 함수 선언이 있어야 합니다. 또한 경우 매개 변수를 사용 하는 함수를 명시적으로 선언 해야 `void` 매개 변수 형식입니다.
