# <a name="compiler-error-c3501"></a>컴파일러 오류 C3501

ProgID ' progid '에 대해 등록 된 typelib가 없습니다.

지정 된 progid의 클래스 ID에 연결 된 형식 라이브러리가 없습니다. 따라서 `#import` 문에이 progid를 전달할 수 없습니다.
