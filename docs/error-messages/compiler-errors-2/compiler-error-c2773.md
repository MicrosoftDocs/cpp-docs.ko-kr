# <a name="compiler-error-c2773"></a>컴파일러 오류 C2773

\#가져오기 및 #using c + + 컴파일러 에서만 사용할 수 있습니다

C 컴파일러를 인식 하지 않습니다는 `#import` 전처리기 지시문입니다. C + + 소스를 컴파일하십시오. 사용 하 여 [/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) 필요한 경우.
