# <a name="compiler-error-c3631"></a>컴파일러 오류 C3631

'function': 관리되는 이벤트 또는 WinRT 이벤트를 오버로드할 수 없습니다.

관리되는 이벤트 또는 WinRT 이벤트를 오버로드할 수 없습니다.

C3631는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
