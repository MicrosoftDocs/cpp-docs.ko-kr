'function': 관리되는 이벤트 또는 WinRT 이벤트를 오버로드할 수 없습니다.

관리되는 이벤트 또는 WinRT 이벤트를 오버로드할 수 없습니다.

C3631 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
