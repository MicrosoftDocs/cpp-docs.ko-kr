# <a name="compiler-error-c2845"></a>컴파일러 오류 C2845

' operator ':이 형식에는 포인터 산술 연산을 사용할 수 없습니다.

관리 되는 클래스에 대 한 포인터를 증가 시킬 수 없습니다.
