# <a name="compiler-error-c2532"></a>컴파일러 오류 C2532

'identifier': 참조에 대 한 잘못 된 한정자

참조 변경 되었습니다. 다른 개체를 가리키는 참조를 수정할 수 없습니다. 에 대 한 포인터를 대신 사용 합니다.
