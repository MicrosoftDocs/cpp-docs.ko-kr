# <a name="compiler-error-c2532"></a>컴파일러 오류 C2532

' identifier ': 참조에 대 한 한정자가 잘못 되었습니다.

참조가 변경 되었습니다. 참조를 수정 하 여 다른 개체를 참조할 수 없습니다. 대신 포인터를 사용 합니다.
