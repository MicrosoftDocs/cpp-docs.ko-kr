# <a name="compiler-error-c2541"></a>컴파일러 오류 C2541
'delete': 삭제: 포인터가 아닌 개체를 삭제할 수 없습니다  
  
 [삭제](../../cpp/delete-operator-cpp.md) 연산자가 포인터가 아닌 개체에 사용 되었습니다.  
  
 다음 샘플에서는 C2541 오류가 생성 됩니다.  
  
```  
// C2541.cpp  
int main() {  
   int i;  
   delete i;   // C2541 i not a pointer  
  
   // OK  
   int *ip = new int;  
   delete ip;  
}  
```
