# <a name="compiler-error-c3500"></a>컴파일러 오류 C3500

invalid ProgID 'progid'

`#import` 문에 잘못 된 progid가 지정 되었습니다. Windows 레지스트리를 확인 하 여 올바른 progid를 지정 하는지 확인 합니다.
