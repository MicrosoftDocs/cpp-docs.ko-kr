# <a name="compiler-error-c3500"></a>컴파일러 오류 C3500
잘못 된 ProgID 'progid'  
  
 으로 지정 되었지만 잘못 된 progid는 `#import` 문. Windows 레지스트리를 지정 하 고 있는지 유효한 확인 검사 progid입니다.
