# <a name="compiler-error-c2542"></a>컴파일러 오류 C2542
'identifier': 클래스 개체의 초기화에 대 한 생성자가 없습니다  
  
 생성자가 없는 초기화와 일치 하는 매개 변수 목록 사용 합니다.  
  
 이 오류는 잘못 된 매개 변수 초기화에 의해 발생할 수 있습니다.
