# <a name="compiler-error-c2542"></a>컴파일러 오류 C2542

' identifier ': 클래스 개체에 초기화를 위한 생성자가 없습니다.

초기화와 일치 하는 매개 변수 목록을 포함 하는 생성자가 없습니다.

이 오류는 초기화 시 잘못 된 매개 변수로 인해 발생할 수 있습니다.
