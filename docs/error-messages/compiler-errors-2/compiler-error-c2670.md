# <a name="compiler-error-c2670"></a>컴파일러 오류 C2670

' identifier ': 함수 템플릿에서는 ' type ' 형식의 매개 변수 번호를 변환할 수 없습니다.

매개 변수를 필요한 형식으로 변환할 수 없습니다.

명시적 변환을 만들면이 오류가 해결 될 수 있습니다.
