# <a name="compiler-error-c2670"></a>컴파일러 오류 C2670
'identifier': 함수 템플릿을 'type' 형식의 매개 변수 번호를 변환할 수 없습니다  
  
 매개 변수가 필요한 형식으로 변환 하지 못했습니다.  
  
 명시적 변환을 만드는 경우이 오류를 해결할 수 있습니다.
