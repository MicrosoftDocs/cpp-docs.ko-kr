# <a name="compiler-error-c2795"></a>컴파일러 오류 C2795
'super::function' 멤버 함수가 아닙니다.  
  
 사용 하려고 할 때마다이 오류 메시지가 표시 [super](../../cpp/super.md) 멤버 함수 이외의 다른 멤버에 액세스 합니다.
