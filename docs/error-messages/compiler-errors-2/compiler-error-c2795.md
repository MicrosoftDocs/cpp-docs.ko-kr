# <a name="compiler-error-c2795"></a>컴파일러 오류 C2795

' super:: function '은 멤버 함수가 아닙니다.

이 오류 메시지는 [super](../../cpp/super.md) 를 사용 하 여 멤버 함수가 아닌 멤버에 액세스 하려고 할 때마다 나타납니다.
