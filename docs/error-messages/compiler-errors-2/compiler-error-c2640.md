# <a name="compiler-error-c2640"></a>컴파일러 오류 C2640
'identifier': __based 한정자를 참조  
  
 `__based` 한정자는 포인터에만 사용할 수 있습니다.  
  
 다음 샘플에서는 C2640 오류가 생성 됩니다.  
  
```  
// C2640.cpp  
void f(int i) {  
    void *vp;  
    int _based(vp) &vr = I;  // C2640  
}  
```
