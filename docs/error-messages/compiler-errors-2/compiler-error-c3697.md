# <a name="compiler-error-c3697"></a>컴파일러 오류 C3697
'한정자':에이 한정자를 사용할 수 없습니다 ' ^'  
  
 추적 핸들 (^)를 사용할 수 없는 한정자에 적용 되었습니다.  
  
 다음 샘플에서는 C3697 오류가 생성 됩니다.  
  
```  
// C3697.cpp  
// compile with: /clr  
using namespace System;  
int main() {  
   String ^__restrict s;   // C3697  
   String ^ s2;   // OK  
}  
```
