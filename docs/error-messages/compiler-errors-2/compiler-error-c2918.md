# <a name="compiler-error-c2918"></a>컴파일러 오류 C2918

'name': WinRT 형식의 게시된 표면에서는 인덱싱된 속성을 사용할 수 없습니다.

WinRT 형식의 게시된 표면에서는 인덱싱된 속성이 지원되지 않습니다.
