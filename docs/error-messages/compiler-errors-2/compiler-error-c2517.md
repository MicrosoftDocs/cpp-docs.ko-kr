# <a name="compiler-error-c2517"></a>컴파일러 오류 C2517

'identifier': 오른쪽 ': ' 정의 되지 않습니다

범위 결정 연산자의 오른쪽에 있는 식별자 (`::`) 클래스, 구조체 또는 공용 구조체는 왼쪽의 정의 된 멤버 여야 합니다. 없는 클래스, 구조체 또는 공용 구조체 라고 하는 경우 전역 범위를 사용 하 여 오른쪽에 대 한 식별자 선언 되어야 합니다.
