# <a name="compiler-error-c2613"></a>컴파일러 오류 C2613
기본 클래스 목록에 후행 ','를 사용할 수 없습니다.  
  
 기본 클래스 목록에서 마지막 base 뒤에 쉼표가 있습니다.
