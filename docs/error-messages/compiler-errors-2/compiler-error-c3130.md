# <a name="compiler-error-c3130"></a>컴파일러 오류 C3130

내부 컴파일러 오류: 삽입 된 코드 블록을 PDB에 쓰지 못했습니다.

컴파일러에서.pdb 파일에는 삽입 된 코드 블록을 쓰지 못했습니다이 오류가 발생 합니다. 오류에 대 한 가장 일반적인 이유는 디스크 공간이 부족 합니다.
