# <a name="compiler-error-c3130"></a>컴파일러 오류 C3130

내부 컴파일러 오류: 삽입 된 코드 블록을 PDB에 쓰지 못했습니다.

이 오류는 컴파일러가 삽입 된 코드 블록을 .pdb 파일에 쓰지 못한 경우에 발생 합니다. 오류의 가장 일반적인 원인은 디스크 공간이 부족 하다는 것입니다.
