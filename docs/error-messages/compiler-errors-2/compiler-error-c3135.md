# <a name="compiler-error-c3135"></a>컴파일러 오류 C3135

' property ': 속성에는 ' const ' 또는 ' volatile ' 형식을 사용할 수 없습니다.

[Const](../../cpp/const-cpp.md) 및 [volatile](../../cpp/volatile-cpp.md) 키워드는 속성에 사용할 수 없습니다.
