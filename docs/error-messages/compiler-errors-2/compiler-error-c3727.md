# <a name="compiler-error-c3727"></a>컴파일러 오류 C3727

' event ': 관리 되는 이벤트는 대리자에 대 한 포인터인 멤버 함수 또는 데이터 멤버 여야 합니다.

.NET 이벤트는 대리자 형식에 대 한 포인터 여야 합니다.
