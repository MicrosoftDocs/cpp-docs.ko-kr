# <a name="compiler-error-c3167"></a>컴파일러 오류 C3167

.NET Framework를 초기화할 수 없습니다. 설치 되어 있는지 확인 하십시오.

.NET Framework이 컴퓨터에 설치 되어 있지 않습니다. .NET Framework를 설치 합니다.
