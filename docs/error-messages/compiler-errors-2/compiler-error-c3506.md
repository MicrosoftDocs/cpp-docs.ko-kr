# <a name="compiler-error-c3506"></a>컴파일러 오류 C3506

typelib가 없습니다. LIBID 'id'에 대 한 등록

형식 라이브러리를 올바르게 등록 되지 않았습니다. Regsvr32.exe를 사용 하 여 형식 라이브러리를 등록 합니다.
