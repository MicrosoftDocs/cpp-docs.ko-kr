# <a name="compiler-error-c3506"></a>컴파일러 오류 C3506

LIBID ' i d '에 대해 등록 된 typelib가 없습니다.

Typelib가 올바르게 등록 되지 않았습니다. regsvr32.exe를 사용 하 여 형식 라이브러리를 등록 합니다.
