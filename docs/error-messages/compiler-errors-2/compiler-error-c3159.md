# <a name="compiler-error-c3159"></a>컴파일러 오류 C3159

'pointer': 값 형식에 대 한 포인터 배열을 선언할 수 없습니다

값 형식에 대 한 포인터 배열을 선언할 수 없습니다.

C3159 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
