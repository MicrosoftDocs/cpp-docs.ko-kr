# <a name="compiler-error-c2892"></a>컴파일러 오류 C2892
지역 클래스 멤버 템플릿을 가질 수 없습니다.  
  
 템플릿 멤버 함수는 함수에 정의 된 클래스에서 유효 하지 않습니다.  
  
 다음 샘플에서는 C2892 오류가 생성 됩니다.  
  
```  
// C2892.cpp  
int main() {  
   struct local {  
      template<class T>   // C2892  
      void f() {}  
   };  
}  
```
