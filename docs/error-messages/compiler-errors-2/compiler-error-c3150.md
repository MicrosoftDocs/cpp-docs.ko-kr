# <a name="compiler-error-c3150"></a>컴파일러 오류 C3150

'element': 클래스, 인터페이스, 배열 또는 포인터에 'attribute'만 적용할 수 있습니다

`__gc` 클래스, 인터페이스 또는 배열에만 사용할 수 있습니다.

C3150 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
