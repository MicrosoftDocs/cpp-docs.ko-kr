# <a name="compiler-error-c2948"></a>컴파일러 오류 C2948

명시적 인스턴스화 ' 지정자 ' 저장소 클래스 지정자는 특수화에 사용할 수 없습니다.

**`extern`** 이전에 명시적으로 인스턴스화된 템플릿 클래스의 특수화에는와 같은 저장소 클래스 지정자를 사용할 수 없습니다.
