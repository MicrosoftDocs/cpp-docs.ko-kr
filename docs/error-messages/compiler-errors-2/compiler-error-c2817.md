# <a name="compiler-error-c2817"></a>컴파일러 오류 C2817
'operator delete'에 대 한 반환 형식은 'void' 이어야 합니다.  
  
오버 로드 된 [delete 연산자](../../standard-library/new-operators.md#op_delete) 함수는 값을 반환할 수 없습니다.
