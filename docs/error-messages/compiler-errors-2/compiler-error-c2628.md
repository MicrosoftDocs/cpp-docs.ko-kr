# <a name="compiler-error-c2628"></a>컴파일러 오류 C2628
'type1' 뒤에 'type2'은 사용할 수 없습니다 (했는지 확인 한 ';'?)  
  
 세미콜론 없을 수 있습니다.  
  
 다음 샘플에서는 C2628 오류가 생성 됩니다.  
  
```  
// C2628.cpp  
class CMyClass {}  
int main(){}   // C2628 error  
```  
  
 해결 방법:  
  
```  
// C2628b.cpp  
class CMyClass {};  
int main(){}  
```
