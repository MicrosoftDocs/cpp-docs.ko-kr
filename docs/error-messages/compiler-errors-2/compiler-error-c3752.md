# <a name="compiler-error-c3752"></a>컴파일러 오류 C3752

'특성 클래스': 특성을 분류할 수 없습니다 이 컨텍스트에서 'keyword'를 사용 하지 않아야 합니다.

사용자 정의 특성을 올바르게 적용 되었습니다.
