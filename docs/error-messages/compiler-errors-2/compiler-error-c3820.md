# <a name="compiler-error-c3820"></a>컴파일러 오류 C3820

> '*name*': 이니셜라이저는 관리 되어야 합니다.

## <a name="remarks"></a>설명

관리되는 형식에 대한 *name* 이니셜라이저도 관리해야 합니다. 이 오류를 해결 하려면 관리 되는 이니셜라이저 메서드를 지정 합니다.
