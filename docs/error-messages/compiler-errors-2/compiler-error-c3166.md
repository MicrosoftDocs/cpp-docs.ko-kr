# <a name="compiler-error-c3166"></a>컴파일러 오류 C3166

'pointer': 'type'의 구성원으로 내부 __gc 포인터에 대 한 포인터를 선언할 수 없습니다

컴파일러는 잘못 된 포인터 선언을 발견 했습니다 (한 `__nogc` 에 대 한 포인터를 `__gc` 포인터.).

C3166 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
