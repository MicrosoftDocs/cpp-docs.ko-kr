# <a name="compiler-error-c3358"></a>컴파일러 오류 C3358

'symbol': 기호를 찾을 수 없습니다.

필요한 기호를 찾을 수 없습니다.

다음 샘플에서는 C3358을 생성합니다.

```
// C3358.cpp
#define __ATLEVENT_H__ 1   // remove this line to resolve the error
#define _ATL_ATTRIBUTES 1
#include "atlbase.h"
#include "atlcom.h"

[event_receiver(com)]
struct A   // C3358
{
   void func();
};

int main()
{
}
```
