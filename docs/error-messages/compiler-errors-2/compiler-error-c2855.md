# <a name="compiler-error-c2855"></a>컴파일러 오류 C2855

미리 컴파일된 헤더와 일치 하지 않는 ' option' 명령줄 옵션

지정 된 명령줄 옵션이 미리 컴파일된 헤더를 만드는 데 옵션에서 다릅니다. 지정 된 명령줄 옵션을 사용 하 여 미리 컴파일된 헤더를 다시 생성 합니다.
