# <a name="compiler-error-c2669"></a>컴파일러 오류 C2669
멤버 함수를 익명 공용 구조체에 사용할 수 없습니다  
  
[익명 공용 구조체](../../cpp/unions.md#anonymous_unions) 멤버 함수를 사용할 수 없습니다.  
  
## <a name="example"></a>예  
다음 샘플에서는 C2669 오류가 생성 됩니다.  
  
```cpp  
// C2669.cpp  
struct X {  
   union {  
      int i;  
      void f() {   // C2669, remove function  
         i = 0;   
      }  
   };  
};  
```
