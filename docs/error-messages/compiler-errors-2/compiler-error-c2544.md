# <a name="compiler-error-c2544"></a>컴파일러 오류 C2544

' () ' 연산자에 ') '가 필요 합니다.

함수 호출 연산자에 왼쪽 괄호가 없습니다.

이 오류는 매크로 확장으로 인해 발생할 수 있습니다.
