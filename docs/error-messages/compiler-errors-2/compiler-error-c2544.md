# <a name="compiler-error-c2544"></a>컴파일러 오류 C2544

예상 ')' (')' 연산자에 대 한

함수 호출 연산자는 왼쪽된 괄호를 누락 되었습니다.

이 오류는 매크로 확장에 의해 발생할 수 있습니다.
