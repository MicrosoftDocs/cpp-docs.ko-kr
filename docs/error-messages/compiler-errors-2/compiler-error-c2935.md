# <a name="compiler-error-c2935"></a>컴파일러 오류 C2935
'class': type-class-id가 전역 함수로 재정의되었습니다.  
  
 제네릭 또는 템플릿 클래스를 전역 함수로 사용할 수 없습니다.  
  
 중괄호가 짝이 맞지 않는 경우 이 오류가 발생할 수 있습니다.  
  
 다음 샘플에서는 C2935를 생성합니다.  
  
```  
// C2935.cpp  
// compile with: /c  
template<class T>  
struct TC {};   
void TC<int>() {}   // C2935  
  
// OK  
struct TC2 {};   
void TC2() {}  
```  
  
 제네릭을 사용할 때도 C2935가 발생할 수 있습니다.  
  
```  
// C2935b.cpp  
// compile with: /clr /c  
generic<class T>   
ref struct GC { };  
void GC<int>() {}   // C2935  
void GC() {}   // OK  
```
