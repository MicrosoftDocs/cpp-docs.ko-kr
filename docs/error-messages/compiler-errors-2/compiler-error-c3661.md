# <a name="compiler-error-c3661"></a>컴파일러 오류 C3661

명시적인 재정의 목록 재정의할 메서드를 찾을 수 없습니다.

명시적 재정의 형식 이름을 하나 이상 지정 합니다.  그러나 재정의 함수의 시그니처와 일치 하는 형식에 필요한 서명과 함께 함수를 찾지 못했습니다.  재정의 하려는 형식 이름을 기반으로, 재정의 함수의 시그니처와 일치 하는 지정된 된 형식에서 하나 이상의 가상 함수 여야 합니다.

자세한 내용은 [명시적으로 재정의](../../windows/explicit-overrides-cpp-component-extensions.md)합니다.
