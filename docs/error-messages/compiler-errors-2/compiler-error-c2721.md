# <a name="compiler-error-c2721"></a>컴파일러 오류 C2721

' 지정자 ': 연산자 키워드 및 형식 사이에 저장소 클래스 지정 자가 잘못 되었습니다.

사용자 정의 형식 변환은 모든 저장소 클래스에 적용 되므로 형식 변환에서 저장소 클래스를 지정할 수 없습니다.
