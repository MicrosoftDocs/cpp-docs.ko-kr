# <a name="compiler-error-c2760"></a>컴파일러 오류 C2760
구문 오류: 'name1' 예상 하지 'name2'  
  
 캐스팅 연산자는 잘못 된 연산자와 함께 사용 됩니다.  
  
 다음 샘플에서는 C2760 오류가 생성 됩니다.  
  
```  
// C2760.cpp  
class B {};  
class D : public B {};  
  
void f(B* pb) {  
    D* pd1 = static_cast<D*>(pb);  
    D* pd2 = static_cast<D*>=(pb);  // C2760  
    D* pd3 = static_cast<D*=(pb);   // C2760  
}  
```
