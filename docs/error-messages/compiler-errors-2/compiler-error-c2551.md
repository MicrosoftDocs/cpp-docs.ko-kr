# <a name="compiler-error-c2551"></a>컴파일러 오류 C2551
'void *' 형식의 경우 명시적 캐스트가 필요합니다.  
  
 암시적 변환에 의해 비 void 포인터에 `void` 포인터가 할당되었습니다. 명시적 캐스트를 사용해야 합니다.
