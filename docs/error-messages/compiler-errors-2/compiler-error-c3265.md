# <a name="compiler-error-c3265"></a>컴파일러 오류 C3265
관리 되는 '관리 '의 구조는 관리 되지 않는 '관리 되지 않는 구문'를 선언할 수 없습니다.  
  
관리 되지 않는 컨텍스트에서 관리 되는 개체를 포함할 수 없습니다.  
  
다음 샘플에서는 c3265:  
  
```  
// C3265_2.cpp  
// compile with: /clr /LD  
#include <vcclr.h>  
  
ref class A { };  
  
class B  
// try the following line instead  
// ref class B   
{  
   A ^a;   // C3265  
   // or embed the managed handle using gcroot  
   // try the following line instead  
   // gcroot<A^> a;  
};  
```
