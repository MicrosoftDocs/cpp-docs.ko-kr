# <a name="compiler-error-c2577"></a>컴파일러 오류 C2577
'member': 소멸자/종료자는 반환 형식을 가질 수 없습니다  
  
 소멸자 또는 종료자의 값을 반환할 수 `void` 또는 기타 형식입니다. 제거는 `return` 소멸자 정의에서 문입니다.  
  
## <a name="example"></a>예  
 다음 샘플에서는 C2577 오류가 발생 합니다.  
  
```  
// C2577.cpp  
// compile with: /c  
class A {  
public:  
   A() {}  
   ~A(){  
      return 0;   // C2577  
   }  
};  
```
