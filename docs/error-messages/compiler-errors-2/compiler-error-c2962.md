# <a name="compiler-error-c2962"></a>컴파일러 오류 C2962

구문 오류: 'token': 템플릿-클래스 멤버 함수 정의는 '}'로 끝나야 합니다.

토큰으로 인해 템플릿 선언에서 구문 오류가 발생했습니다.

이 오류는 짝이 맞지 않는 구분 기호로 인해 발생할 수 있습니다.
