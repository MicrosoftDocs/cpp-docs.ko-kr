# <a name="compiler-error-c3554"></a>컴파일러 오류 C3554
'decltype'을 다른 형식 지정자와 함께 사용할 수 없습니다.  
  
 형식 지정자로 `decltype()` 키워드를 한정할 수는 없습니다. 예를 들어 다음 코드 조각은 C3554 오류를 생성합니다.  
  
```  
int x;  
unsigned decltype(x); // C3554  
```
