# <a name="compiler-error-c3187"></a>컴파일러 오류 C3187

> ' &#95;&#95;func&#95;&#95; ': 함수의 본문 내 에서만 사용할 수 있습니다.

미리 정의된 매크로는 함수의 본문 밖에서 사용할 수 없습니다.

이 오류를 해결하려면 함수의 본문 안으로 매크로를 이동합니다.
