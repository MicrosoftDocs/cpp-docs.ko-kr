# <a name="compiler-error-c3728"></a>컴파일러 오류 C3728

'event': 이벤트에 raise 메서드가 없습니다.

메타 데이터와 같은 언어를 사용 하 여 만든 C#에 정의 된, 포함 된 해당 하는 클래스 외부에서 발생 하는 이벤트를 허용 하지 않습니다 합니다 [#using](../../preprocessor/hash-using-directive-cpp.md) 지시문 및 시각적 개체 C++ CLR 프로그래밍을 사용 하 여 프로그래밍 이벤트를 발생 시 키 려 했습니다.

C#과 같은 언어에서 개발 된 프로그램에서 이벤트를 발생 시키려면 이벤트를 포함 하는 클래스는 이벤트를 발생 시키는 공용 메서드를 정의 해야 합니다.
