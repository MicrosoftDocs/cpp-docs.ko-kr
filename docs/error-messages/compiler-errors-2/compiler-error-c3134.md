# <a name="compiler-error-c3134"></a>컴파일러 오류 C3134

' value ': 특성 인수 ' attribute '의 값은 ' type ' 형식이 잘못 되었습니다.

값을 특성에 할당 하는 동안 구문 오류가 발생 했습니다.
