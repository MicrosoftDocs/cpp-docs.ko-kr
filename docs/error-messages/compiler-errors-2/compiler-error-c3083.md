# <a name="compiler-error-c3083"></a>컴파일러 오류 C3083
'function': 왼쪽의 기호는 ':: ' 형식 이어야 합니다  
  
 함수를 잘못 호출 했습니다.  
  
## <a name="example"></a>예  
 다음 샘플에서는 C3083 오류가 발생 합니다.  
  
```  
// C3083.cpp  
// compile with: /c  
struct N {  
   ~N();  
};  
  
struct N1 {  
   ~N1();  
};  
  
N::N::~N() {}   // C3083  
N1::~N1() {}   // OK  
```
