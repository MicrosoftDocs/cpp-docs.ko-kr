# <a name="compiler-error-c3356"></a>컴파일러 오류 C3356

'attribute': 정규화된 이름을 사용하여 멀티캐스트 특성을 호출할 수 없습니다.

둘 이상의 프로세스(예: 컴파일러 및 ATL 공급자)에서 처리되는 특성을 잘못 지정했습니다.
