# <a name="compiler-error-c2873"></a>컴파일러 오류 C2873

' symbol ': using 선언에는 기호를 사용할 수 없습니다.

**`using`** 지시문에 [namespace](../../cpp/namespaces-cpp.md) 키워드가 없습니다. 이렇게 하면 컴파일러가 [using 지시문](../../cpp/namespaces-cpp.md#using_directives)이 아닌 [using 선언](../../cpp/using-declaration.md) 으로 코드를 잘못 해석 합니다.
