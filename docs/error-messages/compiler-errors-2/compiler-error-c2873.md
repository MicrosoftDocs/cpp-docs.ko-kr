# <a name="compiler-error-c2873"></a>컴파일러 오류 C2873

'symbol': using 선언에서 기호를 사용할 수 없습니다

A `using` 지시문이 없습니다.을 [네임 스페이스](../../cpp/namespaces-cpp.md) 키워드입니다. 코드를 잘못 해석할 컴파일러가이 [선언을 사용 하 여](../../cpp/using-declaration.md) 아닌 [지시문을 사용 하 여](../../cpp/namespaces-cpp.md#using_directives)입니다.
