# <a name="compiler-error-c2550"></a>컴파일러 오류 C2550
'identifier': 생성자 이니셜라이저 목록은 생성자 정의에 사용할 수  
  
 기본 클래스 이니셜라이저 목록 생성자가 아닌 함수의 정의에 사용 됩니다.  
  
 다음 샘플에서는 C2550 오류가 생성 됩니다.  
  
```  
// C2550.cpp  
// compile with: /c  
class C {  
public:  
   C();  
};  
  
class D : public C {  
public:  
   D();  
   void func();  
};  
  
void D::func() : C() {}  // C2550  
D::D() : C() {}   // OK  
```
