# <a name="compiler-error-c2849"></a>컴파일러 오류 C2849
'소멸자': 인터페이스는 소멸자를 사용할 수 없습니다  
  
 Visual c + + [인터페이스](../../cpp/interface.md) 소멸자를 가질 수 없습니다.  
  
 다음 샘플에서는 C2849 오류가 생성 됩니다.  
  
```  
// C2849.cpp  
// compile with: /c  
__interface C {  
   ~C();   // C2849 destructor not allowed in an interface  
};  
```
