# <a name="compiler-error-c2818"></a>컴파일러 오류 C2818
오버로드된 'operator ->'는 'type' 형식을 통해 재귀적으로 적용됩니다.  
  
 재귀를 포함 하는 클래스 멤버 액세스 연산자의 재정의 `return` 문. 다시 정의 하는 `->` 재귀 연산자를 이동 해야 재귀 루틴의 연산자에서 호출 하는 별도 함수에 함수를 재정의 합니다.
