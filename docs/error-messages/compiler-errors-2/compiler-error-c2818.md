# <a name="compiler-error-c2818"></a>컴파일러 오류 C2818

응용 프로그램의 'operator->' 오버 로드 된 'type' 형식에서 재귀적으로는

재귀를 포함 하는 클래스 멤버 액세스 연산자의 재정의 `return` 문입니다. 다시 정의 하는 `->` 재귀를 사용 하 여 연산자를 이동 해야 재귀 루틴을 별도 함수 호출 연산자에서 함수를 재정의 합니다.
