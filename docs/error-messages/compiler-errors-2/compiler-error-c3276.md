# <a name="compiler-error-c3276"></a>컴파일러 오류 C3276

' keyword ': 종료를 처리 하는 동안 __finally/finally 블록 밖으로 점프 하면 정의 되지 않은 동작이 발생 합니다.

이 오류는 [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) 경고와 같습니다. 그러나/clr을 사용 하는 경우에는 [warning](../../preprocessor/warning.md) pragma를 사용 하 여이 상태를 비활성화할 수 없습니다.
