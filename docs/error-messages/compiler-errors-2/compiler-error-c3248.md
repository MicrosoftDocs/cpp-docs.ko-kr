# <a name="compiler-error-c3248"></a>컴파일러 오류 C3248

'function1' : '__sealed'로 선언된 함수를 'function2'로 재정의할 수 없습니다.

파생 클래스에서 **__sealed** 가상 메서드를 재정의하려고 했습니다.

C 3248는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
