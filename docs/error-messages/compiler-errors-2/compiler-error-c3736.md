# <a name="compiler-error-c3736"></a>컴파일러 오류 C3736
'event': 메서드여야 또는 관리 되는 이벤트의 경우 데이터 멤버  
  
 기본 모드와 COM 이벤트 메서드여야 합니다. .NET 이벤트 데이터 멤버를 수도 있습니다.  
  
 다음 샘플에서는 C3736 오류가 생성 됩니다.  
  
```  
// C3736.cpp  
struct A {  
   __event int e();  
};  
  
struct B {  
   int f;   // C3736  
   // The following line resolves the error.  
   // int f();  
   B(A* a) {  
      __hook(&A::e, a, &B::f);  
   }  
};  
  
int main() {  
}  
```
