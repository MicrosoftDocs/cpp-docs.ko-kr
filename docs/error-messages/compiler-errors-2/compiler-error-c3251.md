# <a name="compiler-error-c3251"></a>컴파일러 오류 C3251

값 형식 인스턴스의 기본 클래스 메서드를 호출할 수 없습니다.

다음 오류는 `GetClass` 가 `Microsoft.Runtime.Object`가 아닌 `Microsoft.Runtime.Integer4`의 멤버이기 때문에 발생합니다.
