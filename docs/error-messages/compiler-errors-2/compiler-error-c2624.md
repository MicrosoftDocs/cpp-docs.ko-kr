# <a name="compiler-error-c2624"></a>컴파일러 오류 C2624
'extern' 변수를 선언 하는 로컬 클래스를 사용할 수 없습니다.  
  
 지역 클래스 또는 구조체 선언에 사용할 수 없습니다 `extern` 변수입니다.  
  
 다음 샘플에서는 C2624 오류가 생성 됩니다.  
  
```  
// C2624.cpp  
int main() {  
   struct C {};  
   extern C ac;   // C2624  
}  
```
