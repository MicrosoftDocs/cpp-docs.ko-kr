# <a name="compiler-error-c3603"></a>컴파일러 오류 C3603

' Symbol ': ' Type ' 형식은 아직 지원 되지 않습니다.

관리 되는 개체의 .NET 런타임에서 아직 지원 되지 않는 데이터 형식을 사용 하려고 했습니다.
