# <a name="compiler-error-c2634"></a>컴파일러 오류 C2634
'&class::member' : 참조 멤버에 대한 포인터가 잘못되었습니다.  
  
 참조 멤버에 대 한 포인터 선언 됩니다.  
  
 다음 샘플에서는 C2634 오류가 생성 됩니다.  
  
```  
// C2634.cpp  
int mem;  
struct S {  
   S() : rf(mem) { }  
   int &rf;  
};  
int (S::*pdm) = &S::rf;   // C2634  
```
