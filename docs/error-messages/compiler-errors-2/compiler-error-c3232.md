# <a name="compiler-error-c3232"></a>컴파일러 오류 C3232
'param': 정규화된 이름에는 제네릭 형식 매개 변수를 사용할 수 없습니다.  
  
 제네릭 형식 매개 변수가 잘못 사용되었습니다.  
  
 다음 샘플에서는 C3232를 생성합니다.  
  
```  
// C3232.cpp  
// compile with: /clr  
generic <class T>  
ref class C {  
   typename T::TYPE t;   // C3232  
};  
```
