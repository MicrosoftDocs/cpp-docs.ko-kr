# <a name="compiler-error-c3396"></a>컴파일러 오류 C3396

'attribute': 'namespace'에 사용자 지정 특성이 없습니다.

C3396은 최신 공용 언어 런타임을 사용하지 않고 있다는 것을 나타냅니다.  해결하려면 CLR을 다시 설치합니다.
