# <a name="compiler-error-c2602"></a>컴파일러 오류 C2602
'class::Identifier' 'class'의 기본 클래스의 구성원이 아닙니다.  
  
 `Identifier`클래스는 기본 클래스에서 상속 된 멤버가 없기 때문에 액세스할 수 없습니다.  
  
 다음 샘플에서는 C2602 오류가 생성 됩니다.  
  
```  
// C2602.cpp  
// compile with: /c  
struct X {  
   int x;  
};  
struct A {  
   int a;  
};  
struct B : public A {  
   X::x;   // C2602 B is not derived from X  
   A::a;   // OK  
};  
```
