# <a name="compiler-error-c2792"></a>컴파일러 오류 C2792
'super':이 키워드 뒤에 야 ':: '  
  
 키워드 다음에 올 수 있는 토큰 `__super` 은 `::`합니다.  
  
 다음 샘플에서는 C2792 오류가 생성 됩니다.  
  
```  
// C2792.cpp  
struct B {  
   void mf();  
};  
  
struct D : B {  
   void mf() {  
      __super.();   // C2792  
  
      // try the following line instead  
      // __super::mf();  
   }  
};  
```
