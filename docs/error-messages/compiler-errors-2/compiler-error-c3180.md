# <a name="compiler-error-c3180"></a>컴파일러 오류 C3180

' type name ': 이름이 메타 데이터 한계인 ' limit ' 자를 초과 합니다.

컴파일러가 메타 데이터에서 관리 되는 형식의 이름을 잘랐습니다. 잘림을 수행 하면 `#using` 지시문 (또는 다른 언어의 경우)에서 형식을 사용할 수 없게 됩니다.

형식 이름 제한에는 모든 네임 스페이스 한정자가 포함 됩니다.
