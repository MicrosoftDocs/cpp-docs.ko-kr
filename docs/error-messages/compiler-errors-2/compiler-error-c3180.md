# <a name="compiler-error-c3180"></a>컴파일러 오류 C3180
'type name': 이름이 메타 데이터 한계인 '한계' 자 초과  
  
 컴파일러는 메타 데이터에서 관리 되는 형식에 대 한 이름을 잘립니다. 잘라낼 수 없게 됩니다 종류는 `#using` 지시문 (또는 다른 언어에 해당).  
  
 형식 이름 제한에는 모든 네임 스페이스 한정자가 포함 됩니다.
