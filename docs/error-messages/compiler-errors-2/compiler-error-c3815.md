# <a name="compiler-error-c3815"></a>컴파일러 오류 C3815

' get_accessor ' 메서드의 반환 형식은 setter의 마지막 매개 변수 형식과 일치 해야 합니다.

속성을 선언할 때 메서드의 반환 값은 `get_accessor` set 접근자 메서드 선언의 마지막 매개 변수와 일치 해야 합니다.

C3815는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
