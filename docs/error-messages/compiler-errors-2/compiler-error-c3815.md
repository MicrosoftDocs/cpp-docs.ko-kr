# <a name="compiler-error-c3815"></a>컴파일러 오류 C3815

'get_accessor' 메서드의 반환 형식은 setter의 마지막 매개 변수 형식과 일치 해야

속성의 반환 값을 선언 하는 경우는 `get_accessor` 메서드 선언의 set 접근자 메서드에 마지막 매개 변수와 일치 해야 합니다.

C3815 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
