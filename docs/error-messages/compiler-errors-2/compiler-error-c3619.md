# <a name="compiler-error-c3619"></a>컴파일러 오류 C3619
관리되는 형식 또는 WinRT 형식 내에 템플릿을 선언할 수 없습니다.  
  
관리되는 클래스나 WinRT 클래스 또는 인터페이스에서 클래스 템플릿을 사용할 수 없습니다.  
  
C3619은 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
