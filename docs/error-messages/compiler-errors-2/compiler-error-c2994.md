# <a name="compiler-error-c2994"></a>컴파일러 오류 C2994

템플릿 매개 변수 목록에 명명되지 않은 클래스가 있습니다.

**`class`** 클래스 이름을 지정 하지 않고 키워드를 템플릿 인수로 사용할 수 없습니다.
