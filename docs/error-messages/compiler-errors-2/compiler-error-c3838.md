# <a name="compiler-error-c3838"></a>컴파일러 오류 C3838
명시적으로 'type'에서 상속할 수 없습니다.  
  
 지정 된 `type` 클래스에서 기본 클래스로 사용할 수 없습니다.  
  
## <a name="example"></a>예
 다음 샘플에서는 C3838 오류가 생성 됩니다.  
  
```  
// C3838a.cpp  
// compile with: /clr /c  
public ref class B : public System::Enum {};   // C3838  
```
