# <a name="compiler-error-c3874"></a>컴파일러 오류 C3874

' function '의 반환 형식은 ' t r u e ' 대신 ' i n t ' 여야 합니다.

함수에 컴파일러에서 예상한 반환 형식이 없습니다.

다음 샘플에서는 C3874를 생성 합니다.

```cpp
// C3874b.cpp
double main()
{   // C3874
}
```
