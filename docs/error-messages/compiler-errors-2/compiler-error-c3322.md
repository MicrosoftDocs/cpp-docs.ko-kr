# <a name="compiler-error-c3322"></a>컴파일러 오류 C3322
'property': 'attribute' 특성에 대해 올바른 속성이 아닙니다.  
  
 잘못되었거나 인식할 수 없는 속성(또는 매개 변수)이 특성에 전달되었습니다. 특성에 대한 설명서를 참조하세요.
