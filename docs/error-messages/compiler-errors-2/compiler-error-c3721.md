# <a name="compiler-error-c3721"></a>컴파일러 오류 C3721

'서명': 이벤트에 대 한 호환 되지 않는 서명

이벤트를 올바르게 선언 되었습니다. 자세한 내용은 [__event](../../cpp/event.md)합니다.

C3721 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
