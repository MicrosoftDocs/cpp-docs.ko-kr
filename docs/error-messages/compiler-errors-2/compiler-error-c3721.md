# <a name="compiler-error-c3721"></a>컴파일러 오류 C3721

' signature ': 이벤트의 서명이 호환 되지 않습니다.

이벤트가 잘못 선언 되었습니다. 자세한 내용은 [__event](../../cpp/event.md)를 참조 하세요.

C3721는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
