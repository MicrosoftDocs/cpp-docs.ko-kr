# <a name="compiler-error-c2586"></a>컴파일러 오류 C2586
잘못 된 사용자 정의 변환 구문이: 간접 참조가 잘못 되었습니다  
  
 변환 연산자의 간접 참조는 허용 되지 않습니다.  
  
 다음 샘플에서는 C2586 오류가 생성 됩니다.  
  
```  
// c2586.cpp  
// compile with: /c  
struct C {  
   * operator int();   // C2586  
   operator char();   // OK  
};  
```
