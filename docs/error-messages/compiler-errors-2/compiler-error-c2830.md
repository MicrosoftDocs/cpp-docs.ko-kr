# <a name="compiler-error-c2830"></a>컴파일러 오류 C2830

' operator n e w '에 대 한 배치 매개 변수만 기본값을 사용할 수 있습니다.

[New 연산자](../../standard-library/new-operators.md#op_new) 에 대 한 표준 정식 매개 변수는 기본값을 가질 수 없습니다. 사용자 정의 배치 매개 변수만 기본값을 지정할 수 있습니다.
