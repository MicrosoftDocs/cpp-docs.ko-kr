# <a name="compiler-error-c2734"></a>컴파일러 오류 C2734
'identifier': 경우 하지 extern const 개체를 초기화 합니다  
  
 식별자 선언 된 `const` 초기화 되지 않지만 또는 `extern`합니다.  
  
 다음 샘플에서는 C2734 오류가 생성 됩니다.  
  
```  
// C2734.cpp  
const int j;   // C2734  
extern const int i;   // OK, declared as extern  
```
