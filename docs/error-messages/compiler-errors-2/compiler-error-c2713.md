# <a name="compiler-error-c2713"></a>컴파일러 오류 C2713

> 함수 별로 허용 되는 한 가지 형태의 예외 처리

동일한 함수에서 구조적 예외 처리 ( **`__try`** / **`__except`** )와 c + + 예외 처리 ( **`try`** / **`catch`** )를 사용할 수 없습니다.
