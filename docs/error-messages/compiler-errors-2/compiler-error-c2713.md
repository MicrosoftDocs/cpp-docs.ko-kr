# <a name="compiler-error-c2713"></a>컴파일러 오류 C2713
한 가지 형태의 함수 당 허용 되는 예외 처리  
  
 구조적된 예외 처리를 사용할 수 없습니다 (`__try`/`__except`) 및 c + + 예외 처리 (`try`/`catch`) 같은 함수입니다.
