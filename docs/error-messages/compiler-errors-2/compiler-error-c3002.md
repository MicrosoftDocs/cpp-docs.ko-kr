# <a name="compiler-error-c3002"></a>컴파일러 오류 C3002
'name1 name2': OpenMP 지시문 이름이 여러 개 있습니다.  
  
 여러 지시문 이름은 허용되지 않습니다.  
  
 다음 샘플에서는 C3002를 생성합니다.  
  
```  
// C3002.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp parallel single   // C3002  
}  
```
