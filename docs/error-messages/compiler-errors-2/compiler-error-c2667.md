# <a name="compiler-error-c2667"></a>컴파일러 오류 C2667

' function ': 변환할 수 있는 오버 로드가 없습니다.

오버 로드 된 함수 호출이 모호 하며 확인할 수 없습니다.

오버 로드 된 함수 중 하나에 대 한 함수 호출의 실제 매개 변수를 일치 시키는 데 필요한 변환은 오버 로드 된 다른 모든 함수에 필요한 변환 보다 엄격 하 게 향상 되어야 합니다.
