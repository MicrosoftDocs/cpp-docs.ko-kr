# <a name="compiler-error-c3632"></a>컴파일러 오류 C3632

' event ': 생성에 대 한 이벤트 스타일이 잘못 되었습니다.

모든 구문에서는 [__event](../../cpp/event.md) 선언을 사용할 수 없습니다.

C3632는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
