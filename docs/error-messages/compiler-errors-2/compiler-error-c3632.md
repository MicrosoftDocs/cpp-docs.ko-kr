# <a name="compiler-error-c3632"></a>컴파일러 오류 C3632
'event': 잘못 된 스타일 구문에 대 한 이벤트  
  
 [__event](../../cpp/event.md) 선언이 모든 구문에서 올바르지 않습니다.  
  
 C3632은 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
