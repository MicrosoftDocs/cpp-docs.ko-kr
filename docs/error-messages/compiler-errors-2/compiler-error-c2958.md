# <a name="compiler-error-c2958"></a>컴파일러 오류 C2958

' 위치 '에 있는 왼쪽 구분 기호가 올바르게 일치 하지 않습니다.

구분 기호가 제대로 일치 하지 않습니다.

이 오류는 함수 템플릿 선언에서 괄호가 일치 하지 않기 때문에 발생할 수 있습니다.
