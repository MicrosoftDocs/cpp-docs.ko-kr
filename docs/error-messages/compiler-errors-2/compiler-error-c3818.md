# <a name="compiler-error-c3818"></a>컴파일러 오류 C3818
배열 속성 선언은 'property1' '속성 2' 인덱스 속성 오버 로드 하지 않습니다.  
  
 오버 로드 불가능 속성에 대 한 인덱서를 하나 하 고 다른 하나는 배열 속성 합니다. 
  
 C3818은 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
