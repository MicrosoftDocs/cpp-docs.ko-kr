# <a name="compiler-error-c3818"></a>컴파일러 오류 C3818

배열 속성 선언은 'property1' 인덱스 속성 'property2' 오버 로드 하지 않습니다.

오버 로드 하나 인덱서 이며 다른 배열 속성 때 속성에 대 한 가능한 아닙니다.

C3818 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
