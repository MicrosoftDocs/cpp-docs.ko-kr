# <a name="compiler-error-c2904"></a>컴파일러 오류 C2904
'identifier': 이름이 현재 범위의 템플릿으로 이미 사용되었습니다.  
  
 코드 이름이 중복되지 않았는지 확인합니다.  
  
 다음 샘플에서는 C2904를 생성합니다.  
  
```  
// C2904.cpp  
// compile with: /c  
void X();  // X is declared as a function  
template<class T> class X{};  // C2904  
```
