# <a name="compiler-error-c2651"></a>컴파일러 오류 C2651

' 데이터 형식 ': ' operator ' 왼쪽은 클래스, 구조체 또는 공용 구조체 여야 합니다.

클래스와 같이 템플릿 매개 변수를 사용 하려면 정수 계열 형식 대신 클래스를 사용 하 여 클래스 템플릿을 특수화 합니다.
