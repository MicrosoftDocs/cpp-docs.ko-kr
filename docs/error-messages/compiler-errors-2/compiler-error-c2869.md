# <a name="compiler-error-c2869"></a>컴파일러 오류 C2869
'name': 이미 네임 스페이스로 정의  
  
 네임 스페이스도 이미 사용 되는 이름을 다시 사용할 수 없습니다.  
  
 다음 샘플에서는 C2869 오류가 생성 됩니다.  
  
```  
// C2869.cpp  
// compile with: /c  
namespace A { int i; };  
  
class A {};   // C2869, A is already used  
```
