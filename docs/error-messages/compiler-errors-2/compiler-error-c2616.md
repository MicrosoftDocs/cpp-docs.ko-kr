# <a name="compiler-error-c2616"></a>컴파일러 오류 C2616

' conversion ': lvalue가 아닌 ' type1 '을 (를) const가 아닌 ' type2 ' (으)로 암시적으로 변환할 수 없습니다.

참조는 lvalue가 아닌에서 초기화할 수 없습니다.

이 오류는 ANSI 호환성 ([/za](../../build/reference/za-ze-disable-language-extensions.md))에서 발생 하 고 Microsoft 확장 ( **/ze**)에서 경고를 발생 하는 경우에 발생 합니다.
