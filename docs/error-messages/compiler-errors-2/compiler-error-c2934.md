# <a name="compiler-error-c2934"></a>컴파일러 오류 C2934
'class': type-class-id가 'identifier'의 중첩된 'item'으로 다시 정의되었습니다.  
  
 제네릭 또는 템플릿 클래스를 중첩된 항목으로 사용할 수 없습니다.
