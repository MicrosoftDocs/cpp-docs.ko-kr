# <a name="compiler-error-c3800"></a>컴파일러 오류 C3800

' 선언 ': 속성과 이벤트를 혼합할 수 없습니다.

구문을 속성과 이벤트 둘 다로 선언할 수 없습니다.

C3800는 사용 되지 않는 컴파일러 옵션 **/clr: oldSyntax** 를 사용 하는 경우에만 연결할 수 있습니다.
