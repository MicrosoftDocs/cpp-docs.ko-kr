# <a name="compiler-error-c2510"></a>컴파일러 오류 C2510

'identifier': '::' 왼쪽은 클래스/구조체/공용 구조체이어야 합니다.

클래스, 구조체 또는 공용 구조체 이름은 범위 결정 연산자(`::`) 연산자의 왼쪽에 표시되어야 합니다.
