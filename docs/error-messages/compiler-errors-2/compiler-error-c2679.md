# <a name="compiler-error-c2679"></a>컴파일러 오류 C2679

이항 'operator': 'type' 형식의 오른쪽 피연산자로 사용 하는 연산자 없습니다 (또는 허용 된 변환이)

연산자를 사용하려면 지정된 형식에 대해 오버로드하거나 연산자가 정의된 형식으로의 변환을 정의해야 합니다.

다음 샘플에서는 C2679를 생성합니다.

```
// C2679.cpp
class C {
public:
   C();   // no constructor with an int argument
} c;

class D {
public:
   D(int) {}
   D(){}
} d;

int main() {
   c = 10;   // C2679
   d = 10;   // OK
}
```
