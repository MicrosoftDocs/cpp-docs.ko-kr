# <a name="compiler-error-c2223"></a>컴파일러 오류 C2223

'->identifier ' 왼쪽은 구조체/공용 구조체를 가리켜야 합니다.

`->` 왼쪽의 피연산자가 클래스, 구조체 또는 공용 구조체에 대한 포인터가 아닙니다.

이 오류는 정의 되지 않은 변수 (따라서 형식) 인 왼쪽 피연산자에 의해 발생할 수 있습니다 **`int`** .
