# <a name="compiler-error-c2205"></a>컴파일러 오류 C2205

' identifier ': 블록 범위를 사용 하 여 extern 변수를 초기화할 수 없습니다.

**`extern`** 함수에서 변수를 초기화할 수 없습니다.
