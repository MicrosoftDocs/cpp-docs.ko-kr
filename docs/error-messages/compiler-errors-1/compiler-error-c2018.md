# <a name="compiler-error-c2018"></a>컴파일러 오류 C2018

'hexnumber' 문자를 인식할 수 없습니다.

소스 파일에 16 진수로 식별 되는 예기치 않은 ASCII 문자가 포함 되어 있습니다. 오류를 해결 하려면 문자를 제거 합니다.
