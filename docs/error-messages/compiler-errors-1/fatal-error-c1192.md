# <a name="fatal-error-c1192"></a>심각한 오류 C1192

\#' file '에서 사용 하지 못했습니다.

MSIL (Microsoft 중간 언어) 형식의 파일만 [#using](../../preprocessor/hash-using-directive-cpp.md) 지시문에 전달할 수 있습니다. [/Clr](../../build/reference/clr-common-language-runtime-compilation.md) 컴파일러 옵션을 사용 하면 MSIL 출력 파일을 만들 수 있습니다. 다른 Visual Studio 언어도 MSIL 파일도 생성 합니다.
