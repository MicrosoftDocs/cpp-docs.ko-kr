# <a name="compiler-error-c2371"></a>컴파일러 오류 C2371
'identifier': 재정의. 기본 형식이 다릅니다.  
  
 식별자가 이미 선언되었습니다.  
  
 다음 샘플에서는 C2371을 생성합니다.  
  
```  
// C2371.cpp  
int main() {  
   int i;  
   float i;   // C2371, redefinition  
   float f;   // OK  
}  
```
