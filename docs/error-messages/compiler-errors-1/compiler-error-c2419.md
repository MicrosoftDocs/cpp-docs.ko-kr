# <a name="compiler-error-c2419"></a>컴파일러 오류 C2419
'context'에서 0으로 나머지 연산을 수행했습니다.  
  
 `MOD` 연산자 오른쪽의 매개 변수가 이 컨텍스트에서 0입니다.
