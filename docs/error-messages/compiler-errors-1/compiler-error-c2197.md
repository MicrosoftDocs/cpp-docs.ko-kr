# <a name="compiler-error-c2197"></a>컴파일러 오류 C2197

'function': 호출에 매개 변수가 너무 많습니다.

컴파일러가 함수 호출 매개 변수를 너무 많이 발견했거나 잘못된 함수 선언을 발견했습니다.

다음 샘플에서는 C2197을 생성합니다.

```c
// C2197.c
// compile with: /Za /c
void func( int );
int main() {
   func( 1, 2 );   // C2197 two actual parameters
   func( 2 );   // OK
}
```
