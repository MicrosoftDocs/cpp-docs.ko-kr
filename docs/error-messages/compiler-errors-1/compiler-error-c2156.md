# <a name="compiler-error-c2156"></a>컴파일러 오류 C2156
pragma는 함수 외부에 있어야 합니다.  
  
 전역 수준(함수 본문 외부)에서 지정해야 하는 pragma가 함수 내에 있습니다.  
  
 다음 샘플에서는 C2156을 생성합니다.  
  
```  
// C2156.cpp  
#pragma optimize( "l", on )   // OK  
int main() {  
   #pragma optimize( "l", on )   // C2156  
}  
```
