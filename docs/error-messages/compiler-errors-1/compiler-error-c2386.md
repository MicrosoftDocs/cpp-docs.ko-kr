# <a name="compiler-error-c2386"></a>컴파일러 오류 C2386
'symbol': 이름이 같은 기호가 현재 범위에 이미 있습니다.  
  
 네임스페이스 별칭을 만들려고 했지만 선택한 이름이 이미 있습니다.  
  
 다음 샘플에서는 C2386을 생성합니다.  
  
```  
// C2386.cpp  
namespace A {  
   int k;  
}  
  
int i;  
namespace i = A;   // C2386, i already exists  
```
