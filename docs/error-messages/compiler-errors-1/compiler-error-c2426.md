# <a name="compiler-error-c2426"></a>컴파일러 오류 C2426

' token ': ' context '에 잘못 된 연산자가 있습니다.

토큰은이 컨텍스트에서 연산자로 사용할 수 없습니다. 예를 들어 인덱스 연산자는 중첩 될 수 없습니다.
