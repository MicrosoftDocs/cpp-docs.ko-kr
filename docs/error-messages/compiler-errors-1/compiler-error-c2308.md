# <a name="compiler-error-c2308"></a>컴파일러 오류 C2308

일치 하지 않는 문자열을 연결

두 너비와 비 와이드 문자 문자열 연결에 대해 지정 되었습니다. 비 와이드 문자 문자열을 와이드 문자 문자열을 연결할 수 없습니다.
