# <a name="compiler-error-c2308"></a>컴파일러 오류 C2308

일치 하지 않는 문자열 연결

연결에 대 한 와이드 문자열과 와이드 문자열이 모두 지정 되었습니다. 와이드 문자열과 와이드 문자열을 연결할 수 없습니다.
