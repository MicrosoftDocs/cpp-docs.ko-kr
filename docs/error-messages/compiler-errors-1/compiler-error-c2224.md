# <a name="compiler-error-c2224"></a>컴파일러 오류 C2224
'.identifier' 왼쪽에는 구조체/공용 구조체 형식이 있어야 합니다.  
  
 마침표 (.) 왼쪽에 있는 피연산자는 클래스, 구조체 또는 공용 구조체입니다.  
  
 이 오류는 정의 되지 않은 변수는 왼쪽된 피연산자가 발생할 수 있습니다 (따라서 입력 `int`).
