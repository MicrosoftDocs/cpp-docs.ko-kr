# <a name="fatal-error-c1108"></a>심각한 오류 C1108

> DLL을 찾을 수 없습니다. '*dll 이름*'

지정 된 DLL (*dll 이름*) 경로에서 찾을 수 없습니다.

이 오류를 해결 하려면 시각적 개체를 다시 설치 C++ 또는 컴퓨터에 설치에서 해당.dll 파일에 복사 합니다.
