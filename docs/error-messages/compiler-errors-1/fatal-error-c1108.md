# <a name="fatal-error-c1108"></a>심각한 오류 C1108

> '*dll 이름*' dll을 찾을 수 없습니다.

지정 된 DLL (*dll 이름*)을 경로에서 찾을 수 없습니다.

이 오류를 해결 하려면 Visual C++을 다시 설치 하거나 설치 프로그램에서 해당 .dll 파일을 컴퓨터에 복사 합니다.
