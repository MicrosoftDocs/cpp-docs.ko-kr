# <a name="compiler-error-c2151"></a>컴파일러 오류 C2151

둘 이상의 언어 특성

함수에 호출 규칙을 지정 하는 둘 이상의 키워드 (`__cdecl`, `__stdcall`또는 `__fastcall`)가 있습니다.
