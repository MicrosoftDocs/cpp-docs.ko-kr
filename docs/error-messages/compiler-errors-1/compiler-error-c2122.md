# <a name="compiler-error-c2122"></a>컴파일러 오류 C2122

' identifier ': 이름 목록의 프로토타입 매개 변수가 잘못 되었습니다.

매개 변수가 올바른 형식이 아닙니다. ANSI C는 사용자 정의 형식을 지원 하지 않습니다.
