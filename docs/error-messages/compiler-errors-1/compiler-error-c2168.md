# <a name="compiler-error-c2168"></a>컴파일러 오류 C2168
'function': 내장 함수에 대한 실제 매개 변수가 너무 적습니다.  
  
 `intrinsic` 함수에 대한 참조에 매개 변수가 너무 적습니다.
