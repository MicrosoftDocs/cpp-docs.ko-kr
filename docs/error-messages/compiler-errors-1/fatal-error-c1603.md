# <a name="fatal-error-c1603"></a>심각한 오류 C1603
인라인 어셈블리 분기 대상이 'number' 바이트 만큼 범위를 벗어났습니다  
  
 계산된 간격 JCXZ 또는 JECXZ 명령 및 지정 된 대상 레이블을 128 바이트 보다 큽니다. 레이블을 더 가까운를 명령에 있도록 코드를 업데이트 합니다.
