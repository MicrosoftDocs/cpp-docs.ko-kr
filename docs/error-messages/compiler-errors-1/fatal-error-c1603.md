# <a name="fatal-error-c1603"></a>심각한 오류 C1603

인라인 어셈블리 분기 대상이 ' number ' 바이트 만큼 범위를 벗어났습니다.

JCXZ 또는 JECXZ 명령과 지정 된 대상 레이블 사이의 계산 된 거리가 128 바이트 보다 큽니다. 레이블이 명령에 가까이 있도록 코드를 업데이트 합니다.
