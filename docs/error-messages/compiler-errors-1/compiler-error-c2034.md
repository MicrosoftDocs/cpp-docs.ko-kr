# <a name="compiler-error-c2034"></a>컴파일러 오류 C2034
'identifier': 비트 수에 비해 너무 작음을 비트 필드의 유형  
  
 비트 필드 선언에는 비트 수는 기본 형식의 크기를 초과합니다.  
  
 다음 샘플에서는 C2034 오류가 생성 됩니다.  
  
```  
// C2034.cpp  
struct A {  
   char test : 9;   // C2034, char has 8 bits  
};  
```  
  
 해결 방법:  
  
```  
// C2034b.cpp  
// compile with: /c  
struct A {  
   char test : 8;  
};  
```
