# <a name="fatal-error-c1107"></a>심각한 오류 C1107

' file ' 어셈블리를 찾을 수 없습니다./AI를 사용 하거나 LIBPATH 환경 변수를 설정 하 여 어셈블리 검색 경로를 지정 하십시오.

컴파일러에서 찾을 수 없는 [#using](../../preprocessor/hash-using-directive-cpp.md) 지시문에 메타 데이터 파일이 전달 되었습니다.
