# <a name="compiler-error-c2332"></a>컴파일러 오류 C2332
'typedef': 태그 이름이 없습니다.  
  
 컴파일러는 불완전 한 형식 정을 찾았습니다.  
  
 다음 샘플에서는 C2332 오류가 생성 됩니다.  
  
```  
// C2332.cpp  
// compile with: /c  
struct S {  
   int i;  
};  
  
typedef struct * pS;   // C2332  
typedef struct S* pS;   // OK  
  
int get_S_i(pS p) {  
   return p->i;  
}  
```
