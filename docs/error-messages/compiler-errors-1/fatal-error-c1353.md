# <a name="fatal-error-c1353"></a>심각한 오류 C1353

메타데이터 작업이 실패했습니다. 런타임이 설치되지 않았거나 버전이 일치하지 않습니다.

CLR 또는 Visual Studio를 다시 설치합니다.
