# <a name="fatal-error-c1082"></a>심각한 오류 C1082

filetype 파일을 닫을 수 없습니다. 'file': message

"잘못된 파일 번호"라는 메시지가 표시되면 백그라운드에서 컴파일하는 동안 파일이 전경에서 닫혀 있을 수 있습니다.
