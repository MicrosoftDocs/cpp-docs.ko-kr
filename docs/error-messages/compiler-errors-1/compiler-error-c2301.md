# <a name="compiler-error-c2301"></a>컴파일러 오류 C2301

'->~identifier' 왼쪽은 클래스/구조체/공용 구조체를 가리켜야 합니다.

`->` 연산자 왼쪽에 있는 식이 클래스, 구조체 또는 공용 구조체에 대한 포인터로 계산되지 않습니다.
