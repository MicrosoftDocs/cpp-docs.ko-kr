# <a name="compiler-error-c2219"></a>컴파일러 오류 C2219

구문 오류: 형식 한정자는 ' * ' 뒤에와 야 합니다.

허용 되지 않는 형식 한정자 (`const` 또는 `volatile`)가 표시 됩니다.
