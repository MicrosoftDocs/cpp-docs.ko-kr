# <a name="compiler-error-c2320"></a>컴파일러 오류 C2320

> 액세스 지정자 '*지정자*' 다음에 ': '이 필요 합니다.

키워드 **`public`** , **`protected`** 또는은 **`private`** 뒤에 콜론이와 야 합니다.
