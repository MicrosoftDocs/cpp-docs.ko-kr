# <a name="compiler-error-c2320"></a>컴파일러 오류 C2320
액세스 지정자 'specifier' 다음에 ':'이 필요합니다.  
  
 키워드 `public`, `protected`또는 `private` 은 뒤에 콜론이 와야 합니다.
