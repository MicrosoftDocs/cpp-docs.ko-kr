# <a name="fatal-error-c1601"></a>심각한 오류 C1601

지원되지 않는 인라인 어셈블리 opcode입니다.

이 오류는 불완전 한 설치로 인해 발생 했을 수 있는 컴파일러.exe 파일에 불일치를 나타냅니다. 예를 들어 설치 했을 수 서비스 팩이 설치 되지만 프로세서 팩에 없습니다. 모든 필요한 제품을 설치 합니다.
