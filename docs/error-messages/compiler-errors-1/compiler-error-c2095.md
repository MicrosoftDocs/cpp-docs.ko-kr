# <a name="compiler-error-c2095"></a>컴파일러 오류 C2095

' function ': 실제 매개 변수의 형식은 ' void '입니다. ' number ' 매개 변수를 포함 합니다.

함수에 전달 된 매개 변수가 허용 되지 않는 형식 `void`입니다. 대신 void (`void *`)에 대 한 포인터를 사용 합니다.

`number`는 `void`매개 변수를 나타냅니다.
