# <a name="compiler-error-c2095"></a>컴파일러 오류 C2095

'function': 실제 매개 변수는 'void' 형식: 'number' 매개 변수

함수에 전달 된 매개 변수는 형식 `void`, 허용 되지 않습니다. Void에 대 한 포인터를 사용 하 여 ( `void *`) 대신 합니다.

합니다 `number` 매개 변수를 나타내는 `void`합니다.
