# <a name="compiler-error-c2095"></a>컴파일러 오류 C2095
'function': 실제 매개 변수는 'void' 형식의: 'number' 매개 변수  
  
 함수에 전달 된 매개 변수 형식이 `void`,이 허용 되지 않습니다. Void에 대 한 포인터를 사용 하 여 ( `void *`) 대신.  
  
 `number` 어느 매개 변수가 나타내는 `void`합니다.
