# <a name="fatal-error-c1033"></a>심각한 오류 C1033

프로그램 데이터베이스 pdb를 열 수 없습니다.

이 오류는 디스크 오류, 바이러스 백신 프로그램이 만든 임시 잠금, 완전히 종료 되지 않은 이전 디버거 인스턴스 또는 동일한 파일에 대 한 액세스를 시도 하는 병렬 빌드 mspdbsrv.exe 프로세스가 가능한 다른 원인으로 인해 발생할 수 있습니다.
