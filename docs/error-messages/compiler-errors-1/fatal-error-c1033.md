# <a name="fatal-error-c1033"></a>심각한 오류 C1033

프로그램 데이터베이스 pdb를 열 수 없습니다.

하려고 하는 병렬 빌드 mspdbsrv.exe 프로세스는 다른 원인이 간에 동일한 파일에 액세스할 또는 바이러스 백신 프로그램을 종료 되지 않았음을 완벽 하 게, 디버거 인스턴스를 이전에 만든 임시 잠금 디스크 오류가 발생 하 여이 오류를 발생할 수 있습니다.
