# <a name="fatal-error-c1506"></a>심각한 오류 C1506

복구할 수 없는 블록 범위 지정 오류입니다.

블록을 컴파일하는 데 너무 큽니다.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. 짝이 맞지 않는 괄호

1. 매우 큰 함수 또는 클래스
