# <a name="fatal-error-c1053"></a>심각한 오류 C1053
'\<식별자 >': 함수가 너무 큽니다.  
  
 함수가 너무 커서 컴파일할 수 없습니다.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>아래의 해결 방법 따라 수정합니다.  
  
1.  최적화 없이 컴파일을 시도합니다.  
  
2.  함수를 좀 더 작은 함수로 나눕니다.  
  
3.  인라인 함수에 대한 호출을 줄입니다.
