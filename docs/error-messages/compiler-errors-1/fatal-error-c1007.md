# <a name="fatal-error-c1007"></a>심각한 오류 C1007

옵션에서 인식할 수 없는 플래그 문자열

명령줄 옵션에 잘못된 문자열이 있습니다. **CL** 명령줄 및 환경 변수에서 오류를 확인하세요.
