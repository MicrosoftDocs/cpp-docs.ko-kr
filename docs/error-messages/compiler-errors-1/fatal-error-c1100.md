# <a name="fatal-error-c1100"></a>심각한 오류 C1100

OLE: 시스템 오류 메시지를 초기화할 수 없습니다.

컴파일러가 COM (구성 요소 개체 모델) 라이브러리를 초기화할 수 없습니다. [CoInitialize](/windows/win32/api/objbase/nf-objbase-coinitialize)를 참조 하세요.
