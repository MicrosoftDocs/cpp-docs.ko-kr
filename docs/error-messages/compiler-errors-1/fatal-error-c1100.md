# <a name="fatal-error-c1100"></a>심각한 오류 C1100

OLE를 초기화할 수 없습니다: 시스템 오류 메시지

컴파일러 구성 요소 개체 모델 (COM) 라이브러리를 초기화할 수 없습니다. 참조 [CoInitialize](/windows/desktop/api/objbase/nf-objbase-coinitialize)합니다.
