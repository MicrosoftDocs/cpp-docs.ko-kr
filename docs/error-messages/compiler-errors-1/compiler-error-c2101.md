# <a name="compiler-error-c2101"></a>컴파일러 오류 C2101
상수에 '&'가 있습니다.  
  
 연산자 주소( `&` )에 피연산자로 l 값이 있어야 합니다.  
  
 다음 샘플에서는 C2101을 생성합니다.  
  
```  
// C2101.cpp  
int main() {  
   char test;  
   test = &'a';   // C2101  
   test = 'a';   // OK  
}  
```
