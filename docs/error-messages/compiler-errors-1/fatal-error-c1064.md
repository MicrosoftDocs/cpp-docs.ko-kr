# <a name="fatal-error-c1064"></a>심각한 오류 C1064

컴파일러 한계: 내부 버퍼에서 토큰 오버플로가 발생 했습니다.

식별자가 식별자에 사용 되는 내부 버퍼의 길이를 초과 합니다. 이름을 줄입니다.
