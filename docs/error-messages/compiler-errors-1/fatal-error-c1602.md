# <a name="fatal-error-c1602"></a>심각한 오류 C1602

지원되지 않는 내장 함수입니다.

이 오류는 불완전 한 설치로 인해 발생 했을 수 있는 컴파일러 .exe 파일의 불일치를 나타냅니다. 예를 들어 Service Pack를 설치 했지만 프로세서 팩은 설치 하지 않았을 수 있습니다. 필요한 모든 제품을 설치 합니다.
