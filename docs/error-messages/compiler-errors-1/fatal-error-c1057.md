# <a name="fatal-error-c1057"></a>심각한 오류 C1057

매크로 확장에서 예기치 않은 파일의 끝이 나타났습니다.

컴파일러에서 매크로 호출 오른쪽 괄호를 누락 되었기 때문일 수 있습니다 매크로 호출 인수를 수집 하는 동안 소스 파일의 끝에 도달 합니다.
