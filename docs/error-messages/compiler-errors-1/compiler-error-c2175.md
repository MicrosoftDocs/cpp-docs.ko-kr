# <a name="compiler-error-c2175"></a>컴파일러 오류 C2175

' locale ': 로캘이 잘못 되었습니다.

지정 된 로캘이 잘못 되었습니다. 지원 되는 로캘의 경우 *런타임 라이브러리 참조* 에서 [언어 및 국가/지역 문자열](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) 을 참조 하세요.
