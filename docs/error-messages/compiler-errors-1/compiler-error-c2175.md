# <a name="compiler-error-c2175"></a>컴파일러 오류 C2175

'로캘': 잘못 된 로캘

지정 된 로캘을 올바르지 않습니다. 참조 [언어 및 국가/지역 문자열](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) 에 *런타임 라이브러리 참조* 로캘을 지원 합니다.
