# <a name="fatal-error-c1073"></a>심각한 오류 C1073
(컴파일러 파일 'filename', 줄 번호) 증분 컴파일과 관련 된 내부 오류  
  
 증분 컴파일을 사용 하지 않고 파일을 다시 컴파일하십시오.
