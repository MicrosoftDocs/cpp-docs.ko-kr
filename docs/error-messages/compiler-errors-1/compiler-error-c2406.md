# <a name="compiler-error-c2406"></a>컴파일러 오류 C2406

'identifier': 'context'에 이름을 정의하지 않았습니다.

정의 되지 않은 식별자는 `SIZE`, `LENGTH`또는 멤버 선택 (.) 연산자와 함께 사용 됩니다.
