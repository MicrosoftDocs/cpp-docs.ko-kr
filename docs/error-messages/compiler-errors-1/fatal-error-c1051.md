# <a name="fatal-error-c1051"></a>심각한 오류 C1051

프로그램 데이터베이스 파일 ' pdbfile '에 사용 되지 않는 형식이 있습니다. 삭제 한 후 다시 컴파일하십시오.

컴파일러는 이전 버전 번호가 있는 프로그램 데이터베이스 파일을 업데이트할 수 없습니다. 파일을 삭제 하 고 **/zi** 또는 **/zi** 를 사용 하 여 프로그램을 다시 컴파일합니다. 자세한 내용은 [/Z7,/zi,/zi (디버그 정보 형식)](../../build/reference/z7-zi-zi-debug-information-format.md) 를 참조 하세요.
