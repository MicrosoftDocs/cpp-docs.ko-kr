# <a name="compiler-error-c2166"></a>컴파일러 오류 C2166
l-value가 const 개체를 지정합니다.  
  
 코드에서 `const`로 선언된 항목을 수정하려고 합니다.  
  
 다음 샘플에서는 C2166을 생성합니다.  
  
```  
// C2166.cpp  
int f();  
int main() {  
   ( (const int&) 1 ) = 5;   // C2166  
}  
```
