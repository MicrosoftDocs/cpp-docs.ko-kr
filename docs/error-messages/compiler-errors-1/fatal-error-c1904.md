# <a name="fatal-error-c1904"></a>심각한 오류 C1904

잘못된 공급자 상호 작용: 'file'

이 오류는 특성 공급자 오류를 나타냅니다.

오류 상황을 확인, 문제를 격리 하는 재현 가능한 테스트 사례를 만든 후에 문의 [의견을 보내](/visualstudio/ide/talk-to-us)합니다.
