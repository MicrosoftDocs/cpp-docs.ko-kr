# <a name="compiler-error-c2345"></a>컴파일러 오류 C2345
align(value): 맞춤 값이 잘못되었습니다.  
  
 [align](../../cpp/align-cpp.md) 키워드에 허용 범위 외부의 값을 전달했습니다.  
  
 다음 코드에서는 C2345를 생성합니다.  
  
```  
// C2345.cpp  
// compile with: /c  
__declspec(align(0)) int a;   // C2345  
__declspec(align(1)) int a;   // OK  
```
