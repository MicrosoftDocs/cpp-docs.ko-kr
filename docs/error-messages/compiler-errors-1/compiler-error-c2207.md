# <a name="compiler-error-c2207"></a>컴파일러 오류 C2207
'member': 클래스 템플릿의 멤버는 함수 형식을 가져올 수 없습니다.  
  
 클래스 템플릿의 `member`는 이전에 비정적 데이터 멤버로 구문 분석되었습니다. 멤버 함수로 재정의할 수 없습니다.
