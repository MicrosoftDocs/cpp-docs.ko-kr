# <a name="compiler-error-c2401"></a>컴파일러 오류 C2401

'identifier': 레지스터는 'context'에서 기본이어야 합니다.

간접 메모리 피연산자에 사용되는 레지스터는 이 컨텍스트에서 기본 레지스터여야 합니다.
