# <a name="fatal-error-c1312"></a>심각한 오류 C1312
함수에 조건부 분기가 너무 많습니다. 소스 코드를 단순화하거나 리팩터링하세요.  
  
 코드가 너무 복잡해서 컴파일러에서 처리하려면 스택 메모리를 모두 사용해야 합니다.  코드를 단순화하세요.
