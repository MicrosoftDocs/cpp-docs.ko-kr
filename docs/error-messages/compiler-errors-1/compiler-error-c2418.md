# <a name="compiler-error-c2418"></a>컴파일러 오류 C2418

브라우저 파일 (파일 이름)을 삭제할 수 없습니다.

컴파일러가 브라우저 파일을 삭제할 수 없습니다.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. 파일이 읽기 전용입니다.

1. 다른 프로세스에서 파일을 사용 중입니다.
