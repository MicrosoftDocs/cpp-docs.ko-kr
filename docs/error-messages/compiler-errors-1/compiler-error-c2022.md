# <a name="compiler-error-c2022"></a>컴파일러 오류 C2022

'number': 문자에 비해 너무 큽니다.

백슬래시 뒤에 8 진수 숫자가 (\\) 문자에서는 문자를 나타내는 문자열 상수가 너무 큽니다. 또는 합니다.
