# <a name="fatal-error-c1121"></a>심각한 오류 C1121
CryptoAPI에 대한 호출이 실패했습니다.  
  
 컴파일러는 CryptoAPI에 대 한 호출 만들 호출이 실패 했습니다. Visual Studio를 다시 설치 하 고 가능한 운영 체제를 다시 설치 합니다.
