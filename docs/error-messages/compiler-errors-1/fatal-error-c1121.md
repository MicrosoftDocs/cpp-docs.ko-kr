# <a name="fatal-error-c1121"></a>심각한 오류 C1121

CryptoAPI에 대한 호출이 실패했습니다.

컴파일러가 CryptoAPI를 호출 하 고 호출에 실패 했습니다. Visual Studio를 다시 설치 하 고 운영 체제를 다시 설치할 수 있습니다.
