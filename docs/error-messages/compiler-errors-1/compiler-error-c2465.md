# <a name="compiler-error-c2465"></a>컴파일러 오류 C2465
괄호 안에 무명 형식을 정의할 수 없습니다.  
  
 익명 구조체, 공용 구조체 또는 열거 형식이 괄호 식 내에서 정의됩니다. C++에서는 함수 범위에서 정의가 의미가 없으므로 잘못된 것입니다.
