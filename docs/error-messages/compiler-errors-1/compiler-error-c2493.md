# <a name="compiler-error-c2493"></a>컴파일러 오류 C2493
__based의 형식이  
  
 A `__based` 식 포인터 기반으로 해야 합니다.  
  
 다음 샘플에서는 C2493 오류가 생성 됩니다.  
  
```  
// C2493.cpp  
// compile with: /c  
char mybase;  
int __based(mybase) ptr;   // C2493  
  
// OK  
char * mybase;  
int __based(mybase) * ptr;  
```
