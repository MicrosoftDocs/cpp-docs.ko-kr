# <a name="compiler-error-c2212"></a>컴파일러 오류 C2212

' identifier ': 함수에 대 한 포인터에 __based 사용할 수 없습니다.

`__based`함수에 대 한 포인터를 선언할 수 없습니다. 코드 기반 데이터가 필요한 경우에는 `__declspec` 키워드 또는 `data_seg` pragma를 사용 합니다.
