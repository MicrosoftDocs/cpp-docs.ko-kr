# <a name="fatal-error-c1003"></a>심각한 오류 C1003

오류 수가 number개를 초과했습니다. 컴파일을 중지합니다.

프로그램에 오류가 너무 많아서 복구할 수 없습니다. 컴파일러를 종료해야 합니다.
