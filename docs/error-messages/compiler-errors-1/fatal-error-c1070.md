# <a name="fatal-error-c1070"></a>심각한 오류 C1070
'filename' 파일에서 #if/#endif 쌍이 짝이 맞지 않습니다.  
  
 `#if`, `#ifdef`또는 `#ifndef` 지시문에 해당하는 `#endif`가 없습니다.  
  
 다음 샘플에서는 C1070을 생성합니다.  
  
```  
// C1070.cpp  
#define TEST  
  
#ifdef TEST  
  
#ifdef TEST  
#endif  
// C1070  
```  
  
 해결 방법:  
  
```  
// C1070b.cpp  
// compile with: /c  
#define TEST  
  
#ifdef TEST  
#endif  
  
#ifdef TEST  
#endif  
```
