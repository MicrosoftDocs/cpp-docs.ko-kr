# <a name="compiler-error-c2446"></a>컴파일러 오류 C2446
'operator': 'type1'에서 'type2' 사이의 변환이  
  
 컴파일러는 변환할 수 없는 `type1` 를 `type2`합니다. C/c + + 의미 체계를 위반 하기 때문에 변환 적합할 수 있습니다.
