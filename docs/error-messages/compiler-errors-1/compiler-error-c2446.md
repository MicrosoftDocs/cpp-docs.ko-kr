# <a name="compiler-error-c2446"></a>컴파일러 오류 C2446

'operator': 'type1'에서 'type2'로 변환 되지 않았습니다

컴파일러는 변환할 수 없습니다 `type1` 에 `type2`입니다. C/c + + 의미 체계를 위반 하기 때문에 변환이 적합 하지 않을 수 있습니다.
