# <a name="compiler-error-c2446"></a>컴파일러 오류 C2446

' operator ': ' type1 '에서 ' type2 ' (으)로 변환 하지 않습니다.

컴파일러가 `type1`를 `type2`로 변환할 수 없습니다. C/C++ 의미 체계를 위반 하므로 변환이 적합 하지 않을 수 있습니다.
