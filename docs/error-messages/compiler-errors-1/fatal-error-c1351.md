# <a name="fatal-error-c1351"></a>심각한 오류 C1351

> '*dll*' dll을 로드 하는 동안 오류 발생: 호환 되지 않는 버전

잘못 된 버전의 DLL을 찾았습니다.

이는 설치에 문제가 있음을 나타내고 Visual Studio를 다시 설치 해야 합니다.
