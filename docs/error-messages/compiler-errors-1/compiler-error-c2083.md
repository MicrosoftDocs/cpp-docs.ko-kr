# <a name="compiler-error-c2083"></a>컴파일러 오류 C2083
struct/union 비교가 잘못되었습니다.  
  
 구조체 또는 공용 구조체는 다른 사용자 정의 형식과 직접 비교됩니다. 이는 비교 연산자를 정의했거나 스칼라 형식에 대한 변환이 존재하지 않는 한 허용되지 않습니다.
