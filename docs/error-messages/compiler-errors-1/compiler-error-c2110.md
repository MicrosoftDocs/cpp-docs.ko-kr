# <a name="compiler-error-c2110"></a>컴파일러 오류 C2110
'+': 두 포인터를 더할 수 없습니다.  
  
 더하기( `+` ) 연산자를 사용하여 두 포인터 값을 더하려고 했습니다.  
  
 다음 샘플에서는 C2110을 생성합니다.  
  
```  
// C2110.cpp  
int main() {  
   int a = 0;  
   int *pa;  
   int *pb;  
   a = pa + pb;   // C2110  
}  
```
