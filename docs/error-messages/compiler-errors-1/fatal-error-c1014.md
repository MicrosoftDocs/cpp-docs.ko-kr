# <a name="fatal-error-c1014"></a>심각한 오류 C1014
포함 파일이 너무 많습니다. 수준 = level  
  
 `#include` 지시문의 중첩이 너무 깊습니다. 중첩된 지시문이 열려 있는 파일을 포함할 수 있습니다. 지시문을 포함하는 소스 파일은 한 파일로 계산됩니다.
