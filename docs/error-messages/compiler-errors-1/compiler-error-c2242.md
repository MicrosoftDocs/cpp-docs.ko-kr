# <a name="compiler-error-c2242"></a>컴파일러 오류 C2242

형식 정의 이름은 클래스/구조체/공용 구조체 다음에 올 수 없습니다.

`typedef` 이름은 정규화된 이름의 끝에 나타납니다.
