# <a name="compiler-error-c2020"></a>컴파일러 오류 C2020

' member ': ' class ' 멤버 재정의

기본 클래스 또는 구조체에서 상속 된 멤버가 다시 정의 됩니다. 상속 **`virtual`** 된 멤버는 기본 클래스에서로 선언 된 경우를 제외 하 고 다시 정의할 수 없습니다.
