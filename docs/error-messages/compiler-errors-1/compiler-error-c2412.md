# <a name="compiler-error-c2412"></a>컴파일러 오류 C2412

' label ': 대/소문자를 구분 하지 않는 레이블을 재정의 했습니다.

레이블이 현재 함수에 두 번 이상 정의 되어 있습니다. 레이블의 철자와 해당 참조를 변경 합니다.
