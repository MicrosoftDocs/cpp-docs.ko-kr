# <a name="compiler-error-c2412"></a>컴파일러 오류 C2412

'label': 대/소문자 레이블이 재정의 되었습니다

현재 함수에서 레이블은 두 번 이상 정의 됩니다. 레이블 및 해당 참조의 철자를 변경 합니다.
