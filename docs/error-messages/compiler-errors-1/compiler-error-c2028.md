# <a name="compiler-error-c2028"></a>컴파일러 오류 C2028

구조체/공용 구조체 멤버는 구조체/공용 구조체 안에 있어야 합니다.

구조체 또는 공용 구조체 내에서 구조체 또는 공용 구조체 멤버를 선언해야 합니다.
