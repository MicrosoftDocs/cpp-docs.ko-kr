# <a name="compiler-error-c2174"></a>컴파일러 오류 C2174

' function ': 실제 매개 변수는 ' void ' 형식입니다. 매개 변수 number1, 매개 변수 목록 number2

매개 변수 목록에 전달 된 매개 변수는 매개 변수입니다 `number1` `number2` **`void`** . 매개 변수에는 형식을 사용할 수 없습니다 **`void`** . **`void*`** 대신를 사용 합니다.
