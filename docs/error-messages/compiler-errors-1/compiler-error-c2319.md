# <a name="compiler-error-c2319"></a>컴파일러 오류 C2319
'try/catch'는 복합 문이 뒤에 와야 합니다. '{'가 없습니다.  
  
 `try` 또는 `catch` 블록이 `try` 또는 `catch` 문 다음에 오지 않습니다. 블록은 중괄호로 묶어야 합니다.  
  
 다음 샘플에서는 C2319를 생성합니다.  
  
```  
// C2319.cpp  
// compile with: /EHsc  
#include <eh.h>  
class C {};  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch( C ) ;    // C2319  
   // try the following line instead  
   // catch( C ) {}  
}  
```
