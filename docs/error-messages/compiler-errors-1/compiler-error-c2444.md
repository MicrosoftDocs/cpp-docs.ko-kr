# <a name="compiler-error-c2444"></a>컴파일러 오류 C2444

'identifier': ANSI 프로토타입이 사용되었습니다. '{' 또는 ';'이 있어야 하는데 'type'이 있습니다.

함수 프로토타입 뒤에 형식이 나옵니다.

이 오류는 세미콜론이 또는 중괄호가 누락되어 발생할 수 있습니다.
