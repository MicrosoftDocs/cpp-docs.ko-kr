# <a name="compiler-error-c2295"></a>컴파일러 오류 C2295
이스케이프된 'character': 매크로 정의에서 사용할 수 없습니다.  
  
 매크로 정의에는 지정된 문자와 함께 이스케이프 시퀀스가 포함될 수 없습니다.
