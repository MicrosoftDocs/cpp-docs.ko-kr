# <a name="compiler-error-c2092"></a>컴파일러 오류 C2092
'배열 이름' 배열 요소 형식은 함수 일 수 없습니다.  
  
 함수의 배열은 허용 되지 않습니다. 함수 포인터의 배열을 사용 합니다.  
  
## <a name="example"></a>예  
 다음 샘플에서는 C2092 오류가 생성 됩니다.  
  
```  
// C2092.cpp  
typedef void (F) ();  
typedef F AT[10];   // C2092  
```  
  
## <a name="example"></a>예  
 해결 방법:  
  
```  
// C2092b.cpp  
// compile with: /c  
typedef void (F) ();  
typedef F * AT[10];  
```
