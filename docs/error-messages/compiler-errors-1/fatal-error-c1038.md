# <a name="fatal-error-c1038"></a>심각한 오류 C1038

컴파일러 한계: function: 제어 흐름 상태가 너무 복잡합니다. 함수를 간단하게 만드세요.

함수에 컴파일러에서 처리할 수 있는 것보다 더 많은 제어 흐름 상태가 있습니다. 제어 흐름을 간소화하거나 함수를 좀 더 작은 함수로 분할합니다.
