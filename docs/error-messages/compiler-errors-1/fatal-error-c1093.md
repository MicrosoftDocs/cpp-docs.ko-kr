# <a name="fatal-error-c1093"></a>심각한 오류 C1093

> '*함수 이름*' API 호출에 실패 했습니다. '*호출의 위치*': '*런타임에 텍스트 '가*있습니다.

.NET 함수를 호출 하지 못했습니다. *런타임 문자열의 텍스트* 는 COM 런타임에서 제공 되거나 제공 되지 않을 수 있습니다.

시스템 오류 메시지에 대 한 자세한 내용은 winerror.h 시스템 파일 및 [FormatMessage](/windows/win32/api/winbase/nf-winbase-formatmessage)를 참조 하세요.
