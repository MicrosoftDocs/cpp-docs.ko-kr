# <a name="fatal-error-c1085"></a>심각한 오류 C1085

Filetype 파일을 쓸 수 없습니다. 'file': 메시지

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. 드라이브 읽기 전용입니다.

1. 드라이브가 꽉 찼습니다.

1. 공유 위반입니다.

1. "잘못된 파일 번호"라는 메시지가 표시되면 백그라운드에서 컴파일하는 동안 파일이 전경에서 닫혀 있을 수 있습니다.
