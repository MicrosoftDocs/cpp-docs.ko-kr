# <a name="compiler-error-c2185"></a>컴파일러 오류 C2185

' identifier ': 기반 할당이 잘못 되었습니다.

레지스터 변수 또는 자동 (지역) 변수가 `__based`선언 됩니다. 전역 변수만 `__based`선언할 수 있습니다.
