# <a name="compiler-error-c2077"></a>컴파일러 오류 C2077

비 스칼라 필드 이니셜라이저 ' identifier '

비스칼라 (구조체, 공용 구조체, 배열 또는 클래스)를 사용 하 여 비트 필드를 초기화 하려고 했습니다. 정수 또는 부동 소수점 값을 사용 합니다.
