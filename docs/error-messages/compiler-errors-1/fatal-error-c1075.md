# <a name="fatal-error-c1075"></a>심각한 오류 C1075

파일 끝에 있는 왼쪽 토큰의 짝이 맞지 않습니다.

컴파일러는 파일의 끝에 도달하기 전에 일치하는 *토큰* 을 예상했습니다.

이 오류는 일치하지 않는 대괄호, 중괄호 또는 기타 쌍을 이루는 문자로 인해 발생할 수 있습니다.
