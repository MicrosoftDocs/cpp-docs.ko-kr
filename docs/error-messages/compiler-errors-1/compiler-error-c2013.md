# <a name="compiler-error-c2013"></a>컴파일러 오류 C2013
'>'가 없습니다.  
  
 `#include` 지시문에 닫는 꺾쇠 괄호가 없습니다. 오류를 해결하려면 닫는 대괄호를 추가합니다.  
  
 다음 샘플에서는 C2013을 생성합니다.  
  
```  
// C2013.cpp  
#include <stdio.h    // C2013  
```  
  
 해결 방법:  
  
```  
// C2013b.cpp  
// compile with: /c  
#include <stdio.h>  
```
