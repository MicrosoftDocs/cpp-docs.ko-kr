# <a name="compiler-error-c2411"></a>컴파일러 오류 C2411

' identifier ': ' context '의 구조체/공용 구조체 멤버가 잘못 되었습니다.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. 식별자가이 컨텍스트에서 표시 되는 구조체 또는 공용 구조체의 멤버가 아닙니다.

1. 식별자가 멤버 선택 (.) 연산자를 사용 하 여 지정 된 구조체 또는 공용 구조체의 멤버가 아닙니다.
