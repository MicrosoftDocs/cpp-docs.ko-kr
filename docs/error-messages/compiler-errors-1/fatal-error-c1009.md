# <a name="fatal-error-c1009"></a>심각한 오류 C1009

컴파일러 한계: 매크로가 너무 많이 중첩

컴파일러는 동시에 너무 많은 매크로 확장 하 려 했습니다. 컴파일러의 제한은 256 수준의 중첩 된 매크로입니다. 중첩 된 매크로 보다 간단한 매크로로 분할 합니다.
