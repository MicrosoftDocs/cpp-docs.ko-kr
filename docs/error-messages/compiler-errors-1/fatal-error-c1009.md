# <a name="fatal-error-c1009"></a>심각한 오류 C1009

컴파일러 한계: 매크로가 너무 많이 중첩 되었습니다.

컴파일러가 동시에 너무 많은 매크로를 확장 하려고 했습니다. 컴파일러는 중첩 된 매크로 수준을 256 개로 제한 합니다. 중첩 된 매크로를 간단한 매크로로 분할 합니다.
