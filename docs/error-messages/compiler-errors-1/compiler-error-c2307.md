# <a name="compiler-error-c2307"></a>컴파일러 오류 C2307

pragma 'pragma' 증분 컴파일이 활성화 된 경우 외부 함수 여야 합니다.

배치 해야 합니다는 `data_seg` 증분 컴파일을 사용 하는 경우 함수 간에 pragma입니다.
