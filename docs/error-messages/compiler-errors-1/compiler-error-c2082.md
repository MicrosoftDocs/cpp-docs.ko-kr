# <a name="compiler-error-c2082"></a>컴파일러 오류 C2082
'identifier' 정식 매개 변수 재정의  
  
 함수의 정식 매개 변수가 함수 본문 내에서 다시 선언됩니다. 이 오류를 해결하려면 재정의를 제거합니다.  
  
 다음 샘플에서는 C2082를 생성합니다.  
  
```  
// C2082.cpp  
void func(int i) {  
   int i;   // C2082  
   int ii;   // OK  
}  
```
