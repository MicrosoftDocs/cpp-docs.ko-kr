# <a name="compiler-error-c2459"></a>컴파일러 오류 C2459
'identifier': 정의 하 고 있습니다. 익명 멤버로 서 추가할 수 없습니다.  
  
 클래스, 구조체 또는 공용 구조체는 자체 범위에서 익명 공용 구조체의 멤버에 의해 재정의 됩니다.  
  
 다음 샘플에서는 C2459 오류가 생성 됩니다.  
  
```  
// C2459.cpp  
// compile with: /c  
class C {  
   union { int C; };   // C2459  
   union { int D; };  
};  
```
