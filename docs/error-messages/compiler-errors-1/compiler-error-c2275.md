# <a name="compiler-error-c2275"></a>컴파일러 오류 C2275
'identifier':이 형식 식으로 잘못 사용  
  
 식을 사용 하 여는 `->` 연산자는 `typedef` 식별자입니다.  
  
 다음 샘플에서는 C2275 오류가 생성 됩니다.  
  
```  
// C2275.cpp  
typedef struct S {  
    int mem;  
} *S_t;  
void func1( int *parm );  
void func2() {  
    func1( &S_t->mem );   // C2275, S_t is a typedef  
}  
```
