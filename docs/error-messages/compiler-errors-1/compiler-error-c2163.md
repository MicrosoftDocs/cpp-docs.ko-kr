# <a name="compiler-error-c2163"></a>컴파일러 오류 C2163

' function ': 내장 함수로 사용할 수 없습니다.

`intrinsic` 또는 `function` pragma는 내장 형식으로 사용할 수 없는 함수를 나열 합니다. 예를 들어/clr 프로그래밍을 사용 하는 프로그램을 컴파일할 때 특정 내장 함수를 사용할 수 없습니다.
