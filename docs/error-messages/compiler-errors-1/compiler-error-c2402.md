# <a name="compiler-error-c2402"></a>컴파일러 오류 C2402
'identifier': 레지스터는 'context'에서 인덱스여야 합니다.  
  
 간접 메모리 피연산자에 사용되는 레지스터는 이 컨텍스트에서 인덱스 레지스터여야 합니다.
