# <a name="fatal-error-c1045"></a>심각한 오류 C1045

컴파일러 한계 : 링크 사양이 너무 많이 중첩되었습니다.

중첩된 외부 참조가 컴파일러 한계를 초과했습니다. "C + +"와 같은 외부 링크 형식에서 중첩 된 외부를 사용할 수 **`extern`** 있습니다. 오류를 해결하려면 중첩된 외부 참조 수를 줄입니다.
