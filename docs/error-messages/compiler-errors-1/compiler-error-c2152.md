# <a name="compiler-error-c2152"></a>컴파일러 오류 C2152

' identifier ': 특성이 서로 다른 함수에 대 한 포인터입니다.

하나의 호출 규칙 (, 또는)을 사용 하는 함수에 대 한 포인터는 **`__cdecl`** **`__stdcall`** **`__fastcall`** 다른 호출 규칙을 사용 하 여 함수에 대 한 포인터에 할당 됩니다.
