# <a name="compiler-error-c2152"></a>컴파일러 오류 C2152
'identifier': 특성이 서로 다른 함수에 대 한 포인터  
  
 하나의 호출 규칙을 사용 하는 함수에 대 한 포인터 (`__cdecl`, `__stdcall`, 또는 `__fastcall`) 다른 호출 규칙을 사용 하는 함수에 대 한 포인터에 할당 됩니다.
