# <a name="compiler-error-c2138"></a>컴파일러 오류 C2138

멤버 없이 열거형을 정의할 수 없습니다.

/Za (Microsoft 확장 사용 안 함)를 선택 하면 열거형에 멤버가 하나 이상 있어야 합니다.
