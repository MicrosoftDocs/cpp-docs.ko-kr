# <a name="compiler-error-c2138"></a>컴파일러 오류 C2138

멤버 없이 열거형을 정의 하는 잘못 되었습니다.

열거형 멤버를 하나 이상 있어야 합니다. 때 /Za (Microsoft 확장 사용 안 함)을 선택 합니다.
