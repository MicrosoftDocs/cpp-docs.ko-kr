# <a name="compiler-error-c2333"></a>컴파일러 오류 C2333
'function': 함수 선언; 하는 동안 오류가 발생 했습니다. 함수 본문을 건너뜁니다.  
  
 이 오류는 해당 하는 클래스 내에 정의 된 멤버 함수에 대 한 다른 오류가 발생 한 후 발생 합니다.  
  
 다음 샘플에서는 C2333 오류가 생성 됩니다.  
  
```  
// C2333.cpp  
struct s1 {  
   s1(s1) {}   // C2333  
};  
```
