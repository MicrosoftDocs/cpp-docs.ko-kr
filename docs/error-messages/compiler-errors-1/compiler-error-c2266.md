# <a name="compiler-error-c2266"></a>컴파일러 오류 C2266

'identifier': 비상수 범위 배열 참조를 사용할 수 없습니다.

참조가 비상수 범위를 가진 배열에 대해 선언되었습니다. 배열은 상수 범위를 가져야 합니다.
