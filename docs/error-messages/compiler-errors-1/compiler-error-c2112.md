# <a name="compiler-error-c2112"></a>컴파일러 오류 C2112

'-': 포인터 빼기에는 정수 계열 또는 포인터 피연산자가 있어야 합니다.

서로 다른 형식을 가리키는 포인터를 빼기 위해 시도 했습니다.
