# <a name="fatal-error-c1065"></a>심각한 오류 C1065

컴파일러 한계: 태그가 부족

소스 파일 두 개 65,523 클래스, 구조체, 공용 구조체, 네임 스페이스 또는 열거형을 포함합니다.

파일을 더 작은 파일로 분할 합니다.
