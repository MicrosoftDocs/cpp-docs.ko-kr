# <a name="compiler-error-c2337"></a>컴파일러 오류 C2337
'attribute name': 특성을 찾을 수 없습니다.  
  
 이 버전의 Visual C++에서 지원되지 않는 특성을 사용했습니다.  
  
 다음 샘플에서는 C2337을 생성합니다.  
  
```  
// C2337.cpp  
// compile with: /c  
[emitidl];  
[module(name="x")];  
[grasshopper]   // C2337, not a supported attribute  
class a{};  
```
