# <a name="compiler-error-c2188"></a>컴파일러 오류 C2188

'number': 와이드 문자에 비해 너무 큽니다.

수가 와이드 문자 형식에 대한 크기 제한을 초과합니다. 더 큰 형식을 선택합니다.
