# <a name="compiler-error-c2045"></a>컴파일러 오류 C2045
'identifier': 레이블이 재정의되었습니다.  
  
 레이블이 같은 함수의 여러 문 앞에 나타납니다.  
  
 다음 샘플에서는 C2045를 생성합니다.  
  
```  
// C2045.cpp  
int main() {  
   label: {  
   }  
   goto label;  
   label: {}   // C2045  
}  
```
