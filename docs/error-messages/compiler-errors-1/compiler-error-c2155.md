# <a name="compiler-error-c2155"></a>컴파일러 오류 C2155

'? ': 왼쪽 피연산자가 잘못 되었습니다. 산술 또는 포인터 형식이 필요 합니다.

의 왼쪽에 있는 식을 `?` 0과 비교할 수 없습니다. 0과 비교할 수 있는 산술 또는 포인터 식을 사용 해야 합니다.
