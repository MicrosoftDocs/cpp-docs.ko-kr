# <a name="compiler-error-c2155"></a>컴파일러 오류 C2155

'?': 잘못 된 왼쪽 피연산자를 산술 또는 포인터 형식

왼쪽에 식이 `?` 0과 비교할 수 없습니다. 0과 비교할 수는 산술 또는 포인터 식을 사용 해야 합니다.
