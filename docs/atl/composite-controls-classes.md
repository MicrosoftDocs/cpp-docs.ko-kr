# <a name="composite-controls-classes"></a>복합 컨트롤 클래스

다음 클래스는 복합 컨트롤을 만들기 위한 지원을 제공 합니다.

- [CComCompositeControl](../atl/reference/ccomcompositecontrol-class.md) 에서 파생 된 ActiveX 컨트롤 `CComCompositeControl` 은 표준 대화 상자에서 호스팅됩니다. 이러한 종류의 컨트롤은 다른 컨트롤 (네이티브 Windows 컨트롤 및 ActiveX 컨트롤)을 호스팅할 수 있으므로 복합 컨트롤 이라고 합니다.
