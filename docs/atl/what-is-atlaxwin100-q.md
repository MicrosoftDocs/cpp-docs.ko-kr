# <a name="what-is-atlaxwin100"></a>AtlAxWin100 란?
`AtlAxWin100` ATL의 컨트롤 호스팅 기능을 제공 하는 데 도움이 되는 창 클래스의 이름이입니다. 이 클래스의 인스턴스를 만들 때 창 프로시저를 창에 연결 된 호스트 개체를 만들고 창의 제목으로 지정 하는 컨트롤과 함께 로드 컨트롤 호스팅 API를 자동으로 사용 됩니다.
