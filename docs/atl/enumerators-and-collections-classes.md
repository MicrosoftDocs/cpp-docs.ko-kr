# <a name="enumerators-and-collections-classes"></a>열거자 및 컬렉션 클래스
다음 클래스에서는 COM 컬렉션 및 열거형에 대 한 지원:  
  
-   [CComEnum](../atl/reference/ccomenum-class.md) 정의 COM 열거자 개체 배열을 기반으로 합니다.  
  
-   [CComEnumImpl](../atl/reference/ccomenumimpl-class.md) 열거 되 고 항목이 배열에 저장 되어 있는 COM 열거자 인터페이스에 대 한 구현을 제공 합니다.  
  
-   [CComEnumOnSTL](../atl/reference/ccomenumonstl-class.md) c + + 표준 라이브러리 컬렉션을 기반으로 하는 COM 정의 열거자 개체입니다.  
  
-   [IEnumOnSTLImpl](../atl/reference/ienumonstlimpl-class.md) 열거 되 고 항목 호환 c + + 표준 라이브러리 컨테이너에 저장 되어 있는 COM 열거자 인터페이스에 대 한 구현을 제공 합니다.  
  
-   [ICollectionOnSTLImpl](../atl/reference/icollectiononstlimpl-class.md) 에 대 한 구현을 제공는 **Count**, **항목**, 및 `_NewEnum` 컬렉션 인터페이스의 속성입니다.
