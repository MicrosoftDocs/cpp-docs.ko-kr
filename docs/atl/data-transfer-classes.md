# <a name="data-transfer-classes"></a>데이터 전송 클래스

다음 클래스는 다양 한 유형의 데이터 전송을 지원 합니다.

- [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) 에서는 표준 형식을 사용 하 여 데이터를 검색 및 설정 하는 Uniform Data Transfer을 지원 합니다. Advise 싱크에 대 한 연결을 관리 하 여 데이터 변경 알림을 처리 합니다.

- [Cbindstatuscallback](../atl/reference/cbindstatuscallback-class.md) 비동기 모니커가 개체와의 비동기 데이터 전송에 대 한 정보를 보내고 받을 수 있도록 합니다.
