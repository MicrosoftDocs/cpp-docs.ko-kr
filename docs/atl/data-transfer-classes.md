# <a name="data-transfer-classes"></a>데이터 전송 클래스

다음 클래스는 다양 한 유형의 데이터 전송 지원합니다.

- [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) 지원 균일 한 데이터 사용 하 여 전송 표준 형식을 검색 하 고 데이터를 설정 합니다. 핸들 데이터 싱크는 것이 좋습니다에 대 한 연결을 관리 하 여 알림을 변경 합니다.

- [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) 개체에서 비동기 데이터 전송에 대 한 정보를 수신 하는 비동기 모니커를 허용 합니다.
