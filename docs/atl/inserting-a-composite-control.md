# <a name="inserting-a-composite-control"></a>복합 컨트롤 삽입

**클래스 추가** 대화 상자를 사용 하 여 ATL 개체를 프로젝트에 삽입할 수 있습니다. 솔루션 탐색기에서 프로젝트 이름을 마우스 오른쪽 단추로 클릭 하 고 **추가** 를 가리킨 다음 **클래스 추가** 를 클릭 하 여이 대화 상자에 액세스 합니다.

**클래스 추가** 대화 상자에서 **ATL 컨트롤** 을 선택 합니다. 그러면 [ATL 컨트롤 마법사](../atl/reference/atl-control-wizard.md)가 시작 됩니다. 복합 컨트롤을 만들려면 **옵션** 탭을 선택 하 고 **복합 컨트롤** 확인란을 클릭 합니다.

컨트롤을 보기 위해 기본 HTML 페이지가 생성 됩니다.
