# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule 클래스

이 클래스는 디버깅 인터페이스에 대 한 지원을 제공 합니다.

## <a name="syntax"></a>구문

```
class CAtlDebugInterfacesModule
```

## <a name="remarks"></a>설명

`CAtlDebugInterfacesModule` 디버깅 인터페이스에 필요한 지원을 제공 합니다. 기호를 정의 하는 모든 프로젝트에 포함 되어 [_ATL_DEBUG_QI](debugging-and-error-reporting-macros.md#_atl_debug_qi)합니다.

## <a name="requirements"></a>요구 사항

**헤더:** atlbase.h
