# <a name="default-atl-project-configurations"></a>기본 ATL 프로젝트 구성
ATL 프로젝트 마법사는 기본적으로 두 가지 프로젝트 구성을 만듭니다.  
  
|구성|문자 집합|ATL 사용|  
|-------------------|-------------------|----------------|  
|릴리스|MBCS|DLL|  
|디버그|MBCS|DLL|  
  
 **문자 집합**, **ATL 사용** 모두에서 변경할 수 있습니다는 **프로젝트 설정** 어셈블리 정보는 **일반** 탭 합니다. 또한 구성 관리자를 사용 하 여 자신만 구성을 추가할 수 있습니다. 자세한 내용은 참조 [빌드 구성](/visualstudio/ide/understanding-build-configurations)합니다.
