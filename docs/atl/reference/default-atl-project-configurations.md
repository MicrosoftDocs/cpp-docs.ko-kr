# <a name="default-atl-project-configurations"></a>기본 ATL 프로젝트 구성

ATL 프로젝트 마법사는 기본적으로 두 가지 프로젝트 구성을 만듭니다.

|Configuration|문자 집합|ATL 사용|
|-------------------|-------------------|----------------|
|해제|MBCS|DLL|
|디버그|MBCS|DLL|

**문자 집합**, **ATL 사용** 및 모든 **프로젝트 설정** 대화 상자의 **일반** 탭에서 변경할 수 있습니다. Configuration Manager를 사용 하 여 사용자 고유의 구성을 추가할 수도 있습니다. 자세한 내용은 [빌드 구성](/visualstudio/ide/understanding-build-configurations)을 참조 하세요.
