# <a name="interfaces-atl-control-wizard"></a>인터페이스, ATL 컨트롤 마법사

이 마법사 페이지에서는 컨트롤에서 지 원하는 인터페이스를 식별 합니다. 기본적으로 지원 되는 인터페이스는 대부분의 컨테이너에서 일반적으로 사용 됩니다.

> [!NOTE]
> [옵션](../../atl/reference/options-atl-control-wizard.md) 탭에서 **최소 제어** 를 선택한 경우 **지원 되** 는 목록 상자에 기본적으로 인터페이스가 나타나지 않습니다.

- **지원되지 않음**

   현재 컨트롤에 대해 지원 되지 않는 사용 가능한 인터페이스를 나타냅니다.

- **지원됨**

   현재 컨트롤에 대해 지원 되는 인터페이스를 나타냅니다.

   |전송 단추|설명|
   |---------------------|-----------------|
   |**>**|지원 **되지 않음** 목록에서 현재 선택 된 인터페이스 이름을 **지원 되** 는 목록에 추가 합니다.|
   |**>>**|지원 **되지 않음** 목록에서 사용할 수 있는 모든 인터페이스 이름을 **지원 되** 는 목록에 추가 합니다.|
   |**\<**|**지원** 목록에서 현재 선택 된 인터페이스 이름을 제거 합니다.|
   |**\<\<**|현재 **지원** 목록에 나열 된 모든 인터페이스 이름을 제거 합니다.|
