# <a name="adding-an-atl-simple-object"></a>ATL 단순 개체 추가
프로젝트에 ATL (액티브 템플릿 라이브러리) 개체를 추가 하려면 프로젝트에 ATL 지원을 포함 하는 MFC 응용 프로그램 또는 ATL 응용 프로그램으로 만들어야 합니다. 사용할 수는 [ATL 프로젝트 마법사](../../atl/reference/atl-project-wizard.md) ATL 응용 프로그램을 만드는 또는 [MFC 응용 프로그램에 ATL 개체를 추가](../../mfc/reference/adding-atl-support-to-your-mfc-project.md) MFC 응용 프로그램에 대 한 ATL 지원을 구현 하 합니다.  
  
 처음 만든 또는 사용 하 여 나중에 추가할 때 새로운 ATL 개체에 대 한 COM 인터페이스를 정의할 수는 [인터페이스 구현](../../ide/implement-interface-wizard.md) 클래스 뷰 바로 가기 메뉴에서 명령을 합니다.  
  
### <a name="to-add-an-atl-simple-object-to-your-atl-com-project"></a>ATL COM 프로젝트에 ATL 단순 개체를 추가 하려면  
  
1.  **솔루션 탐색기** 또는 [클래스 뷰](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), ATL 단순 개체를 추가 하려면 프로젝트의 이름을 마우스 오른쪽 단추로 클릭 합니다.  
  
2.  바로 가기 메뉴에서 클릭 **추가**, 클릭 하 고 **클래스 추가**합니다.  
  
3.  에 [클래스 추가](../../ide/add-class-dialog-box.md) 대화 상자의 템플릿 창에서 클릭 **ATL 단순 개체**, 클릭 하 고 **열려** 표시 하는 [ATL 단순 개체 마법사](../../atl/reference/atl-simple-object-wizard.md).  
  
4.  프로젝트에 대 한 추가 옵션에 설정 된 [옵션](../../atl/reference/options-atl-simple-object-wizard.md) ATL 단순 개체 마법사의 페이지입니다.  
  
5.  클릭 **마침** 는 개체를 프로젝트에 추가 합니다.
