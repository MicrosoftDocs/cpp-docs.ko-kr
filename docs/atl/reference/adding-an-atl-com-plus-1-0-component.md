# <a name="adding-an-atl-com-10-component"></a>ATL COM + 1.0 구성 요소 추가

이 마법사는 트랜잭션을 포함 하 여 COM + 1.0 서비스를 지 원하는 프로젝트에 개체를 추가 합니다.

## <a name="to-add-an-atl-com-10-component-to-your-project"></a>프로젝트에 ATL COM + 1.0 구성 요소를 추가 하려면

1. 하나로 **솔루션 탐색기** 또는 [클래스 뷰](/visualstudio/ide/viewing-the-structure-of-code), ATL COM + 1.0 구성 요소를 추가 하려면 프로젝트의 이름을 마우스 오른쪽 단추로 클릭 합니다.

1. 바로 가기 메뉴에서 클릭 **추가**를 클릭 하 고 **클래스 추가**합니다.

1. 에 [클래스 추가](../../ide/add-class-dialog-box.md) 대화 상자의 합니다 **템플릿** 창 클릭 **ATL COM + 1.0 구성 요소**, 클릭 하 고 **열기** 표시 합니다 [ ATL COM + 1.0 구성 요소 마법사](../../atl/reference/atl-com-plus-1-0-component-wizard.md)합니다.
