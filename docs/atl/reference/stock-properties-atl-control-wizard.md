# <a name="stock-properties-atl-control-wizard"></a>스톡 속성, ATL 컨트롤 마법사

이 마법사 페이지에서는 컨트롤에 대해 지원 되는 스톡 속성을 식별 합니다. 기본적으로 속성은 식별 되지 않습니다.

- **지원되지 않음**

   현재 컨트롤에 대해 지원 되지 않는 사용 가능한 속성을 나타냅니다.

- **지원됨**

   현재 컨트롤에 대해 지원 되는 속성을 나타냅니다.

|전송 단추|설명|
|---------------------|-----------------|
|**>**|지원 **되지 않음** 목록에서 현재 선택 된 속성 이름을 **지원 되** 는 목록에 추가 합니다.|
|**>>**|지원 **되지 않음** 목록에서 사용할 수 있는 모든 속성 이름을 **지원 되** 는 목록에 추가 합니다.|
|**\<**|**지원** 목록에서 현재 선택 된 속성 이름을 제거 합니다.|
|**\<\<**|현재 **지원** 목록에 나열 된 모든 속성 이름을 제거 합니다.|
