# <a name="adding-an-atl-ole-db-provider"></a>ATL OLE DB 공급자를 추가합니다.

이 마법사를 사용 하 여 프로젝트에 ATL OLE DB 공급자를 추가 합니다. ATL OLE DB 공급자로 데이터 원본, 세션, 명령 및 행 집합 클래스의 구성 됩니다. ATL COM 응용 프로그램으로 프로젝트를 만든 것 이어야 합니다.

## <a name="to-add-an-atl-ole-db-provider-to-your-project"></a>프로젝트에 ATL OLE DB 공급자를 추가 하려면

1. **클래스 뷰**, 프로젝트를 마우스 오른쪽 단추로 클릭 합니다. 바로 가기 메뉴에서 클릭 **추가** 을 클릭 한 다음 **클래스 추가**합니다.

1. 에 **Visual c + +** 폴더를 두 번 클릭 합니다 **ATL OLE DB Provider** 아이콘 또는 선택 하 고 클릭 **열기**.

   ATL OLE DB 공급자 마법사가 열립니다.

1. 에 설명 된 대로 설정을 정의할 [ATL OLE DB 공급자 마법사](../../atl/reference/atl-ole-db-provider-wizard.md)합니다.

1. 클릭 **완료** 는 프로젝트에서 새로 만든된 OLE DB 공급자 코드를 삽입 하는 마법사를 닫습니다.
