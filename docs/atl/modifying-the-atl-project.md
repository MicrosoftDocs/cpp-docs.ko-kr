# <a name="modifying-the-atl-project"></a>ATL 프로젝트 수정

이 시점에서 복합 컨트롤 프로젝트는 복합 컨트롤에 필요한 개체를 구현 합니다. 다음 단계는 복합 컨트롤이 포함 하는 모든 컨트롤을 추가 하 고 필요한 이벤트를 처리 하는 것입니다.

ActiveX 또는 Windows 컨트롤을 추가 하려면 새 리소스 스크립트를 추가한 다음 대화 상자 편집기를 사용 합니다. 컨트롤 추가 (및 관련 작업)에 대 한 자세한 내용은 [대화 상자 편집기](../windows/dialog-editor.md)를 참조 하십시오.

ActiveX 컨트롤에서 필요한 이벤트를 처리 하려면 [복합 컨트롤에 기능 추가](../atl/adding-functionality-to-the-composite-control.md)를 참조 하세요.
