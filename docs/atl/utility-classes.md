# <a name="utility-classes"></a>유틸리티 클래스
다음과 같은 MFC에 독립적인 유틸리티 클래스가 제공 됩니다.  
  
-   [CImage](../atl-mfc-shared/reference/cimage-class.md) 향상 된 비트맵 지원, 로드 및 이미지를 JPEG, GIF, BMP, 저장 하는 기능을 포함 하 여 제공 하 고 그래픽 PNG (이동식 네트워크)에 서식을 지정 합니다.  
  
-   [CPoint](../atl-mfc-shared/reference/cpoint-class.md) 좌표 (x, y) 쌍을 저장 하기 위한 구현을 제공 합니다.  
  
-   [CRect](../atl-mfc-shared/reference/crect-class.md) 사각형 영역의 좌표를 저장 하기 위한 구현을 제공 합니다.  
  
-   [CSize](../atl-mfc-shared/reference/csize-class.md) 거리, 상대 위치 또는 쌍을 이루는 값을 저장 하기 위한 구현을 제공 합니다.  
  
-   [CString](../atl-mfc-shared/reference/cstringt-class.md) 문자열을 저장 하기 위한 구현을 제공 합니다.  
  
-   [CAdapt](../atl/reference/cadapt-class.md) address-of 연산자를 다시 정의 하는 클래스를 래핑하는 사용 되는 간단한 템플릿입니다.  
  
-   [_U_RECT](../atl/reference/u-rect-class.md) 중 하나를 허용 하는 인수 어댑터 클래스 `RECT` 포인터 또는 포인터의 관점에서 구현 되는 함수에 전달할 수에 대 한 참조입니다.
