# <a name="utility-classes"></a>유틸리티 클래스
다음 MFC에 독립적인 유틸리티 클래스가 제공 됩니다.  
  
-   [CImage](../atl-mfc-shared/reference/cimage-class.md) 개선 된 로드 하 고 GIF, JPEG, BMP에서 이미지를 저장 하는 기능을 포함 하 여 비트맵 지원 제공 및 PNG 이동식 네트워크 그래픽 () 형식을 지정 합니다.  
  
-   [CPoint](../atl-mfc-shared/reference/cpoint-class.md) (x, y) 좌표 쌍을 저장 하는 것에 대 한 구현을 제공 합니다.  
  
-   [CRect](../atl-mfc-shared/reference/crect-class.md) 사각형 영역의 좌표를 저장 하는 것에 대 한 구현을 제공 합니다.  
  
-   [CSize](../atl-mfc-shared/reference/csize-class.md) 거리, 상대 위치 또는 쌍을 이루는 값을 저장 하는 것에 대 한 구현을 제공 합니다.  
  
-   [CString](../atl-mfc-shared/reference/cstringt-class.md) 문자열을 저장 하는 것에 대 한 구현을 제공 합니다.  
  
-   [CAdapt](../atl/reference/cadapt-class.md) 간단한 템플릿으로 address-of 연산자를 다시 정의 하는 클래스를 래핑하는 데 사용 합니다.  
  
-   [_U_RECT](../atl/reference/u-rect-class.md) 를 허용 하는 인수 어댑터 클래스 `RECT` 포인터 또는 참조 포인터 측면에서 구현 되는 함수에 전달할 수 있습니다.
