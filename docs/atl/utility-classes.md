# <a name="utility-classes"></a>유틸리티 클래스

제공 되는 MFC 독립 유틸리티 클래스는 다음과 같습니다.

- [CImage](../atl-mfc-shared/reference/cimage-class.md) JPEG, GIF, BMP 및 PNG (이동식 네트워크 그래픽) 형식으로 이미지를 로드 하 고 저장 하는 기능을 포함 하 여 향상 된 비트맵 지원 기능을 제공 합니다.

- [Cpoint](../atl-mfc-shared/reference/cpoint-class.md) 좌표 (x, y) 쌍을 저장 하기 위한 구현을 제공 합니다.

- [Crect](../atl-mfc-shared/reference/crect-class.md) 사각형 영역의 좌표를 저장 하기 위한 구현을 제공 합니다.

- [Csize](../atl-mfc-shared/reference/csize-class.md) 거리, 상대 위치 또는 쌍을 이루는 값을 저장 하기 위한 구현을 제공 합니다.

- [CString](../atl-mfc-shared/reference/cstringt-class.md) 문자열 저장을 위한 구현을 제공 합니다.

- [Cadapt](../atl/reference/cadapt-class.md) 주소 연산자를 다시 정의 하는 클래스를 래핑하는 데 사용 되는 간단한 템플릿입니다.

- [_U_RECT](../atl/reference/u-rect-class.md) 포인터 `RECT` 또는 참조를 포인터 측면에서 구현 된 함수에 전달할 수 있도록 하는 인수 어댑터 클래스입니다.
