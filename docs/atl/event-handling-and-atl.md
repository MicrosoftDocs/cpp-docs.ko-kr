# <a name="event-handling-and-atl"></a>이벤트 처리 및 ATL

이 섹션에서는 ATL을 사용 하 여 이벤트를 싱크 하는 방법을 보여 줍니다. 이 클래스는 COM 이벤트 처리 원리와 ATL에서 제공 하는 지원을 사용 하 여 이벤트 싱크에 대 한 세부 정보를 다룹니다.

이벤트를 발생 시키고 연결 지점은 구현 하는 방법에 대 한 자세한 내용은 [ATL 연결 요소](../atl/atl-connection-points.md)를 참조 하세요.
