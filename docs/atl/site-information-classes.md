# <a name="site-information-classes"></a>사이트 정보 클래스

다음 클래스를 사용 하 여 개체가 사이트와 통신할 수 있습니다.

- [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) 개체의 사이트에 대 한 포인터를 검색 하 고 설정 합니다. 컨트롤이 아닌 개체에 사용 됩니다.

- [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) 개체의 사이트에 대 한 포인터를 검색 하 고 설정 합니다. 컨트롤에 사용 됩니다.
