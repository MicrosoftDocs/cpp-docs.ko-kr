# <a name="introduction-to-com-and-atl"></a>COM 및 ATL 소개

이 단원에서는 COM 및 ATL에 대해 간략하게 소개 합니다.

## <a name="in-this-section"></a>섹션 내용

[COM 소개](../atl/introduction-to-com.md)<br/>
인터페이스, `IUnknown` 참조 개수, 마샬링 및 집계를 포함 하는 구성 요소 개체 모델의 기본 개념에 대 한 개요를 제공 `QueryInterface` 합니다.

[ATL 소개](../atl/introduction-to-atl.md)<br/>
ATL (액티브 템플릿 라이브러리)이 디자인 된 템플릿 라이브러리 및 ATL 버전 번호에 대해 간략하게 설명 합니다. ATL과 MFC 중 선택에 대 한 권장 사항을 포함 합니다.
