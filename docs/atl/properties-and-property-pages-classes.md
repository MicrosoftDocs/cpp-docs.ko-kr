# <a name="properties-and-property-pages-classes"></a>속성 및 속성 페이지 클래스

다음 클래스는 속성 및 속성 페이지를 지원 합니다.

- [CComDispatchDriver](../atl/reference/atl-typedefs.md#ccomdispatchdriver) 포인터를 통해 개체의 속성을 검색 하거나 설정 합니다 `IDispatch` .

- [CStockPropImpl](../atl/reference/cstockpropimpl-class.md) ATL에서 지 원하는 스톡 속성을 구현 합니다.

- [IPerPropertyBrowsingImpl](../atl/reference/iperpropertybrowsingimpl-class.md) 개체의 속성 페이지에 있는 정보에 액세스 합니다.

- [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) 클라이언트 제공 속성 모음에 개체의 속성을 저장 합니다.

- [IPropertyPageImpl](../atl/reference/ipropertypageimpl-class.md) 속성 시트 내의 특정 속성 페이지를 관리 합니다.

- [IPropertyPage2Impl](../atl/reference/ipropertypage2impl-class.md) 와 유사 `IPropertyPageImpl` 하지만 클라이언트가 속성 페이지에서 특정 속성을 선택할 수도 있습니다.

- [ISpecifyPropertyPagesImpl](../atl/reference/ispecifypropertypagesimpl-class.md) 개체에서 지 원하는 속성 페이지의 Clsid를 가져옵니다.
