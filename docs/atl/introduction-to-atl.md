# <a name="introduction-to-atl"></a>ATL 소개

ATL은 액티브 템플릿 라이브러리, c + + 템플릿 기반 집합이 클래스는 쉽게 만들 수 있습니다, 빠른 구성 요소 개체 모델 (COM) 개체입니다. 포함 하 여 키 COM 기능에 대 한 특수 지원을: 구현의 스톡 [IUnknown](/windows/desktop/api/unknwn/nn-unknwn-iunknown)를 [IClassFactory](/windows/desktop/api/unknwnbase/nn-unknwnbase-iclassfactory)를 [IClassFactory2](/windows/desktop/api/ocidl/nn-ocidl-iclassfactory2), 및 `IDispatch`이중; 인터페이스입니다. 표준 COM 열거자 인터페이스입니다. 연결 지점 분리 인터페이스입니다. 및 ActiveX 컨트롤을 추가 합니다.

ATL 코드는 단일 스레드 개체로, 아파트 모델 개체, 자유 스레드 모델 개체 또는 자유 스레드 및 아파트 모델 개체를 만드는 데 사용할 수 있습니다.
