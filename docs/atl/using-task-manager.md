# <a name="using-task-manager"></a>작업 관리자를 사용 하 여
서비스를 디버깅 하는 가장 간단한 방법 중 하나에 작업 관리자를 사용 하 여입니다. 서비스가 실행 되는 동안 작업 관리자를 시작 하 고 클릭 하 고 **프로세스** 탭 합니다. Exe 파일의 이름을 마우스 오른쪽 단추로 누른 **디버그**합니다. 실행 중인 프로세스에 연결 된 Visual c + +를 시작 합니다. 이제,를 클릭 **중단** 에 **디버그** 메뉴 코드에 중단점을 설정할 수 있도록 합니다. 클릭 **실행** 선택한 중단점을 실행 합니다.
