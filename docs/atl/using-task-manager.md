# <a name="using-task-manager"></a>작업 관리자 사용

서비스를 디버깅 하는 가장 간단한 방법 중 하나는 작업 관리자를 사용 하는 것입니다. 서비스가 실행 되는 동안 작업 관리자를 시작 하 고 **프로세스** 탭을 클릭 합니다. EXE의 이름을 마우스 오른쪽 단추로 클릭 한 다음 **디버그** 를 클릭 합니다. 그러면 실행 중인 프로세스에 연결 된 Visual C++ 시작 됩니다. 이제 **디버그** 메뉴에서 **중단** 을 클릭 하 여 코드에서 중단점을 설정할 수 있습니다. **실행** 을 클릭 하 여 선택한 중단점까지 실행 합니다.
