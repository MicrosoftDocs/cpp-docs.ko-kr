# <a name="error-information-classes"></a>오류 정보 클래스

다음 클래스는 오류 정보를 처리 하는 방법을 나타냅니다.

- [은 isupporterrorinfoimpl](../atl/reference/isupporterrorinfoimpl-class.md) 개체가 [IErrorInfo](/windows/win32/api/oaidl/nn-oaidl-ierrorinfo) 인터페이스를 지원 하는지 여부를 확인 합니다. `IErrorInfo` 클라이언트에 오류 정보를 다시 전파 하도록 허용 합니다.
