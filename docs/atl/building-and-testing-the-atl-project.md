# <a name="building-and-testing-the-atl-project"></a>ATL 프로젝트 빌드 및 테스트

설명 했 듯이 [복합 컨트롤 삽입](../atl/inserting-a-composite-control.md), 프로젝트의 초기 구성 요소 중 하나는 새 복합 컨트롤을 호스트 하는 기본 HTML 페이지입니다. 복합 컨트롤 수정 완료 되 면 클릭 **솔루션 빌드** 또는 **솔루션 다시 빌드** 에서 합니다 **빌드** 메뉴. 프로젝트를 성공적으로 빌드되면, Internet Explorer 또는 다른 브라우저에서 복합 컨트롤 프로젝트의 루트 디렉터리에 있는 HTML 페이지를 로드 하 고 컨트롤의 기능을 테스트 합니다.

또한 테스트 컨테이너 도구 또는 ActiveX 컨트롤을 호스트할 수 있는 다른 응용 프로그램을 사용 하 여 복합 컨트롤을 테스트할 수 있습니다. 테스트 컨테이너에 액세스하는 방법은 [테스트 컨테이너로 속성 및 이벤트 테스트](../mfc/testing-properties-and-events-with-test-container.md) 를 참조하세요.
