# <a name="using-a-template-library"></a>템플릿 라이브러리 사용

템플릿은 매크로와 유사 합니다. 매크로를 사용 하는 경우와 마찬가지로 템플릿을 호출 하면 작성 한 코드에 대 한 적절 한 매개 변수 대체가 확장 됩니다. 그러나 매개 변수로 전달 하는 형식에 따라 새 클래스를 만들 수 있도록 템플릿이이 보다 더 많이 진행 됩니다. 이러한 새 클래스는 템플릿 코드에 표현 된 작업을 수행 하는 형식이 안전한 방법을 구현 합니다.

ATL과 같은 템플릿 라이브러리는 일반적인 c + + 클래스 라이브러리와 다릅니다. 일반적으로 소스 코드 (또는 약간의 지원 런타임)로만 제공 되 고 본질적으로 또는 반드시 계층적으로 계층적으로 제공 되지 않습니다. 원하는 기능을 얻기 위해 클래스에서 파생 하는 대신 템플릿에서 클래스를 인스턴스화합니다.
