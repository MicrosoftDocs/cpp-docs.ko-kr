# <a name="using-a-template-library"></a>템플릿 라이브러리를 사용 하 여

템플릿은 다소 매크로입니다. 매크로와 같이 템플릿을 호출 확장 됩니다 (사용 하 여 적절 한 매개 변수 대체)가 작성 한 코드를 합니다. 그러나 템플릿에 매개 변수로 전달 하는 형식을 기반으로 하는 새 클래스를 만들 수 있도록이 발전 합니다. 이러한 새 클래스에는 템플릿 코드에 표시 된 작업을 수행 하는 형식이 안전한 방법으로 구현 합니다.

ATL 같은 템플릿 라이브러리 달리 기존 c + + 클래스 라이브러리에서 소스 코드로 (또는 소스 코드는 거의 실행된 시간을 지 원하는 사용 하 여)는 일반적으로 제공 되며 기본적으로 또는 반드시 계층적 없습니다. 원하는 기능을 활용 하려면 클래스에서 파생, 대신 클래스 템플릿을 인스턴스화할 수 있습니다.
