# <a name="testing-the-atl-dhtml-control"></a>ATL DHTML 컨트롤 테스트

프로젝트를 만든 후 구축 하 고 샘플 컨트롤을 테스트 합니다. 이 수행 하기 전에 프로젝트를 검사할 클래스 뷰 및 솔루션 탐색기를 사용 합니다. 프로젝트의 요소에 자세히 나와 [DHTML 컨트롤 프로젝트의 요소 식별](../atl/identifying-the-elements-of-the-dhtml-control-project.md)합니다.

#### <a name="to-build-and-test-the-atl-dhtml-control"></a>빌드 및 ATL DHTML 컨트롤 테스트

1. 프로젝트를 빌드합니다. **빌드할** 메뉴에서 클릭 **솔루션 빌드**합니다.

2. 빌드가 완료 되 면 테스트 컨테이너를 엽니다. 참조 [속성 및 이벤트 테스트 컨테이너를 사용 하 여 테스트](../mfc/testing-properties-and-events-with-test-container.md) 테스트 컨테이너에 액세스 하는 방법에 대 한 정보에 대 한 합니다.

3. 테스트 컨테이너의에서 **편집할** 메뉴에서 클릭 **새 컨트롤 삽입**합니다.

4. 에 **컨트롤 삽입** 대화 상자, 목록 상자에서 컨트롤을 선택 합니다. ATL 컨트롤 마법사에 표시 된 짧은 이름에 해당 이름은 기반 기억 하십시오. **확인**을 클릭합니다.

5. 컨트롤을 검사 합니다. 스크롤 막대에는 참고 합니다. 스크롤 막대를 활성화 하려면 컨트롤의 크기를 조정 하려면 컨트롤의 핸들을 사용 합니다.

6. 컨트롤의 단추를 테스트 합니다. 배경색 단추를 나타내는 색으로 변경 합니다.

7. 테스트 컨테이너를 닫습니다.

다음 단계로 [DHTML 컨트롤 수정](../atl/modifying-the-atl-dhtml-control.md)합니다.
