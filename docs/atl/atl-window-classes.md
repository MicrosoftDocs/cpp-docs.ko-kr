# <a name="atl-window-classes"></a>ATL 창 클래스
ATL 사용 하 고 windows를 구현할 수 있는 여러 클래스가 포함 되어 있습니다. 다른 ATL 클래스와 마찬가지로 이러한 클래스는 코드에는 오버 헤드를 부여 하지 않는 효율적인 구현을 제공 합니다.  
  
 이 섹션 ATL 창 클래스에 설명 하 고 사용 하는 방법에 설명 합니다.

 [ATL 창 클래스 소개](../atl/introduction-to-atl-window-classes.md)  
 간략하게 각 ATL 창 클래스를 설명 하 고에 대 한 참조 자료에 대 한 링크를 제공 합니다.  
  
 [창 사용](../atl/using-a-window.md)  
 사용 하는 방법에 설명 `CWindow` 창을 조작 합니다.  
  
 [창 구현](../atl/implementing-a-window.md)  
 메시지 처리기, 메시지 맵 및 사용 하 여 설명 `CWindowImpl`합니다. 클 래 싱 및 하위 클래스에 대 한 세부 정보를 포함합니다.  
  
 [대화 상자 구현](../atl/implementing-a-dialog-box.md)  
 대화 상자 클래스를 추가 하기 위한 두 가지 방법에 설명 하 고 코드 샘플을 보여 줍니다.  
  
 [포함된 창 사용](../atl/using-contained-windows.md)  
 ATL에서 해당 메시지를 자체 클래스에서 처리 하지 않고 컨테이너 개체에 위임 하는 포함 된 창에 설명 합니다.  
  
 [창 특성 이해](../atl/understanding-window-traits.md)  
 Atl에서 창 특성 클래스를 설명합니다. 이러한 클래스는 window 개체를 만드는 데 사용 되는 스타일을 표준화 하기 위한 간단한 방법을 제공 합니다.
