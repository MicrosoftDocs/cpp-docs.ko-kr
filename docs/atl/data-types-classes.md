# <a name="data-types-classes"></a>데이터 형식 클래스

다음 클래스는 c + + 데이터 형식을 래핑합니다.

- [CComBSTR](../atl/reference/ccombstr-class.md) `BSTR` 데이터 형식을 래핑합니다.

- [CComVariant](../atl/reference/ccomvariant-class.md) `VARIANT` 데이터 형식을 래핑합니다.

- [CComCurrency](../atl/reference/ccomcurrency-class.md) 개체를 만들고 관리 하기 위한 메서드 및 연산자를 포함 `CURRENCY` 합니다.
