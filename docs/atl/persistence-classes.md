# <a name="persistence-classes"></a>지 속성 클래스

다음 클래스는 개체 지 속성을 구현 합니다.

- [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) 클라이언트에서 개체의 속성을 로드 하 고 속성 모음에 저장할 수 있습니다.

- [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) 클라이언트에서 개체의 영구 데이터를 로드 하 여 스트림에 저장할 수 있습니다.

- [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) 클라이언트에서 개체의 영구 데이터를 로드 하 여 저장소에 저장할 수 있습니다.
