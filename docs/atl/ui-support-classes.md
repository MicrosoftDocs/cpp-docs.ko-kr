# <a name="ui-support-classes"></a>UI 지원 클래스
다음 클래스에는 일반 UI 지원을 제공합니다.  
  
-   [IDocHostUIHandlerDispatch](../atl/reference/idochostuihandlerdispatch-interface.md) Microsoft HTML 구문 분석 및 렌더링 엔진에 대 한 인터페이스입니다.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) 컨트롤과 컨테이너 통신 하는 주요 메서드를 제공 합니다. 활성화 및 비활성화의 전체 컨트롤을 관리합니다.  
  
-   [IOleInPlaceObjectWindowlessImpl](../atl/reference/ioleinplaceobjectwindowlessimpl-class.md) 전체 컨트롤의 재 활성화를 관리 합니다. 창 없는 컨트롤을 메시지를 받을으로 끌어서 놓기 작업에 참여할 수 있습니다.  
  
-   [IOleInPlaceActiveObjectImpl](../atl/reference/ioleinplaceactiveobjectimpl-class.md) 전체 컨트롤과 해당 컨테이너 간에 통신 하는 데 도움이 됩니다.  
  
-   [IViewObjectExImpl](../atl/reference/iviewobjecteximpl-class.md) 는 컨트롤 자체를 직접 표시 하 고 디스플레이의 변경 사항을 컨테이너에 알리는 수 있도록 합니다. 깜빡임 없는 그리기, 사각형이 아닌 투명 한 컨트롤 및 적중 테스트에 대 한 지원을 제공합니다.
