# <a name="ui-support-classes"></a>UI 지원 클래스

다음 클래스는 일반적인 UI 지원을 제공 합니다.

- [IDocHostUIHandlerDispatch](../atl/reference/idochostuihandlerdispatch-interface.md) Microsoft HTML 구문 분석 및 렌더링 엔진에 대 한 인터페이스입니다.

- [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) 컨테이너가 컨트롤과 통신 하는 데 사용 하는 주요 메서드를 제공 합니다. 내부 컨트롤의 활성화 및 비활성화를 관리 합니다.

- [IOleInPlaceObjectWindowlessImpl](../atl/reference/ioleinplaceobjectwindowlessimpl-class.md) 내부 컨트롤의 다시 활성화를 관리 합니다. 창 없는 컨트롤에서 메시지를 받고 끌어서 놓기 작업에 참여할 수 있도록 합니다.

- [IOleInPlaceActiveObjectImpl](../atl/reference/ioleinplaceactiveobjectimpl-class.md) 내부 컨트롤과 해당 컨테이너 간의 통신을 지원 합니다.

- [Iviewobjecteximpl](../atl/reference/iviewobjecteximpl-class.md) 컨트롤이 자신을 직접 표시 하 고 표시 되는 변경 내용에 대 한 컨테이너를 알리도록 설정 합니다. 깜빡임 없는 그리기, 사각형이 아닌 투명 한 컨트롤 및 적중 테스트에 대 한 지원을 제공 합니다.
