# <a name="mmc-snap-in-classes"></a>MMC Snap-In 클래스

다음 클래스는 MMC (Microsoft Management Console) 스냅인 구성 요소를 개발 하기 위한 지원을 제공 합니다.

- [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) 메뉴 항목 및 도구 모음을 추가 하 고 스냅인 노드에 대 한 명령을 적절 한 처리기 함수에 전달 하는 등의 스냅인 노드 개체를 구현 합니다.

- [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) 스냅인 속성 페이지 개체를 구현 합니다.
