# <a name="mmc-snap-in-classes"></a>MMC 스냅인 클래스

Microsoft Management Console (MMC) 스냅인에서 구성 요소를 개발에 대 한 지원을 제공 하는 다음 클래스:

- [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) 메뉴 항목 및 도구 모음을 추가 하 고 적절 한 처리기 함수에 스냅인에서 노드에 대 한 명령을 전달 등 스냅인 노드 개체를 구현 합니다.

- [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) 스냅인 속성 페이지 개체를 구현 합니다.
