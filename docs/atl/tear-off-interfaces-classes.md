-   [CComTearOffObject](../atl/reference/ccomtearoffobject-class.md) 구현 `IUnknown` 분리 인터페이스입니다.  
  
-   [CComCachedTearOffObject](../atl/reference/ccomcachedtearoffobject-class.md) 구현 `IUnknown` "캐시" 분리 인터페이스에 대 한 합니다.
