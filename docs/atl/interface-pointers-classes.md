# <a name="interface-pointers-classes"></a>인터페이스 포인터 클래스
지정 된 인터페이스 포인터를 관리 하는 다음 클래스:  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) 자동 참조 계산을 수행 합니다.  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) 비슷하지만 `CComPtr`, 하지만 또한 인터페이스를 자동으로 쿼리를 수행 합니다.  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md) COM의 배열을 생성 하는 경우에 유용한 제공 메서드 인터페이스 포인터입니다.  
  
-   [CInterfaceList](../atl/reference/cinterfacelist-class.md) 인터페이스 포인터를 제공 메서드 COM의 목록을 구성할 때 유용 합니다.  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md) 인터페이스 포인터와 전역 인터페이스 테이블 (GIT)를 처리 하기 위한 메서드를 제공 합니다.
