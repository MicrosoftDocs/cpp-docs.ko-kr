# <a name="thread-pooling-classes"></a>스레드 풀링 클래스

다음 클래스는 스레드 풀링을 지원 합니다.

- [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) 여러 스레드 풀 아파트를 지 원하는 EXE 모듈을 구현 합니다.

- [CComApartment](../atl/reference/ccomapartment-class.md) 스레드 풀링된 EXE 모듈에서 아파트를 관리 합니다.

- [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) EXE 모듈의 스레드 선택을 관리 합니다.
