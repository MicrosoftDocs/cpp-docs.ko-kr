# <a name="atl-registry-component-registrar"></a>ATL 레지스트리 구성 요소(등록자)

ATL 등록자는 사용자 지정 인터페이스를 통해 시스템 레지스트리에 최적화 된 액세스를 제공 합니다. 등록자는 자유 스레드된 코드 이며, c + + 클라이언트에 대 한 코드의 정적 링크를 허용 합니다.

> [!NOTE]
> ATL 등록자의 소스 코드는 atlmfc\include\atliface.h.에서 찾을 수 있습니다.
