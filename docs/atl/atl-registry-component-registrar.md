# <a name="atl-registry-component-registrar"></a>ATL 레지스트리 구성 요소(등록자)

ATL 레지스트라는 사용자 지정 인터페이스를 통해 시스템 레지스트리에 대한 최적화된 액세스를 제공합니다. 레지스트라는 무료 스레드이며 C++ 클라이언트에 대한 코드의 정적 연결을 허용합니다.

> [!NOTE]
> ATL 레지스트라의 소스 코드는 atlmfc\include\atliface.h에서 찾을 수 있습니다.
