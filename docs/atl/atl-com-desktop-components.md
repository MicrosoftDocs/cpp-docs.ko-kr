# <a name="atl-com-desktop-components"></a>ATL COM 데스크톱 구성 요소

ATL 참조는 COM (구성 요소 개체 모델) 개체의 프로그래밍을 간소화 하는 템플릿 기반 c + + 클래스 집합인 ATL (액티브 템플릿 라이브러리)을 문서화 합니다. COM은 Windows에서 소프트웨어 구성 요소를 만들고 사용 하기 위한 이진 사양입니다. ATL을 완벽 하 게 활용 하려면 COM을 사용 하는 것이 좋습니다. COM에 대 한 자세한 내용은 [com (구성 요소 개체 모델)](/windows/win32/com/component-object-model--com--portal)을 참조 하세요.
