# <a name="debugging-and-exceptions-classes"></a>디버깅 및 예외 클래스

이러한 클래스는 예외 처리 및 디버깅에 대 한 지원을 제공 합니다.

- [CAtlDebugInterfacesModule](../atl/reference/catldebuginterfacesmodule-class.md) 이 클래스는 디버깅 인터페이스에 대 한 지원을 제공 합니다.

- [CAtlException](../atl/reference/catlexception-class.md) 이 클래스는 ATL 예외를 정의 합니다.
