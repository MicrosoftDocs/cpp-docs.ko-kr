버전 4.0부터 MFC 데이터베이스 프로그래밍 사용 합니다 [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) 하 고 [COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md) 날짜 및 시간 데이터를 나타내기 위해 클래스.
