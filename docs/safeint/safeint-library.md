# <a name="safeint-library"></a>SafeInt 라이브러리

**SafeInt**는 애플리케이션에서 수학 연산을 수행할 때 발생할 수 있는 정수 오버플로를 방지하기 위해 MSVC, GCC 또는 Clang과 함께 사용할 수 있는 이식 가능한 라이브러리입니다. 이 라이브러리의 최신 버전은 [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt)에 있습니다.
