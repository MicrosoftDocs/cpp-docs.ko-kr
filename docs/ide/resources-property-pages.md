 액세스 하는 방법에 대 한 내용은 **리소스** 속성 페이지 참조 [프로젝트 속성 작업](../ide/working-with-project-properties.md)합니다.  
  
 프로그래밍 방식으로 이러한 속성에 액세스 하려면 참조 <xref:Microsoft.VisualStudio.VCProjectEngine.VCResourceCompilerTool>합니다.
