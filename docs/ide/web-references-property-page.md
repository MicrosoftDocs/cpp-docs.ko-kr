# <a name="web-references-property-page"></a>웹 참조 속성 페이지

**웹 참조** 속성 페이지는 XML Web services 프록시 클래스가 생성되는 방법을 지정합니다. 프로젝트에 웹 참조를 추가하는 경우 XML Web services 프록시 클래스가 생성됩니다.

**웹 참조** 속성 페이지에는 다음과 같은 속성이 포함되어 있습니다.

- **출력 파일**

   XML Web services 프록시 클래스를 포함할 파일의 이름입니다.

- **시작 배너 표시 안 함**

   웹 서비스 기술 언어 도구(Wsdl.exe)의 배너를 표시하지 않습니다.

- **네임스페이스**

   생성된 웹 프록시의 이름을 지정합니다.

- **추가 참조**

   프록시 DLL에서 참조하는 추가 DLL을 지정합니다.

**웹 참조** 속성 페이지에 액세스하는 방법에 대한 자세한 내용은 [프로젝트 속성 작업](../ide/working-with-project-properties.md)을 참조하세요.
