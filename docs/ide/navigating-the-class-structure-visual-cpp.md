# <a name="navigate-the-class-structure"></a>클래스 구조 탐색

**클래스 뷰** 또는 **개체 브라우저**를 사용하여 프로젝트 클래스 구조를 탐색할 수 있습니다. 프로젝트 클래스 구조에는 함수, 상속 및 인터페이스가 포함될 수 있습니다. 자세한 내용은 [코드 구조 보기](/visualstudio/ide/viewing-the-structure-of-code)를 참조하세요.

함수 선언으로 이동하려면 클래스 뷰에서 함수 이름을 두 번 클릭합니다. 클래스 정의로 이동하려면 클래스 뷰에서 클래스 이름을 두 번 클릭합니다.
