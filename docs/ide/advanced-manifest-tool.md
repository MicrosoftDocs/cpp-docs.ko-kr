# <a name="advanced-manifest-tool-configuration-properties-ltprojectnamegt-property-pages-dialog-box"></a>구성 속성, 매니페스트 도구, 고급 &lt;Projectname&gt; 속성 페이지 대화 상자
이 대화 상자를 사용 하 여 고급 옵션을 지정 하려면 [Mt.exe](http://msdn.microsoft.com/library/aa375649)합니다.  
  
 이 속성 페이지 대화 상자에 액세스 하려면 프로젝트 또는 속성 시트에 대 한 속성 페이지를 엽니다. 확장 된 **매니페스트 도구** 노드 아래의 **구성 속성**를 선택한 후 **고급**합니다.  
  
## <a name="uielement-list"></a>UI 요소 목록  
 **파일 해시 업데이트**  
 /Hashupdate 옵션을 사용 하 여 매니페스트 도구가에 지정 된 파일의 해시를 계산 지정 `<file>` 요소 및 계산 된 값의 해시 특성을 업데이트 합니다.  
  
 **파일 해시 검색 경로 업데이트 합니다.**  
 참조 되는 파일에 대 한 검색 경로 지정 `<file>` 요소입니다. 이 옵션은 또한 /hashupdate 옵션을 사용 합니다.
