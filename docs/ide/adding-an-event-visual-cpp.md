# <a name="adding-an-event-visual-c"></a>이벤트 추가(Visual C++)
클래스 뷰를 추가할 수 있습니다 사용 하 여 이벤트는 [이벤트 추가 마법사](../ide/add-event-wizard.md) 에서 컨트롤 클래스에만 프로그램 [MFC ActiveX 컨트롤](../mfc/reference/creating-an-mfc-activex-control.md) 프로젝트. 사용 하 여 이벤트 다른 형식의 프로젝트를 추가 하려는 경우는 **이벤트** 단추는 [속성 창](/visualstudio/ide/reference/properties-window)합니다.  
  
### <a name="to-add-an-event-to-your-mfc-activex-control-project"></a>MFC ActiveX 컨트롤 프로젝트에 이벤트를 추가 하려면  
  
1.  클래스 뷰에서 프로젝트에 클래스를 표시 하려면 프로젝트 노드를 확장 합니다.  
  
2.  프로젝트의 컨트롤 클래스를 마우스 오른쪽 단추로 클릭 합니다.  
  
3.  바로 가기 메뉴를 클릭 **추가**, 클릭 하 고 **이벤트 추가** 이벤트 추가 마법사를 표시 합니다.  
  
4.  적절 한 마법사 상자에 이벤트 정보를 제공 합니다.  
  
5.  클릭 **마침** 이벤트 프로젝트에 추가 합니다.
