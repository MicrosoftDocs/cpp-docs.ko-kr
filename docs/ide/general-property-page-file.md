# <a name="general-property-page-file"></a>일반 속성 페이지(파일)

파일을 선택할 때 **솔루션 탐색기**, **일반** 속성 페이지에는 **구성 속성** 노드는 다음 속성이 포함:

**빌드에서 제외**  
현재 구성에 대 한 빌드 파일을 되어야 하는지 여부를 지정 합니다.

프로그래밍 방식으로 이 속성에 액세스하려면 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.ExcludedFromBuild%2A>을 참조하세요.

**도구**  
이 파일을 빌드하는 데 사용 됩니다 하는 도구입니다. 참조 [사용자 지정 빌드 도구 지정](../ide/specifying-custom-build-tools.md) 자세한 정보에 대 한 합니다.

프로그래밍 방식으로 이 속성에 액세스하려면 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.Tool%2A>을 참조하세요.

액세스 하는 방법에 대 한 내용은 **일반** 속성 페이지에는 **구성 속성** 노드를 참조 [프로젝트 속성 작업](../ide/working-with-project-properties.md)합니다.

비 Windows 프로젝트에 대 한 참조 [Linux c + + 속성 페이지 참조](../linux/prop-pages-linux.md)<!-- or [C++ Cross Platform Property Page Reference](../linux/prop-pages-linux.md)-->합니다.
