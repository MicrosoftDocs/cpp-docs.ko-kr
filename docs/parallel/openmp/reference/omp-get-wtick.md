# <a name="ompgetwtick"></a>omp_get_wtick

프로세서 클록 틱 사이의 시간 (초) 수를 반환합니다.

## <a name="syntax"></a>구문

```
double omp_get_wtick( );
```

## <a name="remarks"></a>설명

자세한 내용은 [3.3.2 omp_get_wtick 함수](../../../parallel/openmp/3-3-2-omp-get-wtick-function.md)합니다.

## <a name="example"></a>예제

참조 [omp_get_wtime](../../../parallel/openmp/reference/omp-get-wtime.md) 사용 하는 예제에 대 한 `omp_get_wtick`합니다.
