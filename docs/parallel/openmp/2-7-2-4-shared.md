# <a name="2724-shared"></a>2.7.2.4 공유됨

이 절에 표시 되는 변수를 공유 합니다 *변수 목록* 팀의 모든 스레드 간에 합니다. 팀 내에서 모든 스레드는 동일한 저장소 영역에 대 한 액세스 **공유** 변수입니다.

구문의 합니다 **공유** 절은 다음과 같습니다.

```
shared(variable-list)
```
