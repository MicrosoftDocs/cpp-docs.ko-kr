# <a name="a28---use-of-numthreads-clause"></a>A.28   num_threads 절 사용
다음 예제는 `num_threads` 절 ([섹션 2.3](../../parallel/openmp/2-3-parallel-construct.md) 8 페이지에). 병렬 영역 최대 10 개 스레드가 실행 됩니다.  
  
```  
#include <omp.h>  
main()  
{  
    omp_set_dynamic(1);  
    ...  
    #pragma omp parallel num_threads(10)  
    {  
        ... parallel region ...  
    }  
}  
```
