# <a name="27-data-environment"></a>2.7 데이터 환경
이 섹션에서는 지시문 및 병렬 영역을 실행 하는 동안 다음과 같이 데이터 환경을 제어 하기 위한 여러 절을 제공 합니다.  
  
-   A **threadprivate** 지시문 되도록 파일 범위, 네임 스페이스 범위 또는 블록 범위의 정적 변수 로컬 스레드에 제공 됩니다 (다음 섹션 참조).  
  
-   기간 또는 병렬 작업 공유 구문에 대 한 변수 공유 특성을 제어 하는 대리자에 지정 될 수 있는 절에 설명 된 [섹션 2.7.2](../../parallel/openmp/2-7-2-data-sharing-attribute-clauses.md) 페이지 25입니다.
