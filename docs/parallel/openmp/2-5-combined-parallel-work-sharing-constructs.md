# <a name="25-combined-parallel-work-sharing-constructs"></a>2.5 결합된 병렬 작업 공유 구문

결합 된 병렬 작업 공유 구문은 하나의 작업 공유 생성자를 포함 하는 병렬 영역이 지정 하기 위한 바로 가기입니다. 이러한 지시문의 의미 체계는 동일 명시적으로 지정 합니다는 **병렬** 는 단일 작업 공유 구문 뒤에 지시문입니다.

다음 섹션에서는 결합 된 병렬 작업 공유 구문에 설명 합니다.

- 합니다 **에 대 한 병렬** 지시문입니다.

- 합니다 **섹션에서는 병렬** 지시문입니다.
