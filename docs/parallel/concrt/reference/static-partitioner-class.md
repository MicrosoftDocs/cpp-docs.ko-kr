# <a name="staticpartitioner-class"></a>static_partitioner 클래스

`static_partitioner` 클래스는 `parallel_for`에서 반복하는 범위의 정적 분할을 나타냅니다. 파티셔너는 기본 스케줄러에서 사용할 수 있는 작업자 수만큼의 청크로 범위를 나눕니다.

## <a name="syntax"></a>구문

```
class static_partitioner;
```
