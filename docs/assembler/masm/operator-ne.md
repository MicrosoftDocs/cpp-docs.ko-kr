# <a name="operator-ne"></a>연산자 NE

*Expression1* 가 *식 2* 와 같지 않으면 true (-1)를 반환 하 고, 그렇지 않으면 false (0)를 반환 합니다.

## <a name="syntax"></a>Syntax

> *expression1* **NE** *식*
