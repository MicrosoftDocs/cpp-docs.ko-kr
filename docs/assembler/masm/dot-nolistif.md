# <a name="nolistif"></a>.NOLISTIF

조건이 false (0)로 평가 되는 조건부 블록의 목록을 표시 하지 않습니다.

## <a name="syntax"></a>Syntax

> **.NOLISTIF**

## <a name="remarks"></a>설명

이것이 기본값입니다. 와 동일 [합니다. SFCOND](dot-sfcond.md).
