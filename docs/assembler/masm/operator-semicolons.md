# <a name="operator-"></a>연산자 ;;

매크로 정의에만 표시 되는 매크로에서 *텍스트* 를 주석으로 처리 합니다. 이 목록에는 매크로가 확장 된 *텍스트가* 표시 되지 않습니다.

## <a name="syntax"></a>구문

> **;;** *텍스트*
