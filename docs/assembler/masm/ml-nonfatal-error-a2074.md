# <a name="ml-nonfatal-error-a2074"></a>ML 심각하지 않은 오류 A2074

**세그먼트 레지스터를 통해 레이블에 액세스할 수 없습니다.**

세그먼트 또는 그룹으로 간주 되지 않는 세그먼트 레지스터를 통해 레이블에 액세스 하려고 했습니다.
