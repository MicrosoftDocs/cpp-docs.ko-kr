# <a name="operator--"></a>연산자 -

*Expression1* 를 빼기 *식*으로 반환 합니다. 두 번째 연산자는 *식*의 부호를 반대로 바꿉니다.

## <a name="syntax"></a>구문

> *expression1* **-** *expression2*
>
> __-__ *식*
