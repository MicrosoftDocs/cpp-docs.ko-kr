# <a name="ml-nonfatal-error-a2137"></a>ML 심각하지 않은 오류 A2137

**호출할 인수가 너무 적습니다.**

[INVOKE](invoke.md) 지시문을 사용 하 여 전달 된 인수 수가 호출 되는 프로시저에 대해 프로토타입에 지정 된 필수 매개 변수 개수 보다 작습니다.
