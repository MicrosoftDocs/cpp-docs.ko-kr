# <a name=""></a>\@\@:

간에만 인식할 수 있는 코드 레이블을 정의 *label1* 하 고 *label2*여기서 *label1* 은 코드의 시작 또는 이전 `@@:` 레이블과 *label2* 은 코드의 끝 또는 다음 `@@:` 레이블. 참조 [ \@B](../../assembler/masm/at-b.md) 하 고 [ \@F](../../assembler/masm/at-f.md)합니다.
