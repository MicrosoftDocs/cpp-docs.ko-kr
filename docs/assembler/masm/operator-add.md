# <a name="operator-"></a>operator +
첫 번째 연산자는 반환 *expression1* 플러스 *expression2*합니다. 두 번째 연산자의 부호를 반대로 바꿉니다. *식*합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
      expression1 + expression2  
-expression  
```
