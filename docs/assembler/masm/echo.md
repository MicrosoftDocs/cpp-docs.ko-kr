# <a name="echo"></a>ECHO

표준 출력 장치 (기본적으로 화면)로 *메시지* 를 표시 합니다.

## <a name="syntax"></a>Syntax

> **ECHO** *메시지*

## <a name="remarks"></a>설명

[% OUT](percent-out.md)과 동일 합니다.
