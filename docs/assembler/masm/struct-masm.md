# <a name="struct"></a>STRUCT

지정 된 *필드 선언을*포함 하는 구조체 형식을 선언 합니다. 각 필드는 올바른 데이터 정의 여야 합니다. [STRUC](struc.md)와 동일 합니다.

## <a name="syntax"></a>구문

> *이름* **STRUCT** ⟦*alignment*⟧ ⟦ __,__ **고유**하지 않은 ⟧ *필드 선언*\
> *이름* **끝**

## <a name="remarks"></a>주의

*이름* 인수는 여는 문과 닫는 문에서 동일 해야 합니다.
