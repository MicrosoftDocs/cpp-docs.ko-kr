# <a name="ml-fatal-error-a1016"></a>ML 심각한 오류 A1016

**내부 어셈블러 오류**

ML.exe라는 MASM 드라이버가 시스템 오류를 생성했습니다.

오류가 발생한 환경을 기록하고 Microsoft Corporation에 알립니다. 기술 지원 서비스는 [https://support.microsoft.com/](https://support.microsoft.com/)에서 제공됩니다.
