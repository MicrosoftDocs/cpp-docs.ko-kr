# <a name="operator-this"></a>연산자 THIS
피연산자에 지정 된 반환 `type` 오프셋과 세그먼트 값은 현재 위치 카운터 값으로 동일 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
THIS type  
```
