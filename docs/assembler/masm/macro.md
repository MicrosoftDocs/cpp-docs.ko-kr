# <a name="macro"></a>MACRO

매크로 블록을 *이름* 이라고 표시 하 고 매크로가 호출 될 때 전달 되는 인수에 대 한 *매개 변수* 자리 표시자를 설정 합니다.

## <a name="syntax"></a>구문

> *name***매크로** ⟦*parameter* ⟦ **: 필수** | : =*기본* |  **: VARARG**⟧ ... ⟧\
> *문*\
> **Endm** ⟦*값*⟧

## <a name="remarks"></a>주의

매크로 함수는 호출 하는 문에 *값* 을 반환 합니다.
