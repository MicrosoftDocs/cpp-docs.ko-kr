# <a name="codesize"></a>\@CodeSize

0은 **작음**, **소형**, **컴팩트**및 **플랫** 모델의 경우 0이 고, **중간**, **크고, 큰** **모델의** 경우 1입니다 (숫자는 1).

## <a name="syntax"></a>구문

> **\@CodeSize**
