# <a name="ml-warning-a4012"></a>ML 경고 A4012

**'CODE' 클래스를 사용 하지는 세그먼트에 대 한 줄 번호 정보**

"코드입니다."로 끝나는 클래스 이름이 없는 세그먼트에 지시 했습니다. 어셈블러에는 이러한 지침은 CodeView 정보가 생성 하지 않았습니다.

CodeView "코드입니다."를 사용 하 여 끝나지 않는 클래스 이름의 세그먼트에 코드를 사용 하 여 모듈을 처리할 수 없습니다.
