# <a name="ml-warning-a4012"></a>ML 경고 A4012
**줄 번호 정보 없이 클래스 'CODE' 세그먼트에 대 한**  
  
 "CODE"로 끝나는 클래스 이름이 되어 있지 않은 세그먼트의 지시 했습니다. 어셈블러는이 지침에 대 한 CodeView 정보를 생성 하지 않았습니다.  
  
 CodeView "CODE"로 끝나지 않는 클래스 이름으로 세그먼트의 코드를 사용 하 여 모듈을 처리할 수 없습니다.
