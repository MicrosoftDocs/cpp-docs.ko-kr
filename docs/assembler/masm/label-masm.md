# <a name="label"></a>LABEL

현재 위치 카운터 값과 지정 된 *qualifiedType* 를 *이름* 에 할당 하 여 새 레이블을 만듭니다.

## <a name="syntax"></a>Syntax

> *이름* **레이블** *qualifiedType*
>
> *이름* **레이블** ⟦**NEAR**  |  **FAR**  |  **PROC**⟧ **PTR** ⟦*qualifiedType*⟧
