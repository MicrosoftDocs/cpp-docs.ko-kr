# <a name="label-masm"></a>LABEL (MASM)
현재 위치 카운터 값을 할당 하 여 새 레이블을 만듭니다와 주어진 `type` 를 *이름*합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
