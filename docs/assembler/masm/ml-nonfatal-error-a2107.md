# <a name="ml-nonfatal-error-a2107"></a>ML 심각하지 않은 오류 A2107

**암시적으로 멀리 이동 하거나 가까운 레이블에 대 한 호출을 사용할 수 없습니다.**

다른 세그먼트의 근거리 레이블를 암시적으로 이동 하거나 호출 하려고 했습니다.
