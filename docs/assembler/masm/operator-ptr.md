# <a name="operator-ptr"></a>연산자 PTR

첫 번째 연산자는 *식이* 지정 된 *형식이* 있는 것으로 처리 되도록 합니다. 두 번째 연산자는 *형식* 에 대 한 포인터를 지정 합니다.

## <a name="syntax"></a>Syntax

> *형식* **PTR** *식*
>
> \[*distance*] **PTR** *형식*
