# <a name="real8"></a>REAL8

각 *이니셜라이저*에 대 한 배정밀도 (8 바이트) 부동 소수점 숫자를 할당 하 고 필요에 따라 초기화 합니다.

## <a name="syntax"></a>구문

> **REAL8** *이니셜라이저* ⟦ __,__ *이니셜라이저의* 이름 ... ⟧
