# <a name="real8"></a>REAL8

할당 하 고 필요에 따라 각 배정밀도 (8 바이트) 부동 소수점 숫자를 초기화 *이니셜라이저*합니다.

## <a name="syntax"></a>구문

> 이름 REAL8 이니셜라이저 [[, 이니셜라이저]]...
