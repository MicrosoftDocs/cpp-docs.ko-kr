# <a name="elseif2"></a>ELSEIF2

**SETIF2** 가 **TRUE**인 경우 모든 어셈블리 pass에서 [ELSEIF](elseif-masm.md) 블록이 평가 됩니다.

## <a name="syntax"></a>구문

> **ELSEIF2**

## <a name="remarks"></a>주의

MASM 5.1 vs MASM 6.1의 2 단계 동작에 대 한 자세한 내용은 [IF1 및 IF2](if2.md)를 참조 하세요.
