# <a name="endprolog"></a>.ENDPROLOG

프롤로그 선언의 끝 신호를 보냅니다.

## <a name="syntax"></a>구문

> .ENDPROLOG

## <a name="remarks"></a>설명

간의 지역 외부에서 프롤로그 선언 중 하나를 사용 하는 것 [PROC](../../assembler/masm/proc.md) 프레임 및 합니다. 프롤로그 끝입니다.

자세한 내용은 [MASM (ml64.exe) x64](../../assembler/masm/masm-for-x64-ml64-exe.md)합니다.
