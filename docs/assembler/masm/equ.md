# <a name="equ"></a>EQU

첫 번째 지시문은 *식* 의 숫자 값을 *이름*에 할당 합니다.

## <a name="syntax"></a>구문

> *이름* **같음** *식*
>
> *이름* **같음** __\<__ *텍스트* __>__

## <a name="remarks"></a>주의

나중에 *이름을* 다시 정의할 수 없습니다.

두 번째 지시문은 지정 된 *텍스트* 를 *이름*에 할당 합니다. 이 *이름* 에는 나중에 다른 *텍스트* 를 할당할 수 있습니다. [TEXTEQU](textequ.md)를 참조 하세요.
