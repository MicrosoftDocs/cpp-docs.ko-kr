# <a name="instr"></a>@InStr
첫 번째 항목을 찾습니다 매크로 함수 *string2* 에 *string1*인스턴스에서 시작 *위치* 내 *string1*합니다. 경우 *위치* 표시 되지 않으면의 시작 부분에서 검색을 시작 *string1*합니다. 위치 정수 0을 반환 하면 반환 *string2* 찾을 수 없습니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
@InStr( [[position]], string1, string2 )  
```
