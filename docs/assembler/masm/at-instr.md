# <a name="instr"></a>@InStr

첫 번째 항목을 찾습니다. 매크로 함수 *string2* 에 *string1*부터 *위치* 내 *string1*합니다. 하는 경우 *위치* 나타나지 않으면의 시작 부분에서 검색이 시작 *string1*합니다. 경우는 위치 정수 또는 0을 반환 합니다 *string2* 찾을 수 없습니다.

## <a name="syntax"></a>구문

> @InStr([[위치]], string1, string2)
