# <a name="ifndef"></a>IFNDEF
어셈블리를 부여 *이름* 정의 되지 않았습니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
IFNDEF name  
```  
  
## <a name="remarks"></a>설명  
 참조 [IF](../../assembler/masm/if-masm.md) 전체 구문에 대 한 합니다.
