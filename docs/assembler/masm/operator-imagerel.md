# <a name="operator-imagerel"></a>연산자 IMAGEREL

*식*의 상대 오프셋 이미지를 반환 합니다.

## <a name="syntax"></a>구문

> **Imagerel** *식*

## <a name="remarks"></a>주의

결과 값은 대체로 RVA 또는 상대 가상 주소 라고 합니다.

IMAGEREL은 COFF 개체 내보내기와 함께 사용할 수 있습니다.
