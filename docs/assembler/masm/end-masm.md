# <a name="end"></a>End

모듈의 끝을 표시 하 고 선택적으로 프로그램 진입점을 *Procid* 로 설정 합니다.

## <a name="syntax"></a>Syntax

> **END** ⟦*procid*⟧

## <a name="remarks"></a>설명

*Procid* 인수는 32 비트 MASM 에서만 유효 합니다.
