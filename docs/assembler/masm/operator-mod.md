# <a name="operator-mod"></a>연산자 MOD

*Expression1* 를 *식 2* 로 나눌 때 나머지 (모듈로)의 정수 값을 반환 합니다.

## <a name="syntax"></a>Syntax

> *expression1* **MOD** *식*
