# <a name="substr"></a>SUBSTR

*위치* 에서 시작 하 여 *textitem* 의 부분 문자열을 반환 합니다. *Textitem* 은 리터럴 문자열,가 앞에 오는 상수 `%` 또는 매크로 함수에서 반환 된 문자열 일 수 있습니다.

## <a name="syntax"></a>Syntax

> *이름* **SUBSTR** *textitem*__,__ *position* ⟦__,__ *length*⟧
