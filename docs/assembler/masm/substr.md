# <a name="substr"></a>SUBSTR
부분 문자열을 반환 *textitem*에서 시작 하는 *위치*합니다. *textitem* 앞에 상수는 리터럴 문자열이 될 수는 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
name SUBSTR textitem, position [[, length]]  
```
