# <a name="substr"></a>SUBSTR

부분 문자열을 반환 *textitem*부터 *위치*합니다. *textitem* 리터럴 문자열을 앞에 상수 수를 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.

## <a name="syntax"></a>구문

> SUBSTR textitem, 위치 [[, 길이]] 이름
