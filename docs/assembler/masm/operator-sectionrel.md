# <a name="operator-sectionrel"></a>연산자 SECTIONREL

최종 실행 파일의 대상이 포함 된 섹션을 기준으로 하는 식의 상대적인 섹션 오프셋을 반환 합니다.

## <a name="syntax"></a>구문

> **섹션 rel** *식*

## <a name="remarks"></a>주의

**섹션 rel** 는 COFF 개체 내보내기와 함께 사용할 수 있습니다.
