# <a name="operator-sectionrel"></a>연산자 SECTIONREL

최종 실행 파일 대상이 포함 된 섹션을 기준으로 식의 섹션 상대 오프셋을 반환 합니다.

## <a name="syntax"></a>구문

> SECTIONREL 식

## <a name="remarks"></a>설명

SECTIONREL은 COFF 개체 내보내기만 제공 됩니다.
