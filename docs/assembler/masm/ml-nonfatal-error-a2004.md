# <a name="ml-nonfatal-error-a2004"></a>ML 심각하지 않은 오류 A2004

**기호 형식 충돌: 식별자**

[EXTERNDEF](externdef.md) 또는 [label](label-masm.md) 지시어는 동일한 모듈에서 정의 되었지만 다른 유형으로 정의 된 변수, 기호, 데이터 구조 또는 레이블에 사용 되었습니다.
