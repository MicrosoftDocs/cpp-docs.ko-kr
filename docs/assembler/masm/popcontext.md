# <a name="popcontext"></a>POPCONTEXT

현재 *컨텍스트의* 일부 또는 전체를 복원 합니다 ( [pushcontext](../../assembler/masm/pushcontext.md) 지시문에 의해 저장 됨). *컨텍스트* 는 **가정**, **기**수, **목록**, **CPU**또는 **모두**를 사용할 수 있습니다.

## <a name="syntax"></a>구문

> **POPCONTEXT** *컨텍스트*
