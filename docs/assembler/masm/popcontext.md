# <a name="popcontext"></a>POPCONTEXT

현재 *컨텍스트의* 일부 또는 전체를 복원 합니다 ( [pushcontext](pushcontext.md) 지시문에 의해 저장 됨). *컨텍스트* 는 (32 비트 MASSM만 해당), **기**수, **목록**, **CPU** (32 비트 MASSM만 해당) 또는 **ALL**로 **가정** 될 수 있습니다.

## <a name="syntax"></a>구문

> **POPCONTEXT** *컨텍스트*
