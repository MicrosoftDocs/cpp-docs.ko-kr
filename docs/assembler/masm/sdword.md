# <a name="sdword"></a>SDWORD

각 *이니셜라이저의*저장소의 부호 있는 더블 워드 (4 바이트)를 할당 하 고 선택적으로 초기화 합니다. 또한 형식이 유효한 모든 곳에서 형식 지정자로 사용할 수 있습니다.

## <a name="syntax"></a>구문

> *이름* **sdword** *이니셜라이저* ⟦ __,__ *이니셜라이저의* ... ⟧
