# <a name="comment"></a>COMMENT

구분 기호와 동일한 줄의 모든 *텍스트* 를 주석으로 처리 합니다.

## <a name="syntax"></a>구문

> **주석** *구분 기호* ⟦*text*⟧ \
> ⟦*text*⟧ \
> ⟦*text*⟧ *구분 기호* ⟦*text*⟧
