# <a name="df"></a>DF

각 *이니셜라이저의* 6 바이트의 저장소를 할당 하 고 선택적으로 초기화 합니다. **DF** 는 [fword](fword.md)의 동의어입니다.

## <a name="syntax"></a>Syntax

> ⟦*name*⟧ **DF** *이니셜라이저가* ⟦__,__ *이니셜라이저가* 있습니다. ⟧

## <a name="remarks"></a>설명

또한 형식이 유효한 모든 곳에서 형식 지정자로 사용할 수 있습니다.
