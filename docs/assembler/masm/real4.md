# <a name="real4"></a>REAL4

각 *이니셜라이저의* 단 정밀도 (4 바이트) 부동 소수점 숫자를 할당 하 고 필요에 따라 초기화 합니다.

## <a name="syntax"></a>Syntax

> **REAL4** *이니셜라이저* ⟦__,__ *이니셜라이저의* 이름 ... ⟧
