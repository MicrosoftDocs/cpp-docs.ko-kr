# <a name="version"></a>\@버전

명령줄에서 단일 숫자 (텍스트 매크로)로 보고 되는 MASM의 주 버전 및 부 버전입니다. 예를 들어 MASM 버전 14.23.28107.0는 "1423"을 생성 합니다.

## <a name="syntax"></a>구문

> **\@버전**
