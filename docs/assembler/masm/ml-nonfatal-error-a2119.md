# <a name="ml-nonfatal-error-a2119"></a>ML 심각하지 않은 오류 A2119

**언어 유형을 지정 해야 합니다.**

프로시저 정의 또는 프로토타입에 언어 형식이 제공 되지 않았습니다.

기본 언어 형식이 지정 되지 않은 경우 각 프로시저 정의 또는 프로토타입에서 언어 형식을 선언 해야 합니다. 기본 언어 형식은를 사용 하 여 설정 됩니다 [. 모델](dot-model.md) 지시문, **옵션 언어**또는 ML 명령줄 옵션 **/Gc** 또는 **/gd**.
