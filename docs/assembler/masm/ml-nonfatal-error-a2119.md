# <a name="ml-nonfatal-error-a2119"></a>ML 심각하지 않은 오류 A2119
**언어 형식을 지정 해야 합니다.**  
  
 프로시저 정의 또는 프로토타입 언어 형식을 지정 되지 않았습니다.  
  
 언어 형식 기본 언어 형식이 지정 되지 않은 경우 각 프로시저 정의 또는 프로토타입에 선언 되어야 합니다. 기본 언어 형식 중 하나를 사용 하 여 설정 되어는 [합니다. 모델](../../assembler/masm/dot-model.md) 지시문 **옵션 LANG**, ML 명령줄 옵션 또는 **/Gc** 또는 **/Gd**합니다.
