# <a name="exitm"></a>EXITM

현재 반복 또는 매크로 블록의 확장을 종료 하 고 블록 외부에 있는 다음 문의 어셈블리를 시작 합니다.

## <a name="syntax"></a>구문

> **Exitm** ⟦*textitem*⟧

## <a name="remarks"></a>주의

매크로 함수에서 *textitem* 은 반환 된 값입니다.
