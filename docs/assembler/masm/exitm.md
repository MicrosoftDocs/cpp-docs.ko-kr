# <a name="exitm"></a>EXITM
현재 반복 또는 매크로 블록의 확장을 종료 하 고 문이 블록 밖의 어셈블리를 시작 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
EXITM [[textitem]]   
```  
  
## <a name="remarks"></a>설명  
 매크로 함수에 *textitem* 값이 반환 됩니다.
