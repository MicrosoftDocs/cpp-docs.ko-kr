# <a name="operator-gt"></a>연산자 GT

True (-1)를 반환 합니다 *expression1* 보다 크면 *expression2*, 있지 않으면 false(0)를 반환 합니다.

## <a name="syntax"></a>구문

> expression1 GT expression2
