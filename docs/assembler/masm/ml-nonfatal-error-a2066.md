사용 하 여 세그먼트를 열려는 시도가 있었습니다를 **USE16**를 **USE32**, 또는 **플랫** 지정 된 CPU를 사용 하 여 또는 32 비트에서 16 비트 CPU를 변경 하려면 호환 되지 않은 특성 세그먼트입니다.
