# <a name="ml-nonfatal-error-a2078"></a>ML 심각하지 않은 오류 A2078

**명령에서 FAR 간접 주소 지정을 허용 하지 않습니다.**

조건부 점프 또는 루프는 메모리 피연산자를 사용할 수 없습니다. 상대 주소 또는 레이블을 지정 해야 합니다.
