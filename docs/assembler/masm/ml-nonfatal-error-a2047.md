# <a name="ml-nonfatal-error-a2047"></a>ML 심각하지 않은 오류 A2047

**빈 (null) 문자열**

문자열은 따옴표의 구분 쌍으로 구성 되 고에는 문자가 없습니다.

문자열이 유효 하려면 1-255 문자를 포함 해야 합니다.
