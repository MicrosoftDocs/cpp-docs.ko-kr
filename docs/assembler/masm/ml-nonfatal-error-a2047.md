# <a name="ml-nonfatal-error-a2047"></a>ML 심각하지 않은 오류 A2047
**빈 (null) 문자열**  
  
 문자열 내에 문자가 없는 따옴표의 구분 기호 쌍으로 이루어져 있습니다.  
  
 문자열로 유효 하려면 1-255 자를 포함 해야 합니다.
