# <a name="operator-lt"></a>연산자 LT

True (-1)를 반환 합니다 *expression1* 는 보다 작은 *expression2*, 있지 않으면 false(0)를 반환 합니다.

## <a name="syntax"></a>구문

> expression1 LT expression2
