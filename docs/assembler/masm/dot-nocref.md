# <a name="nocref"></a>.NOCREF

## <a name="syntax"></a>구문

> . NOCREF [[이름 [[, name]]...]]

## <a name="remarks"></a>설명

이름이 지정 된 경우 지정 된 이름만 표시 되지 않습니다. 동일 [합니다. XCREF](../../assembler/masm/dot-xcref.md)합니다.
