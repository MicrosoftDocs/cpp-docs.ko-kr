# <a name="nocref"></a>.NOCREF

## <a name="syntax"></a>구문

> **. Nocref** ⟦*name*⟦ __,__ *이름* ... ⟧⟧

## <a name="remarks"></a>주의

이름이 지정 된 경우 지정 된 이름만 표시 됩니다. 와 동일 [합니다. XCREF](../../assembler/masm/dot-xcref.md).
