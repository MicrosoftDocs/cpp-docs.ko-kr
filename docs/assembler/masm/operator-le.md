# <a name="operator-le"></a>연산자 LE

*Expression1* 가 *식 2*보다 작거나 같으면 true (-1)를 반환 하 고 그렇지 않으면 false (0)를 반환 합니다.

## <a name="syntax"></a>구문

> *expression1* **LE** *식 2*
