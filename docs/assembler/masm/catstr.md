# <a name="catstr"></a>CATSTR

텍스트 항목을 연결 합니다.

## <a name="syntax"></a>구문

>  **⟦** *textitem1* ⟦__,__ *textitem2* 이름 ⟧⟧

## <a name="remarks"></a>주의

각 텍스트 항목은 리터럴 문자열, **%** 앞에 오는 상수 또는 매크로 함수에서 반환 된 문자열이 될 수 있습니다. **Systr** 은 [TEXTEQU](textequ.md)의 동의어입니다.
