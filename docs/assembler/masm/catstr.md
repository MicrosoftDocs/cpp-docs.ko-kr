# <a name="catstr"></a>CATSTR
텍스트 항목을 연결합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## <a name="remarks"></a>설명  
 각 텍스트 항목을 앞에 상수는 리터럴 문자열 수는 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.
