# <a name="catstr"></a>CATSTR

텍스트 항목을 연결합니다.

## <a name="syntax"></a>구문

> CATSTR 이름을 [[textitem1 [[, textitem2]]...]]

## <a name="remarks"></a>설명

각 텍스트 항목 앞에 상수 리터럴 문자열을이 수는 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.
