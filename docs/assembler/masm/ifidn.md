# <a name="ifidn"></a>IFIDN]
텍스트 항목에는 동일한 경우 어셈블리를 부여 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
IFIDN[[I]] textitem1, textitem2  
```  
  
## <a name="remarks"></a>설명  
 경우 **I** 는 경우 비교 된 대/소문자 구분 합니다. 참조 [IF](../../assembler/masm/if-masm.md) 전체 구문에 대 한 합니다.
