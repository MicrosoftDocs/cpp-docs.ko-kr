# <a name="ml-nonfatal-error-a2034"></a>ML 심각하지 않은 오류 A2034

**세그먼트 블록에 있어야 합니다.**

세그먼트 블록 외부에 다음 중 하나가 있습니다.

- 명령

- 레이블 정의

- [이](operator-this.md) 연산자

- [$](dollar.md) 연산자

- 프로시저 정의

- [ALIGN](align-masm.md) 지시문

- [ORG](org.md) 지시문
