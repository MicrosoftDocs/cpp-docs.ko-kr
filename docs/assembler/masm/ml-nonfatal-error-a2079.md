# <a name="ml-nonfatal-error-a2079"></a>ML 심각하지 않은 오류 A2079
**명령 훨씬 직접 주소 지정을 허용 하지 않습니다.**  
  
 서로 다른 세그먼트 또는 그룹에는 조건부 점프 또는 루프 수 없습니다.
