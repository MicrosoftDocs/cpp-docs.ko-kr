# <a name="ml-nonfatal-error-a2079"></a>ML 심각하지 않은 오류 A2079

**명령에서 먼 직접적인 주소 지정을 허용 하지 않습니다.**

조건부 점프 또는 루프는 다른 세그먼트 또는 그룹 일 수 없습니다.
