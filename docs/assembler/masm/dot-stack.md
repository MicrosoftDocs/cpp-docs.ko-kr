# <a name="stack"></a>.STACK
와 함께 사용할 경우 [합니다. 모델](../../assembler/masm/dot-model.md), (세그먼트 이름으로 스택) 스택 세그먼트를 정의 합니다. 선택적 `size` 스택 (기본값 1, 024)에 대 한 바이트 수를 지정 합니다. `.STACK` 지시문을 자동으로 스택 문을 닫습니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
.STACK [[size]]  
```
