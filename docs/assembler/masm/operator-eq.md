# <a name="operator-eq"></a>연산자 EQ
True (-1)를 반환 합니다 *expression1* equals *expression2*, 하거나 그렇지 않은 경우 false (0)를 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
expression1 EQ expression2  
```
