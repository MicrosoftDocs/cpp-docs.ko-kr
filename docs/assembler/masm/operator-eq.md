# <a name="operator-eq"></a>연산자 EQ

*Expression1* 가 *식 2*와 같으면 true (-1)를 반환 하 고 그렇지 않으면 false (0)를 반환 합니다.

## <a name="syntax"></a>구문

> *expression1* **EQ** *식 2*
