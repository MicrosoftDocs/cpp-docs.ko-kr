# <a name="ml-nonfatal-error-a2206"></a>ML 심각하지 않은 오류 A2206

**식에 연산자가 없습니다.**

연산자가 없으므로 식을 평가할 수 없습니다. 이 오류 메시지는 이전 프로그램 오류로 인해 부수적으로 영향을 받을 수도 있습니다.

다음 줄은이 오류를 생성 합니다.

```asm
value1 = ( 1 + 2 ) 3
```
