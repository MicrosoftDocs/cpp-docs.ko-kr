# <a name="ml-nonfatal-error-a2206"></a>ML 심각하지 않은 오류 A2206

**식에서 연산자가 없습니다**

연산자 누락 되었으므로 식을 계산할 수 없습니다. 이 오류 메시지는 이전 프로그램 오류의 부작용을 수도 있습니다.

다음 줄이이 오류를 생성 합니다.

```asm
value1 = ( 1 + 2 ) 3
```
