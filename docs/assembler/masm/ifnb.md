# <a name="ifnb"></a>IFNB

경우에 어셈블리를 부여 *textitem* 비어 있지 않음.

## <a name="syntax"></a>구문

> IFNB textitem

## <a name="remarks"></a>설명

참조 [경우](../../assembler/masm/if-masm.md) 전체 구문에 대 한 합니다.
