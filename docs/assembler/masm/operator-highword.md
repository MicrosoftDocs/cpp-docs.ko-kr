# <a name="operator-highword"></a>연산자 HIGHWORD

낮은 32 비트 *식* 의 상위 16 비트를 반환 합니다. MASM 식은 64 비트 값입니다.

## <a name="syntax"></a>Syntax

> **Highword** *식*
