# <a name="tbyte"></a>TBYTE

각 *이니셜라이저* 에 대해 10 바이트의 저장소를 할당 하 고 선택적으로 초기화 합니다. 형식이 유효한 모든 곳에서 형식을 형식 지정자로 사용할 수도 있습니다. **DT** 는 **1tb** 의 동의어입니다.

## <a name="syntax"></a>Syntax

> ⟦*name*⟧ **1tb** *이니셜라이저* ⟦__,__ *이니셜라이저가* 있습니다. ⟧
