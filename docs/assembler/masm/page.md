# <a name="page"></a>PAGE

첫 번째 지시문은 프로그램 목록의 줄 *길이* 와 문자 *너비* 를 설정 합니다. 인수를 지정 하지 않으면에서 페이지 나누기를 생성 합니다. 두 번째 지시문은 섹션 번호를 증가 시키고 페이지 번호를 1로 다시 설정 합니다.

## <a name="syntax"></a>구문

> **PAGE** ⟦*length*⟧ ⟦ __,__ *width*⟧

> **페이지 +**
