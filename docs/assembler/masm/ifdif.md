# <a name="ifdif-ifdifi"></a>IFDIF, IFDIFI

텍스트 항목에는 다른 경우 어셈블리를 부여 합니다.

## <a name="syntax"></a>구문

> **IFDIF**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>설명

하는 경우 **IFDIFI** 는, 비교는 대/소문자 구분 합니다. 참조 [경우](../../assembler/masm/if-masm.md) 전체 구문에 대 한 합니다.
