# <a name="union"></a>UNION
하나 이상의 데이터 형식의 공용 구조체가 선언합니다. *fielddeclarations* 유효한 데이터 정의 해야 합니다. 생략 된 [종료](../../assembler/masm/ends-masm.md) *이름* 레이블에 중첩 **UNION** 정의 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
      name   
      UNION [[alignment]] [[, NONUNIQUE]]  
   fielddeclarations  
[[name]] ENDS  
```
