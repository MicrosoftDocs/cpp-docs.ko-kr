# <a name="union"></a>UNION

하나 이상의 데이터 형식에 대 한 공용 구조체를 선언 합니다. *필드 선언은* 올바른 데이터 정의 여야 합니다. 중첩 된 **공용 구조체** 정의에서 [end](ends-masm.md) *name* 레이블을 생략 합니다.

## <a name="syntax"></a>구문

> *이름* **UNION** ⟦*alignment*⟧ ⟦ __,__ **고유**하지 않은 ⟧ \
> *필드 선언*\
> ⟦*name*⟧ **end**
