# <a name="ml-fatal-error-a1008"></a>ML 심각한 오류 A1008

 두 매크로 파일 또는 종결 지시문이 끝나기 전에 종료 되지 않았습니다. [ENDM](../../assembler/masm/endm.md) 매크로 블록 외부에서 찾았습니다.  
  
 이 오류의 한 가지 원인은 하기 전에 점 생략은 [합니다. 반복](../../assembler/masm/dot-repeat.md) 또는 [합니다. 반면](../../assembler/masm/dot-while.md)합니다.
