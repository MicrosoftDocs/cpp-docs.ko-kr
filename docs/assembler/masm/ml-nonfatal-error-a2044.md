# <a name="ml-nonfatal-error-a2044"></a>ML 심각하지 않은 오류 A2044

**파일에 잘못 된 문자가 있습니다.**

소스 파일에 연산자나 기타 올바른 문자로 인식 되지 않는 주석, 문자열 또는 리터럴 외부의 문자가 포함 되어 있습니다.
