# <a name="real10"></a>REAL10
할당 및 각각에 대해 10 바이트 부동 소수점 숫자를 선택적으로 초기화 *이니셜라이저*합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```
