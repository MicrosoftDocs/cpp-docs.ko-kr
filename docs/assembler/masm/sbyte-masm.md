# <a name="sbyte-masm"></a>SBYTE (MASM)

각 *이니셜라이저의*서명 된 저장소 바이트를 할당 하 고 필요에 따라 초기화 합니다. 형식이 유효한 모든 곳에서 형식을 형식 지정자로 사용할 수도 있습니다.

## <a name="syntax"></a>구문

> *이름*  **SBYTE** *이니셜라이저* ⟦ __,__ *이니셜라이저의* ... ⟧
