# <a name="ml-nonfatal-error-a2133"></a>ML 심각하지 않은 오류 A2133

**호출에 의해 덮어쓴 값 등록**

레지스터가 프로시저에 인수로 전달 되었지만 다른 인수를 전달 하기 위해 [호출](invoke.md) 에 의해 생성 된 코드에서 레지스터의 내용이 제거 되었습니다.

어셈블러는 AX, AL, AH, EAX, DX, DL, DH 및 EDX 레지스터를 사용 하 여 데이터 변환을 수행할 수 있습니다.

다른 레지스터를 사용 하십시오.
