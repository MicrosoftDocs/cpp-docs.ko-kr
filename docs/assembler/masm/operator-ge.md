# <a name="operator-ge"></a>연산자 GE

*Expression1* 가 *식*1 보다 크거나 같으면 true (-1)를 반환 하 고 그렇지 않으면 false (0)를 반환 합니다.

## <a name="syntax"></a>구문

> *expression1* **GE** *식 2*
