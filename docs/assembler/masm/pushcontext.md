# <a name="pushcontext"></a>PUSHCONTEXT

현재 *컨텍스트의*일부 또는 전체를 저장 합니다. 세그먼트 레지스터는, 기 하 값, 목록 및 cref 플래그 또는 프로세서/보조 프로세서 값을 가정 합니다. *컨텍스트* 는 **가정**, **기**수, **목록**, **CPU**또는 **모두**를 사용할 수 있습니다.

## <a name="syntax"></a>구문

> **Pushcontext** *컨텍스트*
