# <a name="forc"></a>FORC

각 문자에 대해 한 번씩 반복 되는 블록 표시 *문자열*를 사용 하 여 현재 문자 대체 *매개 변수* 각 반복에서.

## <a name="syntax"></a>구문

> FORC *parameter*, \<*string*><br/>
> *statements*<br/>
> ENDM

## <a name="remarks"></a>설명

동일 [IRPC](../../assembler/masm/irpc.md)합니다.
