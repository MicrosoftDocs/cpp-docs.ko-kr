# <a name="forc"></a>FORC

각 반복에서 현재 문자 대체 *매개 변수* 를 사용 하 여 *문자열* 의 각 문자에 대해 한 번씩 반복 되는 블록을 표시 합니다.

## <a name="syntax"></a>Syntax

> **Forc** *매개 변수*__\<__*string*__>__\
> *할당문*\
> **ENDM**

## <a name="remarks"></a>설명

[Irpc](irpc.md)와 동일 합니다.
