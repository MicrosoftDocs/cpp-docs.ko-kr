# <a name="forc"></a>FORC
각 문자에 대해 한 번씩 반복 되는 블록 표시 `string`, 현재 문자 대체 된 `parameter` 각 반복 시.  
  
## <a name="syntax"></a>구문  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## <a name="remarks"></a>설명  
 와 동일 [IRPC](../../assembler/masm/irpc.md)합니다.
