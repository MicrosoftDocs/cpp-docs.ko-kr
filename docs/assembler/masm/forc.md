# <a name="forc"></a>FORC

각 반복에서 현재 문자 대체 *매개 변수* 를 사용 하 여 *문자열*의 각 문자에 대해 한 번씩 반복 되는 블록을 표시 합니다.

## <a name="syntax"></a>구문

> **Forc** *매개 변수*, *문자열* __>__ __\<__ \
> *문*\
> **ENDM**

## <a name="remarks"></a>주의

[Irpc](../../assembler/masm/irpc.md)와 동일 합니다.
