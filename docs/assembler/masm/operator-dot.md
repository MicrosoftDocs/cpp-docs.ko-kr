# <a name="operator-"></a>연산자 .
첫 번째 연산자는 반환 *식* 의 오프셋 더하기 *필드* 해당 구조체 또는 공용 구조체 내에서. 가 가리키는 위치에 값을 반환 하는 두 번째 연산자 *등록* 의 오프셋 더하기 *필드* 해당 구조체 또는 공용 구조체 내에서.  
  
## <a name="syntax"></a>구문  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
