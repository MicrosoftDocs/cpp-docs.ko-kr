# <a name="operator-short"></a>operator SHORT
유형을 설정 *레이블* 짧게 합니다. 모든 점프를 *레이블* 짧아야 (점프 명령에서-128에서 + 127 바이트 ~ 범위 내에서 *레이블*).  
  
## <a name="syntax"></a>구문  
  
```  
  
SHORT label  
```
