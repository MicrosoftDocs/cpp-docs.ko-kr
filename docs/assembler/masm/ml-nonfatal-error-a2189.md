# <a name="ml-nonfatal-error-a2189"></a>ML 심각하지 않은 오류 A2189

**세그먼트 정렬에 잘못 된 조합이 있습니다.**

**ALIGN** 또는 지시어에 지정 된 맞춤이 `EVEN` **세그먼트** 지시문에 지정 된 대로 현재 세그먼트 맞춤 보다 큽니다.
