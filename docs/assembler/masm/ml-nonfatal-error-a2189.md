# <a name="ml-nonfatal-error-a2189"></a>ML 심각하지 않은 오류 A2189
**세그먼트 맞춤으로 잘못 된 조합**  
  
 로 지정 된 맞춤은 **맞춤** 또는 `EVEN` 지시문에 지정 된 대로 현재 세그먼트 맞춤 보다 큽니다.는 **세그먼트** 지시문입니다.
