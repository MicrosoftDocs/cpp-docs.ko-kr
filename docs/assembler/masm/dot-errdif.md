# <a name="errdifi"></a>.ERRDIF[[I]]
텍스트 항목에는 서로 다른 경우 오류가 발생 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>설명  
 경우 **I** 는 경우 비교 된 대/소문자 구분 합니다.
