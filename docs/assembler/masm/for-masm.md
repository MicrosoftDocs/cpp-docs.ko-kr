# <a name="for-masm"></a>FOR (MASM)

각각에 대해 한 번씩 반복 되는 블록 표시 *인수*, 현재 *인수* 교체 *매개 변수* 각 반복에서.

## <a name="syntax"></a>구문

> FOR *parameter* [[:REQ | :=*default*]] , \<*argument* [[, *argument*]]...><br/>
> *statements*<br/>
> ENDM
