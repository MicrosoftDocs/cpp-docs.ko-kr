# <a name="for-masm"></a>FOR (MASM)
각각에 대해 한 번씩 반복 되는 블록 표시 `argument`, 현재의 `argument` 교체 `parameter` 각 반복 시.  
  
## <a name="syntax"></a>구문  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```  
  
## <a name="remarks"></a>설명  
 와 동일 [IRP](../../assembler/masm/irp.md)합니다.
