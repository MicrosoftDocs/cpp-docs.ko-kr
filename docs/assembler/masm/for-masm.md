# <a name="for"></a>FOR

각 *인수*에 대해 한 번씩 반복 되는 블록을 표시 하 고 각 반복에서 현재 *인수* 를 대체 하는 *매개 변수* 를 사용 합니다.

## <a name="syntax"></a>구문

> ⟦ *매개 변수* **: 요청** |  __: =__ *default*⟧ **, \<** _인수_ ⟦ __,__ *argument* ... ⟧ **>** \
> *문*\
> **ENDM**

## <a name="remarks"></a>주의

[IRP](irp.md)와 동일 합니다.
