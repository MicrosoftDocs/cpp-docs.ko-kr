# <a name="ml-nonfatal-error-a2083"></a>ML 심각하지 않은 오류 A2083

**배율 값이 잘못 되었습니다.**

레지스터 소수 자릿수가 1, 2, 4 또는 8이 아닌 지정 되었습니다.
