# <a name="public"></a>PUBLIC

프로그램의 다른 모든 모듈에서 *이름* 으로 지정 된 각 변수, 레이블 또는 절대 기호를 사용할 수 있도록 합니다.

## <a name="syntax"></a>Syntax

> **PUBLIC** ⟦*language-type*⟧ *name* ⟦__,__ ⟦*⟧* *name* ... ⟧

## <a name="remarks"></a>설명

*언어 형식* 인수는 32 비트 MASM 에서만 유효 합니다.
