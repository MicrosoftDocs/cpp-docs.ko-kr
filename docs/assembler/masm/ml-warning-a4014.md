# <a name="ml-warning-a4014"></a>ML 경고 A4014

BSS 세그먼트에서 지원 되지 않는 명령 및 초기화 데이터

BSS 섹션 내에서 초기화 된 데이터를 정의 하려고 했습니다.  BSS 섹션은 이름이 BSS 인 클래스로 정의 됩니다.  여기에는 `.data?`단순화 된 세그먼트가 포함 됩니다.
