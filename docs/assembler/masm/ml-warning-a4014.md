# <a name="ml-warning-a4014"></a>ML 경고 A4014

지침 및 초기화 된 데이터 BSS 세그먼트에서 지원 되지 않습니다

초기화 된 BSS 섹션 내에서 데이터를 정의 하려고 했습니다.  BSS 섹션 이름은 BSS 클래스로 정의 됩니다.  여기에 간소화 된 세그먼트 `.data?`합니다.
