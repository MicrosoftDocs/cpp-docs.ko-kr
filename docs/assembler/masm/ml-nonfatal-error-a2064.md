# <a name="ml-nonfatal-error-a2064"></a>ML 심각하지 않은 오류 A2064

**구조 맞춤은 1, 2, 4, 8 또는 16 이어야 합니다.**

구조 정의에 지정 된 맞춤이 잘못 되었습니다.
