# <a name="group"></a>GROUP

(32 비트 MASM에만 해당) 지정 된 *세그먼트* 를 *이름* 이라는 그룹에 추가 합니다.

## <a name="syntax"></a>Syntax

>  **그룹** *세그먼트* 의 이름을 ⟦__,__ *segment* ... ⟧
