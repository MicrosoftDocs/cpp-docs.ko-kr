# <a name="word"></a>WORD

각 *이니셜라이저의*저장소의 단어 (2 바이트)를 할당 하 고 선택적으로 초기화 합니다. 형식이 유효한 모든 곳에서 형식을 형식 지정자로 사용할 수도 있습니다. [DW](dw.md) 는 **WORD**의 동의어입니다.

## <a name="syntax"></a>구문

> ⟦*name*⟧ **WORD** *이니셜라이저가* ⟦ __,__ *이니셜라이저가* 있습니다. ⟧
