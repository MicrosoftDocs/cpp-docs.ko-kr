# <a name="ml-fatal-error-a1009"></a>ML 심각한 오류 A1009

**줄이 너무 깁니다.**

소스 파일의 줄이 512 자 제한을 초과 했습니다.

줄 연속 문자 (\)와 여러 개의 실제 줄을 연결 하는 경우 결과 논리 줄은 여전히 512 자로 제한 됩니다.
