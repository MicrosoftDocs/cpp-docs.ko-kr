# <a name="ml-fatal-error-a1009"></a>ML 심각한 오류 A1009

**행이 너무 깁니다**

소스 파일의 줄에는 512 자 제한을 초과 했습니다.

줄 연속 문자 (\)를 사용 하 여 여러 실제 줄 연결 하면 결과 논리 줄 경우 512 자로 제한 합니다.
