# <a name="ml-fatal-error-a1009"></a>ML 심각한 오류 A1009
**줄이 너무 길어**  
  
 소스 파일의 선을 512 자를 초과 했습니다.  
  
 여러 실제 줄 줄 연속 문자 (\)로 연결 하는 경우 결과 논리 줄은 여전히 512 자로 제한 됩니다.
