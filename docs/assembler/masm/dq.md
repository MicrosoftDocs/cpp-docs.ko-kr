# <a name="dq"></a>DQ
할당 및 필요에 따라 각각에 대해 8 바이트의 저장소를 초기화 `initializer`합니다. 또한 형식 지정자로 사용할 수 있습니다 형식이 법적 곳입니다. `DQ` 동의어 [QWORD](../../assembler/masm/qword.md)합니다.  
  
## <a name="syntax"></a>구문  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
