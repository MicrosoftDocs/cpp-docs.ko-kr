# <a name="ml-fatal-error-a1005"></a>ML 심각한 오류 A1005

**어셈블러 제한: 매크로 매개 변수 이름 테이블이 꽉 찼습니다.**

매크로에 대 한 매개 변수, 지역 또는 매크로 레이블이 너무 많이 정의 되었습니다. 매크로 이름 테이블에 공간이 더 이상 없습니다.

더 짧거나 더 작은 이름을 정의 하거나 불필요 한 매크로를 제거 합니다.
