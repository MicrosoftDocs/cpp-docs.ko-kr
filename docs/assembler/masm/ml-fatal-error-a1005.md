# <a name="ml-fatal-error-a1005"></a>ML 심각한 오류 A1005
**어셈블러 한계: 매크로 매개 변수 이름 테이블이 꽉 찼습니다.**  
  
 너무 많은 매개 변수, 지역 또는 매크로 레이블을 매크에 대해 정의 되었습니다. 매크로 이름 테이블에 더 많은 공간이 했습니다.  
  
 더 짧은 정의 또는 더 적은 이름 또는 불필요 한 매크로 제거 합니다.
