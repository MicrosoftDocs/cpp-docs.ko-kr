# <a name="operator-xor"></a>연산자 XOR
비트의 결과 반환 **XOR** 에 대 한 작업이 *expression1* 및 *expression2*합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
expression1 XOR expression2  
```
