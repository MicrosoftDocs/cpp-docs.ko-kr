# <a name="operator-xor"></a>연산자 XOR

*Expression1* 및 *식 2* 에 대 한 비트 **XOR** 연산의 결과를 반환 합니다.

## <a name="syntax"></a>Syntax

> *expression1* **XOR** *식 2*
