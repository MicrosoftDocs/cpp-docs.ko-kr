# <a name="includelib-masm"></a>INCLUDELIB (MASM)

## <a name="syntax"></a>구문

> INCLUDELIB libraryname

## <a name="remarks"></a>설명

*libraryname* 세미콜론으로 큰 백슬래시를 포함 하는 경우 꺾쇠 괄호로 묶어야 합니다-기호 적은 보다-보다 기호, 작은따옴표, 큰따옴표를 표시 합니다.
