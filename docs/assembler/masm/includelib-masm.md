# <a name="includelib"></a>INCLUDELIB

현재 모듈이 *libraryname*와 연결 되어야 함을 링커에 알립니다.

## <a name="syntax"></a>구문

> **INCLUDELIB** *libraryname*

## <a name="remarks"></a>주의

백슬래시, 세미콜론, 보다 큼 기호, 보다 작음 기호, 작은따옴표 또는 큰따옴표가 포함 된 경우에는 *libraryname* 를 꺾쇠 괄호로 묶어야 합니다.
