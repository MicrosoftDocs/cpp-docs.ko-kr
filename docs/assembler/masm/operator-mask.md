# <a name="operator-mask"></a>연산자 MASK

*Recordfieldname* 또는 *record* 의 비트가 설정 되 고 다른 모든 비트가 지워지는 비트 마스크를 반환 합니다.

## <a name="syntax"></a>Syntax

> **MASK** {*recordfieldname*  |  *record*}
