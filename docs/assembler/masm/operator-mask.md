# <a name="operator-mask"></a>연산자 MASK
비트 마스크를 반환의 비트 *recordfieldname* 또는 *레코드* 설정 되 고 다른 모든 비트의 선택을 취소 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
MASK {recordfieldname | record}  
```
