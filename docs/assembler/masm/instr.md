# <a name="instr"></a>INSTR

*Textitem1*에서 *textitem2* 의 첫 번째 발생을 찾습니다.

## <a name="syntax"></a>구문

> *이름* **INSTR** ⟦*position* __,__ ⟧ *textitem1* __,__ *textitem2*

## <a name="remarks"></a>주의

시작 *위치* 는 선택 사항입니다. 각 텍스트 항목은 리터럴 문자열, **%** 앞에 오는 상수 또는 매크로 함수에서 반환 된 문자열이 될 수 있습니다.
