# <a name="dd"></a>DD

각 *이니셜라이저의* 저장소에 대해 2 배 워드 (4 바이트)를 할당 하 고 선택적으로 초기화 합니다. **DD** 는 [DWORD](dword.md)의 동의어입니다.

## <a name="syntax"></a>Syntax

> ⟦*name*⟧ **DD** *이니셜라이저* ⟦__,__ *이니셜라이저가* 있습니다. ⟧

## <a name="remarks"></a>설명

형식이 유효한 모든 곳에서 형식을 형식 지정자로 사용할 수도 있습니다.
