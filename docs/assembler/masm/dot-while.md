# <a name="while"></a>.WHILE
블록을 실행 하는 코드를 생성 `statements` 동안 `condition` true로 유지 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
