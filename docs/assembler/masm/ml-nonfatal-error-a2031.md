# <a name="ml-nonfatal-error-a2031"></a>ML 심각하지 않은 오류 A2031

**인덱스 또는 기본 등록 해야 합니다.**

메모리 식에 기본 또는 인덱스 레지스터 없는 레지스터를 사용 하려고 했습니다.

예를 들어 다음 식은이 오류가 발생합니다.

```asm
[ax]
[bl]
```
