# <a name="sqword"></a>SQWORD

각 *이니셜라이저의*8 개의 서명 된 저장소 바이트를 할당 하 고 선택적으로 초기화 합니다. 또한 형식이 유효한 모든 곳에서 형식 지정자로 사용할 수 있습니다.

## <a name="syntax"></a>구문

> *이름* **sqword** *이니셜라이저* ⟦ __,__ *이니셜라이저의* ... ⟧
