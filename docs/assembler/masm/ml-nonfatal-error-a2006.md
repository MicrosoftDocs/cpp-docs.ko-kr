# <a name="ml-nonfatal-error-a2006"></a>ML 심각하지 않은 오류 A2006

**undefined symbol : identifier**

정의 되지 않은 기호를 사용 하려고 했습니다.

다음 중 하나를 발생할 수 있습니다.

- 기호가 정의 되지 않았습니다.

- 필드는 지정 된 구조체의 멤버는 없습니다.

- 기호가 포함 되지 않은 include 파일에 정의 되었습니다.

- 외부 기호 없이 사용한를 [EXTERN](../../assembler/masm/extern-masm.md) 하거나 [EXTERNDEF](../../assembler/masm/externdef.md) 지시문입니다.

- 기호 이름 철자가 틀렸습니다.

- 해당 범위 외부에서 참조 되는 로컬 코드 레이블입니다.
