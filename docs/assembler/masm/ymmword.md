# <a name="ymmword"></a>YMMWORD

Intel 고급 벡터 확장 (avx)를 사용 하 여 256 비트 멀티미디어 피연산자에 사용 됩니다.

## <a name="syntax"></a>구문

> YMMWORD

## <a name="remarks"></a>설명

`YMMWORD` 동일한 형식으로 표시 하기 위해 `__m256` AVX 내장 함수에 대 한 합니다.
