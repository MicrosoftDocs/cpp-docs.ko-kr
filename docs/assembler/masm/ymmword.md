# <a name="ymmword"></a>YMMWORD

Intel AVX (Advanced Vector Extensions) 명령을 사용 하는 256 비트 멀티미디어 피연산자에 사용 됩니다.

## <a name="syntax"></a>구문

> **YMMWORD**

## <a name="remarks"></a>주의

**YMMWORD** 는 AVX 내장 함수에 대해 `__m256`와 동일한 형식을 표시 하기 위한 것입니다.
