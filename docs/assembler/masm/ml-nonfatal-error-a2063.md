# <a name="ml-nonfatal-error-a2063"></a>ML 심각하지 않은 오류 A2063

**2: 식의 제곱만 맞출 수 있습니다.**

[ALIGN](align-masm.md) 지시문으로 지정 된 식이 잘못 되었습니다.

**ALIGN** 식은 2에서 256 사이의 거듭제곱 이어야 하 고 현재 세그먼트, 구조체 또는 공용 구조체의 맞춤 보다 작거나 같아야 합니다.
