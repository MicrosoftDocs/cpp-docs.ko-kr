# <a name="ml-nonfatal-error-a2063"></a>ML 심각하지 않은 오류 A2063

**2의 거듭제곱에만 맞춤을 수 있습니다: 식**

사용 하 여 지정 된 식의 [ALIGN](../../assembler/masm/align-masm.md) 지시문 올바르지 않습니다.

합니다 **ALIGN** 식 2에서 256 사이의 2의 거듭제곱 이어야 하며 현재 세그먼트, 구조체 또는 공용 구조체의 맞춤 보다 작거나 같아야 합니다.
