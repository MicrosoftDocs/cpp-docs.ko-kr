# <a name="ml-nonfatal-error-a2069"></a>ML 심각하지 않은 오류 A2069

**이 명령에 사용할 수 없는 피연산자**

하나 이상의 피연산자는 없음 피연산자를 사용 하는 명령으로 지정 된 것입니다.
