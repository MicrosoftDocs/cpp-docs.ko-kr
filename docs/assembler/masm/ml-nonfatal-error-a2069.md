# <a name="ml-nonfatal-error-a2069"></a>ML 심각하지 않은 오류 A2069

**이 명령에 사용할 수 있는 피연산자가 없습니다.**

피연산자를 사용 하지 않는 명령으로 하나 이상의 피연산자를 지정 했습니다.
