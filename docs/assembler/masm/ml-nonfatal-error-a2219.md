# <a name="ml-nonfatal-error-a2219"></a>ML 심각하지 않은 오류 A2219

> 해제 코드의 오프셋에 대 한 잘못 된 맞춤

## <a name="remarks"></a>설명

피연산자 [ &period;ALLOCSTACK](../../assembler/masm/dot-allocstack.md) 하 고 [ &period;SAVEREG](../../assembler/masm/dot-savereg.md) 8의 배수 여야 합니다.  피연산자 [ &period;SAVEXMM128](../../assembler/masm/dot-savexmm128.md) 하 고 [ &period;SETFRAME](../../assembler/masm/dot-setframe.md) 16의 배수 여야 합니다.
