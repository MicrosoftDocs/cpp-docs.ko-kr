# <a name="operator-lroffset"></a>연산자 LROFFSET

*식* 의 오프셋을 반환 합니다. **오프셋과** 동일 하지만 창에서 코드 세그먼트를 재배치할 수 있는 로더 확인 된 오프셋을 생성 합니다.

## <a name="syntax"></a>Syntax

> **Lroffset** *식*
