# <a name="operator-"></a>연산자 ||

논리적 OR Used only within [.IF](../../assembler/masm/dot-if.md), [.WHILE](../../assembler/masm/dot-while.md), or [.REPEAT](../../assembler/masm/dot-repeat.md) blocks and evaluated at run time, not at assembly time.

## <a name="syntax"></a>구문

> *expression1* **||** *expression2*
