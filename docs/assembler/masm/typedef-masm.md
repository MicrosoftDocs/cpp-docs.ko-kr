# <a name="typedef"></a>TYPEDEF

*Type* 또는 *protoDefinition*와 동일한 새 형식 또는 **프로토콜** 이라는 *이름을*정의 합니다.

## <a name="syntax"></a>구문

> *이름* **TYPEDEF** *형식* | **PROTO** *protoDefinition*
