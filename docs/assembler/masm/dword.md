# <a name="dword"></a>DWORD

각 *이니셜라이저의*저장소에 대해 2 배 워드 (4 바이트)를 할당 하 고 선택적으로 초기화 합니다. **DWORD** 는 [DD](dd.md)의 동의어입니다.

## <a name="syntax"></a>구문

> ⟦*name*⟧ **DWORD** *이니셜라이저* ⟦ __,__ *이니셜라이저가* 있습니다. ⟧

## <a name="remarks"></a>주의

형식이 유효한 모든 곳에서 형식을 형식 지정자로 사용할 수도 있습니다.
