# <a name="textequ"></a>TEXTEQU
할당 *textitem* 를 *이름*합니다. *textitem* 앞에 상수는 리터럴 문자열이 될 수는 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
name TEXTEQU [[textitem]]  
```
