# <a name="textequ"></a>TEXTEQU

*이름*에 *textitem* 을 할당 합니다. *Textitem* 은 리터럴 문자열, `%`앞에 오는 상수 또는 매크로 함수에서 반환 된 문자열이 될 수 있습니다.

## <a name="syntax"></a>구문

> *이름* **TEXTEQU** ⟦*textitem*⟧
