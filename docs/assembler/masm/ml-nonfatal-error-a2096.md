# <a name="ml-nonfatal-error-a2096"></a>ML 심각하지 않은 오류 A2096

**세그먼트, 그룹 또는 세그먼트 레지스터가 필요 합니다.**

세그먼트 또는 그룹이 필요한 데 찾을 수 없습니다.

다음 중 하나가 발생 했습니다.

- 세그먼트 재정의 연산자 ( **:** )로 지정 된 왼쪽 피연산자가 세그먼트 레지스터 (CS, DS, SS, ES, FS 또는 GS), 그룹 이름, 세그먼트 이름 또는 세그먼트 식이 아닙니다.

- [가정](assume.md) 지시문에 올바른 세그먼트 주소, 세그먼트 레지스터, 그룹 또는 특수 **플랫** 그룹이 없는 세그먼트 레지스터가 제공 되었습니다.
