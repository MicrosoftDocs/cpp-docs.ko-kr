# <a name="ml-nonfatal-error-a2096"></a>ML 심각하지 않은 오류 A2096
**세그먼트, 그룹 또는 세그먼트 레지스터 필요 합니다.**  
  
 세그먼트 또는 그룹에 필요한 데 찾을 수 없습니다.  
  
 다음 중 하나가 발생 했습니다.  
  
-   연산자를 재정의 하는 세그먼트와 지정 된 왼쪽된 피연산자 (**:**) 세그먼트 레지스터 (CS, DS, SS, ES, FS, 또는 GS) 그룹 이름, 세그먼트 이름 또는 세그먼트 식이 없습니다.  
  
-   [가정](../../assembler/masm/assume.md) 지시문 유효한 세그먼트 주소, 세그먼트 레지스터, 그룹 또는 특수 없이 세그먼트 레지스터를 지정 된 **플랫** 그룹입니다.
