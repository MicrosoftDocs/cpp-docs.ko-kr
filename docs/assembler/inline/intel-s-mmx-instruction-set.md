# <a name="intel39s-mmx-instruction-set"></a>Intel&#39;s MMX 명령 집합

**Microsoft 전용**

Microsoft C++ 컴파일러를 사용 하 여 인라인 어셈블러에서 INTEL의 MMX (멀티미디어 확장) 명령 집합을 사용할 수 있습니다. 또한 MMX 명령은 디버거의 디스어셈블리에서 지원됩니다. 컴파일러는 함수에 MMX 명령이 포함되어 있지만 멀티미디어 상태를 비우는 EMMS 명령이 포함되어 있지 않은 경우 경고 메시지를 생성합니다. 자세한 내용은 Intel 웹 사이트를 참조하십시오.
