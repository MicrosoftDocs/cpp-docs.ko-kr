# <a name="no_search_namespace-import-attribute"></a>no_search_namespace 가져오기 특성

**C++ 전용**

는 [no_namespace](../preprocessor/no-namespace.md) 특성과 동일한 기능을 포함 하지만, `#import` 지시문을 [auto_search](../preprocessor/auto-search.md) 특성과 함께 사용 하는 형식 라이브러리에 사용 됩니다.

## <a name="syntax"></a>Syntax

> **#import** *형식 라이브러리* **no_search_namespace**
