# <a name="nosearchnamespace"></a>no_search_namespace
**C + + 전용**  
  
와 동일한 기능을 [no_namespace](../preprocessor/no-namespace.md) 하지만 사용 하는 형식 라이브러리에 사용 되는 `#import` 지시문에 [auto_search](../preprocessor/auto-search.md) 특성.  
  
## <a name="syntax"></a>구문  
  
```  
no_search_namespace  
```
