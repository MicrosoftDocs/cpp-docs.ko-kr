# <a name="high_method_prefix-import-attribute"></a>high_method_prefix import 특성

**C++컴퓨터별**

상위 수준 속성과 메서드 명명에 사용될 접두사를 지정합니다.

## <a name="syntax"></a>구문

> **#import** *형식 라이브러리* **high_method_prefix (** "*prefix*" **)**

### <a name="parameters"></a>매개 변수

*접두사*\
사용할 접두사입니다.

## <a name="remarks"></a>설명

기본적으로 상위 수준 오류 처리 속성 및 메서드는 접두사 없이 명명된 멤버 함수에 의해 노출됩니다. 이름은 형식 라이브러리에서 제공됩니다.
