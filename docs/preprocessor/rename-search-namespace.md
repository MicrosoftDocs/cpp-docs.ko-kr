# <a name="rename_search_namespace-import-attribute"></a>rename_search_namespace 가져오기 특성

**C++ 전용**

는 [rename_namespace](../preprocessor/rename-namespace.md) 특성과 동일한 기능을 갖지만, `#import` [auto_search](../preprocessor/auto-search.md) 특성과 함께 지시문을 사용 하는 형식 라이브러리에 사용 됩니다.

## <a name="syntax"></a>Syntax

> **#import** *형식 라이브러리* **rename_search_namespace (** "*NewName*" **)**

### <a name="parameters"></a>매개 변수

*이름*\
네임스페이스의 새 이름입니다.
