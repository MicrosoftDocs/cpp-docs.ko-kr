# <a name="null-directive"></a>Null 지시문
Null 전처리기 지시문은 단일 숫자 기호 (**#**) 줄에 단독으로 합니다. 아무런 영향을 주지 않습니다.

## <a name="syntax"></a>구문

```
#
```
