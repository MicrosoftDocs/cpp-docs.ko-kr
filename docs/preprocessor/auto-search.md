# <a name="autosearch"></a>auto_search
**C + + 전용**  
  
사용 하 여 형식 라이브러리를 참조할 때 지정 `#import` 컴파일러가 암시적 작업을 수행할 수를 다른 형식 라이브러리 참조 및 `#import` 다른 형식 라이브러리에 대 한 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
auto_search  
```
