# <a name="auto_search-import-attribute"></a>auto_search 가져오기 특성

**C++ 전용**

형식 라이브러리가로 참조 되 `#import` 고 자체에서 다른 형식 라이브러리를 참조 하는 경우 컴파일러는 `#import` 다른 형식 라이브러리에 대해 암시적 작업을 수행할 수 있도록 지정 합니다.

## <a name="syntax"></a>Syntax

> **#import** *형식 라이브러리* **auto_search**
