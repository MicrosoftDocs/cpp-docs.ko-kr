# <a name="auto_search-import-attribute"></a>auto_search import 특성

**C++컴퓨터별**

형식 라이브러리가로 `#import` 참조 되 고 자체에서 다른 형식 라이브러리를 참조 하는 경우 컴파일러는 다른 형식 라이브러리에 대해 `#import` 암시적 작업을 수행할 수 있도록 지정 합니다.

## <a name="syntax"></a>구문

> **#import** *형식 라이브러리* **auto_search**
