# <a name="namedguids"></a>named_guids
**C + + 전용**  
  
정의 하 고 폼의 이전 스타일의 GUID 변수를 초기화 하는 컴파일러가 `LIBID_MyLib`, `CLSID_MyCoClass`를 `IID_MyInterface`, 및 `DIID_MyDispInterface`합니다.  
  
## <a name="syntax"></a>구문  
  
```  
named_guids  
```
