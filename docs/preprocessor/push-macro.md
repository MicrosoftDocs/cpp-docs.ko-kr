# <a name="pushmacro"></a>push_macro
값을 저장 된 *macro_name* 이 매크로 대 한 스택의 맨 위에 매크로입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
#pragma push_macro("  
macro_name  
")  
  
```  
  
## <a name="remarks"></a>설명  
 에 대 한 값을 검색할 수 *macro_name* 와 **pop_macro**합니다.  
  
 참조 [pop_macro](../preprocessor/pop-macro.md) 샘플입니다.
