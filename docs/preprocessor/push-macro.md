# <a name="push_macro-pragma"></a>push_macro pragma

이 매크로의 스택 맨 위에 *매크로 이름* 매크로의 값을 저장 합니다.

## <a name="syntax"></a>구문

> **#pragma push_macro ("** _매크로-이름_ **")**

## <a name="remarks"></a>설명

를 사용 하 여 `pop_macro` *매크로 이름* 값을 검색할 수 있습니다.
