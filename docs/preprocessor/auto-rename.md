# <a name="autorename"></a>auto_rename
**C + + 전용**  
  
변수 이름에 잠재적인 이름 충돌을 해결하는 두 개의 밑줄(__)을 추가하여 C++ 예약어의 이름을 바꿉니다.  
  
## <a name="syntax"></a>구문  
  
```  
auto_rename  
```  
  
## <a name="remarks"></a>설명 

이 특성은 하나 이상의 C++ 예약어(키워드 또는 매크로)를 변수 이름으로 사용하는 형식 라이브러리를 가져올 때 사용됩니다.
