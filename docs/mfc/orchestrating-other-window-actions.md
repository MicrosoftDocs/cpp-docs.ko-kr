# <a name="orchestrating-other-window-actions"></a>기타 창 작업 조정

프레임 창은 상황에 맞는 도움말 및 인쇄 미리 보기와 같은 오케스트레이션 semimodal 상태를 제공 합니다. 인쇄 미리 보기의 프레임 창 역할에 대 한 설명은 인쇄 [및 인쇄 미리 보기](printing-and-print-preview.md)를 참조 하세요.
