# <a name="using-ctreectrl"></a>CTreeCtrl 사용

[CTreeCtrl](../mfc/reference/ctreectrl-class.md)클래스로 표시 되는 트리 컨트롤은 문서의 머리글, 인덱스의 항목 또는 디스크의 파일 및 디렉터리와 같은 항목의 계층 목록을 표시 하는 창입니다. 각 항목은 레이블과 선택적 비트맵 이미지로 구성 되며 각 항목에는 연결 된 하위 항목 목록이 있을 수 있습니다. 사용자는 항목을 클릭 하 여 연결 된 하위 항목 목록을 확장 하거나 축소할 수 있습니다. 파일 탐색기의 왼쪽 창에 있는 디렉터리 트리는 트리 컨트롤의 예입니다.
