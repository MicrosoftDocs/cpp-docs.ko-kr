# <a name="using-your-old-toolbars"></a>이전 도구 모음 사용

이전 버전의 Visual C++을 사용 하 여 사용자 지정 된 도구 모음을 만든 경우 클래스 [CToolBar](../mfc/reference/ctoolbar-class.md) 를 새로 구현 하면 문제가 발생할 수 있습니다. 새 기능을 사용 하기 위해 이전 도구 모음을 제공할 필요가 없도록 이전 구현은 계속 지원 됩니다.

DOCKTOOL 샘플에서는 기존 스타일 도구 모음을 사용 하지 않으며, 새 스타일 도구 모음만 사용 합니다.

도구 모음 리소스 편집기를 사용 하 여 이전 스타일 도구 모음을 편집할 수 없습니다.
