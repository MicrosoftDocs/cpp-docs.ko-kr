# <a name="ole-classes"></a>OLE 클래스

OLE 클래스는 다른 응용 프로그램 프레임 워크 클래스와 함께 작동 하 여 ActiveX API에 쉽게 액세스할 수 있으므로 프로그램에서 사용자에 게 ActiveX의 강력한 기능을 제공 하는 쉬운 방법을 제공 합니다. ActiveX를 사용 하 여 다음을 수행할 수 있습니다.

- 사용자가 텍스트, 그래픽, 스프레드시트, 소리 또는 다른 유형의 데이터를 비롯 한 여러 응용 프로그램에서 만든 데이터를 포함 하는 문서를 만들고 편집할 수 있도록 하는 복합 문서를 만듭니다.

- 복합 문서에 포함할 수 있는 OLE 개체를 만듭니다.

- OLE 끌어서 놓기를 사용 하 여 응용 프로그램 간에 데이터를 복사 합니다.

- 자동화를 사용 하 여 한 프로그램을 다른 프로그램으로 제어 합니다.

- ActiveX 컨트롤 및 ActiveX 컨트롤 컨테이너 (이전의 OLE 컨트롤 및 OLE 컨트롤 컨테이너)를 만듭니다.

ActiveX를 지 원하는 클래스의 범주는 다음과 같습니다.

- [OLE 일반 대화 상자 클래스](ole-common-dialog-classes.md)
