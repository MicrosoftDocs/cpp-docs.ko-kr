# <a name="tn050-mfcole-common-dialogs-mfcuix32"></a>TN050: MFC/OLE 일반 대화 상자(MFCUIx32)

이 메모는 원래 일부 문제와 MFC에서 제공 및 사용 되는 OLE 일반 대화 상자의 미래에 대해 설명 했습니다. OLE 일반 대화 상자는 이제 시스템 (OLEDLG.DLL)에 기본 제공 되는 구성 요소로 제공 되며 제품 설명서에 자세히 설명 되어 있습니다.
