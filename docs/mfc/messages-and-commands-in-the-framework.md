# <a name="messages-and-commands-in-the-framework"></a>프레임워크의 메시지 및 명령

Microsoft Windows 용으로 작성 된 응용 프로그램은 "메시지 구동"입니다. 마우스 클릭, 키 입력, 창 이동 등과 같은 이벤트에 대 한 응답으로 Windows는 메시지를 적절 한 창으로 보냅니다. 프레임 워크 응용 프로그램은 Windows 용 다른 응용 프로그램과 마찬가지로 Windows 메시지를 처리 합니다. 하지만 프레임 워크는 메시지 처리를 용이 하 고, 유지 관리 하기 쉽고, 캡슐화 하는 향상 된 기능도 제공 합니다.
