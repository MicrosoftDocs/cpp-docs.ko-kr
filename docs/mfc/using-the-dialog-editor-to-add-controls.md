# <a name="using-the-dialog-editor-to-add-controls"></a>대화 상자 편집기를 사용하여 컨트롤 추가

사용 하 여 대화 상자 템플릿 리소스를 만들 때 합니다 [대화 상자 편집기](../windows/dialog-editor.md), 컨트롤 팔레트에서 컨트롤을 끌어서 대화 상자에 놓으십시오. 이 대화 상자 템플릿 리소스에 해당 컨트롤 형식에 대 한 사양을 추가합니다. 호출을 대화 상자 개체를 생성할 때 해당 `Create` 또는 `DoModal` 멤버 함수를 프레임 워크 Windows 컨트롤을 만들고 화면에 대화 상자 창에 배치 합니다.

할 수 있습니다 [컨트롤을 직접 만들](../mfc/adding-controls-by-hand.md) 하려는 경우. 더 많은 작업입니다.
