# <a name="using-cricheditctrl"></a>CRichEditCtrl 사용

Rich edit 컨트롤은 사용자가 텍스트를 입력 하 고 편집할 수 있는 창입니다. 텍스트에는 문자 및 단락 서식 지정이 할당 될 수 있으며 포함 된 OLE 개체도 포함 될 수 있습니다. Rich edit 컨트롤은 [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) 클래스에 의해 MFC에 표시 됩니다.
