# <a name="servers-user-interface-issues"></a>서버: 사용자 인터페이스 문제

서버 응용 프로그램에는 컨테이너 응용 프로그램에 OLE 항목을 제공 하기 위해 사용자 인터페이스에 추가 해야 하는 다양 한 기능이 있습니다. 서버 응용 프로그램에 추가 해야 하는 메뉴 및 추가 리소스에 대 한 자세한 내용은 [메뉴 및 리소스: 서버 추가](../mfc/menus-and-resources-server-additions.md)를 참조 하세요.
