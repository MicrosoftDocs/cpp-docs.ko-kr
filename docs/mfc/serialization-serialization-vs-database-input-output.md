# <a name="serialization-serialization-vs-database-inputoutput"></a>Serialization: Serialization과 데이터베이스 입출력 비교

이 문서에서는 응용 프로그램이 데이터베이스 응용 프로그램에서와 같이 트랜잭션 별로 데이터를 읽고 쓰도록 하기 때문에 파일 기반 i/o (입/출력) 및 기타 i/o 기술이 적절 한 경우 문서 개체 및 serialization을 사용 하는 시기에 대해 설명 합니다. Serialization을 사용 하지 않는 경우에도 파일 열기, 저장 및 다른 이름으로 저장 명령이 필요 하지 않습니다. 다음 내용을 다룹니다.
