# <a name="tn066-common-mfc-3x-to-40-porting-issues"></a>TN066: 일반적인 MFC 3.x에서 4.0으로 이식 문제

이 기술 노트 MFC를 사용 하 여 작성 된 응용 프로그램을 이식 하는 동안 발생할 수 있는 가장 일반적인 문제를 설명 3.x (MFC 시각적 개체에 포함 된 C++ 2.x) MFC 4.0입니다.
