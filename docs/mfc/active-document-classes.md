# <a name="active-document-classes"></a>액티브 문서 클래스

활성 문서는 Internet Explorer 5.5 등의 웹 브라우저 전체 클라이언트 창 또는 활성 문서를 지 원하는 Microsoft Office 바인더와 같은 활성 컨테이너에서 표시할 수 있습니다.

[CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)<br/>
액티브 문서 인터페이스를 매핑하고 활성 문서 개체를 초기화 및 활성화 합니다.

[CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)<br/>
액티브 문서 서버 전용 OLE 서버 동사를 구현 합니다.

[COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)<br/>
액티브 문서 포함을 구현합니다.
