# <a name="using-cstatusbarctrl"></a>CStatusBarCtrl 사용

상태 표시줄 컨트롤 ([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md))을 사용 하 여 응용 프로그램에 대 한 다양 한 종류의 상태 정보를 반영 하는 컨트롤 창을 만들 수 있습니다. 상태 창은 두 가지 이상의 정보 유형을 표시하는 여러 부분으로 나눌 수 있습니다.
