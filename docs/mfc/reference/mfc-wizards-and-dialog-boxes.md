# <a name="mfc-wizards-and-dialog-boxes"></a>MFC 마법사 및 대화 상자

MFC (Microsoft Foundation Class) 마법사는 다양 한 종류의 구성 요소와 개체에 대 한 상용구 코드를 생성 합니다. **솔루션 탐색기** 에서 프로젝트에 대 한 바로 가기 메뉴를 열고 **추가**, **클래스** 를 선택 하 여 마법사를 실행할 수 있습니다.
