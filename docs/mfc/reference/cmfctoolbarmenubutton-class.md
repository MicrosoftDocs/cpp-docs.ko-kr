## <a name="remarks"></a>설명

A는 `CMFCToolBarMenuButton` 메뉴, 하위 메뉴가 있는 메뉴 항목, 명령을 실행하거나 메뉴를 표시하는 단추 또는 메뉴만 표시하는 단추로 나타날 수 있습니다. 생성자의 `CMFCToolbarMenuButton::CMFCToolbarMenuButton`단추와 연결된 이미지, 텍스트, 메뉴 핸들 및 명령 ID와 같은 매개 변수를 지정하여 메뉴 단추의 동작 및 모양을 결정합니다.

`CMFCToolbarMenuButton` 클래스에서 파생된 사용자 지정 클래스는 [DECLARE_SERIAL](run-time-object-model-services.md#declare_serial) 매크로를 사용해야 합니다. [DECLARE_DYNCREATE](run-time-object-model-services.md#declare_dyncreate) 매크로는 응용 프로그램이 닫히면 오류를 생성합니다.

## <a name="cmfctoolbarmenubuttonsetmenuonly"></a><a name="setmenuonly"></a>CMFCToolBar 메뉴 버튼::세트 메뉴만

단추에 유효한 명령 ID와 하위 메뉴가 모두 있는 경우 단추를 메뉴 단추 또는 분할 단추로 그려둘지 지정합니다.

```cpp
void SetMenuOnly(BOOL bMenuOnly);
```

### <a name="parameters"></a>매개 변수

*b메뉴만*<br/>
【인】 TRUE 는 유효한 명령 ID와 하위 메뉴가 모두있을 때 이 단추를 메뉴 단추로 표시하려면 FALSE가 유효한 명령 ID와 하위 메뉴가 모두있을 때 이 단추를 분할 단추로 표시합니다.

### <a name="remarks"></a>설명

일반적으로 도구 모음 메뉴 단추에 하위 메뉴와 명령 ID가 모두 있는 경우 메뉴에는 주 단추와 연결된 아래쪽 화살표 버튼이 있는 분할 단추로 나타납니다. 이 메서드를 호출 하 고 *bMenuOnlyTRUE,* 대신 단추에 아래쪽 화살표가 있는 단일 메뉴 단추로 나타납니다. 사용자가 두 모드에서 화살표를 클릭하면 하위 메뉴가 열리고 사용자가 어느 모드에서든 단추의 비화살표 부분을 클릭하면 프레임워크가 명령을 실행합니다.

## <a name="cmfctoolbarmenubuttonsetradio"></a><a name="setradio"></a>CMFCToolBar메뉴버튼::셋라디오

도구 모음 메뉴 단추를 선택시 라디오 단추 스타일 아이콘을 표시하도록 설정합니다.

```
virtual void SetRadio();
```

### <a name="remarks"></a>설명

메뉴 단추를 선택 하는 동안 그려집니다, 그것은 [호출 CMFCVisualManager::OnDrawMenuCheck](../../mfc/reference/cmfcvisualmanager-class.md#ondrawmenucheck) 체크 표시 아이콘을 그립니다. 기본적으로 현재 `OnDrawMenuCheck` 시각적 관리자가 메뉴 단추에 확인란 스타일 확인란을 그려달라는 요청이 있습니다. 이 메서드를 호출한 후 현재 시각적 관리자는 메뉴 단추에 라디오 단추 스타일 체크표시를 그립니다. 이 변경 은 취소할 수 없습니다.

이 메서드를 호출하고 메뉴 버튼이 현재 표시되면 새로 고쳐집니다.
