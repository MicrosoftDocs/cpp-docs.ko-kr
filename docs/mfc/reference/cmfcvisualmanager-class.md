# <a name="cmfcvisualmanager-class"></a>CMFCVisualManager 클래스

애플리케이션의 모양을 전역 수준에서 변경하는 기능을 지원합니다. `CMFCVisualManager` 클래스는 일관된 스타일을 사용하여 응용 프로그램의 GUI 컨트롤을 그리는 명령을 제공하는 클래스와 함께 작동합니다. 이러한 다른 클래스를 비주얼 관리자라고 하며 `CMFCBaseVisualManager`에서 상속됩니다.

자세한 내용은 Visual Studio 설치의 **VC \\ s\mfc \\ src \\ mfc** 폴더에 있는 소스 코드를 참조 하세요.

## <a name="syntax"></a>구문

```
class CMFCVisualManager : public CMFCBaseVisualManager
```

## <a name="remarks"></a>설명

`CMFCVisualManager`클래스가 응용 프로그램의 GUI를 제어 하기 때문에 각 응용 프로그램에는의 인스턴스 `CMFCVisualManager` 를 하나 또는에서 파생 된 클래스의 인스턴스 하나를 사용할 수 있습니다 `CMFCVisualManager` . 응용 프로그램은 없이도 작동할 수 있습니다 `CMFCVisualManager` . 정적 메서드를 사용 `GetInstance` 하 여 현재 파생 개체에 대 한 포인터를 가져옵니다 `CMFCVisualManager` .

응용 프로그램의 모양을 변경 하려면 응용 프로그램의 모든 시각적 요소를 그리기 위한 메서드를 제공 하는 다른 클래스를 사용 해야 합니다. 이러한 클래스의 몇 가지 예로는 [CMFCVisualManagerOfficeXP 클래스](../../mfc/reference/cmfcvisualmanagerofficexp-class.md), [CMFCVisualManagerOffice2003 클래스](../../mfc/reference/cmfcvisualmanageroffice2003-class.md)및 [CMFCVisualManagerOffice2007 클래스가](../../mfc/reference/cmfcvisualmanageroffice2007-class.md)있습니다. 응용 프로그램의 모양을 변경 하려면 이러한 시각적 관리자 중 하나를 메서드에 전달 `SetDefaultManager` 합니다. 응용 프로그램이 Microsoft Office 2003의 모양을 모방 하는 방법을 보여 주는 예제는 [CMFCVisualManagerOffice2003 클래스](../../mfc/reference/cmfcvisualmanageroffice2003-class.md)를 참조 하세요.

모든 그리기 메서드는 가상입니다. 이렇게 하면 응용 프로그램의 GUI에 대 한 사용자 지정 비주얼 스타일을 만들 수 있습니다. 사용자 고유의 비주얼 스타일을 만들려는 경우 visual manager 클래스 중 하나에서 클래스를 파생 시키고 변경할 그리기 메서드를 재정의 합니다.

void CMFCSkinsApp::SetSkin (int iIndex)

    if (CMFCVisualManager::GetInstance () != NULL)

        // create the standard visual manager

        // create the first custom visual manager
        break;
    case 2:
        new CMacStyle ();
        // create the second custom visual manager
        break;
    }
    // access the manager and set it properly
    CMFCVisualManager::GetInstance ()->SetLook2000 ();
    CMFCVisualManager::GetInstance ()->RedrawAll ();
}

비활성화 된 도구 모음 이미지에 대해 볼록 모드를 사용 하거나 사용 하지 않도록 설정 합니다.

```cpp
void SetEmbossDisabledImage (BOOL bEmboss = TRUE);
```

### <a name="parameters"></a>매개 변수

*bEmboss*<br/>
진행 비활성화 된 도구 모음 이미지에 대해 볼록 모드를 사용할지 여부를 나타내는 부울 매개 변수입니다.

### <a name="remarks"></a>설명

[Cmfcvisualmanager:: IsEmbossDisabledImage](#isembossdisabledimage) 함수를 사용 하 여 볼록 모드가 사용 되는지 여부를 확인 합니다.

## <a name="cmfcvisualmanagersetfadeinactiveimage"></a><a name="setfadeinactiveimage"></a> CMFCVisualManager:: SetFadeInactiveImage

메뉴 또는 도구 모음에서 비활성 이미지에 대 한 조명 효과를 사용 하거나 사용 하지 않도록 설정 합니다.

```cpp
void SetFadeInactiveImage(BOOL bFade = TRUE);
```

### <a name="parameters"></a>매개 변수

*bFade*<br/>
진행 조명 효과를 사용 하도록 설정할지 여부를 지정 하는 부울 매개 변수입니다.

### <a name="remarks"></a>설명

이 기능은 비활성 이미지가 메뉴 또는 도구 모음에서 페이드 인 된 것으로 표시 되는지 여부를 제어 합니다. [Cmfcvisualmanager:: IsFadeInactiveImage](#isfadeinactiveimage) 메서드를 사용 하 여이 기능을 사용할 수 있는지 여부를 확인 합니다.

## <a name="cmfcvisualmanagersetmenuflatlook"></a><a name="setmenuflatlook"></a> CMFCVisualManager:: SetMenuFlatLook

메뉴 단추를 평면으로 표시할지 여부를 나타내는 플래그를 설정 합니다. 그렇지 않으면 3 차원으로 표시 됩니다.

```cpp
void SetMenuFlatLook(BOOL bMenuFlatLook = TRUE);
```

### <a name="parameters"></a>매개 변수

*bMenuFlatLook*<br/>
진행 메뉴 단추를 평면으로 표시할지 여부를 나타내는 부울 매개 변수입니다.

### <a name="remarks"></a>설명

기본적으로이 기능은 사용 되지 않습니다.

## <a name="cmfcvisualmanagersetmenushadowdepth"></a><a name="setmenushadowdepth"></a> CMFCVisualManager:: SetMenuShadowDepth

void SetMenuShadowDepth(int nDepth);

### <a name="parameters"></a>매개 변수

*nDepth*<br/>

메뉴 그림자의 높이와 너비는 동일 해야 합니다. 기본값은 7 픽셀입니다.
