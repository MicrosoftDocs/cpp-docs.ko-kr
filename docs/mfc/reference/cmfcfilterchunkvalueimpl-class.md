    DWORD cwcStartSource = 0,

*cwcLen소스*<br/>
현재 청크가 파생된 원본 텍스트의 문자 길이입니다. 0 값은 원본 텍스트와 파생 된 텍스트 간의 문자별 대응을 의미합니다. 영하지 않은 값은 그러한 직접적인 대응이 존재하지 않는다는 것을 의미합니다.
