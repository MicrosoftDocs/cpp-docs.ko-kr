##  <a name="cd2dpointf"></a>CD2DPointF::CD2DPointF  
 CPoint 개체에서 CD2DPointF 개체를 만듭니다.  
  
```  
CD2DPointF(const CPoint& pt);    
CD2DPointF(const D2D1_POINT_2F& pt);    
CD2DPointF(const D2D1_POINT_2F* pt); 
CD2DPointF(FLOAT fX = 0., FLOAT fY = 0.);
```  
  
### <a name="parameters"></a>매개 변수

 `fY`
