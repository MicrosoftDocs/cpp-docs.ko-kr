# <a name="data-types-mfc"></a>데이터 형식(MFC)

이 항목에서는 MFC 라이브러리에서 가장 일반적으로 사용 되는 데이터 형식을 나열 합니다. 대부분의 데이터 형식은 플랫폼 SDK (소프트웨어 개발 키트)의 데이터 형식과 동일 하지만 다른 형식은 MFC에 고유 합니다.

Windows SDK와 MFC 모두에서 사용 되는 데이터 형식에 대 한 자세한 내용은             [Windows 데이터 형식](/windows/win32/WinProg/windows-data-types)을 참조 하세요.

MFC 라이브러리에 고유한 데이터 형식은 다음과 같습니다.

- 컬렉션에서 요소의 위치를 나타내는 데 사용 되는 값을 배치 합니다. MFC 컬렉션 클래스에서 사용 됩니다.

- LPCRECT 상수 (수정할 수 없음) 구조체에 대 한 32 비트 포인터                     `RECT` 입니다.
