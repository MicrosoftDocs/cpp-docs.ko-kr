# <a name="data-types-mfc"></a>데이터 형식(MFC)

이 항목에서는 Microsoft Foundation Class 라이브러리에서 가장 일반적으로 사용 되는 데이터 형식을 나열 합니다. 대부분의 데이터 형식은 MFC에 고유한 플랫폼 소프트웨어 개발 키트 (SDK에)와 동일 합니다.

Windows SDK 및 MFC에서 사용 되는 데이터 형식에 대 한 자세한 내용은 [Windows 데이터 형식](/windows/desktop/WinProg/windows-data-types)합니다.

Microsoft Foundation Class 라이브러리에 고유한 데이터 형식은 다음과 같습니다.

- 컬렉션에서 요소의 위치를 나타내는 데 사용 되는 위치는 값 MFC 컬렉션 클래스에서 사용합니다.

- 상수 (예제)에 대 한 32 비트 포인터 LPCRECT `RECT` 구조입니다.
