# <a name="editing-a-message-handler"></a>메시지 처리기 편집

메시지 처리기를 정의한 후에는 멤버 함수의 정의로 이동 하 여 코드를 추가 하거나 수정할 수 있습니다.

[대화 상자 편집기](../../windows/dialog-editor.md)에서 멤버 함수 정의로 이동 하려면 처리기가 이미 정의 된 컨트롤을 두 번 클릭 합니다. 그러면 선택한 컨트롤의 메시지 처리기가 정의 된 파일로 이동 합니다.

[클래스 뷰](/visualstudio/ide/viewing-the-structure-of-code)에서 멤버 함수 정의로 이동 하려면 클래스 뷰에서 함수 이름을 두 번 클릭 합니다.
