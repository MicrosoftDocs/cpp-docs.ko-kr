# <a name="cmdiframewndex-class"></a>CMDIFrameWndEx 클래스

*tabLocation*<br/>
탭 레이블의 위치를 지정 합니다.

*bTabCloseButton*<br/>
탭 닫기 단추를 표시할지 여부를 지정 합니다.

모든 탭 창이 세로로 맞춰질 때 또는 탭 창이 하나만 있는 경우 새 세로 탭 그룹을 만들 수 있습니다.

모든 탭 창이 가로로 정렬 되거나 탭 창이 하나만 있는 경우 새 가로 탭 그룹을 만들 수 있습니다.

탭 창에 탭이 둘 이상 있는 경우에만 탭을 이전 그룹으로 이동할 수 있습니다.

탭 창에 탭이 둘 이상 있는 경우에만 탭을 다음 그룹으로 이동할 수 있습니다.

## <a name="cmdiframewndexremovepanefromdockmanager"></a><a name="removepanefromdockmanager"></a> CMDIFrameWndEx:: RemovePaneFromDockManager

void RemovePaneFromDockManager(

## <a name="cmdiframewndexsavemdistate"></a><a name="savemdistate"></a> CMDIFrameWndEx:: SaveMDIState

현재 MDI 탭 그룹의 레이아웃과 이전에 열린 문서의 목록을 저장 합니다.

```
virtual BOOL SaveMDIState(LPCTSTR lpszProfileName);
```

### <a name="parameters"></a>매개 변수

*lpszProfileName*<br/>
진행 프로필 이름을 지정 합니다.

### <a name="return-value"></a>반환 값

저장에 성공 하면 TRUE입니다. 저장 하지 못한 경우 FALSE입니다.

### <a name="remarks"></a>설명

MDI 탭 및 그룹의 상태를 로드 하거나 저장 하 고 열려 있는 문서 목록을 저장 하려면 다음을 수행 합니다.

- `SaveMDIState`주 프레임이 닫힐 때 호출 됩니다.

- 주 프레임을 만들 때 [CMDIFrameWndEx:: LoadMDIState](#loadmdistate) 를 호출 합니다. 이 호출에 권장 되는 위치는 주 프레임이 처음으로 표시 되기 전에입니다.

- `CWinAppEx::EnableLoadWindowPlacement(FALSE);`이전 호출`pMainFrame->LoadFrame (IDR_MAINFRAME);`

- `CWinAppEx::ReloadWindowPlacement(pMainFrame)`뒤에 `LoadMDIState` 를 호출 하 여 레지스트리에 저장 된 위치에 주 프레임을 표시 합니다.

- `GetDocumentName` `CMDIChildWndEx` 응용 프로그램에서 파일로 저장 되지 않은 문서를 표시 하는 경우 파생 클래스에서를 재정의 합니다. 반환 된 문자열은 레지스트리에 문서 식별자로 저장 됩니다. 자세한 내용은 [CMDIChildWndEx:: GetDocumentName](../../mfc/reference/cmdichildwndex-class.md#getdocumentname)를 참조 하세요.

- [CMDIFrameWndEx:: CreateDocumentWindow](#createdocumentwindow) 를 재정의 하 여 레지스트리에서 로드 될 때 문서를 올바르게 만듭니다. 에 대 한 매개 변수는 `CreateDocumentWindow` 앞에서 반환 된 문자열입니다 `GetDocumentName` .
