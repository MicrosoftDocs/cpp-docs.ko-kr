##  <a name="savemdistate"></a>  CMDIFrameWndEx::SaveMDIState

MDI 탭 그룹의 현재 레이아웃 및 이전에 열린된 문서의 목록을 저장합니다.

```
virtual BOOL SaveMDIState(LPCTSTR lpszProfileName);
```

### <a name="parameters"></a>매개 변수

*lpszProfileName*<br/>
[in] 프로필 이름을 지정합니다.

### <a name="return-value"></a>반환 값

저장 성공 하면 TRUE입니다. FALSE 이면 저장 하지 못했습니다.

### <a name="remarks"></a>설명

으로 로드 하거나 MDI 탭 및 그룹의 상태 및 열린된 문서의 목록을 저장 하려면 다음을 수행 합니다.

- 호출 `SaveMDIState` 주 프레임을 닫을 때

- 호출 [cmdiframewndex:: Loadmdistate](#loadmdistate) 주 프레임이 만들어질 때. 이 호출에 대 한 권장 되는 위치에는 주 프레임은 처음으로 표시 되기 전에입니다.

- 호출 `CWinAppEx::EnableLoadWindowPlacement(FALSE);` 하기 전에 `pMainFrame->LoadFrame (IDR_MAINFRAME);`

- 호출 `CWinAppEx::ReloadWindowPlacement(pMainFrame)` 후 `LoadMDIState` 레지스트리에 저장 된 위치에 있는 주 프레임을 표시 합니다.

- 재정의 `GetDocumentName` 에 `CMDIChildWndEx`-응용 프로그램 파일로 저장 되지 않은 문서를 표시 하는 경우 클래스를 파생 합니다. 문서 식별자로 반환된 된 문자열 레지스트리에 저장 됩니다. 자세한 내용은 [CMDIChildWndEx::GetDocumentName](../../mfc/reference/cmdichildwndex-class.md#getdocumentname)합니다.

- 재정의 [CMDIFrameWndEx::CreateDocumentWindow](#createdocumentwindow) 레지스트리에서 로드 될 때 문서를 올바르게 만드는 합니다. 매개 변수를 `CreateDocumentWindow` 문자열인는 `GetDocumentName` 이전에 반환 합니다.
