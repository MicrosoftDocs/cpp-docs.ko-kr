# <a name="cmfctabctrl-class"></a>CMFCTabCtrl Class

클래스는 `CMFCTabCtrl` 탭 컨트롤에 대 한 기능을 제공 합니다. 탭 컨트롤은 맨 위 또는 아래에 평면 또는 3차원 탭이 포함된 도킹 가능한 창을 표시합니다. 탭은 텍스트와 이미지를 표시하고 활성화된 경우 색을 변경할 수 있습니다.

## <a name="syntax"></a>구문

```
class CMFCTabCtrl : public CMFCBaseTabCtrl
```

## <a name="remarks"></a>설명

클래스는 다음을 `CMFCTabCtrl` 지원합니다.

- 공유 가로 스크롤 막대가 있는 3D, 플랫 및 플랫이 포함된 탭 컨트롤 스타일입니다.

- 창의 위쪽 또는 아래쪽에 있는 탭입니다.

- 텍스트, 이미지 또는 텍스트와 이미지를 표시하는 탭입니다.

- 탭이 활성화되어 있을 때 색상이 변경되는 탭입니다.

- 조정 가능한 탭의 테두리 크기가 변경됩니다.

- 분리 가능한 탭 창입니다.

이 `CMFCTabCtrl` 클래스는 대화 상자와 함께 사용할 수 있지만 Microsoft Excel 및 Visual Studio와 같은 도킹 컨트롤 막대를 사용하는 응용 프로그램을 위한 것입니다. 자세한 내용은 [CDockablePane 클래스를](../../mfc/reference/cdockablepane-class.md)참조하십시오.

다음 단계에 따라 응용 프로그램에서 조정 가능한 도킹 탭 컨트롤을 추가합니다.

1. [CTabbedPane 클래스의 인스턴스를](../../mfc/reference/ctabbedpane-class.md)만듭니다.

1. [CDockablePane::만들기를](../../mfc/reference/cdockablepane-class.md#create)호출합니다.

1. [CBaseTabbedPane을](../../mfc/reference/cbasetabbedpane-class.md#addtab) 사용 하 여:추가 탭 또는 [CMFCBaseTabCtrl::삽입 탭](../../mfc/reference/cmfcbasetabctrl-class.md#inserttab) 새 탭을 추가 합니다.

1. 현재 도킹 탭 컨트롤이 주 프레임 창에서 도킹할 수 있도록 [CBasePane::EnableDocking을](../../mfc/reference/cbasepane-class.md#enabledocking) 호출합니다.

1. [CFrameWndEx::DockPane로](../../mfc/reference/cframewndex-class.md#dockpane) 호출하여 메인 프레임에서 탭된 창을 도킹합니다.

도킹 컨트롤 막대로 탭된 창을 만드는 방법에 대한 예는 [CTabbedPane 클래스를](../../mfc/reference/ctabbedpane-class.md)참조하십시오. 도킹되지 않는 컨트롤로 사용하려면 `CMFCTabCtrl` `CMFCTabCtrl` 개체를 만든 다음 [CMFCTabCtrl::Create를](#create)호출합니다.

【인】 TRUE활성 탭의 레이블을 표시하기 위해 굵은 글꼴을 사용합니다. FALSE는 표준 글꼴을 사용하여 레이블을 표시합니다. 기본값은 TRUE입니다.

### <a name="remarks"></a>설명

## <a name="cmfctabctrlsetdrawframe"></a><a name="setdrawframe"></a>CMFCTabCtrl:::세트드로우프레임

```cpp
void SetDrawFrame(BOOL bDraw=TRUE);
```

### <a name="parameters"></a>매개 변수

*b 그리기*<br/>
【인】 임베디드 막대 주위에 프레임 사각형을 표시하는 TRUE; 그렇지 않으면 false입니다. 기본값은 TRUE입니다.

### <a name="remarks"></a>설명

## <a name="cmfctabctrlsetflatframe"></a><a name="setflatframe"></a>CMFCTabCtrl::세트플랫프레임

탭 영역 주위에 플랫 또는 3D 프레임을 그릴지 여부를 지정합니다.

```cpp
void SetFlatFrame(
    BOOL bFlat=TRUE,
    BOOL bRepaint=TRUE);
```

### <a name="parameters"></a>매개 변수

*b플랫*<br/>
【인】 탭 영역 주위에 플랫(2D) 프레임을 그리는 트루; FALSE는 3차원(3D) 프레임을 그립니다. 기본값은 TRUE입니다.

*bRepaint*<br/>
【인】 TRUE는 즉시 창을 다시 그릴 수 있습니다. 그렇지 않으면 false입니다. 기본값은 TRUE입니다.

### <a name="remarks"></a>설명

## <a name="cmfctabctrlsetimagelist"></a><a name="setimagelist"></a>CMFCTabCtrl:::세트이미지리스트

이미지 목록을 지정합니다.

```
virtual BOOL SetImageList(
    UINT uiID,
    int cx=15,
    COLORREF clrTransp=RGB(255, 0, 255));

virtual BOOL SetImageList(HIMAGELIST hImageList);
```

### <a name="parameters"></a>매개 변수

*uiID*<br/>
【인】 이미지 목록을 포함하는 비트맵 리소스의 ID입니다.

*Cx*<br/>
【인】 각 이미지의 너비(픽셀)입니다. 기본값은 15입니다.

*clrTransp*<br/>
【인】 투명 한 이미지 색상입니다. 이 색상인 이미지 부분은 투명합니다. 기본값은 색상 자홍색, RGB(255,0,255)입니다.

*hImageList*<br/>
[in] 사전 로드된 이미지 목록에 대한 핸들입니다.

### <a name="return-value"></a>Return Value

이 메서드가 성공하면 TRUE입니다. FALSE 탭 컨트롤 플랫 스타일을 사용 하 여 만들어집니다 또는 첫 번째 메서드 오버 로드 *uiID* 매개 변수에 의해 지정 된 비트 맵을 로드할 수 없는 경우.

### <a name="remarks"></a>설명

이 메서드를 사용하여 탭 컨트롤에 대한 이미지 목록을 설정합니다. 이미지 목록의 이미지가 탭 레이블 옆에 표시됩니다. 이 메서드는 탭 의 크기를 조정하여 이미지와 텍스트를 모두 포함하도록 탭 높이를 다시 계산합니다.

탭 컨트롤에 의해 상속되는 [CMFCBaseTabCtrl::AddTab](../../mfc/reference/cmfcbasetabctrl-class.md#addtab) 메서드를 사용하여 표시할 이미지의 인덱스를 지정합니다.

## <a name="cmfctabctrlsetresizemode"></a><a name="setresizemode"></a>CMFCTabCtrl:::세트 사이즈 모드

현재 탭 컨트롤의 크기를 조정한 다음 컨트롤을 다시 표시하는 방법을 지정합니다.

```cpp
void SetResizeMode(ResizeMode resizeMode);
```

### <a name="parameters"></a>매개 변수

*크기 조정 모드*<br/>
【인】 탭 컨트롤의 `CMFCTabCtrl::ResizeMode` 크기를 조정하는 방법을 지정하는 열거형 값 중 하나입니다. 가능한 값 목록은 비고의 표를 참조하십시오.

### <a name="remarks"></a>설명

*resizeMode* 매개 변수는 다음 `ResizeMode` 열거 값 중 하나가 될 수 있습니다.

|속성|Description|
|----------|-----------------|
|RESIZE_NO|탭 컨트롤의 크기를 조정할 수 없습니다.|
|RESIZE_VERT|탭 컨트롤의 크기는 세로로 조정될 수 있지만 가로로는 크기를 조정할 수 없습니다.|
|RESIZE_HORIZ|탭 컨트롤의 크기는 가로로 조정될 수 있지만 세로로는 크기를 조정할 수 없습니다.|

## <a name="cmfctabctrlsettabmaxwidth"></a><a name="settabmaxwidth"></a>CMFCTabCtrl::세트탭맥스폭

탭된 창에서 최대 탭 너비를 지정합니다.

```cpp
void SetTabMaxWidth(int nTabMaxWidth);
```

### <a name="parameters"></a>매개 변수

*nTabMax폭*<br/>
【인】 최대 탭 너비(픽셀)입니다.

### <a name="remarks"></a>설명

이 메서드를 사용하여 탭된 창에서 각 탭의 너비를 제한합니다. 이 방법은 탭에 레이블이 매우 긴 경우에 유용합니다. [CMFCTabCtrl](../../mfc/reference/cmfctabctrl-class.md) 클래스 생성자는 최대 탭 너비를 0으로 초기화하므로 실제로 너비가 제한되지 않음을 의미합니다.

## <a name="cmfctabctrlstopresize"></a><a name="stopresize"></a>CMFCTabCtrl::정지 조정

```cpp

```

### <a name="parameters"></a>매개 변수

*bCancel*<br/>

## <a name="cmfctabctrlsynchronizescrollbar"></a><a name="synchronizescrollbar"></a>CMFCTabCtrl::동기화스크롤바

탭 컨트롤에 플랫 탭을 표시하는 가로 스크롤 막대를 그립니다.

```
BOOL SynchronizeScrollBar(SCROLLINFO* pScrollInfo = NULL);
```

### <a name="parameters"></a>매개 변수

*p스크롤정보*<br/>
【아웃】 [SCROLLINFO](/windows/win32/api/winuser/ns-winuser-scrollinfo) 구조 또는 NULL에 대한 포인터입니다. 이 메서드가 반환되고 이 매개 변수가 NULL이 아닌 경우 구조에 스크롤 막대의 모든 매개 변수가 포함됩니다. 기본값은 NULL입니다.

### <a name="return-value"></a>Return Value

이 메서드가 성공하면 TRUE입니다. 그렇지 않으면 false입니다.

### <a name="remarks"></a>설명

이 메서드는 플랫 탭을 표시 하는 탭 컨트롤만 영향을 줍니다. 스크롤 막대는 모든 탭에 동시에 영향을 미칩니다.
