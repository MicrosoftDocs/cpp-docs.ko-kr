# <a name="cframewndex-class"></a>CFrameWndEx 클래스

겹쳐진 Windows SDI(단일 문서 인터페이스) 또는 팝업 프레임 창의 기능을 구현하고 창 관리를 위한 멤버를 제공합니다. [CFrameWnd](../../mfc/reference/cframewnd-class.md) 클래스를 확장 합니다.

## <a name="syntax"></a>구문

```
class CFrameWndEx : public CFrameWnd
```
