# <a name="cframewndex-class"></a>CFrameWndEx 클래스

겹쳐진 Windows SDI(단일 문서 인터페이스) 또는 팝업 프레임 창의 기능을 구현하고 창 관리를 위한 멤버를 제공합니다. [CFrameWnd](../../mfc/reference/cframewnd-class.md) 클래스를 확장합니다.

## <a name="syntax"></a>구문

```
class CFrameWndEx : public CFrameWnd
```

## <a name="cframewndexshowpane"></a><a name="showpane"></a>CFrameWndEx::쇼파인

지정된 창을 표시하거나 숨깁니다.

```cpp
void ShowPane(
    CBasePane* pBar,
    BOOL bShow,
    BOOL bDelay,
    BOOL bActivate);
```

### <a name="parameters"></a>매개 변수

*pBar*<br/>
【인】 표시하거나 숨길 컨트롤 막대에 대한 포인터입니다.

*bShow*<br/>
【인】 TRUE이면 응용 프로그램에 컨트롤 막대가 표시됩니다. 그렇지 않으면 응용 프로그램이 컨트롤 막대를 숨깁니다.

*bDelay*<br/>
【인】 TRUE인 경우 프레임워크에서 [CFrameWndEx::adjustDockingLayout를](#adjustdockinglayout)호출할 때까지 도킹 레이아웃 조정을 지연합니다. 그렇지 않으면 도킹 레이아웃을 즉시 다시 계산합니다.

*bActivate*<br/>
【인】 TRUE인 경우 컨트롤 막대를 활성화합니다. 그렇지 않으면 컨트롤 막대를 비활성 상태로 표시합니다.
