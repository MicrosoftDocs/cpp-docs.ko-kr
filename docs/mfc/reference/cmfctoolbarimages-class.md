```
    HBITMAP hbmp,
    BOOL bSetBitPerPixel=FALSE);

    const CMFCToolBarImages& imageList,

```

### <a name="parameters"></a>매개 변수

*hbmp*<br/>
진행 추가할 비트맵에 대 한 핸들입니다.

*bSetBitPerPixel*<br/>
진행 `CMFCToolBarImages` 개체가 새 이미지의 색 농도 (픽셀 당 비트)를 사용 하는 경우 TRUE입니다. `CMFCToolbarImages` 개체가 현재 색 농도를 유지 하면 FALSE입니다.

*imageList*<br/>
진행 추가할 이미지를 포함 하는 `CMFCToolbarImages` 개체에 대 한 참조입니다.

*nIndex*<br/>
진행 추가할 이미지의 소스 `CMFCToolbarImages` 개체에 있는 인덱스입니다.

### <a name="return-value"></a>Return Value

```
BOOL Load(
    UINT uiResID,
    HINSTANCE hinstRes=NULL,
    BOOL bAdd=FALSE);

BOOL Load(
    LPCTSTR lpszBmpFileName,
    DWORD nMaxFileSize = 819200);
```

### <a name="parameters"></a>매개 변수

*uiResID*<br/>
진행 비트맵 리소스의 ID입니다.

*Hin Res*<br/>
진행 리소스 DLL의 인스턴스입니다.

*bAdd*<br/>
진행 로드 된 비트맵을 기존 비트맵에 추가 하려면 TRUE이 고, 기존 비트맵을 바꾸려면 FALSE입니다.

*lpszBmpFileName*<br/>
진행 비트맵을 로드할 디스크 파일에 대 한 경로입니다.

*nMaxFileSize*<br/>
진행 비트맵 파일의 최대 바이트 수입니다. 또는 0은 파일 크기에 관계 없이 비트맵을 로드 합니다. 파일 크기가이 최대 크기를 초과 하면 메서드에서 FALSE를 반환 하 고 비트맵을 로드 하지 않습니다.

### <a name="return-value"></a>Return Value

비트맵이 성공적으로 로드 되었으면 TRUE이 고, 그렇지 않으면입니다. 그렇지 않으면 FALSE입니다.

### <a name="remarks"></a>설명

파일에 읽기 전용 특성이 있는 경우에는 이미지 목록이 읽기 전용으로 표시 됩니다.

```
    CAfxDrawState& ds,

    0)
    BOOL bFadeInactive=FALSE);
```
