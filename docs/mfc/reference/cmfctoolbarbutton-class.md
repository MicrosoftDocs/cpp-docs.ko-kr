## <a name="remarks"></a>설명

`CMFCToolbarButton` 개체는 도구 모음에 있는 컨트롤입니다. 해당 동작은 일반 단추와 유사 합니다. 이 개체에 이미지 및 텍스트 레이블을 할당할 수 있습니다. 도구 모음 단추에는 명령 ID를 사용할 수도 있습니다. 사용자가 도구 모음 단추를 클릭 하면 프레임 워크에서이 ID가 지정 하는 명령을 실행 합니다.

일반적으로 도구 모음 단추를 사용자 지정할 수 있습니다. 즉, 사용자가 도구 모음에서 단추를 다른 도구 모음으로 끌어 오고 텍스트 레이블 및 이미지를 복사, 붙여넣기, 삭제 및 편집할 수 있습니다. 사용자가 도구 모음을 사용자 지정 하는 것을 방지 하기 위해 도구 모음을 두 가지 방법 중 하나로 잠글 수 있습니다. `bLocked` [Cmfctoolbar:: loadtoolbar](../../mfc/reference/cmfctoolbar-class.md#loadtoolbar)를 호출할 때 플래그를 TRUE로 설정 하거나, [CMFCToolBarButton:: SetProtectedCommands](#setprotectedcommands) 메서드를 사용 하 여 개별 단추의 명령 ID를 보호 된 명령의 전역 목록에 추가 합니다.

`CMFCToolBarButton`개체는 응용 프로그램의 도구 모음 이미지의 전역 컬렉션에서 이미지를 표시 합니다. 이러한 컬렉션은 [Cmfctoolbar 클래스](../../mfc/reference/cmfctoolbar-class.md)의 부모 도구 모음에서 유지 관리 됩니다. 자세한 내용은 [Cmfc추가 정보 이미지 클래스](../../mfc/reference/cmfctoolbarimages-class.md)를 참조 하세요.

사용자가 도구 모음 단추를 클릭 하면 해당 부모 도구 모음에서 마우스 메시지를 처리 하 고 단추에 적절 한 작업을 전달 합니다. 단추에 유효한 명령 ID가 있으면 부모 도구 모음에서 WM_COMMAND 메시지를 부모 프레임으로 보냅니다.

클래스는 [cmfcCMFCToolBarEditBoxButton menubutton 클래스](../../mfc/reference/cmfctoolbarmenubutton-class.md), [클래스](../../mfc/reference/cmfctoolbareditboxbutton-class.md)및 [CMFCToolBarComboBoxButton 클래스](../../mfc/reference/cmfctoolbarcomboboxbutton-class.md)와 같은 다른 도구 모음 단추 클래스의 기본 클래스입니다. `CMFCToolBarButton`

이 메서드는 새 경계 사각형을 설정한 후 [CMFCToolBarButton:: OnMove](#onmove) 메서드를 호출 합니다.

##  <a name="setstyle"></a>  CMFCToolBarButton::SetStyle

단추의 스타일을 설정 합니다.

```
virtual void SetStyle(UINT nStyle);
```

### <a name="parameters"></a>매개 변수

*nStyle*<br/>
진행 단추의 새 스타일입니다.

### <a name="remarks"></a>설명

기본 구현에서는 [CMFCToolBarButton:: m_nStyle](#m_nstyle) 데이터 멤버를 *nstyle*으로 설정 합니다. 스타일 변경을 처리 하기 위해 추가 처리를 수행 하려면이 메서드를 재정의 합니다. 유효한 스타일 플래그 목록은 [ToolBar 컨트롤 스타일](toolbar-control-styles.md) 을 참조 하세요.

##  <a name="setvisible"></a>  CMFCToolBarButton::SetVisible

단추의 표시 여부를 지정 합니다.

```
void SetVisible(BOOL bShow=TRUE);
```

### <a name="parameters"></a>매개 변수

*bShow*<br/>
진행 단추를 표시 하거나 숨길지 여부를 지정 하는 부울 값입니다. 이 매개 변수가 TRUE 이면 단추가 표시 됩니다. 매개 변수가 FALSE 이면 단추가 숨겨집니다.

### <a name="remarks"></a>설명

특정 도구 모음 단추를 숨기 거 나 표시 하려면이 함수를 사용 합니다. 이 메서드를 호출한 후 [Cpane:: AdjustSizeImmediate](../../mfc/reference/cpane-class.md#adjustsizeimmediate) 메서드를 호출 합니다.

##  <a name="show"></a>  CMFCToolBarButton::Show

단추를 표시 하거나 숨깁니다.

```
void Show(BOOL bShow);
```

### <a name="parameters"></a>매개 변수

*bShow*<br/>
진행 단추를 표시 하거나 숨길지 여부를 지정 하는 부울 값입니다. 이 매개 변수가 TRUE 이면 단추가 표시 됩니다. 매개 변수가 FALSE 이면 단추가 숨겨집니다.

### <a name="remarks"></a>설명

프레임 워크는이 메서드를 호출 하 여 부모 도구 모음의 크기가 조정 될 때 도구 모음 단추의 표시 여부를 업데이트 합니다. 단추가 더 이상 도구 모음 범위 내에 맞지 않는 경우 프레임 워크는 *Bshow* 를 FALSE로 설정 하 여이 메서드를 호출 합니다. 프레임 워크는 단추의 크기를 다시 조정 하 고 나면이를 TRUE로 설정 하 여이 메서드를 호출 합니다.

[CMFCToolBarButton:: SetVisible](#setvisible) 메서드를 사용 하 여 단추의 일반 표시 유형을 설정 합니다.

이 메서드는 단추의 표시 상태를 업데이트 한 후 [CMFCToolBarButton:: OnShow](#onshow) 메서드를 호출 합니다.
