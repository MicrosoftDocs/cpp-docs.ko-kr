## <a name="remarks"></a>설명

`CMFCToolbarButton` 개체는 도구 모음에 있는 컨트롤입니다. 그 동작은 일반 버튼과 유사합니다. 이 개체에 이미지와 텍스트 레이블을 할당할 수 있습니다. 도구 모음 단추에는 명령 ID가 있을 수도 있습니다. 사용자가 도구 모음 단추를 클릭하면 프레임워크는 이 ID가 지정한 명령을 실행합니다.

일반적으로 도구 모음 단추를 사용자 지정할 수 있습니다. 사용자가 도구 모음을 사용자 지정하지 못하도록 두 가지 방법 중 하나로 도구 모음을 잠글 수 있습니다. `bLocked` [CMFCToolBar::LoadToolBar를](../../mfc/reference/cmfctoolbar-class.md#loadtoolbar)호출할 때 플래그를 TRUE로 설정하거나 [CMFCToolBarButton::SetProtectedCommands](#setprotectedcommands) 메서드를 사용하여 보호된 명령의 전역 목록에 개별 단추의 명령 ID를 추가합니다.

`CMFCToolBarButton`개체는 응용 프로그램에서 도구 모음 이미지의 전역 컬렉션에서 이미지를 표시합니다. 이러한 컬렉션은 상위 도구 [모음CMFCToolBar 클래스에서](../../mfc/reference/cmfctoolbar-class.md)유지 관리합니다. 자세한 내용은 [CMFCToolBarImages 클래스를](../../mfc/reference/cmfctoolbarimages-class.md)참조하십시오.

사용자가 도구 모음 단추를 클릭하면 상위 도구 모음에서 마우스 메시지를 처리하고 단추에 적절한 작업을 전달합니다. 단추에 유효한 명령 ID가 있는 경우 상위 도구 모음은 WM_COMMAND 메시지를 부모 프레임으로 보냅니다.

클래스는 `CMFCToolBarButton` [CMFCToolBarMenuButton 클래스,](../../mfc/reference/cmfctoolbarmenubutton-class.md) [CMFCToolBar편집상자 단추 클래스](../../mfc/reference/cmfctoolbareditboxbutton-class.md)및 [CMFCToolBarComboBoxButton 클래스와](../../mfc/reference/cmfctoolbarcomboboxbutton-class.md)같은 다른 도구 모음 단추 클래스의 기본 클래스입니다.

【인】 도구 모음 단추의 상위 창입니다.

*데이터*<br/>
【인】 도구 `CAccessibilityData` 모음 단추의 내게 필요한 옵션 데이터로 채워진 개체입니다.

### <a name="return-value"></a>Return Value

이 메서드는 TRUE를 반환합니다.

### <a name="remarks"></a>설명

도구 모음 단추에서 내게 필요한 옵션 데이터를 제공하지 않는 경우 FALSE를 반환하려면 이 메서드를 재정의합니다.

## <a name="cmfctoolbarbuttonsetclipboardformatname"></a><a name="setclipboardformatname"></a>CMFCToolBar 단추::세트 클립보드 형식 이름

전역 클립보드 형식의 이름을 바꿉니다.

```
static void __stdcall SetClipboardFormatName(LPCTSTR lpszName);
```

### <a name="parameters"></a>매개 변수

*lpszName*<br/>
【인】 전역 클립보드 형식의 새 이름입니다. NULL이 될 수 없습니다.

### <a name="remarks"></a>설명

이 방법을 사용하면 여러 응용 프로그램에서 끌어서 놓기 작업이 발생할 수 있습니다. 각 응용 프로그램은 동일한 클립보드 형식 이름을 제공해야 합니다.

프레임 [워크가 CMFCToolBarButton::GetClipboardFormat을](#getclipboardformat)호출하기 전에이 메서드를 호출해야합니다.

도구 모음 단추가 구분 기호인 경우 *iImage는* 구분 기호 단추의 새 너비를 나타냅니다.

*iImage가* 0보다 적으면 이미지 그리기를 비활성화하고 단추의 텍스트 레이블을 그릴 수 있습니다.

## <a name="cmfctoolbarbuttonsetprotectedcommands"></a><a name="setprotectedcommands"></a>CMFCToolBar 단추::보호 명령 설정

static void SetProtectedCommands(const CList<UINT,UINT>& lstCmds);

### <a name="parameters"></a>매개 변수

[CMFCToolBarButton::GetProtectedCommand메서드를](#getprotectedcommands) 사용하여 보호된 명령 목록을 검색합니다.

## <a name="cmfctoolbarbuttonsetvisible"></a><a name="setvisible"></a>CMFCToolBar 단추::가시설정

단추를 볼 수 있는지 여부를 지정합니다.

```
void SetVisible(BOOL bShow=TRUE);
```

### <a name="parameters"></a>매개 변수

*bShow*<br/>
【인】 단추를 표시하거나 숨길지 여부를 지정하는 부울 값입니다. 이 매개 변수가 TRUE이면 단추가 표시됩니다. 매개 변수가 FALSE이면 단추가 숨김입니다.

### <a name="remarks"></a>설명

이 함수를 사용하여 특정 도구 모음 단추를 숨기거나 표시합니다. [CPane::adjustSize이 메서드를](../../mfc/reference/cpane-class.md#adjustsizeimmediate) 호출 한 후 즉시 메서드를 호출 합니다.

## <a name="cmfctoolbarbuttonshow"></a><a name="show"></a>CMFCToolBar 버튼::표시

단추를 표시하거나 숨깁니다.

```
void Show(BOOL bShow);
```

### <a name="parameters"></a>매개 변수

*bShow*<br/>
【인】 단추를 표시하거나 숨길지 여부를 지정하는 부울 값입니다. 이 매개 변수가 TRUE이면 단추가 표시됩니다. 매개 변수가 FALSE이면 단추가 숨김입니다.

### <a name="remarks"></a>설명

프레임워크는 이 메서드를 호출하여 상위 도구 모음의 크기를 조정할 때 도구 모음 단추의 가시성을 업데이트합니다. 단추를 더 이상 도구 모음의 경계 내에 맞지 않는 경우 프레임워크는 *bShow를* FALSE로 설정하여 이 메서드를 호출합니다. 단추크기를 조정한 후 도구 모음의 경계 에 다시 맞을 때 프레임워크는 *bShow를* TRUE로 설정하여 이 메서드를 호출합니다.

[CMFCToolBarButton::SetVisible](#setvisible) 메서드를 사용하여 단추의 일반적인 가시성을 설정합니다.

이 메서드는 단추의 가시성 상태를 업데이트한 후 [CMFCToolBarButton::OnShow](#onshow) 메서드를 호출합니다.
