# <a name="adding-an-mfc-message-handler"></a>MFC 메시지 처리기 추가

클래스 [마법사](mfc-class-wizard.md) 또는 **CLass 보기의** **속성** 창을 사용하여 클래스에 메시지 처리기(Windows 메시지를 처리하는 멤버 함수)를 추가하고 Windows 메시지를 메시지 처리기에 매핑할 수 있습니다. 모든 대화 [상자 컨트롤에 대한 이벤트 처리기를](../../windows/adding-event-handlers-for-dialog-box-controls.md)추가할 수도 있습니다.

**클래스 마법사** 또는 **속성** 창(클래스 **보기)을**사용하여 메시지 및 이벤트 처리 기능을 정의하면 메시지 디스패치 테이블(또는 메시지 맵)과 클래스 헤더 파일을 자동으로 업데이트할 수 있습니다.

> [!NOTE]
> 클래스 마법사를 사용하여 ATL 클래스에 메시지 처리기를 추가할 수 있습니다. 그러나 일부 결과는 다를 수 있습니다. 자세한 내용은 ATL 메시지 [처리기 추가 ATL](../../atl/adding-an-atl-message-handler.md)항목을 참조하십시오.
