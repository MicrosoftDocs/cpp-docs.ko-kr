# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>MFC DLL 마법사가 생성하는 클래스 및 함수

MFC DLL 마법사가 생성하는 코드는 만드는 DLL의 종류와 선택한 옵션에 따라 달라집니다. MFC DLL 마법사는 두 형태의 일반 MFC DLL에 대해 동일한 코드를 생성합니다.

|DLL의 종류|옵션|클래스|Functions|
|-----------------|------------|-------------|---------------|
|[확장명](../../build/extension-dlls-overview.md)|None|None|`DllMain`|
|[일반](../../build/regular-dlls-dynamically-linked-to-mfc.md)|None|에서 파생된 응용 프로그램 클래스`CWinApp`|None|
|[일반](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Automation|에서 파생된 응용 프로그램 클래스`CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[확장명](../../build/extension-dlls-overview.md)|창 소켓|None|`DllMain`|
|[일반](../../build/regular-dlls-dynamically-linked-to-mfc.md)|창 소켓|에서 파생된 응용 프로그램 클래스`CWinApp`|`InitInstance`호출이 포함되어 있습니다.`AfxSocketInit`|
