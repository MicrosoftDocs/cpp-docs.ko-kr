### <a name="remarks"></a>설명

프레임 워크는 자동으로 개체의 인스턴스를 만들고 `CPaneContainerManager` [CPaneDivider 클래스](../../mfc/reference/cpanedivider-class.md) 개체 또는 [CMultiPaneFrameWnd class](../../mfc/reference/cmultipaneframewnd-class.md) 개체에 포함 합니다.

`CPaneContainerManager`클래스는 [CPaneContainer](../../mfc/reference/cpanecontainer-class.md) 개체에서 빌드된 이진 트리의 루트에 대 한 포인터를 저장 합니다.
