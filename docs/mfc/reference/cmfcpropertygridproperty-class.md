# <a name="cmfcpropertygridproperty-class"></a>CMFCPropertyGridProperty 클래스

   자세한 내용은 Visual Studio 설치의 **VC \\ s\mfc \\ src \\ mfc** 폴더에 있는 소스 코드를 참조 하세요.

## <a name="syntax"></a>구문

```
class CMFCPropertyGridProperty : public CObject
```

*bAdjustLayout*<br/>
진행 속성의 레이블 및 값을 그리는 방법을 다시 계산 하려면 TRUE로 설정 하 고 속성을 그립니다. FALSE 이면 기존 계산을 사용 하 여 속성을 그립니다. 기본값은 TRUE입니다.
