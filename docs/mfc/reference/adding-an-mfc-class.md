# <a name="adding-an-mfc-class"></a>MFC 클래스 추가

프로젝트에 Microsoft 기초 클래스(MFC) 라이브러리 클래스에서 파생된 클래스를 추가하려면 [클래스 마법사에서](mfc-class-wizard.md) **클래스 추가** 단추를 사용합니다. 새 클래스의 이름을 지정하고 기본 클래스를 선택하고 연결된 대화 상자의 ID를 선택합니다(있는 경우). 마법사는 헤더 파일과 구현 파일을 만들고 프로젝트에 추가합니다.

> [!NOTE]
> MFC 지원을 사용하여 응용 프로그램을 처음 만든 경우 ATL COM 응용 프로그램에 [MFC 클래스를](../../atl/reference/mfc-support-in-atl-projects.md)추가할 수 있습니다. MFC가 지원하는 Win32 프로젝트에 MFC 클래스를 추가할 수도 있습니다.

### <a name="to-add-an-mfc-class-to-your-project"></a>프로젝트에 MFC 클래스를 추가하려면

클래스 보기에서 프로젝트 이름을 마우스 오른쪽 단추로 클릭합니다. **클래스 마법사를** 클릭한 다음 **클래스 추가를** 클릭하여 MFC 클래스 추가 대화 상자를 [엽니다.](mfc-add-class-wizard.md) **클래스 마법사를** 사용하여 메시지 처리기 또는 새 메서드를 추가합니다.
