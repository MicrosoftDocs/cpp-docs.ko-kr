# <a name="adding-an-mfc-class"></a>MFC 클래스 추가
프로젝트에 Microsoft Foundation 클래스 (MFC) 라이브러리 클래스에서 파생 된 클래스를 추가 하려면 사용 된 **클래스 추가** 에서 사용할 수 있는 명령 [클래스 뷰](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925)합니다. 새 클래스의 이름을 지정 하 고 기본 클래스를 선택한 다음 ID (있는 경우) 연결 되어 있는 대화 상자를 선택 합니다. 코드 마법사 헤더 파일과 구현 파일 만들고 프로젝트에 추가 합니다.  
  
> [!NOTE]
>  경우 ATL COM 응용 프로그램에 MFC 클래스를 추가할 수 있습니다 처음 [MFC 지 원하는 응용 프로그램을 만든](../../atl/reference/mfc-support-in-atl-projects.md)합니다. 또한 MFC 지원을 지 Win32 프로젝트를 MFC 클래스를 추가할 수 있습니다.  
  
### <a name="to-add-an-mfc-class-to-your-project"></a>MFC 클래스를 프로젝트에 추가 하려면  
  
1.  클래스 뷰에서 프로젝트 이름을 마우스 오른쪽 단추로 클릭 합니다. 클릭 **추가** 클릭 하 고 **클래스 추가** 열려는 [클래스 추가](../../ide/add-class-dialog-box.md) 대화 상자.  
  
2.  템플릿 창에서 선택 **MFC 클래스** 누릅니다는 **추가** 단추입니다.  
  
3.  새 클래스에 대 한 설정을 정의 [MFC 클래스 마법사](../../mfc/reference/mfc-add-class-wizard.md) 대화 상자.  
  
4.  클릭 **마침** 는 마법사를 닫고 클래스 뷰에서 새 클래스를 확인 합니다. 마법사에서 생성 된 파일을 볼 수 있습니다 **솔루션 탐색기**합니다.
