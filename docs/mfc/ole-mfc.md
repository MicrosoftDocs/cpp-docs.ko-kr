# <a name="ole-mfc"></a>OLE(MFC)

프로그램에서 OLE 기능을 구현 하면 다음과 같은 여러 가지 방법으로 사용자 인터페이스에 영향을 줍니다.

- 시각적 편집 (내부 활성화)은 프로그램 창에 있는 다른 프로그램의 사용자 인터페이스를 표시 하 고 다른 프로그램의 항목을 사용 하 여 프로그램의 메뉴를 수정 합니다.

- 끌어서 놓기 기능을 사용 하면 사용자가 windows와 프로그램 사이에서 개체를 끌 수 있습니다.

- 추적기는 시각적 개체 편집 및 끌어서 놓기를 수행 하는 동안 개체의 상태에 대 한 시각적 표시를 제공 합니다.
