# <a name="ole-mfc"></a>OLE(MFC)

프로그램에서 OLE 기능 구현 여러 가지 방법으로 사용자 인터페이스를 영향을 줍니다.

- 비주얼 편집 (전체 활성화) 프로그램의 창에 다른 프로그램의 사용자 인터페이스를 표시 하 고 다른 프로그램에서 항목을 사용 하 여 프로그램의 메뉴를 수정 합니다.

- 끌어서 놓기 내 및 windows 간에 프로그램 간에 개체를 끌어 놓을 수 있습니다.

- 추적기 비주얼 편집 하는 동안 개체의 상태를 시각 신호 제공 끌어서 놓습니다.
