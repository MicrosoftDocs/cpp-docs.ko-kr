# <a name="root-class-cobject"></a>루트 클래스: CObject

MFC (Microsoft Foundation Class) 라이브러리에 있는 대부분의 클래스는 클래스 계층 구조의 루트에 있는 단일 기본 클래스에서 파생 됩니다. `CObject` 는 오버 헤드가 매우 낮으므로 여기에서 파생 된 모든 클래스에 많은 유용한 기능을 제공 합니다. 및 해당 기능에 대 한 자세한 내용은 `CObject` [CObject 사용](../mfc/using-cobject.md)을 참조 하세요.

[CObject](../mfc/reference/cobject-class.md)<br/>
대부분의 MFC 클래스의 궁극적인 기본 클래스입니다. 에서는 데이터를 직렬화 하 고 클래스에 대 한 런타임 정보를 가져올 수 있습니다.

[CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)<br/>
런타임에 개체의 정확한 클래스를 확인 하는 데 사용 되는 구조체입니다.
