# <a name="status-bars"></a>상태 표시줄

상태 표시줄은 사용자의 작업을 중단 하지 않고 메시지 및 유용한 정보를 사용자에 게 표시할 수 있는 장소를 응용 프로그램에 제공 합니다. 일반적으로 창 아래쪽에 표시 되는 상태 표시줄에는 "표시기" 및 "메시지 줄"을 포함 하는 "창"이 있습니다. 표시기는 스크롤 잠금, 매크로 기록 설정 여부 등의 상태를 제공 합니다. 상태 표시줄의 메시지 줄에는 프로그램 상태 또는 사용자가 마우스로 가리키고 있는 도구 모음 단추 또는 메뉴 항목에 대 한 정보가 표시 될 수 있습니다.

MFC 응용 프로그램 마법사에서 **초기 상태 표시줄** 옵션을 선택 하 여 프로그램에서 상태 표시줄을 만듭니다.
