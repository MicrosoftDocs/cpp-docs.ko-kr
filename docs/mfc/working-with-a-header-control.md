# <a name="working-with-a-header-control"></a>헤더 컨트롤 사용

헤더 컨트롤 ([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md))을 사용 하는 쉬운 방법은 목록 컨트롤과 함께 사용 하는 것입니다. 이 항목의 뒷부분에 나오는 [CListCtrl 사용](../mfc/using-clistctrl.md) 을 참조 하세요. 헤더 컨트롤을 단독으로 사용할 수도 있습니다. MFC `InitCommonControls` 는를 호출 합니다. 주요 태스크는 다음과 같습니다.

- [헤더 컨트롤 만들기](../mfc/creating-the-header-control.md)

- [헤더 컨트롤에 항목 추가](../mfc/adding-items-to-the-header-control.md)

- [헤더 컨트롤에서 항목 순서 지정](../mfc/ordering-items-in-the-header-control.md)

- [헤더 컨트롤 알림 처리](../mfc/processing-header-control-notifications.md)

Header 컨트롤 개체가 부모 뷰나 대화 상자 클래스에 포함 되어 있으면 부모가 소멸 될 때 컨트롤이 제거 됩니다.
