# <a name="how-the-framework-searches-message-maps"></a>프레임워크가 메시지 맵을 검색하는 방법

프레임 워크는 메시지 맵 테이블에서 들어오는 메시지와 일치 하는 항목을 검색 합니다. 각 메시지에 대 한 메시지 맵 항목을 작성 한 후 클래스에서 해당 처리기를 처리 하 고 쓰도록 하려면 프레임 워크가 자동으로 처리기를 호출 합니다. 다음 항목에서는 메시지 맵 검색에 대해 설명 합니다.

- [메시지 맵을 찾을 장소](where-to-find-message-maps.md)

- [파생된 메시지 맵](derived-message-maps.md)

- [메시지, 명령 Id 또는 컨트롤 Id의 범위를 하나의 처리기로 매핑](handlers-for-message-map-ranges.md)
