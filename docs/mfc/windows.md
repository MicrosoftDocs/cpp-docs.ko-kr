# <a name="windows"></a>Windows

이 문서의 패밀리는 MFC 프레임 워크의 창 개체를 다룹니다. 모든 MFC 창은 프레임 창, 뷰, 대화 상자 및 컨트롤을 포함 하 여 [CWnd](../mfc/reference/cwnd-class.md)클래스에서 파생 됩니다.

문서의 첫 번째 그룹은 [창 개체](../mfc/window-objects.md) 에 대해 일반적으로 설명 합니다. C + + 창 개체에 대 한 일반 정보, 해당 개체를 캡슐화 하는 방법 `HWND` 및 자식 창과 같이 사용자 고유의 창을 만들 때 사용 하는 방법에 대 한 자세한 내용은이 그룹을 참조 하세요.

문서의 두 번째 그룹은 [프레임 창](../mfc/frame-windows.md)(특히 콘텐츠 주위에 프레임을 배치 하는 창)을 설명 합니다. MFC 프레임 워크에서 프레임 창과 프레임 창을 관리 하는 방법에 대 한 자세한 내용은이 그룹을 참조 하세요.
