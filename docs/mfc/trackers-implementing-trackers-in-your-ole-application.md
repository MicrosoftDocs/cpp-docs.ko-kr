# <a name="trackers-implementing-trackers-in-your-ole-application"></a>추적기: OLE 애플리케이션에서 추적기 구현

추적기는 사용자가 OLE 클라이언트 항목을 조작할 수 있도록 그래픽 인터페이스를 제공 합니다. 다른 tracker 스타일을 사용 하 여 OLE 클라이언트 항목은 빗금 테두리, 크기 조정 핸들 또는 다양 한 시각적 효과를 사용 하 여 표시할 수 있습니다. 이 문서에서는 다음을 설명합니다.

- [코드에서 추적을 구현 하는 방법](../mfc/how-to-implement-tracking-in-your-code.md)입니다.

- [고무 줄무늬 및 추적기](../mfc/rubber-banding-and-trackers.md).

또한이 문서에서는 추적기에 스타일을 사용 하는 방법을 설명 합니다. 또한 MFC OLE 샘플 [OCLIENT](../overview/visual-cpp-samples.md)에 대 한 여러 참조를 만듭니다.
