# <a name="windows-sockets"></a>Windows 소켓

이 문서 제품군에서는 Windows 소켓의 MFC 구현을 다룹니다. MFC는 Windows 소켓 API를 사용 하 여 네트워크 응용 프로그램 프로그래밍을 지 원하는 두 가지 클래스를 제공 합니다. [Casyncsocket](../mfc/reference/casyncsocket-class.md) 클래스는 WINDOWS 소켓 API 하나를 캡슐화 하 여 고급 네트워크 프로그래머에 게 가장 강력한 기능과 유연성을 제공 합니다. [CSocket](../mfc/reference/csocket-class.md) 클래스는 [CArchive](../mfc/reference/carchive-class.md) 개체와 데이터를 serialize 하기 위한 단순화 된 인터페이스를 제공 합니다.
