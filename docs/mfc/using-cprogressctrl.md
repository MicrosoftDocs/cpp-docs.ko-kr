# <a name="using-cprogressctrl"></a>CProgressCtrl 사용
긴 작업의 진행률을 표시 하는 진행률 컨트롤을 사용할 수 있습니다. 작업이 진행 됨에 따라 시스템 강조 색으로 점차적으로 채워진 사각형입니다.  
  
 진행률 컨트롤은 MFC 클래스로 나타납니다 [CProgressCtrl](../mfc/reference/cprogressctrl-class.md)합니다.  
  
 크기와 위치를 부모 창 (일반적으로 대화 상자)을 지정 하 고 id입니다. 처음에 진행률 컨트롤을 만들 때 사용 하 여는 `dwStyle` 매개 변수를 지정할 수도 있습니다는 컨트롤에 대 한 다양 한 창 스타일 및 스타일을 채우는 방법을 합니다.
