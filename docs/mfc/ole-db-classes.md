# <a name="ole-db-classes"></a>OLE DB 클래스

MFC의 OLE DB 지원은 현재 [Coledbrecordview](reference/coledbrecordview-class.md)클래스로 구성 됩니다. `COleDBRecordView`[CRowset](../data/oledb/crowset-class.md) 개체에 직접 연결 된 폼 뷰를 통해 컨트롤의 데이터베이스 레코드를 표시 합니다. OLE DB 소비자 템플릿에 대 한 자세한 내용은 [OLE DB 소비자 템플릿 목록](../data/oledb/ole-db-consumer-templates-reference.md)을 참조 하십시오.
