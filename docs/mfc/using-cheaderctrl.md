# <a name="using-cheaderctrl"></a>CHeaderCtrl 사용

[클래스 CHeaderCtrl로](../mfc/reference/cheaderctrl-class.md)표시되는 헤더 컨트롤을 사용하여 열 목록에 대한 열 헤더를 표시합니다. 예를 들어 헤더 컨트롤은 스프레드시트에서 열 컨트롤을 구현하는 데 유용합니다.

헤더 컨트롤은 일반적으로 "헤더 항목"이라고 하는 부분으로 나뉘며 각 항목은 텍스트 또는 숫자의 관련 열에 대한 제목을 지정합니다. 설정한 스타일에 따라 사용자가 헤더 항목을 조작할 수 있는 여러 가지 직접적인 방법을 제공할 수 있습니다.

> [!NOTE]
> [CListCtrl은](../mfc/reference/clistctrl-class.md) 포함된 헤더 컨트롤을 제공하며 [CListView는](../mfc/reference/clistview-class.md) MFC 클래스에 캡슐화합니다. `CListCtrl` 일반적으로 직접 그릴 `CHeaderCtrl` 목록에 레이블을 지정하는 데 사용하는 것이 좋습니다.
