# <a name="writing-mfc-applications"></a>MFC 애플리케이션 작성

이 문서에서는 응용 프로그램을 개발 하기 위해 수행 하는 초기 단계에 대해 설명 합니다. 먼저 작성 중인 응용 프로그램의 종류를 결정 해야 합니다. [응용 프로그램 디자인 선택](../mfc/application-design-choices.md)에서 몇 가지 선택 사항을 설명 했습니다. 응용 프로그램은 다음과 같습니다.

- 인터넷 또는 인트라넷에서 실행

- 클라이언트 또는 서버에서 실행

- 브라우저에서 실행 하거나 독립 실행형 응용 프로그램으로 실행

- COM 또는 활성 기술 사용

- WinInet 또는 비동기 모니커를 사용 하 여 데이터 다운로드

결정에 따라 응용 프로그램에 적합 한 클래스가 결정 됩니다. 또한 응용 프로그램 마법사를 실행 하 여 응용 프로그램을 만들 때 선택한 내용을 확인 하는 데 도움이 됩니다.

인터넷 응용 프로그램에 대 한 초기 디자인을 결정 한 후에는 응용 프로그램 마법사를 사용 하 여 시작할 수 있습니다. 응용 프로그램 마법사를 사용 하 여 다음 문서에 설명 된 대로 기본 응용 프로그램을 만들고 코드를 수정 합니다.

- ActiveX 컨트롤의 경우 [인터넷의 Activex 컨트롤](../mfc/activex-controls-on-the-internet.md)을 참조 하세요.

다음 문서에서는 프로그래밍 작업을 시작 하는 데 도움이 되는 지침을 제공 합니다.

- [응용 프로그램 디자인 선택](../mfc/application-design-choices.md)

- [인터넷의 비동기 모니커](../mfc/asynchronous-monikers-on-the-internet.md)

- [WinInet 기본 사항](../mfc/wininet-basics.md)
