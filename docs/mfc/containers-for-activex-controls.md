# <a name="containers-for-activex-controls"></a>ActiveX 컨트롤에 대한 컨테이너

ActiveX 컨트롤 포함을 지원 하기만 하면 다른 응용 프로그램에서 Visual C++에서 개발한 ActiveX 컨트롤을 사용할 수 있습니다. 나열 된 버전부터 시작 되는 다양 한 Microsoft 응용 프로그램은 ActiveX 컨트롤 포함을 지원 합니다.

>[!IMPORTANT]
> ActiveX는 새로운 개발에 사용 하지 않아야 하는 레거시 기술입니다. ActiveX를 대체 하는 최신 기술에 대 한 자세한 내용은 [Activex Controls](activex-controls.md)을 참조 하세요.

> [!NOTE]
> 다음 목록은 ActiveX 컨트롤을 지 원하는 응용 프로그램의 전체 목록이 아니라 테스트에서 가장 자주 사용 되는 집합을 나타냅니다.

ActiveX 컨트롤 포함을 지 원하는 응용 프로그램은 다음과 같습니다.

- Internet Explorer 3(sp3) 이상

- Visual Basic 4.x 이상

- Visual C++ 4.x 이상

- 95 이상 액세스

- Excel 97 이상

- 97 단어 이상

- 97 이상 액세스

- FrontPage 97 이상

- PowerPoint 97 이상

- Visual InterDev 97 이상

다음은 ActiveX 컨트롤 포함을 지 원하는 타사 응용 프로그램입니다.

- PowerBuilder

- Delphi

- C + + 작성기

- Netscape navigator 용 NCompass 플러그 인
