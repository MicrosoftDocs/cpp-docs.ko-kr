# <a name="containers-for-activex-controls"></a>ActiveX 컨트롤에 대한 컨테이너

ActiveX 컨트롤 을 지원하는 한 다른 응용 프로그램에서 Visual C++에서 개발된 ActiveX 컨트롤을 사용할 수 있습니다. 나열된 버전으로 시작하여 많은 Microsoft 응용 프로그램이 ActiveX 제어 억제를 지원합니다.

>[!IMPORTANT]
> ActiveX는 새로운 개발에 사용해서는 안 되는 레거시 기술입니다. ActiveX를 대체하는 최신 기술에 대한 자세한 내용은 [ActiveX 컨트롤](activex-controls.md)을 참조하십시오.

> [!NOTE]
> 다음 목록은 ActiveX 컨트롤을 지원하지만 테스트에서 가장 자주 사용되는 집합을 나타내는 응용 프로그램의 전체 목록은 아닙니다.

ActiveX 제어 억제를 지원하는 응용 프로그램은 다음과 같습니다.

- 인터넷 익스플로러 3.x 이상

- 비주얼 베이직 4.x 이상

- 비주얼 C++ 4.x 이상

- 액세스 95 이상

- 엑셀 97 이상

- 단어 97 이상

- 액세스 97 이상

- 프론트페이지 97 이상

- 파워 포인트 97 이상

- 비주얼 인터데브 97 이상

다음은 ActiveX 제어 억제를 지원하는 Microsoft 이외의 응용 프로그램입니다.

- PowerBuilder

- 델파이

- C ++ 빌더

- 넷스케이프 네비게이터를 위한 NCompass 플러그인
