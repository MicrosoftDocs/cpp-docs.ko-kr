# <a name="commonly-added-member-functions"></a>일반적으로 추가되는 멤버 함수

대화 상자에 OK 또는 Cancel 이외의 누름 단추이 포함 되어 있는 경우에는 대화 상자 클래스에서 메시지 처리기 멤버 함수를 작성 하 여 생성 하는 컨트롤 알림 메시지에 응답 해야 합니다. 예제를 보려면 [Scribble](../overview/visual-cpp-samples.md) 샘플 프로그램을 참조 하세요. 대화 상자의 다른 컨트롤에서 제어 알림 메시지를 처리할 수도 있습니다.
