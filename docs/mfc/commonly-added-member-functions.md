# <a name="commonly-added-member-functions"></a>일반적으로 추가되는 멤버 함수

대화 상자에서 확인 또는 취소 이외의 누름 단추를 포함 하는 경우 생성 하는 컨트롤 알림 메시지에 응답 하 여 대화 상자 클래스의 메시지 처리기 멤버 함수를 작성 해야 합니다. 예를 들어 참조를 [Scribble](../visual-cpp-samples.md) 샘플 프로그램입니다. 또한 대화 상자에서 다른 컨트롤의 컨트롤 알림 메시지를 처리할 수 있습니다.
