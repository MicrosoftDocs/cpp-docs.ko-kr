# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>클립보드: 각 클립보드 메커니즘을 사용하는 경우
클립보드를 사용 하 여 이러한 지침을 따릅니다.  
  
-   이제 OLE 클립보드 메커니즘을 사용 하 여 나중에 새 기능을 사용 하도록 합니다. 표준 클립보드 API 유지 되는 동안 데이터 전송의 대체 하는 OLE 메커니즘입니다.  
  
-   OLE 응용 프로그램을 작성 하거나 OLE 기능 중와 같은 끌어서 놓을 경우 OLE 클립보드 메커니즘을 사용 합니다.  
  
-   OLE 클립보드 메커니즘을 사용 하 여 OLE 형식을 제공 하는 경우.
