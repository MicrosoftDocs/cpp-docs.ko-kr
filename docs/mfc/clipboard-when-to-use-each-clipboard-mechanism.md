# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>클립보드: 각 클립보드 메커니즘을 사용하는 경우

클립보드를 사용 하 여 이러한 지침을 따릅니다.

- 나중에 새로운 기능을 사용 하도록 설정 하려면 이제 OLE 클립보드 메커니즘을 사용 합니다. 표준 클립보드 API를 유지 관리 하는 동안 OLE 메커니즘은 데이터 전송의 미래입니다.

- OLE 응용 프로그램을 작성 하는 OLE 기능 중 일부가, 같은 끌어서 놓을 경우 OLE 클립보드 메커니즘을 사용 합니다.

- OLE 형식 제공 하는 경우 OLE 클립보드 메커니즘을 사용 합니다.
