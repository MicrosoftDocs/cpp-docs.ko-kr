# <a name="file-and-database-classes"></a>파일 및 데이터베이스 클래스

이러한 클래스를 사용 하 여 데이터베이스 또는 디스크 파일에 정보를 저장할 수 있습니다. 유사한 기능을 제공 하는 세 가지 데이터베이스 클래스 집합 (OLE DB, ODBC 및 DAO)이 있습니다. OLE DB 그룹은 OLE DB를 사용 하 여 구현 되 고 OLE DB 소비자 템플릿과 함께 작동 하며, DAO 그룹은 데이터 액세스 개체를 사용 하 여 구현 되 고 ODBC 그룹은 오픈 데이터베이스 연결을 사용 하 여 구현 됩니다. 표준 파일, 활성 스트림 및 HTML 스트림을 조작 하기 위한 클래스 집합도 있습니다.

다음 클래스 범주는 데이터 지 속성을 지원 합니다.

- [파일 i/o 클래스](../mfc/file-i-o-classes.md)

- [OLE DB 클래스](../mfc/ole-db-classes.md)

- [DAO 클래스](../mfc/dao-classes.md)

- [ODBC 클래스](../mfc/odbc-classes.md)
