# <a name="file-and-database-classes"></a>파일 및 데이터베이스 클래스

이러한 클래스를 사용 하면 데이터베이스나 디스크 파일에 대 한 정보를 저장할 수 있습니다. 데이터베이스 클래스의 세 가지-OLE DB, ODBC 및 DAO-유사한 기능을 제공 합니다. OLE DB 그룹 OLE DB를 사용 하 여 구현 되 고 OLE DB 소비자 템플릿 작동, DAO 그룹 데이터 액세스 개체를 사용 하 여 구현 됩니다 및 ODBC 그룹 Open Database Connectivity를 사용 하 여 구현 됩니다. 표준 파일, Active 스트림 및 HTML 스트림을 조작 하기 위한 클래스 집합도 있습니다.

클래스의 다음 범주 데이터 지 속성을 지원 합니다.

- [파일 I/O 클래스](../mfc/file-i-o-classes.md)

- [OLE DB 클래스](../mfc/ole-db-classes.md)

- [DAO 클래스](../mfc/dao-classes.md)

- [ODBC 클래스](../mfc/odbc-classes.md)
