# <a name="header-items-in-a-header-control"></a>헤더 컨트롤의 헤더 항목

헤더 컨트롤 ([CHeaderCtrl](reference/cheaderctrl-class.md))을 구성 하는 헤더 항목의 모양 및 동작에 대 한 제어를 크게 제어할 수 있습니다. 각 헤더 항목에는 문자열, 비트맵 이미지, 연결 된 이미지 목록의 이미지 또는이와 연결 된 응용 프로그램 정의 32 비트 값이 있을 수 있습니다. 문자열, 비트맵 또는 이미지가 헤더 항목에 표시 됩니다.

[CHeaderCtrl:: GetItem](reference/cheaderctrl-class.md#getitem) 및 [CHeaderCtrl:: SetItem](reference/cheaderctrl-class.md#setitem)에 대 한 호출을 사용 하 여 [CHeaderCtrl:: InsertItem](reference/cheaderctrl-class.md#insertitem) 를 호출 하거나 기존 항목을 수정 하 여 새 항목의 모양과 내용을 사용자 지정할 수 있습니다.
