# <a name="messages"></a>메시지

메시지 루프를 `Run` 클래스의 멤버 함수 `CWinApp` 대기 중인 다양 한 이벤트에 의해 생성 된 메시지를 검색 합니다. 예를 들어 사용자가 마우스를 클릭 하면 Windows 마우스 왼쪽된 단추를 누를 때 WM_LBUTTONDOWN 및 WM_LBUTTONUP 마우스 왼쪽된 단추를 놓을 때와 같은 여러 마우스 관련 메시지를 보냅니다. 응용 프로그램 메시지 루프의 프레임 워크의 구현에 해당 하는 창 메시지를 디스패치합니다.

메시지의 중요 한 범주에 설명 되어 있습니다 [메시지 범주](../mfc/message-categories.md)합니다.
