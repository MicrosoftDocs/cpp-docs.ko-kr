# <a name="mfc-technical-notes"></a>MFC 기술 참고 사항

기술 참고 사항은 프로그래머 들이 작성 한 문서입니다.

각 기술 정보는 MFC 설명서의 나머지 부분에서 다루지 않는 문제 또는 기능을 설명 합니다. 제공 된 기술 정보에는 사용자의 정보에 대 한 요청과 MFC 개발자가 원하는 고급 정보를 제공 합니다.

기술 참고 사항을 검색 하는 방법에는 두 가지가 있습니다.

- [번호로 기술 참고 사항](technical-notes-by-number.md)

- [범주별 기술 참고 사항](technical-notes-by-category.md)
