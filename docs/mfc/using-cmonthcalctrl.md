# <a name="using-cmonthcalctrl"></a>CMonthCalCtrl 사용

월 달력 컨트롤[(CMonthCalCtrl)](../mfc/reference/cmonthcalctrl-class.md)달력 같은 사용자 인터페이스를 구현 합니다. 해당 컨트롤은 날짜를 입력하거나 선택하는 매우 직관적인 인식 방법을 사용자에게 제공합니다. 또한 기존의 데이터 형식을 사용하여 컨트롤에서 날짜 정보를 가져오고 설정하는 방법을 애플리케이션에 제공합니다. 기본적으로 Month Calendar 컨트롤에는 현재 요일과 월이 표시됩니다. 그러나 사용자는 이전 및 다음 달로 스크롤하고 특정 월 및/또는 연도를 선택할 수 있습니다.

> [!NOTE]
> 프로젝트에서 클래스를 `CMonthCalCtrl` 사용하려면 AFXDTCTL을 포함해야 합니다. H, 일반적으로 *stdafx.h*.
