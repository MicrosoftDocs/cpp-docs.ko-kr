# <a name="tn051-using-ctl3d-now-and-in-the-future"></a>TN051: 현재 및 향후의 CTL3D 사용

이전에 CTL3D.H 및 MFC에 대해 설명한이 기술 참고 사항은 이제 사용 되지 않습니다. 컨트롤의 3D 효과는 MFC에 의해 자동으로 구현 됩니다.
