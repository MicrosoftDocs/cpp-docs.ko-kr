# <a name="tn051-using-ctl3d-now-and-in-the-future"></a>TN051: 현재 및 향후의 CTL3D 사용

앞에서 설명한 바의 CTL3D 및 MFC는이 기술 노트는 이제 사용 되지 않습니다. 컨트롤에 대 한 3 차원 효과 MFC에서 자동으로 구현 됩니다.
