# <a name="using-ccomboboxex"></a>CComboBoxEx 사용

확장 된 콤보 상자 컨트롤은 항목 이미지에 대 한 기본 지원을 제공 하는 표준 콤보 상자 컨트롤의 확장입니다. 이러한 이미지를 사용 하 여 현재 선택 된 항목과 선택 되지 않은 항목과 같은 콤보 상자에 있는 개별 항목의 상태를 나타낼 수 있습니다. 항목 이미지에 쉽게 액세스할 수 있도록 컨트롤에서는 이미지 목록 지원을 제공 합니다.

이 컨트롤을 사용 하 여 수동으로 항목 그래픽을 그리지 않고도 콤보 상자의 기능을 제공할 수 있습니다.
