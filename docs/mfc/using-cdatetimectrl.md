# <a name="using-cdatetimectrl"></a>CDateTimeCtrl 사용

날짜 및 시간 선택기 컨트롤[(CDateTimeCtrl)은](../mfc/reference/cdatetimectrl-class.md)특정 날짜를 입력하거나 선택하는 직관적이고 인식 할 수있는 방법을 구현합니다. 컨트롤의 기본 인터페이스는 콤보 상자와 기능면에서 유사합니다. 그러나 사용자가 컨트롤을 확장하면 월 달력 컨트롤이 기본적으로 나타나사용자가 특정 날짜를 지정할 수 있습니다. 날짜를 선택하면 월 캘린더 컨트롤이 자동으로 사라집니다.

> [!NOTE]
> 프로젝트에서 `CDateTimePicker` 클래스와 `CMonthCalCtrl` 클래스를 모두 사용하려면 AFXDTCTL을 포함해야 합니다. H, 일반적으로 프로젝트의 STDAFX에 있습니다. H 파일.
