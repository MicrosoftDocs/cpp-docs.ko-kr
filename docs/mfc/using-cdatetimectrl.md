# <a name="using-cdatetimectrl"></a>CDateTimeCtrl 사용

날짜 및 시간 선택 컨트롤 ([CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md))은 특정 날짜를 입력 하거나 선택 하는 직관적이 고 인식 가능한 메서드를 구현 합니다. 컨트롤의 주요 인터페이스는 콤보 상자의 기능과 유사 합니다. 그러나 사용자가 컨트롤을 확장 하면 monthcalendar 컨트롤이 기본적으로 표시 되어 사용자가 특정 날짜를 지정할 수 있습니다. 날짜를 선택 하면 month calendar 컨트롤이 자동으로 사라집니다.

> [!NOTE]
>  프로젝트에 `CDateTimePicker` 및 `CMonthCalCtrl` 클래스를 모두 사용 하려면 AFXDTCTL를 포함 해야 합니다. 일반적으로 프로젝트의 STDAFX.H에 있습니다. H 파일.
