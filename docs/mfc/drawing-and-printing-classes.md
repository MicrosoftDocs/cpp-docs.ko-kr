# <a name="drawing-and-printing-classes"></a>클래스 그리기 및 인쇄
Windows에서 모든 그래픽은 출력은 DC(장치 컨텍스트)라는 가상 그리기 영역에 그려집니다. MFC는 비트맵, 브러시, 색상표, 펜 같은 Windows 그리기 도구를 캡슐화할 뿐만 아니라 다양한 DC 형식을 캡슐화하는 클래스를 제공합니다.
