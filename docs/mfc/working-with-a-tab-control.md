# <a name="working-with-a-tab-control"></a>탭 컨트롤 사용

탭 컨트롤 ([Ctabctrl](../mfc/reference/ctabctrl-class.md))을 사용 하는 가장 쉬운 방법은 대화 상자 편집기를 사용 하 여 대화 상자 템플릿 리소스에 추가 하는 것입니다. 탭 컨트롤을 단독으로 사용할 수도 있습니다. MFC `InitCommonControls` 는를 호출 합니다. 주요 태스크는 다음과 같습니다.

- [탭 컨트롤 만들기](../mfc/creating-the-tab-control.md)

- [탭 컨트롤에 탭 추가](../mfc/adding-tabs-to-a-tab-control.md)

- [탭 컨트롤 알림 메시지 처리](../mfc/processing-tab-control-notification-messages.md)

탭 컨트롤 개체가 부모 뷰나 대화 상자 클래스에 포함 되어 있으면 부모가 소멸 될 때 컨트롤이 제거 됩니다.
