# <a name="tn044-mfc-support-for-dbcs"></a>TN044: DBCS에 대한 MFC 지원

이 기술 참고 사항은 "더블 바이트 문자 집합" 또는 DBCS에 대 한 MFC의 지원에 대해 설명 했습니다. 이 정보 뿐만 아니라 MFC의 UNICODE 지원에 대 한 정보는 이제 *클래스 라이브러리 참조* 에서 사용할 수 있습니다.
