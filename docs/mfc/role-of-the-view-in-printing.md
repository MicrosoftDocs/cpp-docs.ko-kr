# <a name="role-of-the-view-in-printing"></a>인쇄에서 뷰의 역할

또한 보기는 관련된 문서를 인쇄 합니다. 두 가지 중요 한 역할을 재생 합니다.

보기:

- 에서는 동일한 [OnDraw](../mfc/reference/cview-class.md#ondraw) 화면의 그리기에 대 한 프린터에 그리는 코드입니다.
