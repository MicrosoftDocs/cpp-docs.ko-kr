# <a name="role-of-the-view-in-printing"></a>인쇄에서 뷰의 역할

또한 보기는 연결 된 문서를 인쇄 하는 두 가지 중요 한 역할을 담당 합니다.

보기:

- 는 동일한 [OnDraw](../mfc/reference/cview-class.md#ondraw) 코드를 사용 하 여 화면에 그리기 위해 프린터에 그립니다.

- 문서를 인쇄 페이지로 나누기를 관리 합니다.

인쇄 및 인쇄의 보기 역할 정보에 대 한 자세한 내용은 [인쇄 및 인쇄 미리 보기](../mfc/printing-and-print-preview.md)를 참조 하세요.
