원래 하는 방법을 설명 하는이 참고 MFCANS32 합니다. ANSI 인터페이스를 제공 하는 DLL을 주로 유니코드 수많은 32 비트 OLE 합니다. 이 DLL은 MFC에서 더 이상 사용 됩니다.
