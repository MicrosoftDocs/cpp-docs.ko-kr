# <a name="window-dialog-and-control-classes"></a>창, 대화 상자 및 컨트롤 클래스

클래스 `CWnd` 및 해당 파생 클래스는 Windows 창에 대 한 핸들 인 `HWND`를 캡슐화 합니다. `CWnd`은 단독으로 사용 하거나 새 클래스를 파생 시키기 위한 기준으로 사용할 수 있습니다. 클래스 라이브러리에서 제공 하는 파생 클래스는 다양 한 종류의 창을 나타냅니다.

[에서 파생되지 않은](../mfc/reference/cwnd-class.md)<br/>
모든 창에 대 한 기본 클래스입니다. `CWnd`에서 파생 된 클래스 중 하나를 사용 하거나 해당 클래스에서 직접 클래스를 파생 시킬 수 있습니다.
