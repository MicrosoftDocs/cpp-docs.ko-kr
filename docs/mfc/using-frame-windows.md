# <a name="using-frame-windows"></a>프레임 창 사용

MFC 프레임 워크는 파일 메뉴의 새로 만들기 및 열기 명령을 구현 하는 과정의 일부로 문서 프레임 창, 해당 뷰 및 문서를 만듭니다. 프레임 워크는 대부분의 프레임 창이 작동 하므로 이러한 창을 만들고 사용 하 고 제거 하는 데에는 작은 역할만 재생 합니다. 그러나 특별 한 용도로 고유한 프레임 창 및 자식 창을 명시적으로 만들 수 있습니다.
