# <a name="tab-controls-and-property-sheets"></a>탭 컨트롤 및 속성 시트

속성 시트는 사용자에 게 최대 24 개의 대화 상자 템플릿 리소스를 표시할 수 있는 다중 페이지 대화 상자 또는 "탭 대화 상자"입니다. 속성 시트의 예제를 보려면 Windows 표시 속성 대화 상자 또는 다음 MFC 샘플 응용 프로그램을 참조 하십시오.

[CMNCTRL1: 공용 컨트롤 클래스 (1 부)를 보여 줍니다.](../overview/visual-cpp-samples.md)

[CMNCTRL2: 공용 컨트롤 클래스를 보여 줍니다. 2 부](../overview/visual-cpp-samples.md)

속성 시트는 MFC 클래스 [CPropertySheet](../mfc/reference/cpropertysheet-class.md)를 사용 하 여 쉽게 구현할 수 있습니다.
