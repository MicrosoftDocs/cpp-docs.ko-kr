# <a name="ole-related-classes"></a>OLE 관련 클래스
이러한 클래스는 다양 한 입력 및 출력 파일에 대 한 예외에 이르는 하는 다른 서비스를 제공 합니다.  
  
 [COleObjectFactory](../mfc/reference/coleobjectfactory-class.md)  
 다른 컨테이너에서 요청 될 때 항목을 만드는 데 사용 합니다. 이 클래스 팩터리를 포함 하 여 더 구체적인 형식에 대 한 기본 클래스로 사용 `COleTemplateServer`합니다.  
  
 [COleMessageFilter](../mfc/reference/colemessagefilter-class.md)  
 동시성으로 OLE Lightweight 원격 프로시저 호출 LRPC ()을 관리 하는 데 사용 합니다.  
  
 [COleStreamFile](../mfc/reference/colestreamfile-class.md)  
 COM을 사용 하 여 `IStream` 인터페이스를 `CFile` 복합 파일에 대 한 액세스. 이 클래스 (에서 파생 된 `CFile`) MFC serialization OLE 구조적 저장소를 사용할 수 있습니다.  
  
 [CRectTracker](../mfc/reference/crecttracker-class.md)  
 이동, 크기 조정, 및 내부 항목의 방향 허용 하는 데 사용 합니다.
