# <a name="drawing-tool-classes"></a>그리기 도구 클래스
이러한 클래스를 그릴 장치 컨텍스트에서 사용 되는 그리기 도구를 캡슐화 합니다.  
  
 [CGdiObject](../mfc/reference/cgdiobject-class.md)  
 GDI 그리기 도구에 대 한 기본 클래스입니다.  
  
 [CBrush](../mfc/reference/cbrush-class.md)  
 디바이스 컨텍스트에서 현재 브러시도 선택할 수 있는 GDI 브러시를 캡슐화 합니다. 브러시 끄는 개체의 내부를 채우는 데 사용 됩니다.  
  
 [CPen](../mfc/reference/cpen-class.md)  
 디바이스 컨텍스트에서 현재 펜으로 선택할 수 있는 GDI 펜을 캡슐화 합니다. 펜 개체의 테두리 선 그리기에 사용 됩니다.  
  
 [CFont](../mfc/reference/cfont-class.md)  
 디바이스 컨텍스트에서 현재 글꼴로 선택할 수 있는 GDI 글꼴을 캡슐화 합니다.  
  
 [CBitmap](../mfc/reference/cbitmap-class.md)  
 비트맵을 조작 하기 위한 인터페이스를 제공 하는 GDI 비트맵을 캡슐화 합니다.  
  
 [CPalette](../mfc/reference/cpalette-class.md)  
 응용 프로그램과 같은 디스플레이 색 출력 장치 간 인터페이스로 사용 하기 위해 GDI 색상표를 캡슐화합니다.  
  
 [CRectTracker](../mfc/reference/crecttracker-class.md)  
 표시 하 고 조정 하 고 사각형 개체를 이동에 대 한 사용자 인터페이스를 처리 합니다.
