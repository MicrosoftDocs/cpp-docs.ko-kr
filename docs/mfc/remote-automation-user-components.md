# <a name="remote-automation-user-components"></a>원격 자동화 사용자 구성 요소
각 클라이언트 컴퓨터 클라이언트 프로그램 및 모든 지원 해야 하는 Dll이 포함 되어 있는지 확인 해야 합니다. 서버 응용 프로그램 및 모든 지원 해야 하는 Dll 서버 컴퓨터에 있는지 확인 해야 합니다. 마지막으로 서버 프로그램 RAC 관리자를 실행 하는 연결을 구성 하려면 먼저 각 클라이언트 컴퓨터에 등록 되어 있는지 확인 해야 합니다. 프로그램 (대부분 됩니다)으로 자체 등록을 등록 하려면 클라이언트 컴퓨터에서 서버 프로그램을 실행할 필요 있습니다. 실패를 제공 하는 등록 파일을 실행 하거나 레지스트리를 수동으로 편집 해야 합니다.
