# <a name="character-comparison"></a>문자 비교
다음 팁을 사용 합니다.  
  
-   올바르게 작동 ASCII 문자로 알려진된 선행 바이트를 비교 합니다.  
  
    ```  
    if( *sz1 == 'A' )  
    ```  
  
-   두 개의 알 수 없는 문자를 비교 Mbstring.h에 정의 된 매크로 중 하나를 사용 하려면 필요 합니다.  
  
    ```  
    if( !_mbccmp( sz1, sz2) )  
    ```  
  
     이렇게 하면 두 바이트는 더블 바이트 문자 같은지 비교 됩니다.
