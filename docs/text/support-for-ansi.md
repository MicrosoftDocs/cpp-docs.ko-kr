# <a name="support-for-ansi"></a>ANSI 지원

대부분의 MFC 클래스와 메서드는 ANSI 문자 집합을 지원 하지만, MFC 프레임 워크 전체가 유니코드 문자 집합만 지원 하기 위해 점차적으로 진화 하 고 있습니다. Windows Vista 및 Windows 공용 컨트롤 버전 6.1의 향상 된 기능으로 인해 여러 ANSI 클래스 및 메서드에 대 한 지원이 더 이상 사용 되지 않습니다.  자세한 내용은 [사용 되지 않는 ANSI api](../mfc/deprecated-ansi-apis.md) 및 [유니코드 지원](../text/support-for-unicode.md)을 참조 하세요.
