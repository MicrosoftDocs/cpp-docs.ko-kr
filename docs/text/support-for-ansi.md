# <a name="support-for-ansi"></a>ANSI 지원

대부분의 MFC 클래스와 메서드는 ANSI 문자 집합을 지원하지만 MFC 프레임워크는 전반적으로 점차 유니코드 문자 집합만 지원하도록 변화하고 있습니다. 지속적으로 향상된 Windows Vista 및 Windows 공용 컨트롤 6.1 버전에서는 여러 ANSI 클래스 및 메서드에 대하여 더이상 지원하지 않습니다.  자세한 내용은 [사용되지 않는 ANSI API](../mfc/deprecated-ansi-apis.md)와 [유니코드 지원](../text/support-for-unicode.md)을 참조합니다.
