# <a name="support-for-ansi"></a>ANSI 지원

대부분의 MFC 클래스 및 메서드를 전체적으로 MFC 프레임 워크는 유니코드만 지 원하는 방식으로 진화 하 고 점진적으로 하지만 ANSI 문자 집합 지원 문자 집합입니다. Windows Vista 및 Windows 공용 컨트롤 버전 6.1에에서 지속적으로 향상 된 기능,으로 인해 여러 ANSI 클래스 및 메서드에 대 한 지원 사용 되지 않습니다.  자세한 내용은 [사용 되지 않는 ANSI Api](../mfc/deprecated-ansi-apis.md) 하 고 [유니코드 지원](../text/support-for-unicode.md)합니다.
