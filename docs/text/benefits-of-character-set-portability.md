# <a name="benefits-of-character-set-portability"></a>문자 집합 이식성의 이점

현재 응용 프로그램을 internationalize 하려는 경우에도 MFC 및 C 런타임 이식성 기능을 사용할 수 있습니다.

- 코딩 간단는 코드 베이스를 유연 하 게 만듭니다. 나중에 유니코드 나 MBCS로 쉽게 이동할 수 있습니다.

- 유니코드를 사용 하 여 Windows 용 응용 프로그램을 보다 효율적으로 만들 수 있습니다. Windows에서는 유니코드를 사용 하기 때문에 운영 체제에 전달 된 비유니코드 문자열을 변환 해야 하므로 오버 헤드가 발생 합니다.
