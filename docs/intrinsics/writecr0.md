# <a name="writecr0"></a>__writecr0
**Microsoft 전용**  
  
 값이 기록 `Data` CR0 레지스터를 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
void writecr0(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Data`  
 CR0 레지스터에 쓸 값입니다.
