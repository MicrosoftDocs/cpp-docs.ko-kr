# <a name="writecr8"></a>__writecr8
**Microsoft 전용**

## <a name="syntax"></a>구문  
  
```  
void writecr8(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Data`  
 CR8 레지스터에 쓸 값입니다.
