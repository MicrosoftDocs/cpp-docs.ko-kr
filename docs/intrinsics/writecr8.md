# <a name="writecr8"></a>__writecr8

**Microsoft 전용**

값을 씁니다 `Data` CR8 등록 합니다.

## <a name="syntax"></a>구문

```
void writecr8(
   unsigned __int64 Data
);
```

#### <a name="parameters"></a>매개 변수

*Data*<br/>
[in] CR8 레지스터에 쓸 값입니다.

## <a name="requirements"></a>요구 사항

|내장 함수|아키텍처|
|---------------|------------------|
|`__writecr8`|X64|

**헤더 파일** \<intrin.h >
