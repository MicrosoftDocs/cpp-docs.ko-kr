# <a name="readcr0"></a>__readcr0
**Microsoft 전용**  
  
 CR0 레지스터를 읽고 해당 값을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
unsigned long __readcr0(void);  /* X86 */  
unsigned __int64 __readcr0(void);  /* X64 */  
  
```  
  
## <a name="return-value"></a>반환 값  
 CR0 레지스터의 값입니다.
