# <a name="indwordstring"></a>__indwordstring
**Microsoft 전용**  
  
 사용 하 여 지정 된 포트에서 데이터를 읽고는 `rep insd` 명령입니다.  
  
## <a name="syntax"></a>구문  
  
```  
void __indwordstring(  
   unsigned short Port,  
   unsigned long* Buffer,  
   unsigned long Count  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Port`  
 읽을 수는 포트입니다.  
  
 [out] `Buffer`  
 포트에서 읽은 데이터를 여기에 기록 됩니다.  
  
 [in] `Count`  
 데이터를 읽을 바이트 수입니다.
