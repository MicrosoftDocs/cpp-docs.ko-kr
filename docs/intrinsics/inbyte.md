# <a name="inbyte"></a>__inbyte
**Microsoft 전용**  
  
 생성 된 `in` 명령 1 바이트를 반환 하 여 지정 된 포트에서 읽은 `Port`합니다.  
  
## <a name="syntax"></a>구문  
  
```  
unsigned char __inbyte(  
   unsigned short Port  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Port`  
 읽을 수는 포트입니다.  
  
## <a name="return-value"></a>반환 값  
 지정된 된 포트에서 읽은 바이트입니다.
