# <a name="addfsbyte-addfsword-addfsdword"></a>__addfsbyte, __addfsword, __addfsdword
**Microsoft 전용**  
  
 값의 시작 부분에 상대적인 오프셋으로 지정 된 메모리 위치에 추가 `FS` 세그먼트입니다.  
  
## <a name="syntax"></a>구문  
  
```  
void __addfsbyte(   
   unsigned long Offset,   
   unsigned char Data   
);  
void __addfsword(   
   unsigned long Offset,   
   unsigned short Data   
);  
void __addfsdword(   
   unsigned long Offset,   
   unsigned long Data   
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Offset`  
 시작 부분에서 오프셋 `FS`합니다.  
  
 [in] `Data`  
 메모리 위치에 추가할 값입니다.  
  
## <a name="requirements"></a>요구 사항  
  
|내장 함수|아키텍처|  
|---------------|------------------|  
|`__addfsbyte`|x86|  
|`__addfsword`|x86|  
|`__addfsdword`|x86|  
  
## <a name="remarks"></a>설명  
 이러한 루틴은 내장 함수로 사용할 수 있습니다.
