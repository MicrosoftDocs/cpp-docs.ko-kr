# <a name="outbytestring"></a>__outbytestring
**Microsoft 전용**  
  
 생성 된 `rep outsb` 첫 번째 전송 하는 명령 `Count` 가리키는 데이터의 바이트 `Buffer` 로 지정 된 포트에 `Port`합니다.  
  
## <a name="syntax"></a>구문  
  
```  
void __outbytestring(   
   unsigned short Port,   
   unsigned char* Buffer,   
   unsigned long Count   
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Port`  
 데이터를 보내야 하는 포트입니다.  
  
 [in] `Buffer`  
 데이터 전송 지정된 된 포트입니다.  
  
 [in] `Count`  
 데이터를 보낼 수의 바이트 수입니다.
