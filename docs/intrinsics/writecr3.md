# <a name="writecr3"></a>__writecr3
**Microsoft 전용**  
  
 값이 기록 `Data` CR3 레지스터에 있습니다.  
  
## <a name="syntax"></a>구문  
  
```  
void writecr3(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Data`  
 CR3 레지스터에 쓸 값입니다.
