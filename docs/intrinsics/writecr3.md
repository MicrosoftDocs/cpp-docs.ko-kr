# <a name="writecr3"></a>__writecr3

**Microsoft 전용**

값을 씁니다 `Data` CR3 등록 합니다.

## <a name="syntax"></a>구문

```
void writecr3(
   unsigned __int64 Data
);
```

#### <a name="parameters"></a>매개 변수

*Data*<br/>
[in] CR3 레지스터에 쓸 값입니다.

## <a name="requirements"></a>요구 사항

|내장 함수|아키텍처|
|---------------|------------------|
|`__writecr3`|x86, x64|

**헤더 파일** \<intrin.h >

## <a name="remarks"></a>설명

이 내장 함수는 커널 모드에서만 사용할 수 있으며 루틴은 내장 함수로만 사용할 수 있습니다.
