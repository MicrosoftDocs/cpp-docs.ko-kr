# <a name="incgsbyte-incgsword-incgsdword-incgsqword"></a>__incgsbyte, __incgsword, __incgsdword, __incgsqword
**Microsoft 전용**  
  
 시작 부분에 상대적인 오프셋으로 지정 된 메모리 위치에 값에 추가 `GS` 세그먼트입니다.  
  
## <a name="syntax"></a>구문  
  
```  
void __incgsbyte(   
   unsigned long Offset   
);  
void __incgsword(   
   unsigned long Offset   
);  
void __incgsdword(   
   unsigned long Offset  
);  
void __incgsqword(   
   unsigned long Offset   
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 [in] `Offset`  
 시작 부분에서 오프셋 `GS`합니다.
