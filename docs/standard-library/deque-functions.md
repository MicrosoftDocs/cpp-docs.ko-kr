## <a name="swap"></a>  swap

두 deque의 요소를 교환합니다.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

*left*<br/>
`deque` 형식의 개체입니다.

*right*<br/>
`deque` 형식의 개체입니다.
