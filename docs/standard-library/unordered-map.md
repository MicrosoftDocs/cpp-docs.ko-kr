# <a name="ltunorderedmapgt"></a>&lt;unordered_map&gt;

## <a name="syntax"></a>구문

```cpp
#include <unordered_map>
```

> 합니다 \<unordered_map > 라이브러리 사용을 `#include <initializer_list>` 문입니다.
