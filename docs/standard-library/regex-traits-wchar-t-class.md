# <a name="regex_traitsltwchar_tgt-class"></a>regex_traits&lt;wchar_t&gt; 클래스

`regex_traits`에 대 한의 특수화입니다 **`wchar_t`** .

## <a name="syntax"></a>구문

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>설명

클래스는 형식의 요소에 대 한 클래스 템플릿 [regex_traits](../standard-library/regex-traits-class.md) 의 명시적 특수화로, **`wchar_t`** 이 형식의 개체를 조작 하는 라이브러리 함수를 활용할 수 있습니다.
