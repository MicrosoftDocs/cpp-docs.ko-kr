# <a name="container-classreverse_iterator"></a>Container Class::reverse_iterator

> [!NOTE]
> Microsoft C++ 설명서의 이 항목은 C++ 표준 라이브러리에서 사용되는 컨테이너의 비기능적 예제입니다. 자세한 내용은 [C++ 표준 라이브러리 컨테이너](../standard-library/stl-containers.md)를 참조하세요.

제어되는 시퀀스의 역방향 반복기로 사용될 수 있는 개체를 설명합니다.

## <a name="syntax"></a>구문

```cpp
typedef T7 reverse_iterator;
```

## <a name="remarks"></a>주의

여기서는 지정 되지 않은 형식 `T7`에 대 한 동의어로 설명 됩니다 (일반적으로 `reverse_iterator` **\<** [반복기](../standard-library/container-class-iterator.md) **>** ).
