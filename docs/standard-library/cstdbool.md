# <a name="ltcstdboolgt"></a>&lt;cstdbool&gt;

C 표준 라이브러리 헤더를 포함 \<stdbool.h> 하 고 네임 스페이스에 연결 된 이름을 추가 합니다 `std` .

> [!NOTE]
> 헤더는 \<stdbool.h> c + +의 키워드인 매크로를 정의 하므로이를 포함 하면 아무런 효과가 없습니다. \<stdbool.h>헤더는 c + +에서 사용 되지 않습니다. \<cstdbool>헤더는 c + + 17에서 더 이상 사용 되지 않으며 초안 c + + 20 표준에서 제거 되었습니다.

## <a name="requirements"></a>요구 사항

**헤더:**\<cstdbool>

**네임스페이스:** std

## <a name="remarks"></a>설명

이 헤더를 포함 하면 C 표준 라이브러리 헤더의 외부 링크를 사용 하 여 선언한 이름이 네임 스페이스에 선언 됩니다 `std` .
