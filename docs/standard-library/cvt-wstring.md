# <a name="ltcvtwstringgt"></a>&lt;cvt/wstring&gt;

`<cvt/wstring>`이전 버전의 Visual Studio에서 헤더는 stdext:: cvt 네임 스페이스의 클래스 템플릿 [wstring_convert 클래스](../standard-library/wstring-convert-class.md) 를 정의 했습니다. 이 헤더는 이전 버전과 호환성을 위해 유지됩니다. 새 코드는 [\<locale>](../standard-library/locale.md) 네임 스페이스의에 정의 된 클래스의 버전을 사용 해야 합니다. `std`
