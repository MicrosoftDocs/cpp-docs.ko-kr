# <a name="container-classconst_reference"></a>Container Class::const_reference

> [!NOTE]
> Microsoft C++ 설명서의 이 항목은 C++ 표준 라이브러리에서 사용되는 컨테이너의 비기능적 예제입니다. 자세한 내용은 [C++ 표준 라이브러리 컨테이너](../standard-library/stl-containers.md)를 참조하세요.

제어되는 시퀀스의 요소에 대한 상수 참조로 사용될 수 있는 개체를 설명합니다.

## <a name="syntax"></a>구문

```cpp
typedef T3 const_reference;
```

## <a name="remarks"></a>주의

여기서는 지정 되지 않은 형식 `T3` (일반적으로 `Alloc::const_reference`)의 동의어로 설명 되어 있습니다.
