# <a name="char_traitsltchargt-struct"></a>char_traits&lt;char&gt; 구조체

형식의 요소에 대 한 템플릿 구조체 **char_traits \<CharType>** 의 특수화 인 구조체입니다 **`char`** .

## <a name="syntax"></a>구문

```cpp
template <>
struct char_traits<char>;
```

## <a name="remarks"></a>설명

특수화를 사용 하면 구조체에서이 형식의 개체를 조작 하는 라이브러리 함수를 활용할 수 있습니다 **`char`** .
