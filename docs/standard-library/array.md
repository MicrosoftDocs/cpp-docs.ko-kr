# <a name="ltarraygt"></a>&lt;array&gt;

컨테이너 템플릿 클래스 **array**와 여러 지원 템플릿을 정의합니다.

## <a name="requirements"></a>요구 사항

**헤더:** \<array>

**네임스페이스:** std

> [!NOTE]
> \<array > 라이브러리는 또한 `#include <initializer_list>` 문을 사용합니다.
