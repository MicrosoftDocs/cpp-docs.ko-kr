# <a name="ltarraygt"></a>&lt;array&gt;

컨테이너 클래스 템플릿 **배열과** 여러 지원 템플릿을 정의 합니다.

## <a name="requirements"></a>요구 사항

**헤더:**\<array>

**네임스페이스:** std

> [!NOTE]
> \<array>또한 라이브러리는 문을 사용 `#include <initializer_list>` 합니다.
