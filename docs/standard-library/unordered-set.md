# <a name="ltunordered_setgt"></a>&lt;unordered_set&gt;

## <a name="requirements"></a>요구 사항

**헤더:**\<unordered_set>

**네임스페이스:** std

> [!NOTE]
> \<unordered_set>또한 라이브러리는 문을 사용 `#include <initializer_list>` 합니다.
