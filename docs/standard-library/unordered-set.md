# <a name="ltunorderedsetgt"></a>&lt;unordered_set&gt;

## <a name="requirements"></a>요구 사항

**헤더:** \<unordered_set>

**네임스페이스:** std

> [!NOTE]
> Unordered_set \<> 라이브러리는 `#include <initializer_list>` 문을 사용 하기도 합니다.
