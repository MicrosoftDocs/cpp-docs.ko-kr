# <a name="regextraitsltchargt-class"></a>regex_traits&lt;char&gt; 클래스

Char에 `regex_traits` 대 한의 특수화입니다.

## <a name="syntax"></a>구문

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>설명

클래스는 **char** 형식의 요소에 대 한 템플릿 클래스 [regex_traits](../standard-library/regex-traits-class.md) 의 명시적 특수화로,이 형식의 개체를 조작 하는 라이브러리 함수를 활용할 수 있도록 합니다.

## <a name="requirements"></a>요구 사항

**헤더:** \<regex>

**네임스페이스:** std
