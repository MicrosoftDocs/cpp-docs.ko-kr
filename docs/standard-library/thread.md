# <a name="ltthreadgt"></a>&lt;thread&gt;

표준 헤더를 포함 \<스레드 >는 클래스를 정의 하려면 `thread` 및 다양 한 지원 기능입니다.

## <a name="syntax"></a>구문

```cpp
#include <thread>
```

> [!NOTE]
> 사용 하 여 컴파일된 코드에서 **/clr**,이 헤더 ï ´ ù.
