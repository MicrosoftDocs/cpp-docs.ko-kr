# <a name="container-classmax_size"></a>Container Class::max_size

> [!NOTE]
> Microsoft C++ 설명서의 이 항목은 C++ 표준 라이브러리에서 사용되는 컨테이너의 비기능적 예제입니다. 자세한 내용은 [C++ 표준 라이브러리 컨테이너](../standard-library/stl-containers.md)를 참조하세요.

개체가 제어할 수 있는 가장 긴 시퀀스의 길이를 제어되는 시퀀스의 길이와 관계없이 상수 시간으로 반환합니다.

## <a name="syntax"></a>구문

```cpp
size_type max_size() const;
```
