# <a name="ltdequegt"></a>&lt;deque&gt;

컨테이너 클래스 템플릿 deque 및 여러 지원 템플릿을 정의 합니다.

## <a name="requirements"></a>요구 사항

**헤더**: \<deque>

> [!NOTE]
> 또한 \<deque > 라이브러리는 `#include <initializer_list>` 문을 사용 합니다.
