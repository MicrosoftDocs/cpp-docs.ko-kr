# <a name="container-classconst_iterator"></a>Container Class::const_iterator

> [!NOTE]
> Microsoft C++ 설명서의 이 항목은 C++ 표준 라이브러리에서 사용되는 컨테이너의 비기능적 예제입니다. 자세한 내용은 [C++ 표준 라이브러리 컨테이너](../standard-library/stl-containers.md)를 참조하세요.

제어되는 시퀀스의 상수 반복기로 사용될 수 있는 개체를 설명합니다.

## <a name="syntax"></a>구문

```cpp
typedef T6 const_iterator;
```

## <a name="remarks"></a>주의

`T6`지정 되지 않은 형식에 대 한 동의어로 여기에 설명 되어 있습니다.
