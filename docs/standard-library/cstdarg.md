# <a name="ltcstdarggt"></a>&lt;cstdarg&gt;

표준 C 라이브러리 헤더 \<stdarg.h>를 포함하고 `std` 네임스페이스에 연결된 이름을 추가합니다.

## <a name="syntax"></a>구문

```cpp
#include <cstdarg>

```

## <a name="remarks"></a>설명

이 헤더를 포함하는 경우 표준 C 라이브러리 헤더의 외부 링크를 사용하여 선언한 이름이 `std` 네임스페이스에도 선언됩니다.
