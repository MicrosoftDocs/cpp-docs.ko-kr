# <a name="ltciso646gt"></a>&lt;ciso646&gt;

C 표준 라이브러리 헤더 \<iso646 >를 포함 하 고 `std` 네임 스페이스에 연결 된 이름을 추가 합니다.

> [!NOTE]
> 이 호환성 헤더는의 C++키워드인 이름을 정의 하므로이를 포함 하면 아무런 효과가 없습니다. Iso646 > 헤더는에서 C++더 이상 사용 되지 않습니다. \< \<Ciso646 > 헤더는 초안 c + + 20 표준에서 제거 됩니다.

## <a name="requirements"></a>요구 사항

**헤더:** \<ciso646>

**네임스페이스:** std

## <a name="remarks"></a>설명

이 헤더를 포함하면 C 표준 라이브러리 헤더의 외부 링크를 사용하여 선언한 이름이 `std` 네임스페이스에 선언됩니다.
