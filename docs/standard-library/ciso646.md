# <a name="ltciso646gt"></a>&lt;ciso646&gt;

C 표준 라이브러리 헤더를 포함 하 \<iso646.h> 고 네임 스페이스에 연결 된 이름을 추가 합니다 `std` .

> [!NOTE]
> 이 호환성 헤더는 c + +의 키워드인 이름을 정의 하므로이를 포함 하면 아무런 효과가 없습니다. \<iso646.h>헤더는 c + +에서 사용 되지 않습니다. \<ciso646>헤더는 초안 c + + 20 표준에서 제거 됩니다.

## <a name="requirements"></a>요구 사항

**헤더:**\<ciso646>

**네임스페이스:** std

## <a name="remarks"></a>설명

이 헤더를 포함 하면 C 표준 라이브러리 헤더의 외부 링크를 사용 하 여 선언한 이름이 네임 스페이스에 선언 됩니다 `std` .
