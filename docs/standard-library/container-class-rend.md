# <a name="container-classrend"></a>Container Class::rend

> [!NOTE]
> Microsoft C++ 설명서의 이 항목은 C++ 표준 라이브러리에서 사용되는 컨테이너의 비기능적 예제입니다. 자세한 내용은 [C++ 표준 라이브러리 컨테이너](../standard-library/stl-containers.md)를 참조하세요.

구성원 함수는 시퀀스의 첫 번째 요소(또는 빈 시퀀스의 끝 바로 다음)를 가리키는 역방향 반복기를 반환하여 역방향 시퀀스의 끝을 지정합니다.

## <a name="syntax"></a>구문

```cpp
const_reverse_iterator rend() const;

reverse_iterator rend();
```
