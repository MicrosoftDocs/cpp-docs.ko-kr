# <a name="container-classiterator"></a>Container Class::iterator

> [!NOTE]
> Microsoft C++ 설명서의 이 항목은 C++ 표준 라이브러리에서 사용되는 컨테이너의 비기능적 예제입니다. 자세한 내용은 [C++ 표준 라이브러리 컨테이너](../standard-library/stl-containers.md)를 참조하세요.

제어되는 시퀀스의 반복기로 사용될 수 있는 개체를 설명합니다.

## <a name="syntax"></a>구문

```cpp
typedef T5 iterator;
```

## <a name="remarks"></a>주의

`T5`지정 되지 않은 형식에 대 한 동의어로 여기에 설명 되어 있습니다. `iterator` 형식의 개체를 [const_iterator](../standard-library/container-class-const-iterator.md)형식의 개체로 캐스팅할 수 있습니다.
