# <a name="container-classbegin"></a>Container Class::begin

> [!NOTE]
> 이 항목은 c + + 표준 라이브러리에서 사용 되는 컨테이너의 작동 하지 않는 예제로 Visual Studio c + + 설명서에 있습니다. 자세한 내용은 [C++ 표준 라이브러리 컨테이너](../standard-library/stl-containers.md)를 참조하세요.

시퀀스의 첫 번째 요소(또는 빈 시퀀스의 끝 바로 다음)를 가리키는 반복기를 반환합니다.

## <a name="syntax"></a>구문

```cpp
const_iterator begin() const;

iterator begin();
```
