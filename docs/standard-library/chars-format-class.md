# <a name="charsformat-class"></a>chars_format 클래스

기본 숫자 변환에 대 한 부동 소수점 형식입니다.

## <a name="syntax"></a>구문

```cpp
enum class chars_format {
    scientific = unspecified ,
    fixed = unspecified ,
    hex = unspecified ,
    general = fixed | scientific
};
```
