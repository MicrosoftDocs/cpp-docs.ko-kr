# <a name="ltiosfwdgt"></a>&lt;iosfwd&gt;
iostreams 전체에서 사용되는 여러 템플릿 클래스에 대한 전방 참조를 선언합니다. 이러한 템플릿 클래스는 모두 다른 표준 헤더에 정의되어 있습니다. 이 헤더는 해당 정의가 아니라 해당 선언 중 하나가 필요한 경우에만 명시적으로 포함합니다.  
  
## <a name="syntax"></a>구문  
  
```  
#include <iosfwd>  
  
```
