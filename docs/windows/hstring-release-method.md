# <a name="hstringrelease-method"></a>HString::Release 메서드

## <a name="syntax"></a>구문  
  
```  
void Release() throw()  
```  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
