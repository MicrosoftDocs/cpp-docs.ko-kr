# <a name="criticalsectionisvalid-method"></a>CriticalSection::IsValid 메서드
현재 임계 영역이 유효한지 여부를 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```  
bool IsValid() const;  
```  
  
## <a name="return-value"></a>반환 값  
 기본적으로 항상 반환 **true**합니다.  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
