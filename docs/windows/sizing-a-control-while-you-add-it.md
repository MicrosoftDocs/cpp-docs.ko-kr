# <a name="sizing-a-control-while-you-add-it"></a>컨트롤을 추가할 때 크기 조정
### <a name="to-size-a-control-while-you-add-it"></a>컨트롤을 추가할 때 크기를 조정 하려면  
  
1.  컨트롤에서 선택 된 [도구 상자 창](/visualstudio/ide/reference/toolbox)합니다.  
  
2.  대화 상자에 새 컨트롤의 왼쪽 위 모퉁이 원하는 (십자)는 커서를 놓습니다.  
  
3.  클릭 하 여 대화 상자에서 컨트롤의 왼쪽 위 모퉁이 고정 하려면 마우스 단추를 길게 누른 다음 컨트롤이 원하는 크기가 될 때까지 커서를 오른쪽 아래로 끕니다.  
  
    > [!NOTE]
    >  실제로 그릴 컨트롤의 네 모퉁이 고정할 수 있습니다. 이 절차 예를 들어 왼쪽 위 모퉁이 사용 합니다.  
  
4.  마우스 단추를 놓습니다. 컨트롤의 대화 상자에서 지정한 크기에 settles 합니다.  
  
    > [!TIP]
    >  컨트롤의 테두리의 크기 조정 핸들을 이동 하 여 대화 상자에 삭제 한 후 컨트롤 크기를 조정할 수 있습니다. 자세한 내용은 참조 [각 컨트롤 크기 조정](../windows/sizing-individual-controls.md)합니다.  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](/dotnet/framework/resources/index) 에 *.NET Framework 개발자 가이드입니다.* 수동으로 관리 되는 프로젝트에 리소스 파일을 추가, 리소스 액세스, 정적 리소스 표시 및 속성에 리소스 문자열 할당에 대 한 정보를 참조 하십시오. [데스크톱 앱에 대 한 리소스 파일 만들기](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오.
