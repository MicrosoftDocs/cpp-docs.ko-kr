# <a name="attribute-requirements"></a>특성 요구 사항
C + + 특성에 대해 나열 된 요구 사항을 프로젝트 형식, 컴파일러 설정 및 특성에 대 한 필요한 정보를 다른 작업에 대해 설명 합니다. 정보 범주에 대 한 설명은 다음과 같습니다.
  
> [!NOTE]
> 또한 특성을 사용 하는 클래스에서 파생 된 클래스에서 특성을 사용 하 여 지원 되지 않습니다.
  
## <a name="header"></a>헤더
 이 필드는 특성을 사용 하려면 먼저 포함 해야 하는 헤더 파일을 나열 합니다.
  
## <a name="project"></a>프로젝트
 이 필드 특성을 사용할 수 있는 프로젝트 형식을 설명 합니다.
  
## <a name="compiler"></a>컴파일러
 이 필드에이 특성을 사용할 수 있어야 하는 컴파일러 옵션을 제공 합니다.
