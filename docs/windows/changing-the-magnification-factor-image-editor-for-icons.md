# <a name="changing-the-magnification-factor-image-editor-for-icons"></a>확대 비율 변경(아이콘에 대한 이미지 편집기)
기본적으로 이미지 편집기는 실제 크기 및 실제 크기 6 배 오른쪽 창에서 보기의 왼쪽된 창에서 보기를 표시 합니다. 확대 비율 (작업 영역 맨 아래에 상태 표시줄에 표시) 이미지의 실제 크기와 표시 크기입니다. 기본 비율 6 이며 범위는 1부터 10 까지입니다.  
  
### <a name="to-change-the-magnification-factor"></a>확대 비율을 변경 하려면  
  
1.  선택 된 **이미지 편집기** 창 확대 비율을 변경 합니다.  
  
2.  에 [이미지 편집기 도구 모음](../windows/toolbar-image-editor-for-icons.md)의 오른쪽 화살표를 클릭는 [확대 도구](../windows/toolbar-image-editor-for-icons.md) 하위 메뉴에서 원하는 확대 비율을 선택 하 고: **1 X**, **2 배**, **6 X**, 또는 **8 X**합니다.  
  
    > [!NOTE]
    >  에 나열 되지 않은 확대 비율을 선택 하 고 **확대** 도구를 액셀러레이터 키를 사용 합니다.

[데스크톱 앱에 대 한 리소스 파일 만들기](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오.
