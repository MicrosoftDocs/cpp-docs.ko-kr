합니다 **이미지** 편집기는 반전 된 색을 사용 하 여 이미지로 형태가 나타나는 방법을 알릴 수 있도록 이미지의 선택한 부분에서 색을 반전 하는 편리한 방법을 제공 합니다.

### <a name="to-invert-colors-in-the-current-selection"></a>현재 선택 영역의 색을 반전하려면

1. 에 **이미지** 메뉴에서 클릭 **색 반전**합니다.
