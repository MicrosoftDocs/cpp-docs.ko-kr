# <a name="copying-a-device-image-image-editor-for-icons"></a>장치 이미지 복사(아이콘에 대한 이미지 편집기)
### <a name="to-copy-a-device-image"></a>장치 이미지 복사  
  
1.  에 **이미지** 메뉴를 클릭 하 여 **장치 이미지 열기** 현재 이미지 목록에서 이미지를 선택 합니다. 예를 들어 32 × 32, 16 색 버전의 아이콘을 선택 합니다.  
  
2.  현재 표시 된 아이콘 이미지 복사 (**CTRL + C**).  
  
3.  아이콘의 다른 이미지를 다른 엽니다 **이미지 편집기** 창. 예를 들어 16 × 16, 아이콘의 16 색 버전을 엽니다.  
  
4.  아이콘 이미지를 붙여 넣습니다 (**CTRL + V**) 하나에서 **이미지 편집기** 다른 창. 더 작은 크기에 더 큰 크기를 붙여 넣는 경우 이미지 크기를 조정 하는 아이콘 핸들을 사용할 수 있습니다.
