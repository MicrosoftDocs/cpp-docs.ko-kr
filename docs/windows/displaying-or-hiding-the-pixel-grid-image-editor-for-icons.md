# <a name="displaying-or-hiding-the-pixel-grid-image-editor-for-icons"></a>픽셀 모눈 표시 또는 숨기기(아이콘에 대한 이미지 편집기)
4 이상인 확대 비율이 모든 이미지 편집기 창에 대 한 이미지의 개별 픽셀을 구분 하는 표를 표시할 수 있습니다.  
  
> [!TIP]
>  도구 모음 단추 위로 커서를 올려 놓으면 도구 설명이 표시 됩니다. 이러한 팁은 각 단추의 기능을 식별할 수 있습니다.  
  
### <a name="to-display-or-hide-the-pixel-grid"></a>표시 하거나 숨기려면 픽셀 모눈  
  
1.  에 **이미지** 메뉴를 클릭 하 여 **눈금 설정**합니다.  
  
2.  선택 된 **픽셀 모눈** 모눈을 표시 하거나 표를 숨기려면 확인란의 선택을 취소 하려면 확인란 합니다.
