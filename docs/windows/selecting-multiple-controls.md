# <a name="selecting-multiple-controls"></a>여러 개의 컨트롤 선택
### <a name="to-select-multiple-controls"></a>여러 컨트롤을 선택 하려면  
  
1.  에 [도구 상자 창](/visualstudio/ide/reference/toolbox), 선택는 **포인터** 도구입니다.  
  
2.  대화 상자에서 선택 하려는 컨트롤 주위의 선택 상자에 대 한 포인터를 끕니다.  
  
     마우스 단추를 놓을 때 모든 안쪽과 선택 상자가 선택 되어 교차 제어 합니다.  
  
     \- 또는 -  
  
-   SHIFT 키를 누른 채 선택 영역에 포함 하려는 컨트롤을 클릭 합니다.  
  
     \- 또는 -  
  
-   Ctrl 키를 누른 채 선택 영역에 포함 하려는 컨트롤을 클릭 합니다.  
  
#### <a name="to-remove-a-control-from-a-group-of-selected-controls-or-to-add-a-control-to-a-group-of-selected-controls"></a>선택한 컨트롤의 그룹에서 컨트롤을 제거 하려면 또는 컨트롤 선택 된 컨트롤의 그룹을 추가 하려면  
  
1.  선택한 컨트롤의 그룹에서 SHIFT 키를 누른 채 고 기존 선택 영역에 추가 하거나 제거할 컨트롤을 클릭 합니다.  
  
    > [!NOTE]
    >  CTRL 키를 누른 채 하 고 선택 영역 내에서 컨트롤을 클릭 하면 선택에서 기준 컨트롤을 제어 하는 설정 됩니다. 자세한 내용은 참조 [기준 컨트롤 지정](../windows/specifying-the-dominant-control.md)합니다.  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](/dotnet/framework/resources/index) 에 *.NET Framework 개발자 가이드입니다.* 수동으로 관리 되는 프로젝트에 리소스 파일을 추가, 리소스 액세스, 정적 리소스 표시 및 속성에 리소스 문자열 할당에 대 한 정보를 참조 하십시오. [데스크톱 앱에 대 한 리소스 파일 만들기](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오.
