# <a name="editing-managed-resource-files-c"></a>관리 되는 리소스 파일 (c + +) 편집

관리 되는 리소스 (.resx) 파일에는 XML 파일입니다. 관리 되는 리소스 파일에서 프로젝트에 추가 되는 경우는 **새 항목 추가** 대화 상자를 **관리 되는 리소스 편집기** 기본적으로 열립니다.
