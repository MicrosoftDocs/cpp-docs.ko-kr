# <a name="creatormapservername-data-member"></a>CreatorMap::serverName 데이터 멤버
WRL 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.  
  
## <a name="syntax"></a>구문  
  
```  
const wchar_t* serverName;  
```  
  
## <a name="remarks"></a>설명  
 CreatorMap에 대 한 서버 이름을 저장합니다.
