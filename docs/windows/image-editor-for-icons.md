# <a name="image-editor-for-icons-c"></a>아이콘 (C++)에 대 한 이미지 편집기

**솔루션 탐색기**에서 .ico, .bmp, .png 등의 이미지 파일을 선택 하면 코드 **편집기**에서 코드 파일을 여는 것과 같은 방식으로 이미지가 이미지 **편집기** 에서 열립니다. **이미지 편집기** 탭이 활성화 되 면 이미지를 만들고 편집할 수 있는 여러 도구가 포함 된 도구 모음이 표시 됩니다. 이미지 **편집기** 도구 모음에서 **이미지** 메뉴와 도구의 명령을 사용 하 여 비트맵, 아이콘 및 커서와 함께 이미지를 GIF 또는 JPEG 형식으로 편집할 수 있습니다.

그래픽 리소스는 응용 프로그램에 대해 정의 하는 이미지입니다. 도형을 사용 하 여 자유형 또는 draw를 그릴 수 있습니다. 편집, 대칭 이동 또는 크기 조정을 위한 이미지의 일부를 선택 하거나 선택한 이미지 부분에서 사용자 지정 브러시를 만들어 해당 브러시로 그릴 수 있습니다. 이미지 속성을 정의 하 고, 다른 형식으로 이미지를 저장 하 고, 이미지를 한 형식에서 다른 형식으로 변환할 수 있습니다.

> [!NOTE]
> **이미지 편집기**를 사용 하 여 32 비트 이미지를 볼 수 있지만 편집할 수는 없습니다.

또한 **이미지 편집기** 와 [바이너리 편집기](binary-editor.md) 를 사용 하 여 관리 되는 프로젝트의 리소스 파일 작업을 수행할 수 있습니다. 편집할 관리되는 리소스는 연결된 리소스여야 합니다. Visual Studio 리소스 편집기에서는 포함 된 리소스 편집을 지원 하지 않습니다.

새 그래픽 리소스를 만들 뿐 아니라 편집용으로 [기존 이미지를 가져온](../windows/how-to-copy-resources.md#import-and-export-resources) 다음 프로젝트에 추가할 수 있습니다. 프로젝트의 일부가 아닌 이미지를 열고 편집 하 여 [독립 실행형 이미지를 편집할](../windows/editing-an-image-outside-of-a-project-image-editor-for-icons.md)수도 있습니다.

**이미지 편집기**에 대 한 자세한 내용은 [아이콘 또는 기타 이미지를 만드는](../windows/creating-an-icon-or-other-image-image-editor-for-icons.md)방법, [이미지 편집](../windows/selecting-an-area-of-an-image-image-editor-for-icons.md), [그리기 도구 사용](../windows/using-a-drawing-tool-image-editor-for-icons.md), [색으로 작업](../windows/working-with-color-image-editor-for-icons.md)및 [액셀러레이터 키](../windows/accelerator-keys-image-editor-for-icons.md)를 참조 하세요.

> [!NOTE]
> 응용 프로그램에서 사용할 수 있는 여러 애니메이션, 비트맵 및 아이콘을 포함 하는 **Visual Studio 이미지 라이브러리** 를 무료로 다운로드할 수 있습니다. 라이브러리를 다운로드 하는 방법에 대 한 자세한 내용은 [Visual Studio 이미지 라이브러리](/visualstudio/designers/the-visual-studio-image-library)를 참조 하십시오.

## <a name="image-menu"></a>이미지 메뉴

이미지 **편집기** 가 활성화 된 경우에만 표시 되는 **이미지** 메뉴에는 이미지 편집, 색상표 관리 및 **이미지 편집기** 창 옵션 설정에 대 한 명령이 있습니다. 또한 아이콘 및 커서로 작업할 때 장치 이미지를 사용 하기 위한 명령을 사용할 수 있습니다.

|명령|설명|
|---|---|
|**색 반전**|색을 반전 합니다.|
|**좌우 대칭**|이미지나 선택 영역을 좌우 대칭 이동합니다.|
|**상하 대칭**|이미지나 선택 영역을 상하 대칭 이동합니다.|
|**90도 회전**|이미지나 선택 영역을 90도 회전합니다.|
|**색 창 표시**|이미지에 사용할 색을 선택할 수 있는 **색** 창을 엽니다.|
|**선택 항목을 브러시로 사용**|이미지의 일부에서 사용자 지정 브러시를 만들 수 있습니다.<br/><br/>선택 항목은 이미지 전체에서 선택 영역에 있는 색을 배포 하는 사용자 지정 브러시가 됩니다. 선택 항목의 복사본은 끌기 경로를 따라 유지 됩니다. 더 천천히 끌면 복사본이 더 많이 생성 됩니다.|
|**선택 영역 복사 및 개요**|현재 선택한 항목의 복사본을 만들고 윤곽선을 그립니다.<br/><br/>배경색이 현재 선택 영역에 포함 되어 있으면 투명 하 게 선택 된 경우 제외 됩니다.
|**색 조정**|이미지에 사용 하는 색을 사용자 지정할 수 있는 **사용자 지정 색 선택기**를 엽니다.|
|**색상표 로드**|이전에 pal 파일에 저장 된 색상표 색을 로드할 수 있는 **색 색상표 로드** 대화 상자를 엽니다.|
|**색상표 저장**|색상표 색을 pal 파일에 저장 합니다.|
|**불투명 그리기**|이 옵션을 선택 하면 현재 선택 항목이 불투명 하 게 됩니다.<br/><br/>이 옵션의 선택을 취소 하면 현재 선택 항목을 투명 하 게 만듭니다.|
|**도구 모음 편집기**|[새 도구 모음 리소스 대화 상자](../windows/new-toolbar-resource-dialog-box.md)를 엽니다.|
|**그리드 설정**|이미지에 대 한 그리드를 지정할 수 있는 **그리드 설정** 대화 상자를 엽니다.|
|**새 이미지 형식**|[ 새\<장치 > 이미지 형식 대화 상자](../windows/new-device-image-type-dialog-box-image-editor-for-icons.md)를 엽니다.<br/><br/>단일 아이콘 리소스는 다양 한 크기의 여러 이미지를 포함할 수 있으며 windows는 표시 되는 방식에 따라 적절 한 아이콘 크기를 사용할 수 있습니다. 새 장치 유형은 아이콘의 크기를 수정 하지 않고 아이콘 안에 새 이미지를 만듭니다. 아이콘 및 커서에만 적용 됩니다.|
|**현재 아이콘/커서 이미지 형식**|사용 가능한 첫 9 개의 커서 또는 아이콘 이미지를 나열 하는 하위 메뉴를 엽니다. 하위 메뉴의 마지막 명령은 **추가** [장치 > 이미지 \<열기 대화 상자](../windows/open-device-image-dialog-box-image-editor-for-icons.md)를 엽니다.|
|**이미지 형식 삭제**|선택한 장치 이미지를 삭제 합니다.|
|**도구**|**이미지 편집기** 도구 모음에서 사용할 수 있는 모든 도구를 포함 하는 하위 메뉴를 시작 합니다.|

**그리드 설정** 대화 상자를 사용 하 여 이미지에 대 한 그리드 설정을 지정 하 고 편집 된 이미지 위에 눈금선을 표시할 수 있습니다. 선은 이미지를 편집 하는 데 유용 하지만 이미지 자체의 일부로 저장 되지 않습니다.

|속성|설명|
|---|---|
|**픽셀 그리드**|이 확인란을 선택 하면 **이미지 편집기**에서 각 픽셀 주위에 표가 표시 됩니다.<br/><br/>표는 4 × 이상 해상도로만 표시 됩니다.|
|**타일 그리드**|이를 선택 하면 표 간격 값으로 지정 된 **이미지 편집기**의 픽셀 블록 주위에 표가 표시 됩니다.|
|**Width**|각 타일 블록의 너비를 지정 합니다.<br/><br/>이 속성은 일정 한 간격으로 정렬 된 여러 이미지를 포함 하는 비트맵을 그릴 때 유용 합니다.|
|**높이**|각 타일 블록의 높이를 지정 합니다.<br/><br/>이 속성은 일정 한 간격으로 정렬 된 여러 이미지를 포함 하는 비트맵을 그릴 때 유용 합니다.|

## <a name="toolbar"></a>Toolbar

**이미지 편집기** 도구 모음에는 그리기, 그리기, 텍스트 입력, 지우기 및 뷰 조작을 위한 도구가 포함 되어 있습니다. 또한 각 도구를 사용 하기 위한 옵션을 선택할 수 있는 옵션 선택 기가 포함 되어 있습니다. 예를 들어 다양 한 브러시 너비, 확대 비율 및 선 스타일에서 선택할 수 있습니다.

**이미지 편집기** 도구 모음에서 사용할 수 있는 모든 도구는 메뉴 **이미지** > **도구**에서도 사용할 수 있습니다. **이미지 편집기** 도구 모음 및 **옵션** 선택기를 사용 하려면 원하는 도구나 옵션을 선택 합니다.

![이미지 편집기 도구 모음](../mfc/media/vcimageeditortoolbar.gif "Vcimageeditortoolbar")<br/>
**이미지 편집기** 도구 모음

> [!TIP]
> 도구 모음 단추 위에 커서를 놓으면 도구 설명이 나타납니다. 이러한 팁은 각 단추의 기능을 식별 하는 데 도움이 될 수 있습니다.

[키보드](../windows/accelerator-keys-image-editor-for-icons.md)에서 많은 그리기 도구를 사용할 수 있으므로 **이미지 편집기** 도구 모음을 숨기는 것이 유용한 경우도 있습니다.

- **이미지 편집기** 도구 모음을 표시 하거나 숨기려면 메뉴 **보기** > **도구 모음** 으로 이동 하 여 **이미지 편집기**를 선택 합니다.

> [!NOTE]
> 현재 프로젝트 또는 솔루션의 이미지 파일을 **이미지 편집기**에서 열지 않은 경우이 도구 모음의 요소를 사용할 수 없는 것으로 나타납니다.

### <a name="option-selector"></a>옵션 선택기

**옵션** 선택기를 사용 하 여 선, 브러시 스트로크 등의 너비를 지정할 수 있습니다. **옵션** 선택기 단추의 아이콘은 선택한 도구에 따라 달라 집니다.

![이미지&#45;편집기 도구 모음에서 도형 선택기 그리기](../mfc/media/vcimageeditortoolbaroptionselector.gif "Vcimageeditortoolbaroptionselector")<br/>
**이미지 편집기** 도구 모음의 **옵션** 선택기

### <a name="text-tool"></a>텍스트 도구

**텍스트 도구** 대화 상자를 사용 하 여 커서, 비트맵 또는 아이콘 리소스에 텍스트를 추가할 수 있습니다.

이 대화 상자에 액세스 하려면 **이미지 편집기** 를 열고 메뉴 **이미지** > **도구**로 이동한 다음 **텍스트 도구** 명령을 선택 합니다.

> [!TIP]
> **텍스트 도구** 대화 상자를 마우스 오른쪽 단추로 클릭 하 여 표준 Windows 명령 목록을 포함 하는 기본 바로 가기 메뉴에 액세스할 수 있습니다.

**텍스트 도구 글꼴** 대화 상자를 열어 커서 글꼴의 글꼴, 스타일 또는 크기를 변경할 수 있습니다. **텍스트** 영역에 표시 되는 텍스트에 변경 내용이 적용 됩니다.

이 대화 상자에 액세스 하려면 **텍스트 도구** 대화 상자에서 **글꼴** 단추를 선택 합니다. 사용할 수 있는 속성은 다음과 같습니다.

|속성|설명|
|---|---|
|**글꼴**|사용 가능한 글꼴을 나열 합니다.|
|**글꼴 스타일**|지정 된 글꼴에 사용할 수 있는 스타일을 나열 합니다.|
|**Size**|지정 된 글꼴에 사용할 수 있는 크기를 나열 합니다.|
|**샘플**|지정 된 글꼴 설정을 사용 하 여 텍스트가 표시 되는 방법의 샘플을 보여 줍니다.|
|**스크립트**|지정 된 글꼴에 사용할 수 있는 언어 스크립트를 나열 합니다.<br/><br/>다른 언어 스크립트를 선택 하면 다국어 문서를 만들 때 해당 언어에 대 한 문자 집합을 사용할 수 있게 됩니다.|

#### <a name="to-change-the-font-of-text-on-an-image"></a>이미지의 텍스트 글꼴을 변경 하려면

Windows 응용 프로그램의 아이콘에 텍스트를 추가 하 고 텍스트의 글꼴을 조작 하는 방법의 예는 다음과 같습니다.

1. Windows Forms 응용 C++ 프로그램을 만듭니다. 자세한 내용은 [방법: Windows Forms 응용 프로그램](/previous-versions/visualstudio/visual-studio-2008/s69bf10x(v%3dvs.90))을 만듭니다. *응용 프로그램 .ico* 파일은 기본적으로 프로젝트에 추가 됩니다.

1. **솔루션 탐색기**에서 *.ico*파일을 두 번 클릭 합니다. **이미지 편집기** 가 열립니다.

1. 메뉴 **이미지** > **도구로** 이동 하 여 **텍스트 도구**를 선택 합니다.

1. **텍스트 도구** 대화 상자에서 빈 텍스트 영역 *C++* 을 입력 합니다. 이 텍스트는 **이미지 편집기**에서 *.ico* 의 왼쪽 위 모퉁이에 있는 크기 조정 가능한 상자에 나타납니다.

1. **이미지 편집기**에서 크기 조정 가능한 상자를 *app-v* 의 가운데로 끌어 텍스트 가독성을 향상 시킵니다.

1. **텍스트 도구** 대화 상자에서 **글꼴** 단추를 선택 합니다.

1. **텍스트 도구 글꼴** 대화 상자에서 다음을 수행 합니다.

   - **글꼴** 목록 상자에 나열 된 사용 가능한 글꼴 목록에서 **Times New Roman** 을 선택 합니다.

   - **글꼴 스타일** 목록 상자에 나열 된 사용 가능한 글꼴 스타일 목록에서 **굵게** 를 선택 합니다.

   - **크기** 목록 상자에 나열 된 사용 가능한 요소 크기 목록에서 **10** 을 선택 합니다.

   - **확인**을 선택합니다. **텍스트 도구 글꼴** 대화 상자가 닫히고 새 글꼴 설정이 텍스트에 적용 됩니다.

1. **텍스트 도구** 대화 상자에서 **닫기** 를 선택 합니다. 텍스트 주위에 있는 크기 조정 가능한 상자가 **이미지 편집기**에서 사라집니다.

텍스트 영역에는 리소스의 일부로 표시 되는 텍스트가 표시 됩니다. 처음에는이 영역이 비어 있습니다.

> [!NOTE]
> **투명 한 배경을** 설정 하면 해당 텍스트만 이미지에 배치 됩니다. **불투명 배경을** 설정 하면 배경색으로 채워진 경계 사각형이 텍스트 뒤에 배치 됩니다.

## <a name="window-panes"></a>창

**이미지 편집기** 창에는 두 개의 창을 구분 하는 분할 막대와 함께 두 개의 이미지 뷰가 표시 됩니다. 분할줄을 한 쪽에서 다른 쪽으로 끌면 창의 상대적 크기를 변경할 수 있습니다. 활성 창에는 선택 테두리가 표시됩니다.

한 뷰는 실제 크기 이며 다른 뷰는 기본 확대 비율 6으로 확대 됩니다. 이러한 두 창의 뷰가 자동으로 업데이트 됩니다. 한 창에서 변경한 내용은 다른 창에 즉시 표시 됩니다. 두 개의 창에서 이미지의 확대 된 보기를 쉽게 수행할 수 있습니다 .이를 통해 개별 픽셀을 구분 하 고 이미지의 실제 크기 보기에 대 한 작업의 효과를 확인할 수 있습니다.

왼쪽 창에는 이미지의 기본 1:1 확대 뷰를 표시 하는 데 필요한 만큼의 공간 ( **이미지** 창의 절반까지)이 사용 됩니다. 오른쪽 창에는 기본 6:1 확대 확대/축소 이미지가 표시 됩니다. **이미지 편집기** 도구 모음의 **확대** 도구를 사용 하거나 액셀러레이터 키를 사용 하 여 각 창에서 배율을 변경할 수 있습니다.

**이미지 편집기** 창의 더 작은 창을 확대 하 고 두 개의 창을 사용 하 여 큰 이미지의 여러 영역을 표시할 수 있습니다. 창 내부를 선택 하 여 선택 합니다.

분할 막대에 포인터를 놓고 분할 막대를 오른쪽 이나 왼쪽으로 이동 하 여 창의 상대 크기를 변경할 수 있습니다. 한 창 에서만 작업 하려는 경우 분할 막대를 양쪽으로 이동할 수 있습니다.

**이미지 편집기** 창이 4 이상의 배율로 확대 된 경우 이미지의 개별 픽셀을 구분 하는 픽셀 모눈을 표시할 수 있습니다.

### <a name="to-change-the-magnification-factor"></a>확대 비율을 변경 하려면

기본적으로 **이미지 편집기** 는 왼쪽 창의 실제 크기와 오른쪽 창의 뷰를 실제 크기의 6 배로 표시 합니다. 확대/축소 비율 (작업 영역 아래쪽의 상태 표시줄에 표시)은 이미지의 실제 크기와 표시 된 크기의 비율입니다. 기본 요소는 6이 고 범위는 1에서 10 사이입니다.

1. 확대/축소 비율을 변경할 **이미지 편집기** 창을 선택 합니다.

1. **이미지 편집기** 도구 모음에서 **확대** 도구 오른쪽의 화살표를 선택 하 고 하위 메뉴에서 확대/축소 비율을 선택 합니다. **1x**, **2x**, **6X**또는 **8x**입니다.

   > [!NOTE]
   > **확대** 도구에 나열 된 것과 다른 확대 비율을 선택 하려면 액셀러레이터 키를 사용 합니다.

### <a name="to-display-or-hide-the-pixel-grid"></a>픽셀 그리드를 표시 하거나 숨기려면

확대 비율이 4 이상인 모든 **이미지 편집기** 창의 경우 이미지의 개별 픽셀을 구분 하는 그리드를 표시할 수 있습니다.

1. 메뉴 **이미지** > **그리드 설정**으로 이동 합니다.

1. **픽셀 그리드** 확인란을 선택 하 여 표를 표시 하거나 상자의 선택을 취소 하 여 표를 숨깁니다.
