# <a name="disabling-guides"></a>안내선 비활성화
맞춤 안내선의 효과 사용 하지 않도록 설정 하려면 특수 키 마우스와 함께에서 사용할 수 있습니다. ALT 키를 사용 하 여 선택한 가이드의 맞춤 효과를 사용 하지 않습니다. SHIFT 키를 사용 하 여 안내선 이동 기준된 컨트롤을 가이드와 함께 이동할 수 없습니다.  
  
### <a name="to-disable-the-snapping-effect-of-the-guides"></a>맞춤 안내선의 효과 사용 하지 않도록 설정 하려면  
  
1.  대화 컨트롤 메뉴는 **ALT** 키입니다.  
  
### <a name="to-move-guides-without-moving-the-snapped-controls"></a>안내선 기준된 컨트롤을 이동 하지 않고 이동 하려면  
  
1.  이 가이드 채로 끕니다는 **SHIFT** 키입니다.  
  
### <a name="to-turn-off-the-guides"></a>이러한 가이드를 해제 하려면  
  
1.  **형식** 메뉴 선택 **가이드 설정**합니다.  
  
2.  에 [안내선 설정 대화 상자](../windows/guide-settings-dialog-box.md)아래 **레이아웃 안내선**선택, **None**합니다.  
  
    > [!NOTE]
    >  눈금자를 두 번 수도 있습니다는 **가이드 설정** 대화 상자.  
  
 \- 또는 -  
  
-   에 **형식** 메뉴를 클릭 하 여 **안내선 설정/해제**합니다.
