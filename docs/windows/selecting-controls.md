# <a name="selecting-controls"></a>컨트롤 선택
크기에 컨트롤을 선택, 정렬, 이동, 복사 또는 삭제할 수 및 다음 원하는 작업을 수행 합니다. 대부분의 경우에서 크기 조정 및 맞춤 도구를 사용 하려면 둘 이상의 컨트롤을 선택 해야는 [대화 상자 편집기 도구 모음](../windows/showing-or-hiding-the-dialog-editor-toolbar.md)합니다.  
  
 컨트롤을 선택, 실선 (활성)와 그 주위에 회색된 테두리에 비어 있는 (비활성) "크기 조정 핸들" 작은 사각형 모양의 하는 경우에 선택 테두리가 나타납니다. 기준 컨트롤에는 고정 크기 조정 핸들이; 여러 컨트롤을 선택 하면 다른 선택 된 컨트롤 빈 크기 조정 핸들을 갖습니다.  
  
 크기 조정 하거나 "주요 컨트롤"를 사용 하 여 대화 상자 편집기 여러 컨트롤을 맞출 때 다른 컨트롤의 크기와 정렬 방법을 확인 하려면. 기본적으로 기준 컨트롤에는 선택한 첫 번째 컨트롤이입니다.  
  
-   [여러 개의 컨트롤 선택](../windows/selecting-multiple-controls.md)  
  
-   [기준 컨트롤 지정](../windows/specifying-the-dominant-control.md)  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](/dotnet/framework/resources/index) 에 *.NET Framework 개발자 가이드입니다.* 수동으로 관리 되는 프로젝트에 리소스 파일을 추가, 리소스 액세스, 정적 리소스 표시 및 속성에 리소스 문자열 할당에 대 한 정보를 참조 하십시오. [데스크톱 앱에 대 한 리소스 파일 만들기](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오.
