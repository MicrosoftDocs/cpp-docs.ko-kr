# <a name="helpfile"></a>helpfile
형식 라이브러리에 대 한 도움말 파일의 이름을 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
[ helpfile(  
   "filename"  
) ]  
```  
  
### <a name="parameters"></a>매개 변수  
 *filename*  
 도움말 항목이 포함 된 파일의 이름입니다.
