# <a name="eventtargetarrayeventtargetarray-destructor"></a>EventTargetArray::~EventTargetArray 소멸자

## <a name="syntax"></a>구문

```cpp

```

## <a name="remarks"></a>설명

현재 초기화 취소 **EventTargetArray** 클래스입니다.
