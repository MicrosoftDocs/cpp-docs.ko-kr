# <a name="deleting-menus-and-menu-commands"></a>메뉴 및 메뉴 명령 삭제
### <a name="to-delete-a-menu-or-menu-command"></a>메뉴 또는 메뉴 명령을 삭제하려면  
  
1.  메뉴 이름 또는 명령을 마우스 오른쪽 단추로 클릭합니다.  
  
2.  바로 가기 메뉴에서 **삭제** 를 선택합니다.  
  
    > [!NOTE]
    >  마찬가지로 바로 가기 메뉴를 사용하여 복사, 잘라내기, 붙여넣기, 새로 삽입, 구분 기호 삽입, ID 편집, 팝업으로 보기, 니모닉 확인 등의 다른 작업을 수행할 수 있습니다.  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](/dotnet/framework/resources/index) 에 *.NET Framework 개발자 가이드입니다.*  
  
 요구 사항  
  
 Win32
