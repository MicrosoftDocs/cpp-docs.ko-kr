# <a name="mutextraitsunlock-method"></a>MutexTraits::Unlock 메서드

## <a name="syntax"></a>구문  
  
```  
inline static void Unlock(  
   _In_ Type h  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `h`  
 뮤텍스 개체에 대 한 핸들입니다.
