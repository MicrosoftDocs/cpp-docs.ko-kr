# <a name="mutextraitsunlock-method"></a>MutexTraits::Unlock 메서드
공유 리소스의 한 독점적인 제어권을 해제합니다.  
  
## <a name="syntax"></a>구문  
  
```  
inline static void Unlock(  
   _In_ Type h  
);  
```  
  
### <a name="parameters"></a>매개 변수  
 *h*  
 뮤텍스 개체에 대 한 핸들입니다.
