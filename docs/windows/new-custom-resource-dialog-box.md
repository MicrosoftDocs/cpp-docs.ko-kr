# <a name="new-custom-resource-dialog-box"></a>새 사용자 지정 리소스 대화 상자
새 사용자 지정 리소스를 만들 수 있습니다.  
  
 **리소스 종류**  
 사용자 지정 리소스 종류의 이름을 입력할 수에 대 한 텍스트 상자를 제공 합니다. Visual c + + 이름을 종료 하면 자동으로 대문자로 시작는 **새 사용자 지정 리소스** 대화 상자.  
  
## <a name="remarks"></a>설명  
 리소스 이름 또는에서 예약 된 ID를 지정 하지 않으면 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]합니다. 예약 된 이름은 DESIGNINFO, HWB 및 TEXTINCLUDE 이며 예약 된 ID는 255입니다.
