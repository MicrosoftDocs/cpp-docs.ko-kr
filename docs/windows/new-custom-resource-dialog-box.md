# <a name="new-custom-resource-dialog-box-c"></a>새 사용자 지정 리소스 대화 상자 (c + +)

C + + 프로젝트에서 새 사용자 지정 리소스를 만들 수 있습니다.

### <a name="resource-type"></a>리소스 형식

사용자 지정 리소스 형식 이름을 입력할 텍스트 상자를 제공 합니다. Visual c + + 이름을 종료할 때 자동으로 대문자로 시작 합니다 **새 사용자 지정 리소스** 대화 상자.

## <a name="remarks"></a>설명

리소스 이름 또는 Visual Studio에서 예약 된 ID를 지정 하지 마십시오. 예약된 된 이름 DESIGNINFO, HWB, 되며 TEXTINCLUDE, 고 예약된 ID는 255 까지입니다.
