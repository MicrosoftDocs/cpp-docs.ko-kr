# <a name="creating-a-new-dialog-box"></a>새 대화 상자 만들기
### <a name="to-create-a-new-dialog-box"></a>새 대화 상자를 만들려면  
  
1.  [리소스 뷰](../windows/resource-view-window.md),.rc 파일을 마우스 오른쪽 단추로 클릭 한 다음 선택 **리소스 추가** 바로 가기 메뉴에서.  
  
    > [!NOTE]
    >  프로젝트에 .rc 파일이 아직 없는 경우 [새 리소스 스크립트 파일 만들기](../windows/how-to-create-a-resource-script-file.md)를 참조하세요.  
  
2.  에 **리소스 추가** 대화 상자에서 **대화** 에 **리소스 종류** 목록을 클릭 합니다 **새로**합니다.  
  
     더하기 기호 (+) 대화 상자 리소스 유형을 옆에 있으면 의미 대화 상자 템플릿을 사용할 수 있습니다. 템플릿 목록 확장, 서식 파일을 선택 하 고, 새로 만들기를 더하기 기호를 클릭 합니다.  
  
     대화 상자 편집기에서 새 대화 상자가 열립니다.  
  
     수도 있습니다 [편집 하기 위한 대화 상자 편집기에서 기존 대화 상자를 열어](../windows/viewing-and-editing-resources-in-a-resource-editor.md)합니다.
