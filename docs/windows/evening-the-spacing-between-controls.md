# <a name="evening-the-spacing-between-controls"></a>컨트롤 사이의 간격을 동일하게 지정
대화 상자 편집기를 사용 하면 선택한 가장 바깥쪽 컨트롤 간에 균등 하 게 공간 컨트롤에 있습니다.  
  
### <a name="to-even-the-spacing-between-controls"></a>컨트롤 사이의 간격을 일정  
  
1.  다시 정렬 하려면 원하는 컨트롤을 선택 합니다.  
  
2.  **형식** 메뉴에서 선택 **간격 똑같이**, 다음 간격 맞춤 중 하나를 선택 합니다.  
  
    -   **건너서**: 맨 왼쪽 및 오른쪽에 있는 컨트롤을 선택 사이의 균등 하 게 합니다.  
  
    -   **아래로**: 맨 위의 및 선택 된 가장 아래쪽에 있는 컨트롤 사이의 균등 하 게 합니다.
