# <a name="specifying-the-dominant-control"></a>기준 컨트롤 지정
선택 된 컨트롤에는 먼저 중요 컨트롤이입니다.  
  
### <a name="to-specify-the-dominant-control"></a>기준 컨트롤 지정 하려면  
  
1.  키를 누른 채는 **CTRL** 키 크기와 다른 컨트롤의 위치를 정하는 데 사용 하려는 컨트롤을 클릭 하 고 *첫 번째*합니다.  
  
     **참고** 기준 컨트롤의 크기 조정 핸들은 실선이 하위 컨트롤의 핸들은 비어 있습니다. 모든 크기 조정 이나 정렬 기준 컨트롤을 기반으로 합니다.  
  
### <a name="to-change-the-dominant-control"></a>기준 컨트롤을 변경 하려면  
  
1.  현재 선택 된 모든 컨트롤 외부를 클릭 하 여 현재 선택을 모두 취소 합니다.  
  
2.  다른 컨트롤을 먼저 선택 하면 이전 절차를 반복 합니다.  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](/dotnet/framework/resources/index) 에 *.NET Framework 개발자 가이드입니다.* 수동으로 관리 되는 프로젝트에 리소스 파일을 추가, 리소스 액세스, 정적 리소스 표시 및 속성에 리소스 문자열 할당에 대 한 정보를 참조 하십시오. [데스크톱 앱에 대 한 리소스 파일 만들기](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오.
