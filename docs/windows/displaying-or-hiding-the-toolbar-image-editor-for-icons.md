# <a name="displaying-or-hiding-the-toolbar-image-editor-for-icons"></a>도구 모음 표시 또는 숨기기(아이콘에 대한 이미지 편집기)
많은 그리기 도구를 사용할 수 있기 때문는 [키보드](../windows/accelerator-keys-image-editor-for-icons.md), 이미지 편집기 도구 모음을 숨기려면 경우가 유용 합니다.  
  
### <a name="to-display-or-hide-the-image-editor-toolbar"></a>표시 하거나 숨기려면 이미지 편집기 도구 모음  
  
1.  에 **보기** 메뉴 클릭 **도구 모음** 선택 **이미지 편집기**합니다.  
  
    > [!NOTE]
    >  현재 프로젝트에서이 도구 모음에서 요소가 이미지 파일이 사용할 수 없게 표시 되는 또는 솔루션 이미지 편집기에서 열려 있지 않으면입니다. 참조 [아이콘 또는 다른 이미지를 만드는](../windows/creating-an-icon-or-other-image-image-editor-for-icons.md), 프로젝트에 이미지 파일 추가에 대 한 내용은 합니다.  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](/dotnet/framework/resources/index) 에 *.NET Framework 개발자 가이드입니다.* 수동으로 관리 되는 프로젝트에 리소스 파일을 추가, 리소스 액세스, 정적 리소스 표시 및 속성에 리소스 문자열 할당에 대 한 정보를 참조 하십시오. [데스크톱 앱에 대 한 리소스 파일 만들기](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오. [전역화 및 지역화.NET Framework 응용 프로그램](/dotnet/standard/globalization-localization/index)합니다.
