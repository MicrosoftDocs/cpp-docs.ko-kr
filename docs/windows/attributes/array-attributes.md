# <a name="array-attributes"></a>배열 특성

다음 특성은 배열 또는 `SAFEARRAY`s에 적용 됩니다.

|attribute|Description|
|---------------|-----------------|
|[library_block](library-block.md)|.Idl 파일의 라이브러리 블록 안에 구문을 배치 합니다.|
|[satype](satype.md)|`SAFEARRAY` 구조의 데이터 형식을 지정 합니다.|
|[string](string-cpp.md)|1 차원 **char**, **wchar_t**, `byte` 또는 해당 배열에 대 한 포인터가 문자열로 처리 되어야 함을 나타냅니다.|
