# <a name="switch_is"></a>switch_is

Union 멤버를 선택 하는 union 판별 역할을 하는 식 또는 식별자를 지정 합니다.

## <a name="syntax"></a>구문

```cpp
[switch_is]
```

## <a name="remarks"></a>설명

**Switch_is** C++ 특성은 [switch_is](/windows/win32/Midl/switch-is) MIDL 특성과 동일한 기능을 포함 합니다.
