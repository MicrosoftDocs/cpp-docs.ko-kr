# <a name="ole-db-consumer-attributes"></a>OLE DB 소비자 특성

OLE DB 소비자 특성은 [OLE DB 소비자 템플릿을](../../data/oledb/ole-db-consumer-templates-reference.md)기반으로 코드를 삽입하여 테이블 열기, 명령 실행 및 데이터 액세스와 같은 작업을 수행하는 작업 OLE DB 소비자를 만듭니다.

|attribute|Description|
|---------------|-----------------|
|[db_accessor](db-accessor.md)|행 집합에 열을 바인딩하고 해당 접근자 맵에 바인딩합니다.|
|[db_column](db-column.md)|지정된 열을 행 집합에 바인딩합니다.|
|[db_command](db-command.md)|OLE DB 명령을 실행합니다.|
|[db_param](db-param.md)|지정된 멤버 변수를 입력 또는 출력 매개 변수와 연결합니다.|
|[db_source](db-source.md)|공급자를 통해 데이터 원본에 대한 연결을 만들고 캡슐화합니다.|
|[db_table](db-table.md)|OLE DB 테이블을 엽니다.|
