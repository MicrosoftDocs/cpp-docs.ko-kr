# <a name="attributes-by-group"></a>그룹별 특성

C + + 특성은 다음과 같은 기능 그룹으로 구성 됩니다.

|특성|설명|
|---------------|-----------------|
|[COM 특성](com-attributes.md)|COM 개발의 다양 한 영역을 지원 하 고 공용 언어 런타임 개발을 .NET Framework 하는 코드를 삽입 합니다.|
|[IDL 특성](idl-attributes.md)|마법사를 사용 하지 않고 해당 파일의 구조와 구문을 몰라도 소스 코드 파일에서 .idl 파일을 수정할 수 있습니다.|
|[OLE DB 소비자 특성](ole-db-consumer-attributes.md)|[OLE DB 소비자 템플릿을](../../data/oledb/ole-db-consumer-templates-reference.md)기반으로 하는 코드를 삽입 하 여 테이블 열기, 명령 실행, 데이터 액세스 등의 작업을 수행 하는 작업 OLE DB 소비자를 만듭니다.|
|[컴파일러 특성](compiler-attributes.md)|Microsoft c + + 컴파일러에서 제공 됩니다.|

.NET Framework 응용 프로그램에 대 한 사용자 지정 특성을 정의 하는 방법에 대 한 자세한 내용은 [사용자 정의 특성](../../extensions/user-defined-attributes-cpp-component-extensions.md)을 참조 하세요.
