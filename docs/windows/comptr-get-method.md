# <a name="comptrget-method"></a>ComPtr::Get 메서드
이 사용 하 여 연결 된 인터페이스에 대 한 포인터를 검색 **ComPtr**합니다.  
  
## <a name="syntax"></a>구문  
  
```  
T* Get() const;  
```  
  
## <a name="return-value"></a>반환 값  
 이 사용 하 여 연결 된 인터페이스에 대 한 포인터 **ComPtr**합니다.
