# <a name="comptrget-method"></a>ComPtr::Get 메서드
이 ComPtr과 연결된 인터페이스에 대한 포인터를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
T* Get() const;  
```  
  
## <a name="return-value"></a>반환 값  
 이 ComPtr과 연결 된 인터페이스 포인터입니다.
