# <a name="modulegetobjectcount-method"></a>Module::GetObjectCount 메서드

이 모듈에서 관리되는 개체 수를 가져옵니다.

## <a name="syntax"></a>구문

```cpp
virtual long GetObjectCount() const;
```

## <a name="return-value"></a>반환 값

이 모듈에서 관리되는 현재 개체 수입니다.

**헤더:** module.h
