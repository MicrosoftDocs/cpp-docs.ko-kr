# <a name="semaphoretraitsunlock-method"></a>SemaphoreTraits::Unlock 메서드
공유 리소스의 버전 제어 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
inline static void Unlock(  
   _In_ Type h  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `h`  
 Semaphore 개체에 대 한 핸들입니다.
