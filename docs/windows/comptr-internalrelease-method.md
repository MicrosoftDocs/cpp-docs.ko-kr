# <a name="comptrinternalrelease-method"></a>ComPtr::InternalRelease 메서드
연결 된이 인터페이스에서 COM 릴리스 작업을 수행 **ComPtr**합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
void InternalRelease();  
```  
  
## <a name="remarks"></a>설명  
 이 메서드는 보호 됩니다.
