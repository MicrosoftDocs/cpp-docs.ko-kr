# <a name="safeint-library"></a>SafeInt 라이브러리

**SafeInt** 라이브러리는 응용 프로그램이 수학 연산을 수행할 때 발생할 수 있는 정수 오버플로를 방지하는 데 도움이 됩니다. 이 라이브러리의 최신 버전은 [ https://github.com/dcleblanc/SafeInt ](https://github.com/dcleblanc/SafeInt)에 있습니다.
