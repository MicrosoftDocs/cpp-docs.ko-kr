# <a name="maxis"></a>max_is
유효한 배열 인덱스에 대 한 최대값을 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
      [ max_is(  
   "expression"  
) ]  
```  
  
#### <a name="parameters"></a>매개 변수  
 *식*  
 하나 이상의 C 언어 식입니다. 빈 인수 슬롯 허용 됩니다.  
  
## <a name="remarks"></a>설명  
 **max_is** c + + 특성에 동일한 기능을는 [max_is](http://msdn.microsoft.com/library/windows/desktop/aa367074) MIDL 특성입니다.
