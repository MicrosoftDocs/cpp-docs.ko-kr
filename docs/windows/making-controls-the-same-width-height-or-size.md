# <a name="making-controls-the-same-width-height-or-size"></a>컨트롤의 너비, 높이, 크기를 동일하게 만들기
기준 컨트롤의 크기에 따라 컨트롤 그룹의 크기를 조정할 수 있습니다. 수도 있습니다 [각 캡션 텍스트의 크기에 따라 컨트롤의 크기를 조정](../windows/sizing-individual-controls.md)합니다.  
  
### <a name="to-make-controls-the-same-width-height-or-size"></a>동일한 너비, 높이 또는 크기 제어 하려면  
  
1.  [컨트롤 선택](../windows/selecting-multiple-controls.md) 크기를 조정 하려면.  
  
     시리즈에서 먼저 선택 된 컨트롤 기준이 되는 컨트롤입니다. 그룹의 컨트롤의 최종 크기 기준 컨트롤의 크기에 따라 달라 집니다. 기준 컨트롤을 선택에 대 한 자세한 내용은 참조 하십시오. [기준 컨트롤 지정](../windows/specifying-the-dominant-control.md)합니다.  
  
2.  **형식** 메뉴 선택 **같은 크기로**, 다음 명령 중 하나를 선택 합니다.  
  
    -   **둘 다**  
  
    -   **높이**  
  
    -   **너비**
