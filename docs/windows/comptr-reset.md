# <a name="comptrreset"></a>ComPtr::Reset

이 사용 하 여 연결 된 인터페이스 포인터에 대 한 모든 참조를 해제 **ComPtr**합니다.

## <a name="syntax"></a>구문

```cpp
unsigned long Reset();
```

## <a name="return-value"></a>반환 값

해제된 참조 수입니다(있는 경우).

## <a name="requirements"></a>요구 사항

**헤더:** client.h

**네임스페이스:** Microsoft::WRL
