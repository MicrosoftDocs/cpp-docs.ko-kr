# <a name="override-specifiers--ccli-and-ccx"></a>Override 지정자 (C + + /cli 및 C + + /cli CX)

*Override 지정자* 수정 방법을 상속 된 형식 및 상속 된 형식의 멤버가 파생된 형식에서 작동 합니다.

## <a name="all-runtimes"></a>모든 런타임

### <a name="remarks"></a>설명

재정의 지정자에 대한 자세한 내용은 다음을 참조하십시오.

- [abstract](../windows/abstract-cpp-component-extensions.md)

- [new (의 new 슬롯 vtable)](../windows/new-new-slot-in-vtable-cpp-component-extensions.md)

- [override](../windows/override-cpp-component-extensions.md)

- [sealed](../windows/sealed-cpp-component-extensions.md)

- [재정의 지정자 및 네이티브 컴파일](../dotnet/how-to-declare-override-specifiers-in-native-compilations-cpp-cli.md)

**추상** 하 고 **sealed** 유효 형식 선언에는 적용 되지 재정의 지정자로 합니다.

기본 클래스 함수를 명시적으로 재정의 하는 방법에 대 한 내용은 [명시적으로 재정의](../windows/explicit-overrides-cpp-component-extensions.md)합니다.
