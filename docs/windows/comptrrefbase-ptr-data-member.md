# <a name="comptrrefbaseptr-data-member"></a>ComPtrRefBase::ptr_ 데이터 멤버

WRL 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.

## <a name="syntax"></a>구문

```cpp
T* ptr_;
```

## <a name="remarks"></a>설명

현재 템플릿 매개 변수로 지정 된 형식에 대 한 포인터입니다.

## <a name="remarks"></a>설명

**ptr_** 보호 된 데이터 멤버입니다.
