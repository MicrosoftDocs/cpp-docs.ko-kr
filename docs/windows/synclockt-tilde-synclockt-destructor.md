# <a name="synclocktsynclockt-destructor"></a>SyncLockT::~SyncLockT 소멸자
WRL 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.  
  
## <a name="syntax"></a>구문  
  
```

```

 이 소멸자에는 현재 SyncLockT 인스턴스 잠금을 해제합니다.
