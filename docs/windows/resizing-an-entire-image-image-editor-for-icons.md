# <a name="resizing-an-entire-image-image-editor-for-icons"></a>전체 이미지 크기 조정(아이콘에 대한 이미지 편집기)
### <a name="to-resize-an-entire-image-using-the-properties-window"></a>속성 창을 사용 하 여 전체 이미지 크기를 조정 하려면  
  
1.  이미지 변경 하려면 속성을 엽니다.  
  
2.  에 **너비** 및 **높이** 상자에 [속성 창](/visualstudio/ide/reference/properties-window)를 원하는 크기를 입력 합니다.  
  
     이미지의 크기를 늘릴 경우 이미지 편집기를 오른쪽으로 하향 이미지 중 하나 또는 둘 다를 확장 하 고 새 영역을 현재 배경색으로 채웁니다. 이미지 늘어나지 않습니다.  
  
     이미지의 크기를 줄이면 이미지 편집기의 오른쪽 또는 아래쪽 가장자리에 있는 이미지를 자릅니다.  
  
    > [!NOTE]
    >  일부 선택 영역이 하지만 전체 이미지, 크기를 조정할 너비 및 높이 속성을 사용할 수 있습니다.

[데스크톱 앱에 대 한 리소스 파일 만들기](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오.
