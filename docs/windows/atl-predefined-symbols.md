# <a name="atl-predefined-symbols"></a>미리 정의된 ATL 기호

이러한 기호는 ATL 헤더 파일에 정의 되지만 표준 Windows 응용 프로그램 함수 및 작업을 지원 합니다. 이러한 기호는 주로 대화 상자와 함께 사용 됩니다.

[대화 상자 편집기에서 대화](../windows/dialog-editor.md)상자와 컨트롤을 사용 하는 경우 이러한 기호는 공용 컨트롤과 연결 된 [속성 창](/visualstudio/ide/reference/properties-window) 에 나타납니다. 예를 들어 대화 상자에 **취소** 단추가 있는 경우 해당 명령은 **속성** 창에서 IDCANCEL 기호와 연결 됩니다.

|||
|-|-|
|IDABORT|조절 대화 상자, 중단 단추|
|IDC_STATIC|조절 정적 컨트롤|
|IDCANCEL|조절 대화 상자, 취소 단추|
|IDIGNORE|조절 대화 상자, 무시 단추|
|IDNO|조절 대화 상자, 단추 없음|
|IDOK|조절 대화 상자, 확인 단추|
|IDR_ACCELERATOR1|리소스나 액셀러레이터 키 테이블|
|IDRETRY|조절 대화 상자, 다시 시도 단추|
|IDS_PROJNAME|문자열 현재 응용 프로그램 이름|
|IDYES|조절 대화 상자, 예 단추|

## <a name="requirements"></a>요구 사항

ATL
