# <a name="propget"></a>propget
속성 접근자 함수를 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
[propget]  
  
```  
  
## <a name="remarks"></a>설명  
 **propget** c + + 특성에 동일한 기능을는 [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL 특성입니다.  
  
## <a name="example"></a>예  
 예를 참조 [바인딩 가능한](../windows/bindable.md) 의 샘플 사용에 대 한 **propget**합니다.
