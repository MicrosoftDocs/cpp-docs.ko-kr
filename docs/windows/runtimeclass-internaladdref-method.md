# <a name="runtimeclassinternaladdref-method"></a>RuntimeClass::InternalAddRef 메서드

## <a name="syntax"></a>구문  
  
```  
ULONG InternalAddRef();  
```  
  
## <a name="return-value"></a>반환 값  
 결과 참조 횟수입니다.
