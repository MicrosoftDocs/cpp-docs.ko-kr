# <a name="guide-settings-dialog-box-c"></a>안내선 설정 대화 상자 (c + +)

## <a name="layout-guides"></a>레이아웃 안내선

레이아웃 안내선에 대 한 설정을 표시합니다.

### <a name="none"></a>없음

레이아웃 도구를 숨깁니다.

### <a name="rulers-and-guides"></a>눈금자 및 안내선

레이아웃 도구;를 사용 하면 추가 눈금자 눈금자의 안내선을 배치할 수 있습니다. 기본 가이드는 여백, 끌어 이동할 수 있습니다. 눈금자의 안내선을 배치에서 클릭 합니다. 가이드를 통해 또는 옆에 있는 컨트롤을 이동 하는 경우에 "맞춤"를 제어 합니다. 컨트롤에 연결 된 후에 가이드를 사용 하 여 이동 합니다. 컨트롤에는 양쪽에 대 한 가이드에 연결 된 지침을 이동 하 고 컨트롤 크기가 조정 됩니다.

### <a name="grid"></a>표

레이아웃 표를 만듭니다. 새 컨트롤 표에 자동으로 정렬 됩니다.

## <a name="grid-spacing"></a>모눈 간격

대화 상자 단위 (Dlu) 모눈 간격에 대 한 설정을 표시합니다.

### <a name="width-dlus"></a>너비: Dlu

Dlu의 레이아웃 모눈의 너비를 설정합니다. 가로 DLU는 4로 나눈 대화 상자 글꼴의 평균 너비입니다.

### <a name="height-dlus"></a>높이: Dlu

Dlu의 레이아웃 모눈의 높이 설정합니다. 세로 DLU은 평균 8로 나눈 대화 상자 글꼴 높이입니다.
