# <a name="makeallocatormakeallocator-constructor"></a>MakeAllocator::MakeAllocator 생성자
WRL 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.  
  
## <a name="syntax"></a>구문  
  
```  
MakeAllocator();  
```  
  
## <a name="remarks"></a>설명
