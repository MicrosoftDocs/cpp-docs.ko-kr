# <a name="array-attributes"></a>배열 특성
배열에는 다음과 같은 특성이 적용 또는 **SAFEARRAY**s입니다.  
  
|특성|설명|  
|---------------|-----------------|  
|[library_block](../windows/library-block.md)|.Idl 파일의 라이브러리 블록 내부는 구문을 배치합니다.|  
|[satype](../windows/satype.md)|SAFEARRAY 구조체의 데이터 형식을 지정합니다.|  
|[string](../windows/string-cpp.md)|나타냅니다는 1 차원 `char`, `wchar_t`, **바이트** (또는 동일한) 배열 또는 이러한 배열에 대 한 포인터를 문자열로 취급 되어야 합니다.|
