# <a name="handletraitsclose-method"></a>HANDLETraits::Close 메서드

## <a name="syntax"></a>구문  
  
```  
inline static bool Close(  
   _In_ Type h  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `h`  
 닫기에 대 한 핸들입니다.
