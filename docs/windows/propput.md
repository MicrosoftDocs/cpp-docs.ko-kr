# <a name="propput"></a>propput
속성 설정 함수를 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
[propput]  
  
```  
  
## <a name="remarks"></a>설명  
 **propput** c + + 특성에 동일한 기능을는 [propput](http://msdn.microsoft.com/library/windows/desktop/aa367146) MIDL 특성입니다.  
  
## <a name="example"></a>예제  
 예를 참조 [바인딩 가능한](../windows/bindable.md) 의 샘플 사용에 대 한 **propput**합니다.
