# <a name="propput"></a>propput
속성 설정 함수를 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
[propput]  
  
```  
  
## <a name="remarks"></a>설명  
 **propput** c + + 특성에 동일한 기능을는 [propput](http://msdn.microsoft.com/library/windows/desktop/aa367146) MIDL 특성입니다.
