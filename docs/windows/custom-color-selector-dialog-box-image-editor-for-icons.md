# <a name="custom-color-selector-dialog-box-image-editor-for-icons"></a>사용자 지정 색 선택 대화 상자(아이콘에 대한 이미지 편집기)
이 대화 상자에는 이미지에 사용할 색을 사용자 지정할 수 있습니다. 자세한 내용은 참조 [사용자 지정 또는 색 변경](../windows/customizing-or-changing-colors-image-editor-for-icons.md)합니다.  
  
 **그라데이션 색 디스플레이**  
 선택한 색의 값을 변경 합니다. 십자선을 변경 하려면 색에 배치 합니다. 그런 다음 슬라이더를 위나 아래로 이동 변경 명도 또는 색의 RGB 값입니다.  
  
 **명도 막대**  
 그라데이션 색 디스플레이 상자에서 선택한 색의 명도 설정 합니다. 클릭 끌어서 밝기 막대를 흰색 화살표 또는 아래쪽 저렴 한 가격입니다. 색 상자 사용자가 선택한 색을 표시 하 고 명도 영향을 주는 다음 항목 들을 설정할 수 있습니다.  
  
 **색**  
 정의 하는 색의 색상 (색 휠 값)를 나열 합니다. 값 범위는 0부터 240 여기서 0은 빨간색, 60은 노란색, 120은 녹색, 180은 녹청, 200은 자홍, 240는 파란색입니다.  
  
 **색상**  
 정의 하는 색의 색상 (색 휠 값)를 나열 합니다. 값 범위는 0부터 240 여기서 0은 빨간색, 60은 노란색, 120은 녹색, 180은 녹청, 200은 자홍, 240는 파란색입니다.  
  
 **토요일**  
 정의 하는 색의 채도 값을 지정 합니다. 채도 색에 지정한 색상입니다. 값은 0에서 240 까지입니다.  
  
 **명도**  
 정의 하는 색의 명도 (밝기)를 나열 합니다. 값은 0에서 240 까지입니다.  
  
 **빨간색**  
 정의 하는 색의 빨강 값을 지정 합니다. 값 0에서 255 까지입니다.  
  
 **녹색**  
 정의 하는 색의 녹색 값을 지정 합니다. 값 0에서 255 까지입니다.  
  
 **blue**  
 정의 하는 색의 파랑 값을 지정 합니다. 값 0에서 255 까지입니다.
