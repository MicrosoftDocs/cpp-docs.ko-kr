# <a name="colors-window-image-editor-for-icons"></a>색상 창(아이콘에 대한 이미지 편집기)

합니다 **색** 창에 두 부분으로 구성 합니다.

- 합니다 **색상표**를 사용할 수 있는 색을 나타내는 색상 샘플의 배열입니다. 그래픽 도구를 사용 하는 경우 전경색 및 배경색을 선택 하는 샘플을 선택할 수 있습니다.

- 합니다 **색 표시기**, 전경색 및 배경색 및 화면색 및 반전색 선택 기가 보여 줍니다.

   ![색상 창은](../windows/media/vccolorswindow.gif "vcColorsWindow")<br/>
   색상 창

> [!NOTE]
> **색 화면** 하 고 **반전색** 도구 아이콘 및 커서에만 사용할 수 있습니다.

사용할 수는 **색** 창에는 [이미지 편집기 도구 모음](../windows/toolbar-image-editor-for-icons.md)합니다.

## <a name="to-display-the-colors-window"></a>색 창을 표시 하려면

마우스 오른쪽 단추로 클릭는 **이미지 편집기** 창 선택 **색상 창 표시** 바로 가기 메뉴에서.

   \- 또는 -

선택 **색상 창 표시** 에 [이미지 메뉴](../windows/image-menu-image-editor-for-icons.md)합니다.

## <a name="to-hide-the-colors-window"></a>색 창을 숨기려면

창 고정을 해제 합니다. 이 작업을 사용 하 여에 없는 경우 자동 숨기기 창을 통해.

\- 또는 -

선택 된 **닫기** 단추입니다.
