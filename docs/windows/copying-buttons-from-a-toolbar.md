# <a name="copying-buttons-from-a-toolbar"></a>도구 모음에서 단추 복사
### <a name="to-copy-buttons-from-a-toolbar"></a>도구 모음에서 단추 복사 하려면  
  
1.  키를 누른 채는 **CTRL** 키입니다.  
  
2.  에 **도구 모음 보기** 창을 끌어 옵니다 단추 새 위치로 도구 모음에서 위치에 다른 도구 모음입니다.
