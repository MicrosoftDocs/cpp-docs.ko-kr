# <a name="comptrswap-method"></a>ComPtr::Swap 메서드
현재 ComPtr에서 관리하는 인터페이스를 지정된 ComPtr에서 관리하는 인터페이스와 교환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
void Swap(  
   _Inout_ ComPtr&& r  
);  
  
void Swap(  
   _Inout_ ComPtr& r  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `r`  
 ComPtr 합니다.  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** client.h  
  
 **네임스페이스:** Microsoft::WRL
