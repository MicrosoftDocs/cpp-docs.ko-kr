# <a name="text-tool-font-dialog-box-image-editor-for-icons"></a>텍스트 도구 글꼴 대화 상자(아이콘에 대한 이미지 편집기)
글꼴 설정을 사용 하는 경우 지정 된 [텍스트 도구 대화 상자](../windows/text-tool-dialog-box-image-editor-for-icons.md) 의 [이미지 편집기](../windows/image-editor-for-icons.md)합니다.  
  
 이 대화 상자를 사용 합니다.  
  
-   [이미지의 텍스트 글꼴 변경](../windows/changing-the-font-of-text-on-an-image-image-editor-for-icons.md)  
  
 이 대화 상자에 액세스 하려면 클릭는 **글꼴** 텍스트 도구 대화 상자에서 단추입니다.  
  
 **글꼴**  
 사용 가능한 글꼴을 나열합니다.  
  
 **글꼴 스타일**  
 지정된 된 글꼴에 사용할 수 있는 스타일을 나열합니다.  
  
 **Size**  
 지정된 된 글꼴에 사용할 수 있는 포인트 크기를 나열 합니다.  
  
 **샘플**  
 지정한 글꼴 설정을 적용 한 텍스트 예제를 보여 줍니다.  
  
 **스크립트**  
 지정된 된 글꼴에 대 한 사용 가능한 언어 스크립트를 나열합니다. 다른 언어 스크립트를 선택할 때 문자 집합에 대 한 언어 다국어 문서를 만드는 데 사용할 수 있습니다.
