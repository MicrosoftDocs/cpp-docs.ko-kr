# <a name="ciexp"></a>_CIexp

스택 상위 값의 지수를 계산합니다.

## <a name="syntax"></a>구문

```
void __cdecl _CIexp();
```

## <a name="remarks"></a>설명

이 버전의 `exp` 함수는 컴파일러가 이해할 수 있는 특별한 호출 규칙을 가집니다. 복사본이 생성되지 않도록 하며 레지스터 할당을 돕기 때문에 실행 속도를 높입니다.

결과 값이 스택의 맨 위에 푸시됩니다.

## <a name="requirements"></a>요구 사항

**플랫폼:** x86
