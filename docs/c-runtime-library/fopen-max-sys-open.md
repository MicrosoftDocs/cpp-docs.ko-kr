# <a name="fopenmax-sysopen"></a>FOPEN_MAX, _SYS_OPEN

## <a name="syntax"></a>구문

```
#include <stdio.h>
```

## <a name="remarks"></a>주의

이는 동시에 열 수 있는 최대 파일 수입니다. `FOPEN_MAX`는 ANSI와 호환 가능한 이름입니다. `_SYS_OPEN`은 기존 코드와의 호환성을 위해 제공됩니다.
