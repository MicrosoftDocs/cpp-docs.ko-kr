# <a name="environmental-constants"></a>환경 상수

## <a name="syntax"></a>구문

```
#include <stdlib.h>
```

## <a name="remarks"></a>설명

이 상수는 문자열에 대한 환경 길이를 정의합니다.

|상수|의미|
|--------------|-------------|
|`_MAX_ENV`|환경 문자열의 최대 문자열 크기입니다.|
