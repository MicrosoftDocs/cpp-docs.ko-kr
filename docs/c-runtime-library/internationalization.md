# <a name="internationalization"></a>국제화

Microsoft 런타임 라이브러리는 다음과 같은 국제 시장을 위해 앱을 사용자 지정 하는 데 사용할 수 있는 여러 루틴을 제공 합니다.
- [로캘 관련 루틴](../c-runtime-library/locale.md)
- 와이드 문자 루틴
- 멀티 바이트 문자 루틴
- 일반 텍스트 루틴

편의를 위해 대부분의 로캘 관련 루틴은 수행 되는 작업에 따라 분류 됩니다.

멀티 바이트 루틴 및 와이드 문자 루틴은 단일 바이트 문자 (있는 경우)와 함께 설명 됩니다.

ISO646 연산자 대체 항목도 포함 되어 있습니다.
