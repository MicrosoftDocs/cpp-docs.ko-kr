# <a name="rand_max"></a>RAND_MAX

## <a name="syntax"></a>구문

```
#include <stdlib.h>
```

## <a name="remarks"></a>설명

상수 `RAND_MAX`는 `rand` 함수에서 반환할 수 있는 최대값입니다. `RAND_MAX`는 0x7fff 값으로 정의됩니다.
