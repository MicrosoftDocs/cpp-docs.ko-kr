# <a name="csinh-csinhf-csinhl"></a>csinh, csinhf, csinhl

복소수의 쌍곡 사인을 검색합니다.

## <a name="syntax"></a>구문

```C
_Dcomplex csinh(
   _Dcomplex z
);
_Fcomplex csinh(
   _Fcomplex z
);  // C++ only
_Lcomplex csinh(
   _Lcomplex z
);  // C++ only
_Fcomplex csinhf(
   _Fcomplex z
);
_Lcomplex csinhl(
   _Lcomplex z
);
```

### <a name="parameters"></a>매개 변수

*z*<br/>
라디안으로 각도를 나타내는 복소수입니다.

## <a name="return-value"></a>반환 값

하이퍼볼릭 사인 *z*, 라디안에서입니다.

## <a name="remarks"></a>설명

C + +에서는 오버 로드 하므로 오버 로드를 호출할 수 있습니다 **csinh** 및 반환 하는 **_Fcomplex** 하 고 **_Lcomplex** 값입니다. C 프로그램에서 **csinh** 항상 받아서 반환 된 **_Dcomplex** 값입니다.
