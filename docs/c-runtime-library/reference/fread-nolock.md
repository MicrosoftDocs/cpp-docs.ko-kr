# <a name="freadnolock"></a>_fread_nolock

다른 스레드를 잠그지 않고 스트림에서 데이터를 읽습니다.

## <a name="syntax"></a>구문

```C
size_t _fread_nolock(
   void *buffer,
   size_t size,
   size_t count,
   FILE *stream
);
```

### <a name="parameters"></a>매개 변수

*buffer*<br/>
데이터의 저장소 위치입니다.

*size*<br/>
항목 크기(바이트)입니다.

*count*<br/>
읽힐 항목의 최대 수입니다.

*스트림*<br/>
**FILE** 구조체에 대한 포인터입니다.

## <a name="return-value"></a>반환 값

[fread](fread.md)를 참조하세요.

## <a name="remarks"></a>설명

이 함수는 잠기지 않은 버전의 **fread**합니다. 동일 **fread** 다른 스레드의 방해 로부터 보호 되지 않으므로 한다는 점이 다릅니다. 이는 다른 스레드를 잠그는 오버헤드를 유발하지 않으므로 속도가 더 빠를 수 있습니다. 단일 스레드 응용 프로그램과 같은 스레드로부터 안전한 컨텍스트 또는 호출 범위에서 이미 스레드 격리를 처리하는 경우에만 이 함수를 사용합니다.
