# <a name="cprintf"></a>cprintf

이 POSIX 함수는 사용되지 않습니다. 대신 ISO C++와 호환되는 [_cprintf](cprintf-cprintf-l-cwprintf-cwprintf-l.md) 또는 보안이 강화된 [_cprintf_s](cprintf-s-cprintf-s-l-cwprintf-s-cwprintf-s-l.md)를 사용하세요.
