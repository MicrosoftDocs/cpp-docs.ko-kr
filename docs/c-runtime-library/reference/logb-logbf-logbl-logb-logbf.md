# <a name="logb-logbf-logbl-_logb-_logbf"></a>logb, logbf, logbl, _logb, _logbf

부동 소수점 인수의 지수 값을 추출합니다.

## <a name="syntax"></a>구문

```C
double logb(
   double x
);
float logb(
   float x
); // C++ only
long double logb(
   long double x
); // C++ only
float logbf(
   float x
);
long double logbl(
   long double x
);
double _logb(
   double x
);
float _logbf(
   float x
);
```

### <a name="parameters"></a>매개 변수

*x*<br/>
부동 소수점 값입니다.

## <a name="return-value"></a>Return Value

**logb는** 부동 소수점 값으로 표시되는 서명된 정수로 *x의* 편향되지 않은 지수 값을 반환합니다.

## <a name="remarks"></a>설명

**logb** 함수는 *x가* 무한 범위로 표시된 것처럼 부동 소수점 인수 *x의*지수 값을 추출합니다. 인수 *x가* 정규화되지 않으면 정규화된 것처럼 처리됩니다.

C++는 오버로드를 허용하므로 **플로트** 또는 **긴** **이중** 값을 가져가서 반환하는 **로그** 오버로드를 호출할 수 있습니다. C 프로그램에서 **logb는** 항상 **double을**가져와 반환합니다.

|입력|SEH 예외|Matherr 예외|
|-----------|-------------------|-----------------------|
|± QNAN,IND|없음|_DOMAIN|
|± 0|ZERODIVIDE|_SING|

기본적으로 이 함수의 전역 상태는 응용 프로그램에 대한 범위가 조정됩니다. 이를 변경하려면 [CRT의 전역 상태를](../global-state.md)참조하십시오.
