# <a name="_strupr_s-_strupr_s_l-_mbsupr_s-_mbsupr_s_l-_wcsupr_s-_wcsupr_s_l"></a>_strupr_s, _strupr_s_l, _mbsupr_s, _mbsupr_s_l, _wcsupr_s, _wcsupr_s_l

현재 로캘 또는 전달된 지정한 로캘을 사용하여 문자열을 대문자로 변환합니다. 이러한 버전의 [_strupr, _strupr_l, _mbsupr, _mbsupr_l, _wcsupr_l, _wcsupr](strupr-strupr-l-mbsupr-mbsupr-l-wcsupr-l-wcsupr.md)에는 [CRT의 보안 기능](../../c-runtime-library/security-features-in-the-crt.md)에 설명된 대로 향상된 보안 기능이 포함되어 있습니다.

> [!IMPORTANT]
> **_mbsupr_s** 및 **_mbsupr_s_l** Windows 런타임에서 실행되는 응용 프로그램에서 사용할 수 없습니다. 자세한 내용은 [유니버설 Windows 플랫폼 앱에서 지원되지 않는 CRT 함수](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md)를 참조하세요.

```C
errno_t _strupr_s(

);

   size_t numberOfElements,
   _locale_t locale

errno_t _strupr_s(
   char (&str)[size]

errno_t _wcsupr_s(
```

### <a name="parameters"></a>매개 변수

*Str*<br/>
대문자로 변환할 문자열입니다.

*숫자오브엘리먼트*<br/>
버퍼의 크기입니다.

*로캘*<br/>
사용할 로캘입니다.

## <a name="return-value"></a>Return Value

정상적으로 실행되는 경우 0이고 오류가 발생하는 경우 0이 아닌 오류 코드입니다.

이러한 함수는 해당 함수 매개 변수의 유효성을 검사합니다. *str이* **NULL** 포인터인 경우 [매개 변수 유효성 검사에](../../c-runtime-library/parameter-validation.md) 설명된 대로 잘못된 매개 변수 처리기가 호출됩니다. 실행을 계속할 수 있는 경우 함수는 **EINVAL을** 반환하고 **errno를** **EINVAL로**설정합니다. *numberOfElements가* 문자열의 길이보다 적으면 함수는 **ERANGE를** 반환하고 **errno를** **ERANGE로**설정합니다.

## <a name="remarks"></a>설명

**_strupr_s** 함수는 *str의* 각 소문자가 대문자로 변환됩니다. **_wcsupr_s** **_strupr_s**와이드 문자 버전입니다. **_mbsupr_s** **_strupr_s**다바이트 문자 버전입니다.

변환은 로캘의 **LC_CTYPE** 범주 설정에 의해 결정됩니다. 다른 문자는 영향을 받지 않습니다. **LC_CTYPE**대한 자세한 내용은 [setlocale](setlocale-wsetlocale.md)를 참조하십시오. **_l** 접미사가 없는 이러한 함수의 버전은 현재 로캘을 사용합니다. **_l** 접미사가 있는 비전은 대신 전달된 로캘을 사용한다는 점을 제외하고는 동일합니다. 자세한 내용은 [Locale](../../c-runtime-library/locale.md)을 참조하세요.

C++에서는 템플릿 오버로드로 인해 이러한 함수를 사용하는 것이 보다 간단해 집니다. 오버로드는 버퍼 길이를 자동으로 유추할 수 있으며(크기 인수를 지정할 필요가 없어짐), 기존의 비보안 함수를 보다 최신의 보안 대응 함수로 자동으로 바꿀 수 있습니다. 자세한 내용은 [안전한 템플릿 오버로드](../../c-runtime-library/secure-template-overloads.md)를 참조하세요.

이러한 함수의 디버그 라이브러리 버전은 먼저 버퍼를 0xFE로 채웁니다. 이 동작을 사용하지 않으려면 [_CrtSetDebugFillThreshold](crtsetdebugfillthreshold.md)를 사용하세요.

기본적으로 이 함수의 전역 상태는 응용 프로그램에 대한 범위가 조정됩니다. 이를 변경하려면 [CRT의 전역 상태를](../global-state.md)참조하십시오.
