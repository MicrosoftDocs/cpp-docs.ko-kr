# <a name="getcnolock-getwcnolock"></a>_getc_nolock, _getwc_nolock

스트림에서 문자를 읽습니다.

## <a name="syntax"></a>구문

```C
int _getc_nolock(
   FILE *stream
);
wint_t _getwc_nolock(
   FILE *stream
);
```

### <a name="parameters"></a>매개 변수

*스트림*<br/>
입력 스트림입니다.

## <a name="return-value"></a>반환 값

[getc, getwc](getc-getwc.md)를 참조하세요.

## <a name="remarks"></a>설명

이러한 함수는 동일 **getc** 하 고 **getwc** 는 호출 스레드를 잠그지 않는다는 점을 제외 하 고 있습니다. 이들은 다른 스레드를 잠그는 오버헤드를 유발하지 않으므로 속도가 더 빠를 수 있습니다. 단일 스레드 응용 프로그램과 같은 스레드로부터 안전한 컨텍스트 또는 이미 스레드 격리를 처리한 호출 범위에서만 이러한 함수를 사용합니다.

## <a name="example"></a>예제

```C
// crt_getc_nolock.c
// Use getc to read a line from a file.

#include <stdio.h>

int main()
{
    char buffer[81];
    int i, ch;
    FILE* fp;

    // Read a single line from the file "crt_getc_nolock.txt".
    fopen_s(&fp, "crt_getc_nolock.txt", "r");
    if (!fp)
    {
       printf("Failed to open file crt_getc_nolock.txt.\n");
       exit(1);
    }

    for (i = 0; (i < 80) && ((ch = getc(fp)) != EOF)
                         && (ch != '\n'); i++)
    {
        buffer[i] = (char) ch;
    }

    // Terminate string with a null character
    buffer[i] = '\0';
    printf( "Input was: %s\n", buffer);

    fclose(fp);
}
```

### <a name="input-crtgetcnolocktxt"></a>입력: crt_getc_nolock.txt

```Input
Line the first.
Line the second.
```

### <a name="output"></a>출력

```Output
Input was: Line the first.
```
