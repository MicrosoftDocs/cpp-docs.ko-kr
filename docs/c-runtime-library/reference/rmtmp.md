# <a name="rmtmp"></a>_rmtmp
임시 파일을 제거합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
int _rmtmp( void );  
```  
  
## <a name="return-value"></a>반환 값  
 `_rmtmp`는 닫히고 삭제된 임시 파일의 수를 반환합니다.  
  
## <a name="remarks"></a>설명  
 `_rmtmp` 함수는 현재 디렉터리의 모든 임시 파일을 정리합니다. 이 함수는 `tmpfile`로 생성된 파일만 제거합니다. 임시 파일이 생성된 디렉터리에서만 사용하십시오.
