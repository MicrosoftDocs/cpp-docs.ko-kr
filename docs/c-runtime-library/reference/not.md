# <a name="not"></a>not

!에 대한 대안 연산자와 함께 사용되었습니다.

## <a name="syntax"></a>구문

```C

#define not !

```

## <a name="remarks"></a>설명

매크로가 ! 연산자를 생성합니다.

## <a name="example"></a>예제

```cpp
// iso646_not.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 0;

   if (!a)
      cout << "a is zero" << endl;

   if (not(a))
      cout << "a is zero" << endl;
}
```

```Output
a is zero
a is zero
```
