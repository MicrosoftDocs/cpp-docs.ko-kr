# <a name="windows-platforms-crt"></a>Windows 플랫폼(CRT)

Visual Studio용 C 런타임 라이브러리는 최신 버전의 Windows와 Windows Server, Windows 8, Windows Server 2012, Windows 7, Windows Server 2008 및 Windows Vista를 지원하며 32비트(x86)용 Windows XP SP3(서비스 팩 3), 64비트(x64)용 Windows XP SP2(서비스 팩 2)와 32비트(x86)/64비트(x64)용 Windows Server 2003 SP2(서비스 팩 2)를 선택적으로 지원합니다. 이 모든 운영 체제는 Windows 데스크톱 API(Win32)를 지원하며 유니코드를 사용할 수 있습니다. 또한 모든 Win32 애플리케이션에서 MBCS(멀티바이트 문자 집합)를 사용할 수 있습니다.

> [!NOTE]
> Visual Studio 2017에서 **C++를 사용한 데스크톱 개발** 워크로드의 기본 설치에는Windows XP 및 Windows Server 2003 개발에 대한 지원이 포함되지 않습니다. Windows XP 플랫폼 도구 세트를 사용하려면 선택적 구성 요소인 **C++용 Windows XP 지원**을 설치해야 합니다.
