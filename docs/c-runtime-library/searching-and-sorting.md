# <a name="searching-and-sorting"></a>검색 및 정렬

검색 및 정렬을 위해 다음 함수를 사용합니다.

## <a name="searching-and-sorting-functions"></a>검색 및 정렬 함수

|함수|검색 또는 정렬|
|--------------|--------------------|
|[bsearch](../c-runtime-library/reference/bsearch.md)|이진 검색|
|[bsearch_s](../c-runtime-library/reference/bsearch-s.md)|**bsearch**의 안전한 버전입니다.|
|[_lfind](../c-runtime-library/reference/lfind.md)|지정된 값에 대한 선형 검색|
|[_lfind_s](../c-runtime-library/reference/lfind-s.md)|**_lfind**의 안전한 버전입니다.|
|[_lsearch](../c-runtime-library/reference/lsearch.md)|지정된 값에 대한 선형 검색. 찾을 수 없으면 배열에 추가됩니다.|
|[_lsearch_s](../c-runtime-library/reference/lsearch-s.md)|**_lsearch**의 안전한 버전입니다.|
|[qsort](../c-runtime-library/reference/qsort.md)|빠른 정렬|
|[qsort_s](../c-runtime-library/reference/qsort-s.md)|**qsort**의 안전한 버전입니다.|
