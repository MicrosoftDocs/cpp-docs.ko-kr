# <a name="string-manipulation-crt"></a>문자열 조작(CRT)

이러한 루틴은 null로 종료되는 싱글바이트 문자, 와이드 문자 및 멀티바이트 문자 문자열에서 작동합니다. [버퍼 조작](../c-runtime-library/buffer-manipulation.md)에 설명된 버퍼 조작 루틴을 사용하여 null 문자로 끝나지 않는 문자 배열을 사용합니다.
