# <a name="_debug"></a>_DEBUG

/MTd 또는 /MDd를 지정하면 컴파일러는 `_DEBUG`를 정의합니다. 이러한 옵션은 C 런타임 라이브러리 디버그 버전을 지정합니다.

자세한 내용은 [CRT 디버깅 기술](/visualstudio/debugger/crt-debugging-techniques)을 참조하세요.
