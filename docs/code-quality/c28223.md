# <a name="c28223"></a>C28223

> 경고 C28223: 주석, 매개 변수에 필요한 토큰/식별자를 찾지 못했습니다.

이 경고는 주석에 대 한 매개 변수가 식별자 여야 하 고 다른 기호가 발견 되었음을 나타냅니다. 일반적으로 이 경고는 잘못 코딩된 주석 매크로를 나타냅니다. 이 위치에서 C 또는 c + + 키워드를 사용 하면이 오류가 발생 합니다.
