# <a name="c28232"></a>C28232

> 경고 C28232: \_ Pre \_ , \_ Post \_ 또는 \_ Deref가 \_ 주석에 적용 되지 않았습니다.

이 경고는 `_Pre_`, `_Post_` 또는 `_Deref_` 연산자가 주석 식에 있지만 후속 기능 주석이 없음을 의미합니다. 수정자가 무시되었지만 이는 잘못 코딩된 주석을 나타냅니다.
