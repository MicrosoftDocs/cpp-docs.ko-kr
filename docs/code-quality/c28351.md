# <a name="c28351"></a>C28351

> 경고 C28351: 주석은 \<annotation> 동적 값 (변수)을 조건에 사용할 수 없는 위치를 설명 합니다.

이 경고는 C/C++ 형식을 검사하는 위치에 주석이 적용될 때 보고됩니다.

`_When_`의 식은 상수로 평가됩니다. 는 `_When_` 무시 됩니다.
