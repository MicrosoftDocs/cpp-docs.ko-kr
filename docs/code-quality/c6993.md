# <a name="c6993"></a>C6993

> 경고 C6993: 코드 분석에서 OpenMP 구문을 무시 합니다. 단일 스레드 코드 분석

이 경고는 코드 분석기에서 분석할 수 없는 OpenMP pragma가 발생한 것을 나타냅니다.
