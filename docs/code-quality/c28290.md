# <a name="c28290"></a>C28290

> 경고 C28290: 함수에 대 한 주석에 실제 매개 변수 개수 보다 많은 외부 참조가 있습니다.

코드 분석 도구는 이 함수에 대한 주석에 실제 매개 변수 수보다 더 많은 외부 참조가 포함된 경우 이 경고를 보고합니다.
