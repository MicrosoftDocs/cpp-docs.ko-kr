# <a name="c26800"></a>C26800

> 경고 C26800: 개체에서 이동 된 \<잠금 >를 사용 합니다.

경고 C26800는에서 이동 된 후 변수를 사용할 때 트리거됩니다. 변수가 rvalue 참조로 함수에 전달 된 후에서 이동 된 것으로 간주 됩니다. 할당, 소멸 및 std:: vector:: clear와 같은 일부 상태 재설정 함수와 같은 사용에 대 한 몇 가지 합법적인 예외가 있습니다.

## <a name="example"></a>예제

다음 코드는 C26800을 생성 합니다.

```cpp

#include <utility>

struct X {
    X();
    X(const X&);
    X(X&&);
    X &operator=(X&);
    X &operator=(X&&);
    ~X();
};

template<typename T>
void use_cref(const T&);

void test() {
    X x1;
    X x2 = std::move(x1);
    use_cref(x1);                // @expected(26800)
}
```

## <a name="example"></a>예제

다음 코드는 C26800을 생성 하지 않습니다.

```cpp

#include <utility>

struct MoveOnly {
    MoveOnly();
    MoveOnly(MoveOnly&) = delete;
    MoveOnly(MoveOnly&&);
    MoveOnly &operator=(MoveOnly&) = delete;
    MoveOnly &operator=(MoveOnly&&);
    ~MoveOnly();
};

template<typename T>
void use(T);

void test() {
    MoveOnly x;
    use(std::move(x)); // no 26800
}
```
