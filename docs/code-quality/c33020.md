# <a name="c33020"></a>C33020

> 경고 C33020: 잘못 된 HRESULT 사용이 검색 되었을 가능성이 높습니다.

이는 HRESULT 반환 함수가 FALSE 또는 false를 반환 한다는 것을 나타내는 높은 신뢰 경고입니다.

## <a name="example"></a>예제

```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return FALSE; // C33020
}
```

이러한 경고는 적절 한 HRESULT 값을 사용 하 여 수정 됩니다.
```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return E_FAIL; // OK
}
```
