# <a name="c26498-use_constexpr_for_functioncall"></a>C26498 USE_CONSTEXPR_FOR_FUNCTIONCALL

컴파일 시간 계산이 필요한 경우이 함수 호출% function%에서 `constexpr`를 사용할 수 있습니다. [ C++ 핵심 지침 con. 5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-constexpr)를 참조 하십시오.
