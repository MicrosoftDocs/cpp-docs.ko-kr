# <a name="c28211"></a>C28211

> 경고 C28211: SAL_context에 대 한 정적 컨텍스트 이름이 필요 합니다.

이 경고는 `_Static_context_` 주석의 피연산자가 도구 정의 컨텍스트의 이름 이어야 함을 나타냅니다. 일반적인 사용의 경우 이 경고가 발생하지 않습니다.
