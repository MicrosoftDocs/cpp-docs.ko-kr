# <a name="c33001"></a>C33001

> 경고 C33001: 변형 ' v a r '가 초기화 되지 않은 경우 (' expr ' 식) 삭제 되었습니다.

초기화 된 변형이 필요한 VariantClear와 같은 API에 초기화 되지 않은 변형이 전달 되는 경우이 경고가 트리거됩니다.

## <a name="example"></a>예제

```cpp
#include <Windows.h>

HRESULT foo(bool some_condition)
{
    VARIANT var;

    if (some_condition)
    {
        //...
        VariantInit(&var);
        //...
    }

    VariantClear(&var);     // C33001
}
```

이러한 경고는 올바르게 초기화 된 변형에 대해서만 VariantClear가 호출 되도록 하 여 수정 됩니다.
```cpp
#include <Windows.h>

HRESULT foo(bool some_condition)
{
    VARIANT var;

    if (some_condition)
    {
        //...
        VariantInit(&var);
        //...
        VariantClear(&var);     // C33001
    }
}
```
