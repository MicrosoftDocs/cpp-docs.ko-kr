# <a name="c26461-use_const_pointer_arguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:

'% Function% ' 함수의 포인터 인수 '% argument% '은 (는) `const`에 대 한 포인터로 표시 될 수 있습니다. [ C++ 핵심 지침 con. 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref)을 참조 하세요.
