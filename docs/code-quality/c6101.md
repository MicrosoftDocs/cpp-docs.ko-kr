# <a name="c6101"></a>C6101

> 경고 C6101: 초기화 되지 않은 메모리 반환

함수를 통한 성공적인 경로는 명명된 `_Out_` 매개 변수를 설정하지 않습니다. 이 메시지는 해당 함수가 항상 성공함을 나타내는 SAL 주석을 기반으로 생성됩니다. `_Out_`분석기는 `_Out_` 함수가 호출 되기 전에 매개 변수가 초기화 되지 않은 데이터 라고 가정 하 고 함수가 매개 변수를 더 이상 초기화 되지 않도록 설정 하므로 성공/실패 표시를 반환 하지 않는 함수는 모든 매개 변수를 설정 해야 합니다. 함수가 성공/실패를 나타내는 경우는 실패의 경우 `_Out_` 매개 변수를 설정할 필요가 없고 초기화되지 않은 위치를 찾아서 방지할 수 있습니다. 어느 경우에도 목적은 초기화되지 않은 위치를 읽지 않도록 방지하는 것입니다. 함수에서 나중에 사용 되는 매개 변수를 건드리지 않는 경우 `_Out_` 함수를 호출 하기 전에 매개 변수를 초기화 하 고 주석으로 표시 `_Inout_` 하거나 `_Pre_null_` 적절 한 경우 보다 명시적으로 또는를 사용 해야 합니다 `_Pre_satisfies_()` . "부분 성공"은 `_When_` 주석으로 처리할 수 있습니다. 자세한 내용은 [SAL 주석을 사용 하 여 c/c + + 코드 오류 줄이기](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)를 참조 하세요.
