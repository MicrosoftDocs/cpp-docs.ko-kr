# <a name="c28215"></a>C28215

> 경고 C28215: typefix는 이미 typefix가 있는 매개 변수에 적용할 수 없습니다.

`__typefix` 주석을 해당 주석이 이미 있는 매개 변수에 적용하면 오류가 발생합니다. `__typefix` 주석은 일부 특수한 경우에서만 사용되며, 일반적인 사용 시에는 이 경고가 표시되지 않습니다.
