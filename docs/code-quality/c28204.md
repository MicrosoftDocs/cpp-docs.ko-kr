# <a name="c28204"></a>C28204

> 경고 C28204: \<함수 >:이 오버 로드와 \<파일 이름 > (\<줄 >)에 있는 하나는 \<paramname >에 대해 주석이 지정 됩니다. 또는 모두에 주석을 추가 해야 합니다.

이 경고는 주석에 오류가 있는 경우 보고됩니다.
