# <a name="c28216"></a>C28216

> 경고 C28216: \_검사\_반환\_ 주석은 특정 함수 매개 변수에 대 한 사후 조건에만 적용 됩니다.

`_Check_return_` 주석이 `__post` 이외의 컨텍스트에 적용되었습니다. `__post`(또는 `__drv_out`) 한정자가 필요하거나 잘못 배치되었을 수 있습니다.
