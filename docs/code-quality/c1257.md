# <a name="c1257"></a>C1257

> 경고 C1257: 플러그 인 초기화에 실패 했습니다.

코드 분석 도구는 분석 중인 코드가 아니라 플러그 인에 내부 오류가 있을 때 이 경고를 보고합니다.
