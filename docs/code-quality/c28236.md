# <a name="c28236"></a>C28236

> 경고 C28236: 주석을 식에 사용할 수 없습니다.

이 경고는 식 컨텍스트에서 주석으로 선언 된 함수를 사용 하려고 했음을 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
