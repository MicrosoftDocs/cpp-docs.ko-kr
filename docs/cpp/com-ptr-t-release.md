# <a name="comptrtrelease"></a>_com_ptr_t::Release
**Microsoft 전용**  
  
 호출의 **릴리스** 의 멤버 함수 **IUnknown** 캡슐화 된 인터페이스 포인터입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
void Release( );  
  
```  
  
## <a name="remarks"></a>설명  
 호출 `IUnknown::Release` 캡슐화 된 인터페이스 포인터를 발생 시키는 `E_POINTER` 이 인터페이스 포인터가 있으면 오류 **NULL**합니다.
