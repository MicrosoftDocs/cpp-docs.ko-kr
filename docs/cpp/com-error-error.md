# <a name="comerrorerror"></a>_com_error::Error
**Microsoft 전용**  
  
 생성자에 전달된 `HRESULT`를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
HRESULT Error( ) const throw( );  
  
```  
  
## <a name="return-value"></a>반환 값  
 생성자에 전달된 원시 `HRESULT` 항목입니다.  
  
## <a name="remarks"></a>설명  
 `HRESULT` 개체에서 캡슐화된 `_com_error` 항목을 검색합니다.
