# <a name="bstrt-relational-operators"></a>_bstr_t 관계형 연산자
**Microsoft 전용**  
  
 두 `_bstr_t` 개체를 비교합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
      bool operator!( ) const throw( );   
bool operator==(  
   const _bstr_t& str   
) const throw( );  
bool operator!=(  
   const _bstr_t& str   
) const throw( );  
bool operator<(  
   const _bstr_t& str   
) const throw( );  
bool operator>(  
   const _bstr_t& str   
) const throw( );  
bool operator<=(  
   const _bstr_t& str   
) const throw( );  
bool operator>=(  
   const _bstr_t& str   
) const throw( );  
```  
  
## <a name="remarks"></a>설명  
 이러한 연산자는 두 `_bstr_t` 개체를 사전순으로 비교합니다. 할당 연산자는 반환 **true** 는 비교가 유효 하면 그렇지 않으면 반환 **false**합니다.
