**Microsoft 전용**  
  
 확인 캡슐화 된 `BSTR` NULL 문자열입니다.  
  
## <a name="syntax"></a>구문  
  
```  
bool operator!( ) const throw( );  
```

 그러한 경우 TRUE를 반환 합니다이 고, 그렇지 않으면 FALSE입니다.
