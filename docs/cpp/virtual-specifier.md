# <a name="virtual-specifier"></a>virtual 지정자

[Virtual](../cpp/virtual-cpp.md) 키워드는 비정적 클래스 멤버 함수에만 적용할 수 있습니다. 이는 함수에 대한 호출 바인딩이 런타임까지 지연됨을 의미합니다. 자세한 내용은 [가상 함수](../cpp/virtual-functions.md)를 참조 하세요.
