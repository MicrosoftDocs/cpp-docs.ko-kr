# `_bstr_t::length`

**Microsoft 전용**

종결 null이 포함되지 않는 캡슐화된 `_bstr_t`의 `BSTR`에서 문자 수가 반환됩니다.

## <a name="syntax"></a>구문

```cpp
unsigned int length ( ) const throw( );
```
