# <a name="using-abort"></a>abort 사용
호출 된 [중단](../c-runtime-library/reference/abort.md) 함수 때문에 즉시 종료 합니다. 이 함수는 초기화된 전역 정적 개체에 대한 일반적인 소멸 프로세스를 건너뜁니다. 또한 이 함수는 `atexit` 함수를 사용하여 지정된 모든 특수 처리를 건너뜁니다.
