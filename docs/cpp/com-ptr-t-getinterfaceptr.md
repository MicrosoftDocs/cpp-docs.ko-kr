# <a name="comptrtgetinterfaceptr"></a>_com_ptr_t::GetInterfacePtr
**Microsoft 전용**  
  
 캡슐화된 인터페이스 포인터가 반환됩니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Interface* GetInterfacePtr( ) const throw( );   
Interface*& GetInterfacePtr() throw();  
```  
  
## <a name="remarks"></a>설명  
 NULL 일 수 있는 캡슐화 된 인터페이스 포인터를 반환 합니다.
