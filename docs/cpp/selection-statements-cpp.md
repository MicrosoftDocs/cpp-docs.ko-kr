# <a name="selection-statements-c"></a>선택문 (C++)

If C++ 및 [switch](../cpp/switch-statement-cpp.md)의 [if](../cpp/if-else-statement-cpp.md) 선택 문은 조건에 따라 코드 섹션을 조건부로 실행 하는 방법을 제공 합니다.

[__If_exists](../cpp/if-exists-statement.md) 및 [__if_not_exists](../cpp/if-not-exists-statement.md) 문을 사용 하 여 기호 존재 여부에 따라 코드를 조건부로 포함할 수 있습니다.

각 문에 대한 개별 구문 항목을 참조하십시오.
