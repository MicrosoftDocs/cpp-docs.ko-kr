와 [atexit](../c-runtime-library/reference/atexit.md) 함수 프로그램 종료 전에 실행 하는 종료 처리 함수를 지정할 수 있습니다. `atexit`를 호출하기 전에 초기화된 전역 정적 개체는 종료 처리 함수의 실행 전에 소멸되지 않습니다.
