# <a name="bstrtdetach"></a>_bstr_t::Detach
**Microsoft 전용**  
  
 `BSTR`로 래핑된 `_bstr_t`을 반환하고 `BSTR`을 `_bstr_t`에서 분리합니다.  
  
## <a name="syntax"></a>구문  
  
```  
BSTR Detach( ) throw;  
```  
  
## <a name="return-value"></a>반환 값  
 `BSTR`로 래핑된 `_bstr_t`입니다.  
  
## <a name="example"></a>예  
 참조 [_bstr_t:: assign](../cpp/bstr-t-assign.md) 사용 하는 예제 **분리**합니다.
