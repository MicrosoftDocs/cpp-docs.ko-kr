# <a name="_bstr_tdetach"></a>_bstr_t::Detach

**Microsoft 전용**

`BSTR`로 래핑된 `_bstr_t`을 반환하고 `BSTR`을 `_bstr_t`에서 분리합니다.

## <a name="syntax"></a>구문

```
BSTR Detach( ) throw;
```

## <a name="return-value"></a>Return Value

`BSTR`로 래핑된 `_bstr_t`입니다.

## <a name="example"></a>예제

**Detach** 를 사용 하는 예제는 [_Bstr_t:: Assign](../cpp/bstr-t-assign.md) 을 참조 하십시오.
