# <a name="initializer-lists"></a>이니셜라이저 목록
이니셜라이저 목록 생성자에서 기본 클래스 생성자 전에 이제 호출 됩니다.  
  
## <a name="remarks"></a>설명  
 Visual c + + 2005 이전 Managed extensions for c + +를 컴파일할 경우에 기본 클래스 생성자 이니셜라이저 목록 보다 먼저 호출 되었습니다. 이제로 컴파일할 때 **/clr**, 이니셜라이저 목록은 먼저 호출 됩니다.
