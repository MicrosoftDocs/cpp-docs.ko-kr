# <a name="initializer-lists"></a>이니셜라이저 목록

이니셜라이저 목록 생성자에서 기본 클래스 생성자 보다 먼저 이제 라고 합니다.

## <a name="remarks"></a>설명

Visual c + + 2005 이전 c + +에 대 한 Managed Extensions를 사용 하 여 컴파일하면 기본 클래스 생성자 이니셜라이저 목록 보다 먼저 호출 되었습니다. 이제 사용 하 여 컴파일하면 **/clr**, 이니셜라이저 목록에서 먼저 호출 됩니다.
