# <a name="initonly-ccli"></a>initonly(C++/CLI)
**initonly** 해당 변수 할당을 나타내는 상황에 맞는 키워드는 선언 또는 동일한 클래스의 정적 생성자의 일부로 서만 발생할 수 있습니다.  
  
 다음 예제에서는 `initionly`을 사용하는 방법을 보여 줍니다.  
  
```  
// mcpp_initonly.cpp  
// compile with: /clr /c  
ref struct Y1 {  
   initonly  
   static int staticConst1;  
  
   initonly  
   static int staticConst2 = 0;  
  
   static Y1() {  
      staticConst1 = 0;  
   }  
};  
```
