# <a name="adapter-stlclr"></a>adapter(STL/CLR)
STL/CLR 헤더 `<cliext/adapter>` 두 템플릿 클래스 지정 (`collection_adapter` 및 `range_adapter`), 및 템플릿 함수 `make_collection`합니다.  
  
## <a name="syntax"></a>구문  
  
```  
#include <cliext/adapter>  
```  
  
## <a name="remarks"></a>설명  
  
|클래스|설명|  
|-----------|-----------------|  
|[collection_adapter(STL/CLR)](../dotnet/collection-adapter-stl-clr.md)|클래스 라이브러리 BCL (기본) 컬렉션으로를 범위를 래핑합니다.|  
|[range_adapter(STL/CLR)](../dotnet/range-adapter-stl-clr.md)|BCL 컬렉션으로 범위를 래핑합니다.|  
  
|함수|설명|  
|--------------|-----------------|  
|[make_collection(STL/CLR)](../dotnet/make-collection-stl-clr.md)|반복기 쌍을 사용 하 여 범위 어댑터를 만듭니다.|  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** \<cliext/어댑터 >  
  
 **Namespace:** cliext
