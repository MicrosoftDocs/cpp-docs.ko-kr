# <a name="comptr"></a>com::ptr

CLR 클래스의 멤버로 사용할 수 있는 COM 개체에 대한 래퍼입니다. 또한 래퍼는 COM 개체의 수명 관리를 자동화 하 여 소멸자가 호출 될 때 개체에 대 한 소유 된 참조를 해제 합니다. [CComPtr 클래스](../atl/reference/ccomptr-class.md)와 유사 합니다.

## <a name="syntax"></a>구문

```
#include <msclr\com\ptr.h>
```

## <a name="remarks"></a>주의

[com::p Tr 클래스](../dotnet/com-ptr-class.md) 는 \<msclr\com\ptr.h > 파일에 정의 되어 있습니다.
