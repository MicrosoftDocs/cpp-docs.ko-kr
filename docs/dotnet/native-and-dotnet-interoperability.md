# <a name="native-and-net-interoperability"></a>네이티브 및 .NET 상호 운용성

Visual C++는 동일한 어셈블리 내에서 관리 되는 구문 및 관리 되지 않는 구문이 공존 하 고 상호 운용할 수 있도록 하는 상호 운용성 기능을 지원 합니다. P/Invoke와 같은이 기능 중 일부는 다른 .NET 언어 에서도 지원 되지만 Visual C++에서 제공 하는 대부분의 상호 운용성 지원은 다른 언어에서 사용할 수 없습니다.
