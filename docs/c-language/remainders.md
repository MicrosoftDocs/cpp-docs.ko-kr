# <a name="remainders"></a>나머지

**ANSI 3.3.5** 정수 나누기에 대한 나머지의 부호

나머지의 부호는 피제수의 부호와 동일합니다. 예를 들어 개체에 적용된

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
