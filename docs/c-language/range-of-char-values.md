# <a name="range-of-char-values"></a>char 값의 범위

**ANSI 3.2.1.1** "일반" **char가** **서명된 char** 또는 **서명되지 않은 char와** 동일한 값 범위를 가지고 있는지 여부

모든 부호 있는 문자 값의 범위는 -128~127입니다. 모든 부호 없는 문자 값의 범위는 0~255입니다.

[`/J`](../build/reference/j-default-char-type-is-unsigned.md) 컴파일러 옵션은 **char의** 기본 형식을 **서명된 char에서** **서명되지 않은 char로**변경합니다.
