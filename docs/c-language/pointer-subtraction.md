# <a name="pointer-subtraction"></a>포인터 빼기
**ANSI 3.3.6, 4.1.1** 같은 배열 **ptrdiff_t**의 요소에 대한 두 포인터 사이의 차이를 수용하는 데 필요한 정수 형식  
  
 32비트 x86 플랫폼에서 `ptrdiff_t` 형식 정의는 `int`입니다. 64비트 플랫폼에서 `ptrdiff_t` 형식 정의는 `__int64`입니다.
