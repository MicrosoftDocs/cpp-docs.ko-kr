# <a name="storage-of-arrays"></a>배열 스토리지

배열 형식과 관련된 스토리지는 배열의 모든 요소에 필요한 스토리지입니다. 배열의 요소는 첫 번째 요소부터 마지막 요소까지 연속적으로 증가하는 메모리 위치에 저장됩니다.
