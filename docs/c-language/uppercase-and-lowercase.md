# <a name="uppercase-and-lowercase"></a>대문자와 소문자

**ANSI 3.1.2** 대/소문자 구분이 의미가 있는지 여부

Microsoft C는 컴파일 단위 내 식별자를 대/소문자를 구분하는 것으로 간주합니다.

Microsoft 링커는 대/소문자를 구분합니다. 대/소문자에 따라 일관성 있게 모든 식별자를 지정해야 합니다.
