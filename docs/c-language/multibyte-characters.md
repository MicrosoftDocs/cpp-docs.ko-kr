# <a name="multibyte-characters"></a>멀티바이트 문자

**ANSI 2.2.1.2** 멀티바이트 문자의 상태 전환

멀티바이트 문자는 기본 문자 집합에서 표현되지 않는 외국어 문자를 나타내기 위해 Microsoft C 및 일부 구현에서 사용됩니다. 그러나 Microsoft C에서는 상태 의존 인코딩이 지원되지 않으므로 상태 전환이 발생하지 않습니다. 자세한 내용은 [멀티바이트 및 와이드 문자](../c-language/multibyte-and-wide-characters.md)를 참조하세요.
