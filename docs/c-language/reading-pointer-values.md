# <a name="reading-pointer-values"></a>포인터 값 읽기

**ANSI 4.9.6.2** `fscanf`함수의 **%p** 변환에 대한 입력

**%p** 형식 문자를 지정할 때 `fscanf` 함수는 16진수 ASCII 값에서 올바른 주소로 포인터를 변환합니다.
