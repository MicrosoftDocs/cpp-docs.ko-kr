# <a name="deleting-open-files"></a>열려 있는 파일 삭제

**ANSI 4.9.4.1** 열린 파일에 대한 제거 함수의 효과

제거 함수는 파일을 삭제합니다. 파일이 열려 있으면 이 함수는 실패하고 -1을 반환합니다.
