# <a name="file-position-in-append-mode"></a>추가 모드의 파일 위치

**ANSI 4.9.3** 추가 모드 스트림의 파일 위치 표시기가 처음에 파일의 시작에 배치되는지 아니면 끝에 배치되는지 여부

추가 모드로 파일을 열 때 파일 위치 표시기는 처음에 파일의 끝을 가리킵니다,
