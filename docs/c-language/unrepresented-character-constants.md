# <a name="unrepresented-character-constants"></a>나타낼 수 없는 문자 상수

**ANSI 3.1.3.4** 기본 실행 문자 집합 또는 확장된 문자 집합(와이드 문자 상수의 경우)으로 표현되지 않는 문자 또는 이스케이프 시퀀스를 포함하는 정수 문자 상수의 값

모든 문자 상수 또는 이스케이프 시퀀스를 확장된 문자 집합으로 표현할 수 있습니다.
