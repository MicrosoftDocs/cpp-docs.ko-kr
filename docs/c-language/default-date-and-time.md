# <a name="default-date-and-time"></a>기본 날짜 및 시간

**ANSI 3.8.8** 변환 날짜와 시간을 각각 사용할 수 없을 때`__DATE__` 및 `__TIME__`에 대한 정의

운영 체제가 날짜 및 시간 변환을 제공하지 않는 경우, `__DATE__` 및 `__TIME__`의 기본값은 `May 03 1957` 및 `17:00:00`입니다.
