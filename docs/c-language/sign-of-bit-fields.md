# <a name="sign-of-bit-fields"></a>비트 필드의 부호

**ANSI 3.5.2.1** "일반" `int` 필드를 **signed int** 비트 필드 또는 unsigned int 비트 필드로 처리할지 여부

비트 필드는 부호가 있거나 부호가 없을 수 있습니다. 일반 비트 필드는 서명된 대로 처리됩니다.
