# <a name="abort-function-c"></a>abort 함수 (C)

**ANSI 4.10.4.1** 열린 파일 및 임시 파일과 관련된 **abort** 함수의 동작

**abort** 함수는 열려 있는 파일이나 임시 파일을 닫지 않으며, 스트림 버퍼를 플러시하지 않습니다. 자세한 내용은 [abort](../c-runtime-library/reference/abort.md)를 참조하세요.
