# <a name="clock-function-c"></a>clock 함수 (C)

**ANSI 4.12.2.1** `clock` 함수에 대한 연대

`clock` 함수의 연대는 C 프로그램 실행이 시작되면 (0 값)으로 시작합니다. 1/**CLOCKS_PER_SEC** 단위로 측정되는 횟수를 반환합니다(Microsoft C의 경우 1/1000에 해당).
