# <a name="domain-errors"></a>도메인 오류

**ANSI 4.5.1** 도메인 오류 발생 시 수학 함수가 반환하는 값

ERRNO.H 파일은 도메인 오류 상수 `EDOM`을 33으로 정의합니다.  반환 값에 대한 자세한 내용은 오류를 발생시킨 특정 함수에 대한 도움말 항목을 참조하세요.
