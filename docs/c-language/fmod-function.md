# <a name="fmod-function"></a>fmod 함수

**ANSI 4.5.6.4**`fmod` 함수의 두 번째 인수가 0일 때 도메인 오류가 발생하는지 아니면 0이 반환되는지 여부

`fmod` 함수의 두 번째 인수가 0인 경우 함수는 0을 반환합니다.
