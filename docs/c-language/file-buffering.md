# <a name="file-buffering"></a>파일 버퍼링

**ANSI 4.9.3** 파일 버퍼링의 특징

표준 I/O 기능을 통해 액세스되는 디스크 파일은 완전히 버퍼링됩니다. 기본적으로 버퍼는 512바이트를 보유합니다.
