# <a name="bits-per-character"></a>문자당 비트

**ANSI 2.2.4.2.1** 문자의 비트 수

문자의 비트 수는 매니페스트 상수 **CHAR_BIT**로 표현됩니다. LIMITS.H 파일에서는 **CHAR_BIT**를 8로 정의합니다.
