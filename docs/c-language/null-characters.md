# <a name="null-characters"></a>Null 문자

**ANSI 4.9.2** 이진 스트림에 기록된 데이터에 추가할 수 있는 null 문자의 개수

null 문자가 개수에 제한 없이 이진 스트림에 추가될 수 있습니다.
