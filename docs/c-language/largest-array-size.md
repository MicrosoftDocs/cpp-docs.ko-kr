# <a name="largest-array-size"></a>가장 큰 배열 크기
**ANSI 3.3.3.4, 4.1.1** 배열의 최대 크기를 보유하는 데 필요한 정수의 형식 즉, **size_t**의 크기입니다.  
  
 32비트 x86 플랫폼에서 `size_t` 형식 정의는 `unsigned int`입니다. 64비트 플랫폼에서 `size_t` 형식 정의는 `unsigned __int64`입니다.
