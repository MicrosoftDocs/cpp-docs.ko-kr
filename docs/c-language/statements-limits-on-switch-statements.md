# <a name="statements-limits-on-switch-statements"></a>문: Switch 문의 제한

**ANSI 3.6.4.2** `switch` 문에 있는 **case** 값의 최대 수

Microsoft C는 **switch** 문의 **case** 값 수를 제한하지 않습니다. 이 수는 사용 가능한 메모리에 의해서만 제한됩니다.
