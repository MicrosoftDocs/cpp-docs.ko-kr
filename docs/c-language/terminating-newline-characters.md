# <a name="terminating-newline-characters"></a>줄 바꿈 문자 종결

**ANSI 4.9.2** 텍스트 스트림의 마지막 줄에 종료하는 줄 바꿈 문자가 필요한지 여부

스트림 함수에서는 새 줄 또는 파일의 끝을 줄의 종료 문자로 인식합니다.
