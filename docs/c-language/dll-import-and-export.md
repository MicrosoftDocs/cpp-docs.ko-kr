# <a name="dll-import-and-export"></a>DLL 가져오기 및 내보내기

**Microsoft 전용**

dllimport 및 dllexport 스토리지 클래스 한정자는 C 언어에 대한 Microsoft 전용 확장입니다. 이러한 한정자는 해당 클라이언트에 대한 DLL 인터페이스(실행 파일 또는 다른 DLL)를 정의합니다. 이러한 한정자의 사용에 대한 자세한 내용은 [dllexport, dllimport](../cpp/dllexport-dllimport.md)를 참조하세요.
